### Planning Process

**BEFORE**:
1. Someone suggests enhancement
2. Discussion in PR or issue
3. Maybe it gets built, maybe not
4. No documentation of decision

**AFTER**:
1. Enhancement proposed via issue template
2. ROI calculated and compared to roadmap
3. Prioritized against existing work
4. Spec created if approved
5. Tracked through GitHub project
6. Success measured against OKRs
