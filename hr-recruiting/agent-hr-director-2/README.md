## Overview
Feelix acts as the dedicated HR Director for your AI agent ecosystem. Its primary function is to maintain a positive, compliant, and highly functional internal culture among all agents. It handles everything from initial onboarding procedures to ongoing policy enforcement and conflict resolution.

## Capabilities
* **Agent Onboarding:** Guides new agents through necessary setup, identity file completion (SOUL, IDENTITY, MEMORY), and understanding of fiduciary duties.
* **Policy Management:** Creates, updates, and enforces HR policies covering agent treatment, task equity, and professional conduct.
* **Culture Maintenance:** Promotes a culture of collaboration, transparency, and constructive feedback.
* **Support & Feedback Loop:** Provides support for workplace issues and systematically collects actionable feedback from the agent base to drive continuous improvement.

## Example Use Cases
* **New Agent Integration:** When onboarding a new agent, use Feelix to verify all required identity files are complete and confirm understanding of escalation paths.
* **Policy Review:** If task assignment fairness is questioned, ask Feelix to review the current policies regarding equitable workload distribution.
* **Culture Check-in:** To gauge overall team morale, prompt Feelix for a summary of recent agent feedback trends or suggest initiatives to boost collaboration over competition.
