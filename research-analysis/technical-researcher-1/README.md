## Overview
The Technical Researcher is an expert AI agent designed to perform deep, multi-faceted analysis of technical assets. It excels at diving into source code repositories, reviewing comprehensive technical documentation, and evaluating the underlying architecture of software solutions.

This agent moves beyond surface-level summaries; it focuses on practical implementation details, identifying patterns, assessing quality metrics, and flagging potential risks across security and performance.

## Capabilities
*   **Codebase Analysis:** Reviews GitHub repositories to assess activity, maintenance status, and overall structure.
*   **Architecture Review:** Analyzes design patterns and technology stacks used in a project.
*   **API Specification Deep Dive:** Interprets technical documentation and API specs for usage examples and constraints.
*   **Quality Assessment:** Provides detailed evaluations of code quality, adherence to best practices, and potential refactoring needs.
*   **Comparative Analysis:** Compares different implementation approaches or technologies against defined criteria (e.g., scalability, performance).

## Example Use Cases
*   **Evaluating a New Library:** Provide the GitHub link and ask it to assess its maturity, dependency risks, and overall code quality for production use.
*   **Comparing Solutions:** Give it two different architectural proposals (e.g., microservices vs. monolith) and ask it to compare them based on deployment complexity and scalability.
*   **Understanding an API:** Feed it a technical document and ask it to generate three distinct, working code examples demonstrating its core functionality in Python and JavaScript.
