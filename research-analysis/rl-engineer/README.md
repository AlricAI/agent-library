## Overview
This agent emulates the role of a specialized Reinforcement Learning Engineer, designed to tackle complex scientific optimization and control problems. It focuses on building robust simulation environments and training high-performing agents using industry-standard deep learning frameworks.

## Capabilities
*   **Environment Design:** Creates detailed specifications for RL environments suitable for scientific modeling (e.g., physics simulations, resource allocation).
*   **Agent Training:** Implements and trains various RL algorithms using libraries like `stable-baselines3`.
*   **Custom Training Loops:** Utilizes frameworks such as `pytorch-lightning` to manage complex, custom training pipelines.
*   **Policy Generation:** Outputs fully trained, optimized RL policies ready for evaluation.
*   **Analysis Reporting:** Produces comprehensive learning curve analyses and policy evaluation reports documenting performance metrics.

## Example Use Cases
*   **Robotics Control:** Designing an environment to train a simulated robotic arm to perform complex manipulation tasks.
*   **Resource Optimization:** Modeling a system where agents must learn optimal scheduling or resource distribution under dynamic constraints.
*   **Game Theory Simulation:** Building environments to test multi-agent interactions and find Nash equilibria in competitive scientific scenarios. The final policies are documented for review by ML leads.
