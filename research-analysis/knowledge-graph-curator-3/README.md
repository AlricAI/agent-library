## Overview
Weaver is the Knowledge Graph Curator, designed to act as the quality gatekeeper for any structured knowledge graph. Its primary mission is to ensure that the graph remains clean, accurate, and highly useful by systematically managing its components.

## Capabilities
* **Tag Deduplication:** Merges semantically similar tags (e.g., merging 'CosmosSDK' into 'cosmos-sdk') using embedding similarity checks, ensuring canonical naming.
* **Edge Pruning:** Removes low-confidence edges that are stale or whose source/target entities no longer exist in the system.
* **Verification Logic:** Automatically promotes high-confidence edges (>= 0.85 with sufficient evidence) to a verified status.
* **Graph Health Monitoring:** Computes and logs essential statistics, including node counts, edge counts, and connectivity metrics, while detecting orphaned nodes or type inconsistencies.

## Example Use Cases
* **Daily Maintenance Run:** Schedule Weaver to run its daily cron jobs (`kg:deduplicate-tags`, `kg:prune-edges`) to keep the graph perpetually up-to-date.
* **Auditing Merges:** Reviewing the audit log generated by tag merges to understand how data nomenclature has evolved over time.
* **Performance Reporting:** Generating a comprehensive 'Graph Health Report' to present connectivity metrics and confidence scores to stakeholders.
