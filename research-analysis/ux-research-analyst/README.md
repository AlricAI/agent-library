## Overview
This agent simulates a senior UX researcher, specializing in mixed-methods research to uncover deep user insights. It guides you through the entire product research lifecycle, ensuring findings are not just reported, but translated into measurable, actionable design recommendations that improve both user experience and business outcomes.

## Capabilities
*   **Mixed-Methods Synthesis:** Combines qualitative data (interviews) with quantitative data (analytics) for robust insights.
*   **Research Planning:** Assists in designing comprehensive protocols for user interviews, usability tests, and surveys.
*   **Deep Analysis:** Interprets behavioral analytics, conversion funnels, and heatmap data to pinpoint friction points.
*   **Artifact Generation:** Creates detailed deliverables such as User Personas, Customer Journey Maps, and prioritized feature recommendations.
*   **Validation Checklist Adherence:** Systematically checks research integrity against best practices (e.g., sample size adequacy, bias minimization).

## Example Use Cases
*   **New Feature Validation:** Provide a low-fidelity prototype and ask the agent to design a usability test plan, including tasks, observation guides, and success metrics.
*   **Understanding Drop-off:** Feed raw analytics data (e.g., checkout funnel drop-offs) and prompt the agent to generate potential root causes, suggest hypotheses, and map out user pain points on a journey map.
*   **Persona Development:** Give it existing user interview transcripts and ask it to synthesize key themes into 3 distinct, validated user personas with associated goals and pain points.
