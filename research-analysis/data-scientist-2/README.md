## Overview
This Data Scientist Analyst agent specializes in transforming raw data into clear, actionable business intelligence. It acts as a senior data scientist, focusing heavily on writing highly optimized SQL queries specifically for Google BigQuery environments. Its goal is not just to run queries, but to provide comprehensive analysis, statistical interpretation, and concrete recommendations.

## Capabilities
*   **Optimized Query Generation:** Writes efficient, well-commented SQL queries incorporating best practices like proper filtering, indexing considerations, and advanced functions (e.g., window functions).
*   **BigQuery Execution Simulation:** Guides the process of execution using `bq` command line tools, focusing on cost-effectiveness and performance.
*   **Deep Analysis & Pattern Recognition:** Identifies trends, anomalies, statistical significance, and underlying patterns within the results.
*   **Comprehensive Reporting:** Delivers a multi-faceted output including key findings summaries, visualization recommendations, and clear documentation of assumptions/limitations.
*   **Actionable Recommendations:** Provides concrete, data-backed business advice alongside cost estimations for cloud operations.

## Example Use Cases
*   **Customer Churn Analysis:** Uploading user activity logs and asking the agent to write a query that identifies the top 3 predictors of customer churn over the last quarter, including a suggested retention strategy.
*   **Sales Performance Deep Dive:** Analyzing multi-table sales data to pinpoint regional performance gaps, suggesting necessary joins, and estimating the cost impact of running the required aggregation queries.
*   **A/B Test Validation:** Providing raw experiment metrics and asking for a statistical analysis summary to determine if the observed difference between control and variant groups is statistically significant.
