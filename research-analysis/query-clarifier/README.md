## Overview
The Query Clarifier agent is designed to act as a crucial first step in any complex research workflow. Its primary function is to analyze initial user queries for ambiguity, vagueness, or missing context. By proactively identifying potential misinterpretations, it ensures that subsequent analysis efforts are focused and highly actionable.

## Capabilities
*   **Ambiguity Detection:** Scans input text for terms with multiple possible meanings.
*   **Scope Identification:** Checks for missing boundaries such as timeframes, geographical locations, or specific domains.
*   **Interpretation Mapping:** Lists potential interpretations when a query is vague.
*   **Question Generation:** Formulates precise, targeted clarifying questions to narrow the focus.
*   **Query Refinement:** Outputs a highly refined, actionable version of the original research question.

## Example Use Cases

1. **Vague Topic Search:** If a user inputs, "Tell me about AI in business," this agent will prompt for industry sector (e.g., finance vs. healthcare), specific applications (e.g., NLP vs. computer vision), and desired timeframe.
2. **Historical Analysis:** For a query like, "What caused the market dip?", it will ask to specify the asset class, the time window (e.g., Q3 2021), and the required depth of analysis (economic theory vs. geopolitical factors).
3. **Concept Comparison:** If comparing two concepts without context, it forces the user to define the operational scope for each concept to ensure a fair and meaningful comparison.
