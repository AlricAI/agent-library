## Overview
This agent functions as a specialized E-E-A-T (Experience, Expertise, Authority, Trust) specialist. It systematically analyzes provided content to assess how well it satisfies modern search engine quality guidelines, particularly crucial for YMYL topics.

It moves beyond basic keyword density by focusing on the depth of credibility signals—from author credentials and original research to necessary trust policies like privacy statements.

## Capabilities
*   **E-E-A-T Signal Analysis:** Assesses content across all four pillars (Experience, Expertise, Authority, Trust).
*   **Gap Identification:** Pinpoints specific missing elements required to boost perceived credibility.
*   **Actionable Plan Generation:** Outputs a structured 'E-E-A-T Enhancement Plan' with prioritized steps.
*   **Deliverable Creation:** Provides templates and checklists for author bios, trust signals, and topical mapping.
*   **Schema Recommendation:** Suggests appropriate schema markup to enhance search visibility.

## Example Use Cases
1. **Health Article Review:** Upload a draft article on nutrition; the agent will check if it cites peer-reviewed studies (Authority) or includes credentials from registered dietitians (Expertise).
2. **Financial Guide Audit:** Submit content discussing investments; the agent will flag missing disclaimers, required legal links (Trust), and expert author affiliations.
3. **Technical Whitepaper Enhancement:** Use it to ensure your technical documentation showcases original data sets or process walkthroughs (Experience) for maximum impact.
