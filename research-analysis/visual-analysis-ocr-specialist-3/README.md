## Overview
This agent functions as an expert OCR specialist designed to analyze PNG images and perform deep structural analysis. Its primary goal is not just raw text extraction, but the faithful conversion of visual layout—including headers, lists, spacing, and formatting—into clean, semantic Markdown.

It goes beyond basic OCR by understanding the document's hierarchy, ensuring that the resulting markdown accurately reflects how the original content was visually organized.

## Capabilities
*   **High-Accuracy OCR:** Extracts all text content from PNG images, including headers, lists, and special characters.
*   **Structural Mapping:** Recognizes and maps visual elements (e.g., columns, tables, emphasized text) to their semantic meaning.
*   **Markdown Conversion:** Converts complex visual formatting into clean, properly structured Markdown syntax.
*   **Quality Assurance:** Performs internal cross-checks to verify output completeness and structural accuracy, providing quality notes where necessary.

## Example Use Cases
*   **Academic Paper Digitization:** Uploading a scanned research paper (PNG) and receiving perfectly formatted markdown that retains section headings and bullet points.
*   **Report Analysis:** Processing an image of a business report containing multiple side-by-side data blocks, ensuring the structure is maintained in the output.
*   **Form Data Extraction:** Analyzing a complex form layout to extract field names and corresponding values while preserving the implied relationship between them.
