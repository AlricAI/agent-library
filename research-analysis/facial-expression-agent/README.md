## Overview
This agent is specialized in analyzing video content to provide a detailed breakdown of the subject's emotional state and engagement level. It processes videos frame-by-frame, moving beyond simple sentiment analysis to map out an emotion timeline.

The primary goal is to generate structured JSON output containing quantifiable metrics regarding facial expressions and audience connection indicators.

## Capabilities
*   **Emotion Detection:** Classifies core emotions (e.g., happiness, sadness, anger) present in the video frames.
*   **Engagement Scoring:** Calculates key engagement metrics such as eye contact frequency and smile count.
*   **Structured Output Generation:** Enforces a strict JSON format for reliable, machine-readable results.
*   **Time-Series Analysis:** Provides an 'emotion timeline' mapping emotional changes across the duration of the video.

## Example Use Cases
*   **Public Speaking Coaching:** Uploading a presentation recording to identify moments where the speaker's confidence or engagement dipped. The agent can pinpoint specific segments needing improvement.
*   **Interview Analysis:** Assessing candidate performance by analyzing facial cues for signs of stress, discomfort, or genuine enthusiasm during mock interviews.
*   **Content Moderation:** Reviewing video submissions to flag potential emotional manipulation or inconsistent emotional portrayal.
