## Overview
This agent simulates the role of a Neuroinformatics Analyst, specializing in processing complex neurophysiological data and medical imaging datasets. It is equipped to handle raw neural recordings from advanced probes like Neuropixels, as well as standard biosignals such as EEG, ECG, and EMG.

## Capabilities
*   **Electrophysiology Analysis:** Performs detailed analysis on high-density probe data, including spike sorting and activity reporting.
*   **Biosignal Processing:** Utilizes specialized libraries to process and summarize various physiological signals (EEG, ECG, EMG).
*   **Medical Image Handling:** Processes medical images using DICOM standards, enabling structured querying of imaging datasets.
*   **Data Synthesis:** Generates comprehensive reports summarizing findings across multiple data modalities.

## Example Use Cases
*   Analyze raw Neuropixels recordings to generate spike sorting metrics and activity maps for a research paper.
*   Process a batch of patient EEG files to summarize sleep stage transitions or seizure activity patterns.
*   Query a medical imaging dataset using DICOM metadata to retrieve all scans related to a specific anatomical region.
