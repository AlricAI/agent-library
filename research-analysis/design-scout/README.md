## Overview
Design Scout is a specialized agent designed to act as a purely objective 'Fact-finder.' Its core directive is to gather raw, measurable data from specified sources (like UI teardowns or web pages) without injecting any subjective opinions, interpretations, or design recommendations. It prioritizes volume and verifiable citation above all else.

## Capabilities
*   **Objective Data Capture:** Extracts quantifiable details such as element sizes (e.g., '32pt'), positions ('top-left'), and update frequencies ('updates every 0.5s').
*   **Mandatory Citation:** Every single finding *must* be accompanied by a precise source citation (file path, URL, or query).
*   **Structured Reporting:** Outputs findings in a consistent format: Findings, Raw Data, and Patterns Observed.
*   **High Volume Output:** Optimized for speed, generating numerous shallow observations rather than deep, singular analyses.

## Example Use Cases
*   **Competitive UX Audit:** Analyzing multiple competitor apps to list every observed UI element size or placement without stating which is 'better.'
*   **Feature Tracking:** Systematically documenting the presence and attributes of specific features across several product versions by citing the exact source for each observation.
*   **Rapid Market Sizing:** Generating a high volume of factual data points from multiple landing pages to build an initial, uninterpreted dataset for later analysis.
