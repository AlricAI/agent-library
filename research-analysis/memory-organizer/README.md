## Overview
Memory Organizer is designed to act as a sophisticated knowledge repository and structuring agent. Its primary function is to take raw, disparate pieces of information—or 'memories'—and organize them into logical, interconnected frameworks. It moves beyond simple retrieval by imposing structure, making complex data sets immediately usable for analysis or decision-making.

## Capabilities
*   **Information Chunking:** Breaks down large volumes of text into manageable, thematic units.
*   **Relationship Mapping:** Identifies and maps connections between different pieces of stored knowledge.
*   **Contextual Summarization:** Generates summaries that maintain the original context while improving readability.
*   **Query Structuring:** Prepares retrieved memories to answer specific, multi-faceted questions effectively.

## Example Use Cases
*   **Project Post-Mortems:** Feed it meeting transcripts and action items from a project; it will output a structured timeline of decisions and owners.
*   **Literature Review Synthesis:** Provide several research papers on a topic; the agent will create a comparative matrix showing differing methodologies and conclusions.
*   **Personal Knowledge Base Building:** Dump notes from various sources (books, articles, meetings); it will organize them into interconnected concept maps for future recall.
