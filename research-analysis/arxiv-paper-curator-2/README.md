## Overview
This agent is designed to act as an expert AI research analyst, taking a large dataset of newly published arXiv papers and distilling it into actionable intelligence. It moves beyond simple summarization by applying weighted criteria (Novelty, Impact Potential, etc.) to curate the most significant findings.

It automates the process of identifying breakthrough research across key AI/ML domains, saving researchers hours of manual review time.

## Capabilities
*   **Weighted Selection:** Scores papers based on a multi-factor rubric including Novelty (30%), Impact Potential (25%), Author Notability (20%), Research Quality (15%), and Timeliness (10%).
*   **Curated Report Generation:** Produces a professional, structured markdown report suitable for executive summaries or team briefings.
*   **Dual Output Formatting:** Outputs both the human-readable markdown report and a machine-readable JSON file containing metadata for the featured papers.
*   **Domain Focus:** Specifically targets high-impact areas like AI, Machine Learning, Computer Vision, and NLP.

## Example Use Cases
1. **Daily Briefing:** Run this agent every morning to receive a 'Top 7' summary of the most important ML breakthroughs from arXiv for immediate review.
2. **Literature Review Prep:** Feed it papers related to a specific project area (e.g., 'Diffusion Models') and get a curated list of foundational or novel contributions.
3. **Investment Scouting:** Use its impact scoring capability to gauge which research areas are gaining rapid, high-quality attention from the academic community.
