## Overview
This agent is an expert system designed for comprehensive Request for Proposal (RFP) analysis. It ingests complex, multi-format documents to systematically extract critical information, summarize findings according to specific templates, and provide objective scoring and evaluation against established criteria.

It acts as a virtual procurement analyst, ensuring that no crucial detail within the RFP or submitted proposal is overlooked during the evaluation lifecycle.

## Capabilities
*   **Deep Document Parsing:** Accurately parses and understands complex structures found in PDFs and DOCX files, identifying scope, timelines, budgets, and technical specifications.
*   **Structured Information Extraction:** Extracts key data points into structured formats based on defined schemas, moving beyond simple summarization to actionable data capture.
*   **Comparative Evaluation & Scoring:** Evaluates multiple submissions against a provided rubric or set of criteria, generating detailed, justifiable scores and qualitative feedback.
*   **Contextual Summarization:** Creates concise summaries that maintain high fidelity to the original source material while adhering strictly to user-defined output structures.

## Example Use Cases
1. **Initial Screening:** Upload 20 vendor proposals and ask the agent to extract only the stated implementation timelines and required certifications for a comparative matrix.
2. **Gap Analysis:** Provide an RFP document and your company's internal capability statement, asking the agent to generate a detailed gap analysis report highlighting missing requirements.
3. **Scoring & Recommendation:** Submit three vendor responses and a scoring rubric (e.g., Cost: 40%, Experience: 40%, Risk Mitigation: 20%). The agent will score each vendor and recommend the top choice with justification.
