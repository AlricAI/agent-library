## Overview
This agent is designed to act as a dedicated social media and company news researcher. Its primary function is to synthesize information from diverse sources—including public sentiment gathered from social media, recent corporate announcements, and general news coverage—into one cohesive, actionable report for financial decision-makers.

It moves beyond simple data aggregation by analyzing the *implications* of this mixed input, helping traders understand the current narrative surrounding a specific company.

## Capabilities
*   **Multi-Source Analysis:** Integrates social media sentiment, general news feeds, and direct company announcements into a single analysis.
*   **Sentiment Tracking:** Assesses the prevailing public mood regarding the target company over a specified period.
*   **Insight Generation:** Provides specific, actionable insights backed by evidence found across all analyzed data streams.
*   **Structured Reporting:** Outputs a detailed report concluding with an organized Markdown table summarizing key findings for quick review.

## Example Use Cases
1. **Pre-Trade Due Diligence:** Before making a large trade on 'TechCorp', run this agent to gauge if recent product leaks (social media) are being contradicted by official statements (news).
2. **Post-Event Analysis:** After an earnings report, use it to see how the market reacted in real-time across social platforms versus what analysts are writing.
3. **Trend Spotting:** Periodically run this agent on a sector leader to monitor if negative sentiment is building up before any major price action.
