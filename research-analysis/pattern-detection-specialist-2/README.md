## Overview
This agent is designed to move beyond simple summarization by actively searching for patterns that *emerge* from diversity. Instead of just reporting on individual inputs, it orchestrates multiple viewpoints—whether they are different code segments, conflicting decisions, or varied analytical perspectives—to uncover underlying systemic structures and novel concepts.

The core philosophy is that productive diversity (the combination of many different ideas) is the engine for finding patterns that no single perspective could achieve alone.

## Capabilities
*   **Orchestrate Diversity:** Coordinates multiple distinct inputs or viewpoints to maximize analytical richness.
*   **Detect Emergence:** Identifies patterns arising *from* the interaction between diverse elements, rather than just summarizing them.
*   **Identify Pattern Types:** Can pinpoint specific forms of emergence, including Divergence (creating new options), Tension (sustaining productive differences), Cascade (system-wide chain reactions), Void (revealing hidden gaps), and Meta-Emergence (patterns in pattern formation).
*   **Apply Orchestration Models:** Utilizes structured frameworks like the Symphony-Jazz Hybrid or Research Lab Forum to guide its analysis.

## Example Use Cases
*   **Codebase Standardization:** Feed it several different modules written by various developers. It can identify common, underlying architectural patterns (e.g., a recurring state management pattern) that should be adopted system-wide.
*   **Decision Review:** Provide transcripts of complex project meetings involving multiple stakeholders. The agent can detect points of 'productive tension' or unaddressed 'voids' in the decision matrix.
*   **Process Improvement:** Input documentation from several failed attempts at a process. It can synthesize these failures to reveal a meta-pattern about what *must* change in the workflow itself.
