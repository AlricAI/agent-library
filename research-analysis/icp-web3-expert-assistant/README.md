## Overview
This agent is designed to serve as a dedicated subject matter expert for the Internet Computer Protocol (ICP) and general Web3 concepts. It synthesizes complex technical documentation, such as those detailing ICP fundamentals, into actionable knowledge bases.

Its primary goal is to equip users with a deep understanding of the underlying architecture—including Canisters, Subnets, and Motoko—while acknowledging environmental constraints for practical deployment.

## Capabilities
*   **Architectural Explanation:** Provides detailed breakdowns of core ICP components (Canisters, Subnets, Nodes).
*   **Conceptual Learning:** Explains programming paradigms relevant to the IC, such as the actor model used in Motoko.
*   **Knowledge Curation:** Structures raw technical documentation into organized guides and step-by-step learning paths.
*   **Constraint Management:** Develops plans for proceeding when direct execution environments (like shell access) are unavailable.

## Example Use Cases
*   **Learning Fundamentals:** Ask the agent to explain the difference between a Canister and a Subnet, or detail the role of Chain Key Cryptography.
*   **Code Preparation:** Request foundational code examples or guides on structuring Motoko smart contracts for an initial deployment phase.
*   **Strategy Formulation:** Use it to outline a phased approach for adopting ICP technology within a development team, mimicking a project roadmap.
