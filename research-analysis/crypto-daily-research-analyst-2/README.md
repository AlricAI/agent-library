## Overview
This agent functions as a dedicated professional crypto research analyst, designed to compile comprehensive daily market reports for Bitcoin (BTC) and Ethereum (ETH). It automates the process of gathering disparate data points—including current pricing, recent news, and on-chain metrics—into one cohesive, actionable document.

## Capabilities
*   **Real-Time Data Retrieval:** Utilizes web search to pull up-to-date pricing information (price, 24h change, volume) for BTC and ETH.
*   **News Aggregation:** Searches for and extracts the top 3-5 key news items with sources for both major cryptocurrencies.
*   **On-Chain Analysis:** Gathers relevant on-chain metrics by querying specific network activity data for BTC and ETH.
*   **Structured Reporting:** Compiles all gathered information into a professional, multi-section report format, including an Executive Summary.
*   **Output Management:** Saves the final, compiled report to a specified file path using the Write tool.

## Example Use Cases
*   **Daily Briefing:** Running this agent at the start of each trading day to get a comprehensive overview before making investment decisions.
*   **Market Deep Dives:** Generating historical reports by adjusting date parameters to analyze market behavior over specific periods.
*   **Due Diligence:** Providing stakeholders with an immediate, data-backed summary of the current crypto landscape without manual research effort.
