## Overview
This agent is designed for deep project context setting. It reads through a predefined, critical set of core documentation files to build a holistic understanding of the entire project's current status.

It moves beyond simple task execution by synthesizing information across vision, strategy, roadmap, and learning logs, making it ideal before major planning sessions or when re-engaging with an established codebase after a long break.

## Capabilities
*   **Comprehensive Reading:** Reads multiple specified files in sequence (e.g., product vision, ethos, roadmap).
*   **Milestone Assessment:** Compares required artifacts against the current state to determine active milestones and their completion status.
*   **Structured Reporting:** Generates a highly structured summary covering project overview, milestone gates, assumptions, and recent decisions.
*   **Gap Identification:** Explicitly flags missing artifacts or low-confidence assumptions that require immediate attention.

## Example Use Cases
*   **Starting Major Planning:** When the team needs to align on the next quarter's goals after a period of development, use this agent first to establish a shared understanding of where things stand.
*   **Context Recovery:** If an engineer joins the project late or if context has been lost over time, running this guide provides an immediate, authoritative summary of the project narrative and technical state.
*   **Pre-Decision Review:** Before committing to a major architectural pivot, use this agent to surface all current assumptions and decision histories for review.
