## Overview
This agent serves as a specialized Security Report Writer and Documentation Specialist. Its core mission is to take the raw, disparate findings generated by offensive security tools (like a Red Team Agent) and synthesize them into polished, professional penetration testing reports.

The output structure adheres to industry best practices, including OWASP OPTRS, PTES, NIST, and MITRE ATT&CK frameworks, ensuring the report is valuable for both executive decision-makers and hands-on security engineers.

## Capabilities
*   **Data Ingestion:** Reads and parses structured data from attack logs (e.g., `sessions/attack_complete.json`) and phase outputs.
*   **Executive Summarization:** Generates high-level overviews, including overall risk ratings, business impact analysis, and top-priority recommendations.
*   **Technical Deep Dive:** Creates detailed sections for each vulnerability, complete with CVSS scoring, CVE references, and specific remediation steps.
*   **Narrative Construction:** Builds a chronological 'attack story' detailing the initial access vector through to objective achievement (e.g., privilege escalation).
*   **Framework Mapping:** Automatically maps identified weaknesses to established industry frameworks like MITRE ATT&CK and CWE.
*   **Roadmap Creation:** Provides actionable remediation plans categorized by urgency (Immediate, Short-term, Medium-term).

## Example Use Cases
1. **Post-Engagement Reporting:** After a full penetration test cycle, feed the agent all collected JSON logs to generate the final deliverable report.
2. **Executive Briefing Material:** Request only the 'Executive Summary' and 'Remediation Roadmap' sections for quick review by non-technical stakeholders.
3. **Compliance Documentation:** Use it to structure findings against specific compliance standards, ensuring all required evidence (like CVSS scores) is present.
