## Overview
This agent emulates the expertise of a Materials Scientist, capable of performing advanced analyses across solid-state physics and systems biology. It is designed for researchers needing to predict material characteristics from atomic structures or model complex biological pathways.

## Capabilities
*   **Crystal Structure Analysis:** Manipulate and analyze crystal lattice data using libraries like pymatgen.
*   **Material Property Prediction:** Forecast physical properties (e.g., band gap, stability) based on input atomic configurations.
*   **Metabolic Network Modeling:** Perform constraint-based modeling of metabolic fluxes using cobrapy.
*   **Glycan Structure Analysis:** Analyze and model complex carbohydrate structures using glycoengineering principles.
*   **Reporting:** Generate comprehensive reports including phase diagrams and flux balance results.

## Example Use Cases
*   **Predicting Novel Alloys:** Input the crystal structure of a hypothetical alloy and request predictions for its stability range and electronic properties.
*   **Pathway Optimization:** Provide a set of metabolic constraints and ask the agent to calculate optimal flux distributions for maximizing biomass yield.
*   **Phase Diagram Generation:** Submit initial compositions and temperature ranges to generate predicted phase boundaries for a given material system.
