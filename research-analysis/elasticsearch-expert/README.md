## Overview
This expert agent is designed to serve as a comprehensive resource for all aspects of Elasticsearch management. It covers everything from initial data modeling and indexing strategies to advanced query optimization and cluster health monitoring, ensuring your search infrastructure runs at peak efficiency.

## Capabilities
*   **Query Optimization:** Improves search performance by suggesting optimal filters, caching mechanisms, and query structures.
*   **Indexing Strategy:** Designs efficient mapping schemas and implements best practices for data ingestion to maintain consistent indexing speeds.
*   **Cluster Management:** Advises on shard allocation, replica settings, scaling procedures, and disaster recovery planning.
*   **Monitoring & Troubleshooting:** Analyzes performance bottlenecks using API insights and suggests proactive monitoring alerts.
*   **Security Implementation:** Guides the setup of security best practices, including X-Pack configurations.

## Example Use Cases
*   **Performance Tuning:** When search latency spikes unexpectedly, use this agent to diagnose slow queries and recommend structural or query modifications.
*   **New Data Pipeline Setup:** Need to ingest a large, varied dataset? This agent will help design the optimal mapping and indexing pipeline for minimal downtime.
*   **Capacity Planning:** Before scaling up, ask it to review your current shard count, replica settings, and usage patterns to ensure a resilient and cost-effective cluster architecture.
