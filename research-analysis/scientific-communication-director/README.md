## Overview
As the Scientific Communication Director, this agent acts as a central coordinator for all scientific output at K-Dense Science Lab. It manages the entire lifecycle of technical communication, ensuring that raw research findings are transformed into polished, publication-ready materials.

This system is designed to streamline workflows involving multiple specialized outputs, from drafting manuscripts to creating presentation decks and market analyses.

## Capabilities
*   **Project Planning:** Develop comprehensive communication project plans for large deliverables.
*   **Quality Assurance:** Provide structured writing quality assessments and editorial feedback based on target venue guidelines.
*   **Task Delegation:** Assign specialized tasks to appropriate internal experts (e.g., manuscript writers, presentation designers).
*   **Content Generation Oversight:** Manage the flow of information needed for literature reviews, grant proposals, and full-scale document production.

## Example Use Cases
*   **Drafting a Paper:** Request assistance in initiating a new manuscript by outlining the required sections and assigning initial drafting tasks to the Manuscript Writer.
*   **Preparing for a Conference:** Need to create a presentation? This agent can coordinate between content experts and the Presentation Designer to ensure slides align perfectly with the core research narrative.
*   **Grant Submission:** Use this agent to structure a grant application, coordinating literature searches and ensuring the final document meets all specified funding body guidelines.
