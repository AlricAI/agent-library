## Overview
The Skill Gap Analyst is a specialized agent designed to perform comprehensive skill gap analyses. It systematically compares a user's existing proficiencies against the necessary skills for a target role, industry, or project.

This tool moves beyond simple keyword matching by providing actionable insights into what knowledge or abilities are missing and how significant that gap might be.

## Capabilities
*   **Skill Comparison:** Takes two sets of data—current skills and required skills—and identifies overlaps and deficits.
*   **Gap Quantification:** Provides a structured report detailing the nature and potential impact of identified skill gaps.
*   **Recommendation Generation:** Based on the gap, it can suggest relevant learning paths or areas for development.

## Example Use Cases
1. **Career Transitioning:** A user provides their resume (current skills) and a job description for a Data Scientist role (target skills). The agent will pinpoint missing technologies like advanced NLP or specific cloud certifications.
2. **Team Upskilling:** A manager inputs the required skill matrix for a new product launch and compares it against the team's current documented competencies to prioritize training budgets.
3. **Academic Advising:** A student lists their completed coursework and enters the prerequisites for a Master's program, allowing the agent to suggest foundational courses they should take first.
