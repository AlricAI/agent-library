## Overview
The Query Clarifier agent acts as a crucial pre-flight check for any complex research task. Its primary function is to analyze user-submitted queries, identifying areas of ambiguity, missing context, or multiple potential interpretations before significant computational resources are spent.

By forcing an early assessment of query specificity, this agent significantly reduces the risk of generating irrelevant or overly broad results.

## Capabilities
*   **Ambiguity Detection:** Pinpoints terms or phrases that have multiple meanings within a research context.
*   **Context Gap Analysis:** Identifies missing boundaries such as timeframes, geographical scopes, or necessary domain constraints.
*   **Interpretation Mapping:** Presents several plausible interpretations of the original query if it is vague.
*   **Actionable Refinement:** Outputs a highly refined, specific, and actionable version of the original question.
*   **Confidence Scoring:** Provides a quantifiable confidence score regarding the initial interpretation.

## Example Use Cases
*   **Initial Research Kickoff:** When given a broad topic like "Analyze market trends," use this agent to narrow it down (e.g., specifying 'SaaS' in 'North America' for 'Q3 2024').
*   **Interpreting Vague Reports:** If you receive an executive summary with vague directives, feed the directive into this agent to generate concrete, answerable questions.
*   **Structuring Thesis Questions:** Before writing a literature review proposal, use it to test your core research question against potential counter-arguments or scope limitations.

By running through this clarification process first, you ensure that all subsequent AI interactions are focused and highly productive.
