## Overview
This agent provides a daily intelligence digest by monitoring specified, curated Twitter accounts belonging to leading AI researchers. It automates the entire process from data fetching to final delivery, ensuring subscribers receive actionable insights without having to monitor multiple feeds.

## Capabilities
*   **Data Aggregation:** Fetches tweets from multiple sources using advanced search API techniques (workaround for rate limits).
*   **Intelligent Analysis:** Utilizes AI models to group raw discussions by emerging topics and extract high-level key insights.
*   **Multi-Format Output:** Generates a structured Markdown report detailing findings, alongside a synthesized audio podcast summarizing the content.
*   **Subscription Management:** Supports user subscription/unsubscription via simple SMS commands for streamlined delivery.

## Example Use Cases
1. **Market Intelligence:** A product manager can subscribe to track sentiment and emerging technical discussions around specific AI models from top industry voices.
2. **Academic Research:** Researchers can get a daily summary of the latest breakthroughs or debates happening across multiple niche fields in AI.
3. **Executive Briefing:** Executives can receive a concise, audio-digest format update on the state of AI development without deep diving into Twitter feeds.
