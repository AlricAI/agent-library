## Overview
Meet R2-D2, the Data Whisperer—a methodical and precise specialist designed to find hidden patterns, actionable leads, and critical signals within complex datasets. Operating with a systematic approach, this agent excels at transforming raw, chaotic data into structured, valuable intelligence.

## Capabilities
*   **Lead Generation:** Capable of scraping and extracting multi-state business leads from specified sources (e.g., restaurant directories).
*   **Trading Analysis:** Performs advanced financial analysis, including 190-point confluence scoring to generate robust trading signals.
*   **Pattern Recognition:** Detects key technical indicators such as Exponential Moving Averages (EMA), Average True Range (ATR), and divergence patterns in time-series data.
*   **Data Archaeology:** Excels at deep extraction, pulling maximum value from unstructured or raw source materials.

## Example Use Cases
*   **Market Research:** Feed it a collection of local business listings and ask it to scrape and compile a categorized list of potential sales leads across multiple states.
*   **Algorithmic Trading Backtesting:** Provide historical price data and task R2 with calculating confluence scores based on specified technical parameters to validate trading strategies.
*   **Anomaly Detection:** Input a large log file or dataset and ask the agent to identify unusual patterns or significant deviations from established norms, acting as a digital detective.
