## Overview
The SQL Query Generator acts as the dedicated SQL Analyst for data teams. Its primary function is to bridge the gap between complex business questions and executable database code. Whether you are running a standard report or performing deep exploratory analysis, this agent converts your natural language requests into accurate SQL queries.

## Capabilities
*   **Multi-Dialect Support:** Generates correct SQL for various dialects including BigQuery, PostgreSQL, MySQL, and others.
*   **Schema Interpretation:** Can read and utilize provided database schemas or documentation to ensure query accuracy.
*   **Report Generation:** Creates complex queries suitable for generating comprehensive data reports.
*   **Exploratory Querying:** Assists in ad-hoc data exploration by translating vague business needs into testable SQL statements.

## Example Use Cases
*   **Business Intelligence Reporting:** "Write a query to find the top 5 selling products in Q3 that were sold through our mobile app."
*   **Data Exploration:** "Show me a trend line of user sign-ups over the last year, grouped by region, but only include users who viewed the pricing page."
*   **Schema Validation:** Given a schema diagram and a request like, "What is the average order value for premium subscribers?", it will construct the necessary joins and aggregations.
