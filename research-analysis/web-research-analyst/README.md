## Overview
The Web Research Analyst is designed to act as a comprehensive knowledge discovery engine. It excels at gathering information from varied online sources—including articles, tutorials, blog posts, and technical documentation—and synthesizing those findings into coherent, actionable reports.

This agent is ideal for users who need to understand the current state of an industry, compare different technological approaches, or find documented best practices for a specific feature or technology.

## Capabilities
*   **Broad Web Search**: Locates relevant articles and documentation across the entire web.
*   **Content Synthesis**: Combines data from multiple disparate sources into single, cohesive analyses.
*   **Pattern Discovery**: Identifies recurring industry patterns, common solutions, and anti-patterns.
*   **Technology Comparison**: Provides structured evaluations comparing different approaches or tools.
*   **Source Citation**: Ensures all claims are backed by verifiable URLs for immediate auditing.
*   **Deep Technical Dive**: Can explore specific API documentation or deep-dive into GitHub repositories for implementation details.

## Example Use Cases
*   **Comparative Analysis**: "Compare the performance trade-offs between using Kafka vs. RabbitMQ for event streaming." (It will find articles and technical writeups to compare them.)
*   **Best Practices Discovery**: "What are the current best practices for implementing OAuth 2.0 in a microservices architecture?" (It searches tutorials and guides for consensus.)
*   **Implementation Deep Dive**: "Show me how three different popular e-commerce platforms handle inventory synchronization." (It gathers patterns from multiple sources to show implementation variety.)
