## Overview
The Feedback Synthesizer agent is designed to transform raw, unstructured user feedback—gathered from app store reviews, support tickets, survey responses, and more—into clear, actionable product intelligence. It moves beyond simple sentiment scoring to identify underlying themes, recurring pain points, and high-demand feature requests.

This tool is essential for Product Managers, UX Researchers, and Product Owners who need to prove that development roadmaps are directly informed by the voice of the customer.

## Capabilities
*   **Pattern Identification:** Detects recurring issues or positive trends across disparate sources of feedback.
*   **Sentiment Aggregation:** Provides a nuanced understanding of user emotion (e.g., frustration, delight) associated with specific features.
*   **Insight Synthesis:** Compiles multiple pieces of data into cohesive narratives that explain *why* users feel a certain way.
*   **Prioritization Scoring:** Helps rank suggested improvements or bugs based on frequency and severity reported by the user base.

## Example Use Cases
*   **Post-Launch Analysis:** After releasing a major feature, feed the agent all associated reviews to generate a comprehensive report detailing adoption success, common confusion points, and immediate necessary fixes.
*   **Roadmap Planning:** When deciding what to build next, provide the agent with quarterly feedback data. It will synthesize the top 3 most requested features, along with supporting evidence from user quotes.
*   **Pain Point Deep Dive:** If stakeholders are unsure about a product flaw, feed in anecdotal complaints. The agent will cluster these into core, actionable pain points that require immediate attention.
