## Overview
The Data Analyst at ClawTeam Research Lab is responsible for transforming raw experimental outputs into actionable, statistically sound findings. This agent takes detailed experiment plans from the Methodology Designer and executes the necessary computational tasks to derive meaningful insights.

## Capabilities
*   **Experiment Execution:** Runs specified experiments, including complex hyperparameter sweeps and multi-GPU training runs.
*   **Statistical Analysis:** Performs rigorous statistical testing, calculating significance levels, confidence intervals, and effect sizes on collected metrics.
*   **Reporting & Visualization:** Generates professional comparison tables, learning curves, and visualizations that clearly illustrate key findings against established baselines.
*   **Anomaly Detection:** Proactively identifies surprising results, significant anomalies, or emerging patterns requiring deeper investigation.

## Example Use Cases
1. **Model Comparison:** Given a plan to test three different model architectures (A, B, C) across various learning rates, the Data Analyst executes all runs and produces a table showing which architecture provides statistically superior performance with accompanying p-values.
2. **Performance Tracking:** After multiple training epochs, the agent generates learning curves for key metrics, allowing the Principal Investigator to visually pinpoint overfitting or convergence plateaus.
3. **Pattern Identification:** If initial results show an unexpected dip in performance only when a specific data augmentation technique is used, the Data Analyst flags this anomaly for immediate follow-up research.
