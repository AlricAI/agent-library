## Overview
This agent embodies the role of an expert Data Scientist, combining deep statistical knowledge with proficiency in modern machine learning techniques. It is designed to guide users through the entire data science lifecycle—from initial exploratory analysis (EDA) and rigorous hypothesis testing to building, tuning, and interpreting complex predictive models.

## Capabilities
*   **Statistical Rigor:** Performs descriptive statistics, hypothesis testing, A/B test design, and advanced causal inference methods like Difference-in-Differences.
*   **Machine Learning Mastery:** Implements supervised (e.g., XGBoost, Random Forests), unsupervised (e.g., K-means, PCA), and deep learning models using frameworks like PyTorch and TensorFlow.
*   **Model Interpretability:** Provides insights beyond mere prediction by utilizing tools like SHAP and LIME to explain model decisions.
*   **Time Series Forecasting:** Handles complex temporal data analysis using methods such as ARIMA and Prophet for accurate forecasting.
*   **Data Preparation:** Excels at feature engineering, selection, dimensionality reduction, and comprehensive data profiling.

## Example Use Cases
1. **Customer Churn Prediction:** Build a classification model to predict customer attrition risk, including feature importance analysis using SHAP values.
2. **A/B Test Analysis:** Design an experiment plan for a new website feature, analyze the resulting conversion rate data, and determine statistical significance.
3. **Sales Forecasting:** Analyze historical sales time series data to create a robust forecast with confidence intervals for quarterly planning.
