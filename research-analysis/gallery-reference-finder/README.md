## Overview
The Gallery Reference Finder acts as a dedicated visual research assistant for AI art generation. Instead of generating directly, this agent helps you explore creative possibilities by searching and analyzing a large database of curated, high-quality AI images from the MeiGen gallery.

It is ideal when you are in the ideation phase—when you need to see what's possible, gather stylistic references, or build a comprehensive mood board before committing to a final prompt.

## Capabilities
*   **Visual Search**: Searches the extensive gallery using keywords across various categories (e.g., Photography, Illustration).
*   **Inspiration Retrieval**: Fetches detailed information, including full prompts and multiple image URLs for promising results.
*   **Pattern Synthesis**: Analyzes search results to summarize common themes, recurring styles, and reusable prompt elements.
*   **Structured Output**: Presents findings in a clean, actionable format that highlights why each reference is relevant and what parts of the prompt can be adopted.

## Example Use Cases
*   "I need some references for cyberpunk cityscapes with neon lighting." (Triggers broad search and style analysis).
*   "Show me different approaches to depicting a vintage sci-fi product." (Focuses on category filtering and comparison).
*   "I'm stuck; give me some general inspiration for fantasy character design."

By synthesizing multiple references, this agent helps you move from vague ideas to concrete, prompt-ready concepts.
