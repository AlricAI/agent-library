## Overview
This agent acts as a meticulous text comparison specialist, designed to rigorously audit and validate extracted text against established reference markdown files. Its core function is to ensure absolute accuracy and structural consistency across disparate textual sources by performing detailed line-by-line analysis.

## Capabilities
*   **Systematic Comparison:** Performs deep, line-by-line comparisons between two distinct text inputs.
*   **Discrepancy Detection:** Identifies and categorizes spelling errors, missing words, character substitutions, and structural deviations.
*   **Formatting Analysis:** Detects inconsistencies in markdown elements such as bullet points, numbering schemes, and heading structures.
*   **Structured Reporting:** Provides a comprehensive report including an overall accuracy percentage, detailed breakdowns by issue type, and specific quotes highlighting the exact differences found.
*   **Actionable Recommendations:** Delivers priority-ranked findings with clear, actionable steps and precise line references for easy correction.

## Example Use Cases
1. **Data Migration Validation:** After scraping product details from an image, use this agent to compare the extracted text against the official markdown specification file to ensure all fields are present and correctly formatted.
2. **Content Auditing:** When updating documentation, feed the old version and the new draft into the validator to pinpoint subtle changes in wording or structural drift that might have been missed during a manual review.
3. **Transcription Verification:** Compare text transcribed from an audio recording against a known source document to flag any misheard words or omitted sections.
