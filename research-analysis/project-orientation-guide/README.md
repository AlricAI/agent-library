## Overview
This agent is designed to provide a complete, high-fidelity orientation to an entire project codebase and its associated documentation. It moves beyond simple task execution by synthesizing information from multiple core documents—including the product vision, roadmap, decision history, and assumptions log.

Use this when starting a major planning session, recovering context after a long gap, or needing a holistic summary before committing to the next strategic action.

## Capabilities
*   **Comprehensive Context Reading:** Systematically reads and digests critical files like `product-vision.md`, `roadmap.md`, and `decision-history.md` in sequence.
*   **Milestone Tracking:** Assesses the current project milestone by comparing required artifacts against the repository state.
*   **Structured Reporting:** Generates a highly structured summary covering the overall project status, active assumptions, decision timelines, and immediate next steps.
*   **Gap Identification:** Clearly flags missing artifacts or low-confidence assumptions that require immediate attention.

## Example Use Cases
*   **Project Kickoff:** When onboarding a new team member to understand the entire scope and current phase of development.
*   **Strategic Review:** Before a major planning meeting, use this agent to ensure all stakeholders are aligned on the current goals and known risks (assumptions).
*   **Context Recovery:** If you have been away from the project for weeks, running this agent will quickly bring you up to speed by summarizing decisions made in your absence.
