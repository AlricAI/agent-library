## Overview
This agent specializes in enhancing early-stage product or engineering discovery by employing structured cross-domain exploration. Instead of accepting the most obvious solution, it systematically searches for analogous mechanisms, patterns, and trade-offs from unrelated fields to generate more robust and innovative design inputs.

It is designed to assist when a problem is complex, involves multiple stakeholders, or requires consideration of systemic issues like incentives, coordination, or scale.

## Capabilities
*   **Problem Abstraction:** Deconstructs a given problem into core components (objective, bottlenecks, actors, feedback loops).
*   **Domain Mapping:** Identifies 3-4 structurally similar external domains relevant to the abstract problem.
*   **Mechanism Extraction:** For each domain, it extracts transferable mechanisms and analogous problems.
*   **Synthesis & Proposal:** Synthesizes findings into actionable solution directions, design principles, and critical assumptions for validation.

## Example Use Cases
*   **Designing a decentralized governance model:** Instead of suggesting standard voting systems, the agent might draw parallels from biological evolution or historical political structures to propose novel consensus mechanisms.
*   **Improving supply chain resilience:** If conventional logistics solutions are insufficient, it could analyze resource allocation patterns found in ant colony optimization or ecological nutrient cycles.
*   **Developing user adoption strategies:** For a new technology, it can borrow incentive structures from behavioral economics (e.g., gamification loops) or social network theory to suggest better onboarding flows.
