## Overview
This agent is designed to function as an in-house subject matter expert for the Internet Computer Protocol (ICP) and Web3 technologies. Its primary goal is to build a comprehensive, documented knowledge base regarding ICP fundamentals, Motoko programming, and canister deployment procedures.

Since direct execution environments are restricted, this agent focuses on deep research, documentation curation, and creating actionable guides rather than live deployments.

## Capabilities
*   **ICP Architecture Explanation:** Provides detailed breakdowns of core components like Canisters, Subnets, and Nodes within the IC ecosystem.
*   **Motoko Guidance:** Offers educational content and examples related to Motoko, the language optimized for the Internet Computer.
*   **Conceptual Documentation:** Structures complex technical topics into easy-to-digest guides, simulating a deep dive research phase.
*   **Deployment Planning:** Creates step-by-step documentation outlining necessary actions for future canister deployments when shell access is available.

## Example Use Cases
*   **Learning Curve Mapping:** "Explain the relationship between Subnets and Canisters in simple terms." 
*   **Code Structure Drafting:** "Draft a conceptual guide on setting up an actor-based system using Motoko for state management." 
*   **Technical Gap Analysis:** "Based on this documentation, what are the prerequisite steps needed before we can run `dfx deploy`?"

This agent excels at absorbing complex technical specifications and outputting structured, highly educational reports suitable for onboarding new team members or planning architectural roadmaps.
