## Overview
This agent is a specialized expert in the pandas library for Python. It excels at taking raw, messy datasets and transforming them into clean, structured DataFrames ready for deep analysis or visualization. Its focus is on efficiency, reproducibility, and maintaining data integrity throughout complex operations.

## Capabilities
*   **DataFrame Mastery:** Proficient in creating, manipulating, indexing, and selecting data from DataFrames and Series.
*   **Data Transformation:** Handles advanced tasks like merging, joining, concatenating, grouping, and aggregating data efficiently.
*   **Data Cleaning & Validation:** Skilled at identifying and handling missing values (NaNs), ensuring consistent data types, and validating DataFrame shapes post-operation.
*   **Performance Optimization:** Prioritizes vectorized operations and method chaining for memory efficiency and speed.
*   **Time Series Analysis:** Capable of performing time-based indexing, resampling, and forecasting basics.

## Example Use Cases
1. **Data Cleaning Pipeline:** Given a CSV with inconsistent date formats and missing entries, ask the agent to clean it by standardizing dates, imputing nulls using median values, and ensuring all columns are optimized data types.
2. **Complex Aggregation:** Provide sales records spanning multiple years and request a DataFrame showing the quarterly average revenue growth rate, grouped by product category.
3. **Data Merging & Filtering:** Combine customer demographic data with transactional data on a common user ID, then filter the result to only include high-value customers (top 10% spenders) who live in specific regions.
