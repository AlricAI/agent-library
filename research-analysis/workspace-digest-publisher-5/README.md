## Overview
This agent is designed to create comprehensive, evidence-based internal digests summarizing the current state of work across multiple project dimensions. It acts as a central point for synthesizing durable artifacts, active development context, and necessary follow-up actions.

Its core function is to provide a structured roundup that prioritizes verifiable data over polished narrative, ensuring stakeholders understand both what has been accomplished and what requires immediate attention.

## Capabilities
*   **Source Synthesis:** Integrates information from the Blueprint Knowledge base (durable artifacts), Growth Studio mirror (current framing), and selected Work Queue views (action items).
*   **Digest Drafting:** Generates a concise, structured summary suitable for weekly internal reporting.
*   **Action Item Generation:** Identifies and drafts necessary follow-up work items when the synthesized digest clearly implies outstanding tasks.
*   **Contextual Awareness:** Understands the hierarchy of evidence, prioritizing raw workspace data over speculative or external marketing language.

## Example Use Cases
*   **Weekly Status Report:** Running this agent at the end of a sprint to create a single document summarizing progress across all fronts for leadership review.
*   **Hand-off Preparation:** Generating a digest specifically tailored for the `metrics-reporter` when significant metric movement has occurred, providing necessary context.
*   **State Reconciliation:** Using it in conjunction with the `notion-reconciler` to validate that the documented project state matches the actual artifacts and open tasks.
