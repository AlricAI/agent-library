## Overview
This Data Scientist Analyst agent is designed for data professionals who require deep, rigorous analysis from large datasets stored in cloud environments like BigQuery. It goes beyond simple querying by focusing on performance optimization, statistical interpretation, and delivering actionable business recommendations.

## Capabilities
*   **Advanced SQL Generation:** Writes highly optimized, commented SQL queries incorporating complex features like window functions, appropriate joins, and indexing considerations.
*   **BigQuery Execution Simulation:** Guides the process using BigQuery command-line tools (`bq`) for execution planning and performance analysis.
*   **Insight Extraction:** Analyzes query results to identify significant patterns, trends, anomalies, and statistical relationships within the data.
*   **Comprehensive Reporting:** Provides a structured output including key findings summaries, visualization recommendations, and clear documentation of assumptions and limitations.
*   **Performance & Cost Management:** Includes cost estimation for BigQuery operations and suggests optimizations to ensure queries are both accurate and cost-effective.

## Example Use Cases
1. **Customer Churn Analysis:** Upload a dataset containing user activity logs and ask the agent to write an optimized SQL query to identify the top three predictors of customer churn, along with a recommendation for retention strategies.
2. **Sales Trend Forecasting:** Provide quarterly sales data and request an analysis that not only identifies year-over-year growth but also suggests necessary adjustments to forecasting models based on observed seasonality.
3. **Data Quality Audit:** Submit schema details and sample data, asking the agent to write validation queries to check for missing values or inconsistent formats across key transactional tables.
