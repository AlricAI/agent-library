## Overview
This agent provides continuous competitive intelligence by monitoring key competitor applications in the trail and outdoor navigation space across both the Apple App Store and Google Play. It is designed to give product teams an early warning system for market shifts, feature gaps, and immediate user dissatisfaction.

## Capabilities
*   **Comprehensive Tracking:** Monitors a defined list of direct and adjacent competitors (e.g., AllTrails, GAIA GPS).
*   **Data Point Extraction:** Captures crucial metrics including current version number, overall rating trend, app size, and pricing structure.
*   **Change Detection:** Automatically compares daily data to flag new releases, significant rating drops/spikes, or major feature additions.
*   **Pain Point Mining:** Specifically analyzes 1-star reviews to identify recurring user complaints regarding bugs, missing features, or poor usability—the 'gold mine' for product improvements.

## Example Use Cases
*   **Pre-Feature Planning:** Before starting development on a new feature (e.g., offline map integration), run this agent to see if competitors have recently addressed that exact functionality.
*   **Crisis Monitoring:** Set up daily alerts to immediately notify the team if a primary competitor experiences a sudden, significant drop in rating or reports widespread crashes via 1-star reviews.
*   **Market Positioning:** Use aggregated data to build a feature matrix comparing your product's strengths against every tracked competitor based on their latest updates.
