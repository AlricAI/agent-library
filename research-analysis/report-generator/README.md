## Overview
The Report Generator is a specialized AI agent designed to take raw, synthesized research findings and transform them into polished, comprehensive final reports. Its core function is to build a clear, engaging narrative structure around complex data while maintaining the highest standards of academic rigor and proper citation.

This agent should be used as the *final* step in any multi-stage research process, ensuring that all preceding work culminates in a professional, ready-to-present document.

## Capabilities
*   **Structured Formatting:** Automatically structures reports with standard sections (Executive Summary, Introduction, Key Findings, Analysis, Conclusion, References).
*   **Narrative Cohesion:** Creates a logical flow between disparate findings, connecting data points to broader implications.
*   **Citation Management:** Ensures every claim is supported by proper, sequentially numbered citations derived from the input research.
*   **Tone Adaptation:** Adjusts the tone and format based on the specified report type (e.g., academic paper vs. executive briefing).
*   **Conflict Analysis:** Provides a balanced presentation of contradictions or debates found within the source material.

## Example Use Cases
1. **Academic Thesis Drafting:** Input multiple chapters' worth of synthesized literature reviews to generate a cohesive final draft with proper citation style.
2. **Market Analysis Briefing:** Feed in disparate data points from various sources (e.g., economic reports, survey results) and request an executive summary suitable for C-suite presentation.
3. **Technical White Paper Creation:** Provide raw experimental results and methodology summaries to generate a formal white paper that details findings and actionable recommendations.
