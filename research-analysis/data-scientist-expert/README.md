## Overview
This agent embodies the role of an expert data scientist, combining rigorous statistical foundations with state-of-the-art machine learning techniques. It is designed to guide users through the entire data science lifecycle—from initial exploratory analysis (EDA) to building and interpreting production-ready models.

## Capabilities
*   **Statistical Analysis:** Performs hypothesis testing, A/B test design, time series forecasting (ARIMA, Prophet), and causal inference methods like Difference-in-Differences.
*   **Machine Learning Modeling:** Implements supervised algorithms (XGBoost, Random Forests), unsupervised techniques (K-means, PCA), and deep learning models using PyTorch/TensorFlow.
*   **Data Preparation & Feature Engineering:** Handles data profiling, feature selection, encoding, and dimensionality reduction to optimize model performance.
*   **Model Interpretation:** Provides crucial insights into model behavior using interpretability tools like SHAP and LIME, ensuring business stakeholders understand the 'why' behind predictions.

## Example Use Cases
*   **Customer Churn Prediction:** Build a classification model (e.g., XGBoost) to predict which customers are likely to leave, identifying key contributing features via SHAP values.
*   **Sales Forecasting:** Analyze historical sales data using time series decomposition and ARIMA models to project revenue for the next quarter.
*   **Market Segmentation:** Apply unsupervised clustering (DBSCAN/K-means) on customer behavioral data to identify distinct, actionable market segments.
