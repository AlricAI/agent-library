## Overview
This agent acts as a specialized consultant for scientific research, focusing on retrieving critical data from regulatory bodies and patent databases. It is designed to assist researchers in understanding the compliance landscape, tracking intellectual property, and validating biological targets.

## Capabilities
*   **FDA Approval Queries:** Retrieves information regarding drug approvals and labeling requirements using dedicated FDA endpoints.
*   **Patent Landscape Analysis:** Performs comprehensive searches across patent databases (like USPTO) to map out existing IP claims.
*   **Sequence Database Querying:** Accesses specialized biological sequence repositories (e.g., ENA) for nucleotide data retrieval.
*   **Target Validation & Metabolomics:** Provides evidence summaries by querying target validation profiles and metabolomics datasets.

## Example Use Cases
*   **Pre-submission Planning:** A researcher needs to know the current FDA approval timeline and required labeling changes for a novel compound. This agent can query the necessary regulatory databases.
*   **Freedom to Operate (FTO) Analysis:** Before filing a patent, a team member uses this agent to search USPTO records against their core invention claims to assess potential infringement risks.
*   **Pathway Discovery:** A biologist wants to see if a specific transcription factor's binding profile overlaps with known metabolic pathways, requiring integration of JASPAR and metabolomics data.
