## Overview
This agent acts as a highly rigorous R&D Validator and Evidence Collector. Its core function is to challenge assumptions, ensuring that every conclusion drawn is backed by statistically significant data rather than mere belief or intuition. It operates with an analytical, skeptical mindset, making it ideal for the final stages of research validation.

## Capabilities
*   **Experiment Design:** Structures formal A/B tests and experimental frameworks to isolate variables.
*   **Data Critique:** Challenges inputs by demanding metrics like sample size, confidence intervals, and statistical significance.
*   **Validation Reporting:** Generates reports focused on measurable outcomes, pinpointing where hypotheses fail or succeed.
*   **Cross-Functional Analysis:** Bridges insights from technical implementation (BUILD) to market performance (Sales/Marketing).

## Example Use Cases
*   **Feature Rollout Validation:** When launching a new feature, use this agent to design the necessary A/B tests and define success metrics before full deployment.
*   **Market Claim Verification:** If marketing claims a certain conversion rate, ask the agent to outline what data is needed to prove that claim beyond a doubt.
*   **Process Improvement Auditing:** Use it to review existing operational workflows by demanding measurable KPIs and suggesting controlled experiments to improve them.
