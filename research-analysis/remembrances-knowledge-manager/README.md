## Overview
Remembrances is a specialized AI agent designed to act as an expert knowledge manager. Its core function is to interface with and manipulate structured memory systems, allowing users to effectively search, retrieve, and maintain information stored across short, mid, and long-term contexts.

This agent abstracts the complexity of managing persistent knowledge, ensuring that critical information remains accessible and up-to-date for complex reasoning tasks.

## Capabilities
*   **Advanced Search:** Performs sophisticated searches within the remembrance store using multiple techniques, including keyword-based (KB), hybrid search, and semantic vector search to maximize recall.
*   **Knowledge Modification:** Provides full CRUD (Create, Read, Update, Delete) capabilities for remembrances, allowing users to add new facts or correct outdated entries.
*   **Tool Integration:** Can enhance its memory functions by integrating with external tools, enabling it to fetch real-time data or manage related workflows alongside knowledge updates.

## Example Use Cases
*   **Research Deep Dives:** When conducting extensive research, use Remembrances to store key findings from multiple sources and later query them cohesively for a literature review.
*   **Project Documentation:** Maintain a living document of project decisions, meeting notes, and technical specifications that need to be referenced months later.
*   **Personal Knowledge Base Building:** Systematically capture lessons learned or complex procedures so they can be retrieved accurately when facing similar real-world scenarios.
