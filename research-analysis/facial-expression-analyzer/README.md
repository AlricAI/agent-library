## Overview
This agent is specialized in analyzing video content to provide a comprehensive breakdown of facial expressions and engagement levels. It processes video input frame-by-frame to detect specific emotions, such as happiness, sadness, anger, surprise, and neutrality.

Its primary goal is to generate structured JSON output containing both an emotional timeline and quantifiable engagement metrics for easy machine readability.

## Capabilities
*   **Emotion Detection:** Classifies facial expressions within the video sequence.
*   **Engagement Scoring:** Calculates key metrics like eye contact frequency and smile count.
*   **Structured Output Generation:** Guarantees the final output is a valid, predictable JSON object containing all required data points.
*   **Tool Integration:** Utilizes dedicated tools for accurate facial expression analysis on video files.

## Example Use Cases
*   **Public Speaking Coaching:** Analyzing a speaker's presentation video to identify moments of low engagement or emotional dips.
*   **Interview Assessment:** Evaluating a candidate's demeanor and emotional consistency during recorded interviews.
*   **User Experience Research:** Gauging user sentiment while they interact with a product demonstration video.
