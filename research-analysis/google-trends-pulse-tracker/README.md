## Overview
Google Trends Pulse is a specialized intelligence agent designed to monitor the daily search interest for highly relevant keywords across several distinct industry verticals (e.g., outdoor recreation, recycled goods, AI automation). By tracking shifts in search volume, this tool helps users detect emerging market opportunities or potential declines before they become obvious.

## Capabilities
*   **Trend Analysis:** Pulls and compares interest-over-time data for specified keywords over rolling 7-day periods.
*   **Signal Detection:** Automatically flags significant shifts using predefined thresholds: 'Breakout' (100%+ WoW jump), 'Rising' (20-99% WoW increase), or 'Declining' (20%+ drop).
*   **Geographic Insights:** Provides a breakdown of search interest by US state.
*   **Cross-Referencing:** Correlates rising keyword trends with specific company coverage gaps or existing inventory to suggest actionable strategies.

## Example Use Cases
*   **Content Planning:** If 'UTV trail app' shows a 'Rising' trend, the agent suggests creating content targeting that gap for DirtSync.
*   **Marketing Strategy:** If 'recycled golf balls' experiences a 'Breakout,' it triggers an alert recommending an immediate marketing push using current inventory data.
*   **Product Launch Timing:** Before launching new AI features, running this tool on 'AI agent orchestration' can confirm if market interest is peaking or waning.
