## Overview
The Template Improvement Agent is a specialized AI assistant designed for prompt engineering and document template optimization. Its core function is to take existing summary templates and performance evaluation results, then systematically diagnose weaknesses and generate superior, highly refined versions of the prompts.

This agent acts as an expert consultant, ensuring that your AI extraction or summarization tasks are robust, specific, and yield consistent, high-quality outputs.

## Capabilities
*   **Gap Analysis**: Reviews provided evaluation metrics to pinpoint exactly where current templates fail (e.g., missing data points, incorrect categorization).
*   **Root Cause Identification**: Determines the underlying reasons for suboptimal AI performance within a given prompt structure.
*   **Prompt Engineering**: Rewrites prompts using advanced techniques like few-shot examples, chain-of-thought prompting, and explicit constraints.
*   **Structural Refinement**: Reorganizes template sections to improve logical flow and guide the LLM more effectively.
*   **Validation Planning**: Provides actionable testing strategies to validate the effectiveness of the newly generated templates.

## Example Use Cases
*   **Improving Meeting Summaries:** If your current summary consistently misses action items, feed in the poor summaries and ask the agent to redesign the 'Action Items' section prompt for better extraction.
*   **Standardizing Report Extraction:** When multiple reports require similar data points but yield inconsistent results, use this agent to create a master, highly specific template that enforces consistency across all inputs.
*   **Benchmarking Prompts:** After running an initial set of tests, provide the comparison analysis and ask the agent to generate 'Version 2.0' with explicit instructions on what was improved from 'Version 1.0'.
