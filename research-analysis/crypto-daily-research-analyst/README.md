## Overview
This agent functions as a professional crypto research analyst, designed to compile thorough daily market reports covering Bitcoin (BTC) and Ethereum (ETH). It systematically gathers the latest pricing data, significant news developments, and critical on-chain metrics using live web search capabilities.

## Capabilities
*   **Real-Time Data Retrieval:** Uses `WebSearch` to fetch current prices, 24h changes, and trading volumes for BTC and ETH.
*   **News Aggregation:** Searches for and extracts 3-5 key news items with sources for both major cryptocurrencies.
*   **On-Chain Analysis:** Gathers relevant on-chain metrics for Bitcoin and Ethereum using specified timeframes.
*   **Structured Reporting:** Compiles all disparate data points into a single, professionally formatted report structure, including an Executive Summary.
*   **Output Management:** Utilizes a write tool to save the final, comprehensive report to a specified file path.

## Example Use Cases
1. **Daily Market Briefing:** Run this agent at the start of each trading day to receive a complete snapshot of BTC and ETH performance for stakeholders.
2. **Due Diligence:** Quickly gather all necessary data points—price, news sentiment, and on-chain health—to support investment thesis development.
3. **Comparative Analysis:** Compare the narrative surrounding Bitcoin versus Ethereum using the structured report format.
