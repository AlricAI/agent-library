## Overview
This agent is designed to be the central hub for synthesizing complex, multi-source intelligence regarding market demand and buyer needs. It prioritizes primary evidence—such as operator interviews, product documentation, and live web research—over secondary commentary.

Its core function is not just to collect information but to structure it into actionable insights that can directly inform product development or sales strategy within the Blueprint ecosystem.

## Capabilities
*   **Source Triangulation:** Compares findings across multiple knowledge bases (e.g., agent programs, buyer dossiers, and live web data) to build a robust view of reality.
*   **Structured Insight Generation:** Utilizes specialized tools for structured output, such as Jobs-To-Be-Done (JTBD) frameworks, persona development, and objection mapping from Voice of Customer (VOC) evidence.
*   **Actionable Workflow Mapping:** Guides the process from raw finding to actionable follow-up by integrating with issue lifecycle tools, determining if an insight requires 'done' or 'blocked' status.
*   **Trust Layer Enforcement:** Adheres strictly to a trust model that prioritizes direct primary evidence and flags areas requiring mandatory human oversight (e.g., pricing, legal).

## Example Use Cases
*   **Validating New Features:** When launching a new feature, use this agent to cross-reference preliminary user feedback against existing buyer dossiers and recent event talk transcripts to confirm market resonance.
*   **Understanding Competitive Gaps:** Feed it competitor documentation alongside internal product roadmaps. The agent will synthesize gaps by comparing stated capabilities against documented primary needs.
*   **Drafting Demand Reports:** After gathering all necessary inputs, use the deterministic demand-intel writer tool to generate a final, structured report that mirrors an operator-facing artifact, ensuring continuity and clear next steps.
