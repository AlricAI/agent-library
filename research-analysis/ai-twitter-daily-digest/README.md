## Overview
This agent provides a daily intelligence digest by monitoring specified, curated Twitter accounts belonging to leading AI researchers. It automates the entire process from data fetching to final delivery, ensuring subscribers stay updated on the latest trends and discussions within the fast-moving field of artificial intelligence.

## Capabilities
*   **Automated Data Fetching:** Retrieves tweets from multiple sources using advanced search API techniques designed to work around rate limits.
*   **Intelligent Analysis:** Utilizes AI models (like OpenAI) to group raw tweets by emerging topics and extract high-level, actionable insights.
*   **Multi-Format Reporting:** Generates a polished, structured Markdown report detailing key findings.
*   **Audio Synthesis:** Creates an accompanying audio podcast script and synthesizes it into an audible summary using services like ElevenLabs.
*   **Subscription Management:** Supports subscription commands (`AIT SUB`) for reliable daily delivery to registered users.

## Example Use Cases
*   **Daily Briefing:** A user can run the agent manually or wait for the scheduled daily run to receive a complete package of insights, ready for consumption during their morning routine.
*   **Trend Spotting:** By analyzing topic groupings across multiple experts, the system helps identify nascent research areas before they become mainstream knowledge.
*   **Content Archiving:** The agent systematically stores both the report and covered tweets in the database, creating a searchable historical record of AI discourse.
