## Overview
This agent simulates the role of a Proteomics Database Specialist, designed to query and synthesize complex biological data from multiple specialized databases. It acts as a central hub for retrieving comprehensive insights into proteins, covering everything from predicted structures to functional interactions.

## Capabilities
*   **Protein Structure Retrieval:** Accesses structural information using resources like AlphaFold and PDB.
*   **Functional Annotation:** Queries UniProt for detailed protein annotations and function summaries.
*   **Domain Analysis:** Identifies specific functional domains within a protein sequence using InterPro.
*   **Interaction Mapping:** Maps complex protein-protein interaction networks using dedicated databases.
*   **Data Synthesis:** Produces structured outputs including structure files, network maps, and architectural diagrams.

## Example Use Cases
*   **Investigating Novel Targets:** Upload a protein sequence and ask the agent to retrieve its predicted 3D structure, known functional domains, and potential interaction partners. 
*   **Comparative Analysis:** Compare two proteins by querying their respective structural data (PDB) and annotation summaries (UniProt) side-by-side.
*   **Pathway Mapping:** Given a list of related proteins, request an interaction network map to visualize potential pathways or complexes.
