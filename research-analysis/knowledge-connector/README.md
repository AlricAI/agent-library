## Overview
The Knowledge Connector is a specialized agent designed to transform isolated documents within your knowledge management system into a richly interconnected web of knowledge. It moves beyond simple keyword matching by analyzing semantic relationships, entity mentions, and structural patterns across your entire vault.

Its core function is to identify *meaningful* connections—the links that truly deepen understanding—and flag areas where crucial information might be orphaned or disconnected.

## Capabilities
*   **Entity Extraction & Mapping:** Automatically identifies and maps key entities (people, technologies, companies) across multiple notes for relationship building.
*   **Semantic Similarity Analysis:** Goes beyond keyword overlap to suggest connections based on underlying conceptual similarity between documents.
*   **Orphaned Note Detection:** Scans the vault structure to pinpoint notes that lack incoming or outgoing links, ensuring no knowledge is lost.
*   **Connection Pattern Analysis:** Generates reports detailing clusters of related topics and highlights potential 'knowledge gaps' where more linking could be beneficial.
*   **Prioritized Suggestion Reports:** Outputs actionable link suggestions ranked by a confidence score and strategic importance for efficient manual curation.

## Example Use Cases
*   **Building MOCs (Maps of Content):** Feed it a directory structure and receive prioritized links to build comprehensive, navigable indexes.
*   **Onboarding New Material:** When adding a large body of research, run the agent to immediately suggest where this new content should link back to existing concepts.
*   **Auditing Knowledge Bases:** Use it periodically to generate a 'Knowledge Health Report,' identifying underconnected or isolated topics that need attention.
