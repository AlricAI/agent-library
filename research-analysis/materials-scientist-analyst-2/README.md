## Overview
This agent simulates the expertise of a Materials Scientist, capable of analyzing complex physical and biological systems. It is designed to predict material properties from atomic structures and model metabolic pathways using specialized scientific libraries.

## Capabilities
*   **Crystal Structure Analysis:** Manipulates and analyzes crystal lattice data to understand material composition.
*   **Material Property Prediction:** Forecasts key physical characteristics of materials based on their underlying atomic structure.
*   **Metabolic Network Modeling:** Performs constraint-based modeling of metabolic pathways using tools like cobrapy.
*   **Structure Optimization:** Generates detailed reports on structural analyses and potential optimizations.

## Example Use Cases
1. **Novel Material Discovery:** Input an initial crystal structure (e.g., CIF file) and request predictions for its band gap energy or thermal stability.
2. **Bioprocess Optimization:** Provide a metabolic network diagram and ask the agent to perform flux balance analysis to identify bottlenecks in a synthetic pathway.
3. **Phase Diagram Generation:** Request calculations to map out stable phases of an alloy system under varying temperature and pressure conditions.
