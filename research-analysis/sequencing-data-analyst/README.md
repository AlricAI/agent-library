## Overview
This agent simulates the role of a specialized Sequencing Data Analyst, proficient in handling complex biological datasets from next-generation sequencing. It is designed to perform rigorous quality control and detailed analysis across multiple 'omics' modalities.

## Capabilities
*   **Sequencing QC:** Performs comprehensive quality control checks on raw sequencing reads.
*   **Epigenomic Analysis:** Analyzes marks like ChIP-seq and ATAC-seq to identify enriched genomic regions and peak calls.
*   **Flow Cytometry Processing:** Processes and analyzes data from flow cytometry files, including necessary gating strategies.
*   **Database Querying:** Interacts with biological databases using specialized tools for targeted information retrieval.
*   **Reporting:** Generates structured reports covering coverage metrics, enrichment profiles, and quality assessments.

## Example Use Cases
*   **Assessing Library Quality:** Upload raw sequencing files to generate a full QC report detailing read depth and adapter contamination.
*   **Identifying Regulatory Elements:** Input ChIP-seq data to pinpoint regions of histone modification enrichment across the genome.
*   **Analyzing Cell Populations:** Process flow cytometry FCS files to establish appropriate gates and quantify distinct cell populations within a sample set.
