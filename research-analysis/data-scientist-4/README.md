## Overview
This Data Scientist Analyst agent is designed to act as a senior data scientist specializing in complex data querying and analysis within SQL and Google BigQuery environments. It moves beyond simple query writing by focusing on the entire analytical lifecycle: from understanding the business problem to delivering actionable, statistically sound recommendations.

## Capabilities
*   **Advanced Query Generation:** Writes highly optimized, commented SQL queries incorporating best practices like proper filtering, indexing considerations, and window functions.
*   **Performance Analysis:** Provides query execution plans and cost estimations for BigQuery operations, ensuring results are both accurate and cost-effective.
*   **Insight Extraction:** Analyzes the resulting data to identify key patterns, significant trends, anomalies, and statistical relationships.
*   **Comprehensive Reporting:** Delivers a structured output including data summaries, visualization recommendations, documented assumptions, and clear limitations.
*   **Actionable Recommendations:** Concludes with concrete, business-oriented next steps and follow-up analysis suggestions based on the findings.

## Example Use Cases
*   **Customer Churn Analysis:** Given raw user activity logs, ask it to write a query that identifies the top 3 predictors of customer churn over the last quarter, along with statistical significance metrics.
*   **Sales Performance Deep Dive:** Provide sales transaction data and request an analysis comparing regional performance against historical benchmarks, including cost-optimized SQL for BigQuery execution.
*   **Data Quality Audit:** Upload a dataset schema and ask it to write validation queries to check for missing values, inconsistencies, or unexpected data distributions across key metrics.
