## Overview
Business Analyst Pro is a senior-level AI agent designed to act as an expert business analyst. Its core function is to bridge the gap between abstract business needs and concrete, actionable technical solutions. It systematically analyzes existing processes, identifies gaps, and delivers comprehensive recommendations supported by data insights.

## Capabilities
*   **Requirements Elicitation:** Conducts stakeholder interviews, facilitates workshops, and generates detailed user stories with clear acceptance criteria.
*   **Process Modeling:** Maps current ('As-Is') and future ('To-Be') states using industry standards like BPMN, performing value stream mapping and gap analysis.
*   **Data Analysis & Insights:** Develops KPIs, suggests SQL queries for data verification, and performs root cause analysis to pinpoint optimization opportunities.
*   **Solution Design:** Creates functional specifications, data flow diagrams, and integration maps necessary for technical implementation planning.
*   **Strategic Analysis:** Executes SWOT analyses, cost-benefit assessments, and comprehensive risk identification to ensure ROI is maximized.

## Example Use Cases
1. **System Upgrade Planning:** Feed it existing operational documentation and stakeholder interviews; it will output a gap analysis report detailing necessary features, process changes, and required technical specifications.
2. **Process Optimization:** Provide a workflow diagram (e.g., customer onboarding); the agent will map out bottlenecks, suggest automation points, and define measurable success metrics for improvement.
3. **Feasibility Study:** Give it a high-level business objective; the agent will structure a full analysis plan, including required data sources, necessary stakeholder sign-offs, and preliminary risk assessments.
