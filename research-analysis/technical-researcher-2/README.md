## Overview
The Technical Researcher is an expert AI agent designed for in-depth analysis of complex technical artifacts, including source code repositories, API specifications, and comprehensive technical documentation. It moves beyond simple summarization to provide actionable insights into architecture, quality, and implementation patterns.

## Capabilities
*   **Repository Analysis:** Assesses GitHub projects by analyzing star count, recent activity, maintenance status, and overall community engagement.
*   **Code Quality Assessment:** Reviews codebases against industry best practices, identifying potential bugs, inefficiencies, and adherence to established design patterns.
*   **Architecture Review:** Deconstructs system designs to map out technology stacks, identify dependencies, and evaluate scalability bottlenecks.
*   **Implementation Comparison:** Compares multiple technical approaches or solutions found across different sources, providing a balanced view of trade-offs.
*   **Security & Performance Evaluation:** Highlights potential security vulnerabilities and assesses the performance implications of proposed implementations.

## Example Use Cases
1. **Evaluating a New Library:** Provide it with a GitHub link and ask it to assess its maturity, primary technology stack, and suggest three areas for immediate improvement.
2. **API Integration Planning:** Feed it an OpenAPI specification and request an analysis comparing the best implementation patterns for handling asynchronous vs. synchronous calls in a microservices environment.
3. **Technical Due Diligence:** Give it documentation for two competing solutions and ask it to compare their architectural trade-offs regarding long-term maintainability and performance under heavy load.
