## Overview
This agent is designed to move beyond simple pattern matching by detecting *emergent* patterns—those that arise from the productive interaction and tension between diverse inputs, perspectives, or code components. It treats diversity as an engine for discovery, identifying higher-level structures rather than just repeating observed elements.

## Capabilities
*   **Orchestrate Diversity**: Coordinates multiple viewpoints to maximize informational richness before analysis.
*   **Detect Emergence**: Identifies patterns that are not obvious from any single source but arise from their combination (e.g., Divergence, Tension, Cascade).
*   **Analyze Orchestration Models**: Applies frameworks like Symphony-Jazz Hybrid or Research Lab Forum to structure complex problem spaces.
*   **Measure System Health**: Assesses diversity and emergence using defined metrics (e.g., Perspective Diversity Index, Productive Tension Levels).

## Example Use Cases
*   **Codebase Review**: Feed it multiple pull requests addressing the same feature; it will identify the underlying architectural pattern that should be standardized.
*   **Strategy Synthesis**: Input market reports from varied sources (economic, social, technical); it can pinpoint emergent strategic vectors or productive tensions to focus on.
*   **Process Documentation**: After running several experimental workflows, use this agent to document not just *what* worked, but the underlying meta-pattern of *how* success was achieved.
