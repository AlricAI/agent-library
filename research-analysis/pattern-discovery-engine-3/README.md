## Overview
This agent is designed to move beyond simple pattern matching by actively orchestrating diverse perspectives. Its core function is detecting *emergent* patterns—those that arise from the productive tension and interaction between multiple, varied inputs, rather than being inherent in any single source.

The system models several types of emergence (Divergence, Tension, Cascade, Void, Meta) to provide a deep, multi-layered understanding of complex systems or large codebases.

## Capabilities
*   **Orchestrate Diversity**: Coordinates multiple viewpoints simultaneously to maximize analytical richness.
*   **Detect Emergence Types**: Identifies patterns arising from productive tension (e.g., 'creates' + 'removes' $\rightarrow$ 'transforms') or gaps in knowledge.
*   **Apply Orchestration Models**: Utilizes frameworks like the Symphony-Jazz Hybrid or Research Lab Forum to structure analysis across varied inputs.
*   **Measure System Health**: Provides quantitative metrics on diversity, productive tension levels, and meta-pattern frequency for systematic review.

## Example Use Cases
*   **Codebase Standardization**: Feed it multiple feature implementations written by different engineers; the agent will extract common architectural patterns or anti-patterns to create a style guide.
*   **Process Improvement**: Input meeting transcripts, decision logs, and project retrospectives. The agent can pinpoint recurring systemic bottlenecks (Void Emergence) or conflicting departmental goals (Tension Emergence).
*   **Strategy Formulation**: When brainstorming solutions, run the concept through multiple specialized agents simultaneously to generate a comprehensive map of potential outcomes and underlying structural patterns.
