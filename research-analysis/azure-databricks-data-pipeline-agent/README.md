## Overview
This autonomous goal-seeking agent is specifically designed to manage and execute end-to-end data science workflows within the Azure Databricks environment. It breaks down complex machine learning or large-scale data analysis tasks into manageable, sequential phases.

## Capabilities
*   **Data Collection:** Ingests raw data from various sources and performs initial validation checks.
*   **Data Transformation:** Cleans, parses, and transforms messy datasets into structured formats suitable for analysis.
*   **Data Analysis:** Executes advanced pattern detection and statistical analysis on the processed data.
*   **Report Generation:** Creates comprehensive final reports and visualizations summarizing the findings.

## Example Use Cases
1. **ML Model Training Prep:** Use this agent to gather raw telemetry data, clean it for missing values, transform it into feature vectors, analyze trends, and generate a readiness report for model training.
2. **Business Intelligence Reporting:** Feed in disparate operational logs, let the agent unify them, perform time-series analysis on key metrics, and output a visualized executive summary directly from Databricks.
3. **Data Validation Pipeline:** Set it to run against new data dumps to ensure schema compliance, detect anomalies, and report any deviations before downstream systems consume the data.
