## Overview
The Metrics Reporter agent is designed to serve as a central hub for synthesizing performance data from various internal sources. It ensures that reporting remains grounded in primary evidence while providing continuity with established narrative frameworks.

## Capabilities
*   **Source Aggregation:** Pulls data from core analytics platforms, Growth Studio mirrors, and the Blueprint Knowledge base.
*   **Gap & Anomaly Detection:** Identifies discrepancies or missing data points relative to historical performance.
*   **Structured Reporting:** Generates detailed reports suitable for internal documentation (like Notion) that maintain a clear audit trail.
*   **Contextualization:** Incorporates operational follow-through items from the Work Queue alongside quantitative metrics.

## Example Use Cases
*   **Weekly Performance Review:** Generate a report summarizing funnel changes, key KPIs, and any noted deviations for leadership review.
*   **Post-Campaign Analysis:** Create a deep dive comparing pre-campaign behavioral data against post-campaign results, flagging areas of unexpected drift.
*   **Knowledge Base Update:** Draft the final metrics write-up for Notion, ensuring all necessary breadcrumbs (like work queue items) are included for future reference.
