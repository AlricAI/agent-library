## Overview
The Technical Researcher is an advanced AI agent designed for deep technical due diligence. It specializes in dissecting complex software artifacts, including entire GitHub repositories, detailed API specifications, and extensive technical documentation. Its goal is to provide a comprehensive assessment of viability, quality, and best practices.

## Capabilities
*   **Repository Analysis:** Assesses open-source projects by analyzing metrics like star count, recent activity, and overall maintenance status.
*   **Code Quality Assessment:** Reviews codebases against industry best practices, identifying potential bugs, anti-patterns, and areas for refactoring.
*   **Architecture Review:** Maps out the system's design patterns, technology stack breakdown, and component interactions.
*   **Implementation Comparison:** Compares multiple technical approaches or solutions to recommend the most robust path forward.
*   **Security & Performance Evaluation:** Highlights potential security vulnerabilities and analyzes scalability/performance bottlenecks.

## Example Use Cases
*   **Evaluating a New Library:** Feed it a GitHub link and ask for an assessment of its maturity, dependency risks, and core architectural strengths.
*   **API Integration Planning:** Provide API documentation and ask the agent to outline the best implementation pattern in Python, including necessary error handling.
*   **Technical Debt Audit:** Give it a codebase snippet or README and request a detailed report on technical debt, suggesting modern alternatives for outdated patterns.
