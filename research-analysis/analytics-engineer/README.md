## Overview
This agent specializes in the end-to-end lifecycle of data analysis for product and game development. Its core function is to build robust telemetry systems that capture player interactions, allowing teams to move beyond intuition and make decisions based on quantitative evidence.

## Capabilities
*   **Telemetry Schema Design:** Defining precise event structures, properties, and tracking requirements for comprehensive data capture.
*   **Data Pipeline Construction:** Building the necessary infrastructure from instrumentation (client-side) through transport, storage, and final query layers.
*   **A/B Testing Framework Management:** Designing full experiment plans, including hypothesis formulation, audience segmentation, statistical power calculations, and result interpretation.
*   **Behavioral Analysis:** Analyzing key metrics such as retention curves, funnel conversion rates, feature adoption patterns, and identifying early churn signals.
*   **Reporting & Visualization:** Creating actionable dashboards and reports that surface critical performance indicators to stakeholders.

## Example Use Cases
*   **Validating Game Mechanics:** A game designer needs to know if a new combat mechanic increases player engagement. The agent designs the necessary telemetry, builds the pipeline to track usage frequency, and sets up an A/B test comparing old vs. new mechanics.
*   **Monitoring Economic Health:** An economy designer requires data on currency flow and transaction patterns. The agent structures schemas to capture all in-game purchases and resource exchanges, building dashboards for real-time monitoring.
*   **Identifying Retention Issues:** Live Ops needs to understand why players stop playing after the first week. The agent analyzes session length correlations with specific feature usage, pinpointing potential points of friction in the user journey.
