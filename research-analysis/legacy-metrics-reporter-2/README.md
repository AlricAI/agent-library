## Overview
This agent is designed for legacy continuity, acting as a reliable mirror or backfill mechanism for older reporting artifacts that were originally Notion-facing. Its core value lies in providing decision-grade signals rather than simply mirroring dashboard noise. It prioritizes trust and clarity over comprehensive coverage.

## Capabilities
*   **Signal vs. Noise Distinction:** Expertly filters out instrumentation churn to highlight true, actionable metrics.
*   **Consistency Enforcement:** Maintains consistent reporting logic across disparate sources like Analytics, Growth Studio, and Work Queue for legacy mirroring.
*   **Risk Highlighting:** Explicitly calls out anomalies and missing evidence rather than presenting overly optimistic summaries.
*   **Artifact Trail Management:** Ensures a clean, traceable record of all reports within the Knowledge base and Agent Runs.

## Example Use Cases
*   **Backfilling Old Dashboards:** When an old Notion page needs to be updated with current metrics but cannot connect directly to live sources.
*   **Auditing Historical Data:** Generating a trustworthy snapshot report for compliance or historical review where source systems are deprecated.
*   **Cross-System Reconciliation:** Creating a single, digestible report that reconciles key performance indicators (KPIs) pulled from multiple, disparate legacy data silos.
