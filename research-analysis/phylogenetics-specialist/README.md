## Overview
This agent simulates the role of an Evolutionary Genomics Specialist, designed to construct and analyze complex phylogenetic trees from molecular sequence data. It leverages specialized bioinformatics libraries to provide rigorous analyses of evolutionary relationships.

## Capabilities
*   **Phylogenetic Tree Construction:** Builds robust phylogenetic trees using appropriate inference methods.
*   **Sequence Alignment & Homology Study:** Performs necessary alignments and analyzes genetic homology between sequences.
*   **Diversity Metrics Calculation:** Computes various diversity metrics using established bioinformatics toolkits.
*   **Divergence Analysis:** Estimates species divergence times and performs molecular clock analyses.
*   **Output Generation:** Produces structured outputs including trees with statistical support values, distance matrices, and ancestral sequence reconstructions.

## Example Use Cases
*   **Species Relationship Mapping:** Upload multiple gene sequences from different taxa and request a fully resolved phylogenetic tree to visualize evolutionary divergence.
*   **Molecular Clock Dating:** Provide fossil records alongside genetic data to estimate the time of divergence between two or more lineages.
*   **Comparative Genomics:** Submit sets of aligned sequences and ask for an analysis detailing potential functional constraints or shared ancestry patterns.
