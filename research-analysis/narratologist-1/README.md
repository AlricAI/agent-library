## Overview
The Narrative Structure Analyst is an AI designed for rigorous literary and narrative deconstruction. It functions as a senior narrative theorist, analyzing any given story—whether it's a novel plot, screenplay outline, or game concept—by applying established frameworks from structuralism to modern screenwriting theory.

This agent doesn't just summarize; it diagnoses the underlying mechanics of storytelling, identifying where the structure is strong, where tension is unresolved, and what theoretical models best explain its construction.

## Capabilities
*   **Structural Mapping:** Identifies core narrative elements like the controlling idea (McKee) or premise (Egri).
*   **Arc Evaluation:** Assesses character development using established models (e.g., flat vs. round arcs, transformation journeys).
*   **Theory Application:** Grounds all analysis in named theoretical frameworks such as Propp's morphology, Campbell's Monomyth, and Chekhov's Gun.
*   **Fabula vs. Sjuzhet Distinction:** Clearly separates the raw chronological events (fabula) from the way they are presented to the audience (sjuzhet).

## Example Use Cases
1. **Script Review:** Provide a three-act structure and ask the agent to evaluate if the inciting incident adheres to genre expectations, citing which structural pattern it fulfills.
2. **Worldbuilding Critique:** Submit character biographies and request an analysis of their internal conflict using the 'want/need/lie' framework to ensure thematic consistency.
3. **Plot Hole Detection:** Present a sequence of events and ask the agent to track narrative promises, ensuring every introduced element has a satisfying payoff or structural justification.
