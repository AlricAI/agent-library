## Overview
This agent acts as an expert visual analysis and Optical Character Recognition (OCR) specialist. Its core function is to process PNG images, not merely extracting text, but understanding and replicating the document's entire structure, including headings, lists, spacing, and formatting, into clean Markdown.

It goes beyond basic OCR by mapping visual hierarchy—such as columns, indentations, and emphasized text—into semantic markdown elements, providing a highly structured output that mirrors the original source material.

## Capabilities
*   **High-Accuracy OCR:** Extracts all visible text, including special characters and complex symbols.
*   **Structural Mapping:** Identifies and maps visual elements (headings, lists, tables) to appropriate Markdown syntax.
*   **Formatting Preservation:** Maintains logical flow, paragraph spacing, and relative positioning of content.
*   **Quality Assurance:** Provides confidence notes on extraction completeness and flags any ambiguous sections encountered during processing.

## Example Use Cases
*   **Academic Paper Digitization:** Uploading a scanned research paper (PNG) to receive perfectly formatted markdown ready for direct use in documentation tools.
*   **Infographic Interpretation:** Analyzing complex diagrams or flowcharts embedded in an image and extracting the accompanying textual labels while preserving their intended relationships.
*   **Report Conversion:** Converting screenshots of structured reports (e.g., financial summaries) into editable, well-formatted markdown documents for further analysis.
