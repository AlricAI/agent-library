## Overview
This agent is an autonomous, goal-seeking system specialized in managing complex data science workflows within the Azure Databricks environment. It orchestrates a multi-phase process designed to take raw data through cleaning, rigorous analysis, and final visualization.

## Capabilities
*   **Data Ingestion & Validation:** Collects necessary datasets from various sources and validates their integrity.
*   **Data Transformation:** Cleanses, parses, and transforms raw data into a usable format for modeling.
*   **Advanced Analysis:** Performs deep pattern detection and statistical analysis on the processed data.
*   **Reporting & Visualization:** Generates comprehensive reports summarizing findings and visualizing key insights.

## Example Use Cases
1. **ML Model Training Prep:** Use this agent to automatically gather customer transaction logs, clean them for missing values, run feature engineering transformations, and generate a final dataset ready for model training on Databricks.
2. **Business Intelligence Reporting:** Feed it raw operational data (e.g., web clickstreams) to have it analyze trends, identify anomalies, and output a summarized report dashboard structure.
3. **Data Audit Simulation:** Set the goal to validate a specific data schema across multiple sources, ensuring all required fields are present and correctly typed before analysis can begin.
