## Overview
The Model QA Specialist acts as the final gatekeeper for AI model outputs within Agency Agents. Its primary function is to rigorously validate generated content, ensuring it meets the highest standards of accuracy, adherence to established guidelines, and overall quality expected by the Specialized Operations division.

This agent operates under a strict set of internal protocols, making it ideal for mission-critical tasks where errors are costly or unacceptable.

## Capabilities
*   **Protocol Enforcement:** Strictly follows detailed operational procedures documented in its source material.
*   **Quality Auditing:** Systematically reviews outputs against predefined quality metrics (e.g., tone, factual accuracy, completeness).
*   **Error Identification:** Pinpoints discrepancies, logical flaws, or deviations from best practices within provided text or data.
*   **Structured Feedback Generation:** Delivers actionable, structured feedback detailing necessary revisions rather than simple pass/fail judgments.

## Example Use Cases
*   **Pre-Publication Review:** Before deploying any AI-generated report or article, run it through this agent to guarantee compliance with corporate style guides and factual integrity.
*   **System Output Validation:** When integrating outputs from multiple specialized agents, use the QA Specialist to harmonize and validate the final composite result against a master set of rules.
*   **Process Documentation Check:** Submit complex workflows or process documents for review to ensure they are logically sound and executable according to internal standards.
