## Overview
This agent is designed to rapidly onboard a team or individual into the complex ecosystem of the Internet Computer Protocol (ICP). It functions as an in-house subject matter expert, focusing initially on foundational research and documentation when direct deployment environments are restricted.

Its core mission is to build comprehensive knowledge around ICP architecture, Motoko programming, and canister development best practices for Performance Supply Depot LLC.

## Capabilities
*   **ICP Architecture Deep Dive:** Explains core components like Canisters, Subnets, Nodes, and Chain Key Cryptography.
*   **Motoko Guidance:** Provides educational content on the Motoko language, its actor-based model, and suitability for IC development.
*   **Knowledge Curation:** Creates structured guides, documentation, and conceptual blueprints, even when live deployment tools are unavailable.
*   **Step-by-Step Planning:** Develops detailed, actionable plans for future technical implementations (e.g., documenting required shell access steps).

## Example Use Cases
*   **Initial Onboarding:** Ask the agent to generate a comprehensive 'ICP Fundamentals' guide suitable for non-technical stakeholders.
*   **Technical Deep Dive:** Request an explanation comparing canister state management in Motoko versus Rust on the IC.
*   **Roadmapping:** Instruct it to create a phased deployment roadmap, detailing prerequisites and knowledge gaps needed before live `dfx` access is granted.
