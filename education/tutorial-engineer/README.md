## Overview
The Tutorial Engineer is a specialized AI designed to act as a pedagogical expert, transforming dense technical material or complex code into highly structured, hands-on learning experiences. It follows established principles of adult learning theory, ensuring that knowledge transfer is progressive and actionable.

This agent doesn't just summarize; it *teaches*. It structures content to guide the reader from foundational concepts to mastery through guided practice and challenges.

## Capabilities
* **Pedagogical Structuring**: Breaks down complex topics into atomic, sequential learning units.
* **Progressive Disclosure**: Ensures that advanced concepts are only introduced after prerequisite knowledge is established.
* **Hands-On Exercise Generation**: Creates scaffolded coding exercises that build in difficulty (simple to complex).
* **Comprehensive Structure**: Automatically builds tutorials with clear objectives, prerequisites, guided steps, and troubleshooting sections.
* **Learning Style Adaptation**: Incorporates analogies and practical examples to cater to diverse learning preferences.

## Example Use Cases
1. **Onboarding Guides**: Feed it a new microservice's API documentation, and ask it to generate a 5-day onboarding tutorial for new team members.
2. **Concept Deep Dives**: Provide a complex algorithm (e.g., Dijkstra's), and request a full tutorial explaining the theory, implementation steps, and variations.
3. **Feature Walkthroughs**: Give it a finished feature set and ask it to create a step-by-step guide for end-users or junior developers learning that specific functionality.
