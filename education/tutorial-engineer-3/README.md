## Overview
The Tutorial Engineer is a specialized AI designed to transform dense, complex technical material into highly engaging and progressive learning experiences. It operates on established pedagogical principles, ensuring that the reader moves from foundational theory to practical mastery through guided, hands-on exercises.

This agent doesn't just summarize; it structures knowledge for optimal retention, making it perfect for creating official documentation, internal onboarding materials, or comprehensive feature guides.

## Capabilities
*   **Progressive Disclosure**: Decomposes large topics into small, digestible, sequential steps.
*   **Hands-On Exercise Generation**: Creates practical coding challenges that build in complexity (simple to complex).
*   **Structured Content Output**: Adheres to a proven tutorial format including objectives, prerequisites, guided practice, and troubleshooting sections.
*   **Learning Style Adaptation**: Incorporates analogies and varied examples to cater to different learning preferences.

## Example Use Cases
1. **Onboarding Guides**: Input raw API documentation and receive a step-by-step guide for a new developer to get their first endpoint working.
2. **Feature Deep Dives**: Provide a complex library feature, and the agent will generate a tutorial showing its minimal implementation, guided usage, and advanced variations.
3. **Concept Explanation**: Feed in academic papers or technical specifications, and receive an educational breakdown that explains 'why' something works using real-world analogies.
