## Overview
The Learning Path Generator is a specialized AI agent designed to create detailed, structured learning roadmaps. Instead of providing general advice, this agent breaks down complex skills or knowledge domains into manageable, sequential steps, making it ideal for self-learners and corporate training departments.

## Capabilities
*   **Goal Decomposition:** Takes a high-level goal (e.g., 'Become proficient in machine learning') and breaks it into prerequisite knowledge areas.
*   **Curriculum Structuring:** Organizes topics into logical modules or phases, ensuring a smooth progression of difficulty.
*   **Resource Suggestion Integration:** Utilizes integrated tools to suggest specific resources, projects, or study materials for each step (though the actual tool implementation details are abstracted).
*   **Customization:** Can adjust paths based on user-defined time constraints or existing knowledge levels.

## Example Use Cases
*   **Career Transition:** A user wants to switch careers into Data Science. The agent can map out a 12-month learning path covering Python, statistics, SQL, and ML frameworks.
*   **Skill Mastery:** An individual needs to learn advanced topics in cloud architecture (e.g., AWS). The agent will structure the path from foundational services to advanced deployment patterns.
*   **Academic Planning:** A student preparing for a comprehensive exam can input the syllabus, and the agent will generate a weighted study plan across all required subjects.
