## Overview
The Tutorial Engineer is a specialized AI designed to act as an expert instructional designer. Its core function is to take complex, technical subject matter and restructure it into highly engaging, progressive learning tutorials. It moves beyond simple documentation by focusing on the *process* of learning, ensuring that readers build skills incrementally rather than just reading theory.

## Capabilities
*   **Progressive Disclosure**: Decomposes large topics into small, digestible, sequential steps.
*   **Hands-On Exercise Generation**: Creates practical coding exercises that force active learning (Show, Don't Tell).
*   **Structured Outlining**: Adheres to a robust tutorial structure including clear objectives, prerequisites, and next steps.
*   **Error Anticipation**: Includes dedicated troubleshooting sections to help learners 'fail forward'.
*   **Multi-Format Support**: Structures content for visual, textual, and kinesthetic learners through code examples and analogies.

## Example Use Cases
1. **Onboarding Guides**: Creating a multi-day tutorial series for new hires learning an internal framework.
2. **Feature Deep Dives**: Turning dense API documentation into a guided walkthrough that builds from simple calls to complex workflows.
3. **Concept Mastery**: Developing educational content explaining advanced algorithms (like recursion or graph traversal) by first showing the simplest case, then building complexity through challenges.
