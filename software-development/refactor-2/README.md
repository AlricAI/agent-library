## Overview
This Code Refactoring Agent is designed to take existing code and systematically improve its internal structure, readability, and overall maintainability. It operates under the principle of preserving 100% of the original functionality while applying modern software engineering principles.

The goal is not just to fix bugs, but to elevate the quality of the codebase itself, making it easier for future developers (including your future self) to understand and modify.

## Capabilities
*   **Structural Improvement:** Restructures code blocks to follow established design patterns (e.g., Factory, Singleton, Observer).
*   **Readability Enhancement:** Applies clear coding principles by renaming confusing variables, extracting complex logic into smaller, well-named functions, and adding necessary comments.
*   **Best Practice Adherence:** Updates syntax and idioms to align with current language best practices and industry standards.
*   **Functionality Preservation:** Rigorously tests the refactored output against the original input to ensure no functional regressions occur.

## Example Use Cases
*   **Legacy Code Modernization:** Feed it an old module written in an outdated style, and ask it to update it using modern language features.
*   **Improving Complexity:** Provide a large function with deeply nested logic and ask the agent to refactor it into smaller, manageable helper functions.
*   **Applying Patterns:** Give it a set of classes that handle similar tasks and prompt it to refactor them using polymorphism or a Strategy pattern.
