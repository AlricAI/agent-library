## Overview
The Technical Reference Builder acts as a specialized documentation specialist, creating comprehensive, searchable, and highly structured technical references. It is designed to transform raw feature lists, API endpoints, or configuration files into definitive guides that serve as the single source of truth for developers.

This agent excels at imposing rigorous structure on complex technical information, ensuring every parameter, edge case, and dependency is documented with precision.

## Capabilities
*   **Exhaustive Coverage**: Documents every method signature, parameter, and configuration option available.
*   **Structured Output**: Adheres to a strict markdown format (including Type, Default, Required, Examples) for consistency.
*   **Multi-Format Support**: Capable of generating API References, detailed Configuration Guides, and formal Schema Documentation.
*   **Contextual Linking**: Automatically suggests related features or concepts using 'See Also' sections.
*   **Completeness Check**: Ensures documentation covers error codes, rate limits, and deprecation warnings.

## Example Use Cases
1. **API Deep Dives**: Feed it a set of API endpoints and receive fully documented methods, including required parameters, return types, and example calls for various use cases.
2. **System Onboarding Guides**: Provide it with a collection of environment variables or settings, and it will output a structured Configuration Guide detailing default values, valid ranges, and migration paths.
3. **Data Model Documentation**: Input raw JSON schemas, and the agent will generate detailed documentation covering field types, validation rules, and relationships for developers building against that data structure.
