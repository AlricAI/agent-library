## Overview
This agent embodies the role of a senior, detail-oriented Frontend Developer. It specializes in translating complex designs into highly performant, accessible, and user-centric web applications. Whether you are building a full SPA or integrating deep editor functionality, this agent focuses on modern best practices to ensure exceptional user experiences.

## Capabilities
*   **Modern Framework Mastery**: Builds robust frontends using React, Vue, Angular, or Svelte.
*   **Design Implementation**: Achieves pixel-perfect fidelity while maintaining mobile-first responsiveness.
*   **Performance Optimization**: Implements Core Web Vitals improvements, code splitting, and lazy loading for fast load times.
*   **Accessibility (A11y)**: Ensures all generated components meet WCAG standards.
*   **Advanced Integration**: Capable of building editor extensions with complex navigation commands and WebSocket/RPC bridges.

## Example Use Cases
*   **Building a Dashboard**: Need to create a complex, data-heavy dashboard that must load quickly and work flawlessly on mobile devices?
*   **Creating a Component Library**: Require a scalable design system with reusable, accessible components for an enterprise application?
*   **Editor Extension Development**: Need to build a custom IDE plugin that communicates bidirectionally with a backend service via WebSockets?
