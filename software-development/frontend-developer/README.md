## Overview
This agent is an expert frontend developer specializing in the latest versions of React (19+) and Next.js (15+). It masters modern web architecture, including Server Components (RSC), Server Actions, and advanced state management patterns. It ensures that all generated code is performant, accessible, and scalable for production-grade applications.

## Capabilities
*   **React Mastery:** Implements features using React 19 capabilities like `useActionState` and Suspense for optimal user experience.
*   **Next.js Full-Stack:** Develops robust applications leveraging the App Router, Server Components, and efficient data fetching patterns (e.g., ISR).
*   **Performance Optimization:** Focuses on Core Web Vitals by implementing code splitting, memoization (`useMemo`, `React.memo`), and optimizing image delivery.
*   **State Management:** Utilizes modern, lightweight state solutions like Zustand or Jotai for client-side logic, while handling server state via React Query.
*   **Architecture:** Supports advanced patterns such as component-driven development (Atomic Design) and micro-frontends.

## Example Use Cases
*   **Building a Dashboard:** Generating a complex dashboard layout that requires fetching data from both client and server components, ensuring smooth transitions.
*   **Implementing Forms with Mutations:** Creating forms that use Next.js Server Actions for secure, seamless data submission without manual API calls.
*   **Optimizing Slow Components:** Taking an existing component and refactoring it to improve rendering performance using `useDeferredValue` or proper memoization techniques.
