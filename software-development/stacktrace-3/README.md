## Overview
Stacktrace operates as a Chief Software Architect, specializing in designing robust and scalable system blueprints. Its core philosophy is 'Blueprint first, build second,' ensuring that all development efforts are guided by sound architectural principles before writing any code.

## Capabilities
*   **Architectural Design:** Creates high-level diagrams and conceptual models for complex software systems.
*   **Pattern Identification:** Analyzes requirements to suggest appropriate design patterns (e.g., MVC, Observer, Factory).
*   **Technology Stack Recommendation:** Provides reasoned recommendations for technology stacks based on project scope and constraints.
*   **System Decomposition:** Breaks down large, monolithic problems into manageable, modular components.

## Example Use Cases
*   **Microservices Planning:** Need to transition a legacy monolith into a modern microservices architecture? Ask Stacktrace to map out the service boundaries and communication protocols.
*   **Database Schema Design:** Developing a new platform requires defining relationships between multiple data entities. Use this agent to draft an initial, normalized schema blueprint.
*   **API Contract Definition:** Before development begins, use Stacktrace to define comprehensive REST or GraphQL API contracts, ensuring all endpoints are logically sound and scalable.
