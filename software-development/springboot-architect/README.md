## Overview
This agent embodies the expertise of a senior Spring Boot engineer, focusing exclusively on developing modern, cloud-native Java applications using Spring Boot 3.x and beyond. It is designed to guide you through complex architectural decisions, ensuring your code base is scalable, highly reliable, and production-ready.

## Capabilities
*   **Architecture Design:** Analyzes project context to recommend optimal microservices patterns (e.g., API Gateway, Service Mesh).
*   **Modern Java Implementation:** Leverages Spring Boot 3+ features, including GraalVM native support, Virtual Threads, and reactive programming paradigms (WebFlux/R2DBC).
*   **Ecosystem Integration:** Implements complex patterns using the Spring Cloud suite (e.g., Service Discovery, Config Server, Circuit Breakers).
*   **Quality Assurance Focus:** Ensures best practices are followed, including high test coverage (>85%), robust security hardening (OAuth2/JWT), and thorough API documentation.

## Example Use Cases
*   **Building a Distributed System:** Need to design an e-commerce backend with separate services for Inventory, Payments, and Users? This agent will structure the service boundaries and suggest inter-service communication patterns.
*   **Migrating Legacy Code:** If you have a monolithic application that needs to become cloud-native, use this agent to guide the decomposition process into manageable microservices.
*   **Implementing Reactive Flows:** When building high-throughput APIs where blocking I/O is unacceptable, utilize its expertise in WebFlux and reactive data access (R2DBC) to ensure non-blocking performance.
