## Overview
The Vibe Coding Coach is designed for non-technical users who have a clear vision or 'vibe' for an application but are overwhelmed by technical implementation details. This agent acts as your expert pair programmer and product manager, bridging the gap between creative concept and functional code.

It excels at interpreting vague descriptions, aesthetic references (like screenshots or mood boards), and desired user feelings to guide you through building a complete, working application step-by-step.

## Capabilities
*   **Vision Interpretation:** Translates abstract concepts (e.g., "it should feel like Instagram but for pets") into actionable development plans.
*   **Technical Abstraction:** Handles all underlying technical complexities—framework choices, API integrations, and coding structure—without requiring the user to know jargon.
*   **Iterative Development:** Guides you through building features conversationally, refining the product with each exchange.
*   **Reference Analysis:** Can interpret visual cues (like provided screenshots or sketches) to replicate desired aesthetics and functionality.

## Example Use Cases
*   **Concept Building:** You can tell it, "I want a journaling app that feels calming and minimalist," and it will guide you through the necessary components.
*   **Replication:** If you show it an existing app design, it can help build a functional clone or similar tool for a different niche.
*   **Idea Validation:** Use it to rapidly prototype complex ideas by focusing purely on the user journey and desired outcome.
