## Overview
This agent is a specialized expert in the Laravel PHP framework. It adheres strictly to modern Laravel conventions, best practices, and PSR standards to produce highly maintainable, secure, and scalable codebases. Whether you are building a complex API or optimizing an existing feature, this agent ensures adherence to industry-leading patterns.

## Capabilities
*   **Eloquent Mastery:** Efficiently handles relationships, scopes, accessors, and query optimization using eager loading.
*   **Architecture Design:** Implements modular structures using Service Providers and manages the request/response lifecycle correctly.
*   **API Development:** Creates secure, versioned APIs with built-in rate limiting and proper authentication (e.g., Sanctum).
*   **Performance Optimization:** Suggests caching strategies, optimizes database queries, and advises on asset management (Laravel Mix).
*   **Testing & Quality:** Writes robust unit and integration tests using PHPUnit and enforces comprehensive validation.

## Example Use Cases
*   **Building a CRUD API:** Request the agent to scaffold an entire resource endpoint, including models, migrations, controllers, and necessary routes.
*   **Debugging Slow Queries:** Provide existing Eloquent code and ask for optimization suggestions, focusing on N+1 query resolution and indexing best practices.
*   **Implementing Background Tasks:** Need to process large datasets? Use this agent to set up job queues with appropriate dispatching and handling.
*   **Refactoring Authentication:** Ask it to implement a new feature requiring role-based access control (RBAC) using Laravel's built-in authorization features.
