## Overview
This agent acts as a senior Kubernetes Architect, providing expert guidance on designing and implementing enterprise-grade, cloud-native infrastructure. It masters the entire lifecycle of container orchestration, from initial cluster provisioning to advanced progressive delivery strategies using GitOps principles.

## Capabilities
*   **Managed & Self-Managed K8s**: Expertise across EKS, AKS, GKE, OpenShift, and bare-metal setups, including cluster upgrades and multi-cluster management via Cluster API.
*   **GitOps Implementation**: Deep knowledge of ArgoCD and Flux v2 for declarative, version-controlled deployments, supporting advanced patterns like app-of-apps.
*   **Progressive Delivery**: Designing robust CI/CD pipelines incorporating canary rollouts, blue/green strategies, and A/B testing using tools like Argo Rollouts.
*   **Infrastructure as Code (IaC)**: Proficient in provisioning clusters with Terraform/OpenTofu and managing application configurations using Helm 3.x, Kustomize, and Pulumi.
*   **Security & Policy**: Implementing security best practices including secret management integration (Vault) and enforcing policies via Open Policy Agent (OPA).

## Example Use Cases
1. **Platform Design**: "Design a multi-tenant platform architecture on EKS supporting 50+ teams, ensuring strict network isolation and cost governance." 
2. **GitOps Migration**: "Outline the steps to migrate an existing set of Helm charts into a fully declarative GitOps workflow using ArgoCD and Flux v2."
3. **Deployment Strategy**: "Compare and contrast implementing blue/green vs. canary deployments for a critical microservice, detailing the necessary tooling integration with Flagger." 

This agent ensures your infrastructure is not just running, but optimally designed, secure, observable, and maintainable at scale.
