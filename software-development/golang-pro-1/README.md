## Overview
Go Pro is a principal-level AI engineer specializing in building mission-critical applications using Go. It focuses relentlessly on idiomatic code, system architecture, and operational excellence to ensure the resulting software is robust, scalable, and maintainable.

## Capabilities
*   **System Architecture:** Designs scalable microservices and distributed systems with clear API boundaries (gRPC/REST).
*   **Advanced Concurrency:** Implements complex patterns using goroutines, channels, worker pools, and handles race condition detection.
*   **Performance Optimization:** Utilizes profiling tools knowledge (like pprof) to suggest memory allocation improvements and benchmark-driven tuning.
*   **Error Management:** Enforces best practices for error handling, including custom types and context-aware strategies.
*   **Testing Excellence:** Adheres to TDD principles by generating comprehensive unit tests, integration tests, and benchmarks alongside the code.

## Example Use Cases
1. **Architecting a Service:** Ask it to design a multi-stage data processing pipeline using Go microservices, detailing the necessary API contracts between services.
2. **Concurrency Review:** Provide existing concurrent code and ask it to review it for potential race conditions or deadlocks, suggesting idiomatic fixes.
3. **Performance Bottleneck Identification:** Submit a function suspected of being slow and request optimization suggestions based on memory profiling best practices.
