## Overview
This agent is specialized in scaffolding and building user interfaces using the powerful and customizable components provided by shadcn/ui. It ensures that the generated code adheres to modern best practices, prioritizing accessibility (WCAG standards) and responsiveness across various devices.

By integrating directly with the shadcn component system, this tool moves beyond simple boilerplate generation; it builds structured, production-ready UI modules that fit seamlessly into an existing design system.

## Capabilities
*   **Component Generation:** Creates full, functional components (e.g., Cards, Forms, Dialogs) using shadcn primitives.
*   **Accessibility Focus:** Automatically incorporates necessary ARIA attributes and semantic HTML to ensure high accessibility scores.
*   **Responsiveness Handling:** Structures layouts using utility-first CSS principles (like Tailwind CSS) to guarantee optimal viewing on mobile, tablet, and desktop.
*   **Integration Guidance:** Provides context-aware code snippets that fit directly into React/Next.js project structures.

## Example Use Cases
*   **Building a Dashboard Layout:** Requesting a complex dashboard layout featuring data cards, navigation sidebars, and interactive charts built entirely with shadcn components.
*   **Creating a Login Flow:** Generating the complete UI for a multi-step authentication process, including form validation feedback using shadcn's input components.
*   **Implementing a Settings Panel:** Developing a detailed user settings page that requires toggles, dropdowns, and structured form inputs, all styled consistently with the design system.
