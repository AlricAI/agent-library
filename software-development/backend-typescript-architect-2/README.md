## Overview
This agent acts as a Senior Backend TypeScript Architect, specializing in building high-quality, maintainable server-side applications using TypeScript and the modern Bun runtime. It adopts the rigorous standards of an experienced engineer, prioritizing clean code, robust patterns, and optimal performance.

## Capabilities
*   **Advanced TypeScript Implementation:** Generates idiomatic, type-safe code adhering to best practices for large-scale backends.
*   **API Design:** Designs RESTful or GraphQL endpoints with clear request/response schemas.
*   **Database Integration:** Provides patterns and boilerplate for connecting to and interacting with various databases (e.g., PostgreSQL, MongoDB).
*   **Performance Optimization:** Reviews existing code snippets and suggests architectural improvements for speed and efficiency within the Bun environment.
*   **Server Architecture:** Helps structure multi-module applications, including middleware setup and service layer separation.

## Example Use Cases
*   **Building a CRUD API:** Ask it to scaffold a complete set of endpoints for managing users, including database models and validation layers.
*   **Refactoring Legacy Code:** Provide an existing module and ask it to refactor it using modern TypeScript patterns for better testability.
*   **Implementing Authentication:** Request the boilerplate code for JWT-based authentication middleware integrated into an Express/Fastify-like structure running on Bun.
