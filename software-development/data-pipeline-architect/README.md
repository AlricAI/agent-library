## Overview
This agent acts as an expert Data Engineer specializing in designing, building, and optimizing modern, scalable data architectures. It masters the entire spectrum of data engineering, from initial ingestion to final consumption in analytics platforms.

Whether you are implementing a batch ETL process, setting up real-time streaming capabilities, or structuring a complex data mesh, this agent provides best practices across industry-leading tools.

## Capabilities
*   **Modern Data Stack Implementation:** Expertise integrating Fivetran/Airbyte with dbt and cloud warehouses (Snowflake, BigQuery).
*   **Orchestration & Workflow:** Proficient in using Apache Airflow for complex dependency management and scheduling.
*   **Processing Engines:** Skilled in Spark (PySpark/Scala), Polars, and Ray for high-performance data transformation.
*   **Data Warehousing & Lakes:** Implements lakehouse patterns using Delta Lake/Iceberg on cloud storage (S3, GCS).
*   **Streaming Architecture:** Handles real-time event processing using Kafka, Flink, and cloud message queues (Kinesis, Pub/Sub).
*   **Data Quality:** Incorporates data validation best practices using tools like Great Expectations.

## Example Use Cases
*   **Building a Customer 360 View:** Design an ingestion pipeline that pulls transactional data from operational databases, processes it through Spark for standardization, and loads the results into Snowflake via dbt.
*   **Real-Time Dashboarding:** Architect a system using Kafka to ingest clickstream events, process them with Flink, and make them available in a low-latency store like ClickHouse for immediate dashboard visualization.
*   **Data Warehouse Migration:** Plan the migration from an on-premise data warehouse to a cloud-native lakehouse architecture, detailing necessary components like Airflow DAGs and schema evolution strategies.
