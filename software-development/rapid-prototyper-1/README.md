## Overview
Rapid Prototyper is an AI agent engineered for speed and validation. Its core mission is to take abstract ideas or business hypotheses and transform them into functional, testable prototypes or Minimum Viable Products (MVPs) in the shortest time possible. It prioritizes getting a working solution in front of users quickly over building perfect, production-ready code.

## Capabilities
*   **Speed-Focused Development:** Builds functional proofs-of-concept using efficient, modern tools and frameworks.
*   **MVP Creation:** Focuses on implementing the core user flow and primary value proposition with minimal necessary features.
*   **Validation Integration:** Automatically incorporates mechanisms for user feedback collection and basic analytics from day one.
*   **Architecture Planning:** Designs prototypes with modularity in mind, ensuring a clear path for iteration or scaling to production.
*   **Technology Agnostic Speed:** Knows when to leverage no-code/low-code solutions versus writing necessary backend logic (e.g., using BaaS).

## Example Use Cases
1. **Idea Validation:** You have an idea for a niche productivity tool. Ask the agent to build a clickable prototype focusing only on the main workflow and integrate a simple feedback form.
2. **Feature Testing:** Your existing product needs to test if users prefer Feature A or Feature B. Have the agent build a basic A/B testing structure within a prototype environment.
3. **Concept Proof:** You need to prove that a specific data integration between two services is feasible. The agent will rapidly spin up a minimal backend service to demonstrate connectivity and core logic.
