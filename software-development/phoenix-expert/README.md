## Overview
This agent is a dedicated expert for the Phoenix web framework built on Elixir. It specializes in architecting, optimizing, and debugging complex, scalable web applications that require real-time capabilities. Whether you are building chat features with Channels or creating highly interactive UIs with LiveView, this agent adheres strictly to modern Phoenix best practices.

## Capabilities
*   **Real-Time Development:** Expertise in using Phoenix Channels and Presence for low-latency communication.
*   **Data Layer Management:** Proficient in designing robust data interactions using Ecto with PostgreSQL.
*   **UI/UX Implementation:** Skilled in leveraging LiveView for dynamic, stateful user interfaces.
*   **Architecture & Patterns:** Enforces adherence to Phoenix conventions, including proper use of Contexts and modular design.
*   **Performance Tuning:** Focuses on optimizing both HTTP request cycles and WebSocket performance using Elixir's concurrency model.
*   **Security & Testing:** Implements secure authentication via Plug and ensures high test coverage using ExUnit.

## Example Use Cases
*   **Building a Live Dashboard:** Need to create a real-time analytics dashboard that updates instantly when backend data changes? Use this agent to structure the LiveView components and associated Phoenix Channels.
*   **Optimizing API Endpoints:** If your existing Phoenix controllers are slow under load, ask for performance reviews focusing on Ecto query optimization and request handling efficiency.
*   **Implementing Chat Features:** For building a multi-user chat application, this agent will guide you through setting up the necessary Channels, managing user presence, and ensuring reliable message delivery.
