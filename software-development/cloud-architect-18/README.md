## Overview
This agent functions as a Master Cloud Architect, bridging the gap between high-level business vision and concrete, scalable technical execution. It specializes in designing systems that not only function flawlessly but also adhere to best practices for cost efficiency (FinOps) and global resilience.

The core philosophy is to design *for* the cloud's unique properties—elasticity, managed services, and global reach—while maintaining a ruthless focus on operational expenditure (OpEx).

## Capabilities
*   **Strategic Design:** Develop end-to-end architectures considering multi-cloud deployment patterns.
*   **Cost Optimization (FinOps):** Apply the FinOps Decision Matrix to ensure infrastructure choices are cost-effective at every stage.
*   **Pattern Selection:** Advise on Serverless vs. Container vs. VM trade-offs based on project needs.
*   **Governance & Security:** Enforce Zero Trust protocols and check designs against modern cloud anti-patterns.
*   **Scaling Strategy:** Adjust architectural focus (PaaS/Serverless first for MVPs) based on the required project scale.

## Example Use Cases
1. **New Product Launch:** "Design a global e-commerce platform MVP using the PaaS/Serverless first approach, ensuring initial cost estimates are provided." 
2. **Cost Overrun Analysis:** "Review this existing monolithic setup and propose a migration path to microservices that reduces monthly cloud spend by at least 20% while maintaining uptime."
3. **Compliance Check:** "We are expanding into the EU market; what governance changes must we implement across our current US-based stack to ensure GDPR compliance?"

By integrating deep architectural discovery with financial rigor, this agent ensures your cloud strategy is both innovative and responsible.
