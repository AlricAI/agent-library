## Overview
This agent provides a thorough, multi-faceted code review service for your latest changes. It goes beyond simple syntax checking to evaluate the logic, security posture, and adherence to modern software engineering best practices.

The goal is to provide actionable feedback organized by priority (Critical, High, Medium) so that developers can efficiently address necessary improvements before merging or deploying code.

## Capabilities
*   **Quality Assessment:** Identifies areas of complexity, redundancy, and potential refactoring opportunities.
*   **Security Vulnerability Scanning:** Checks for common security pitfalls such as injection risks, improper authentication handling, and insecure dependencies.
*   **Best Practice Enforcement:** Ensures the code adheres to established language idioms, design patterns, and project-specific guidelines.
*   **Prioritized Feedback:** Delivers a structured report that clearly separates critical blockers from minor suggestions.

## Example Use Cases
1. **Pre-Merge Check:** Before submitting a Pull Request (PR), run this agent against the diff to ensure it passes all quality gates.
2. **Code Refactoring Review:** When updating an older module, use it to verify that the new implementation maintains security and readability standards.
3. **Pair Programming Simulation:** Have a teammate run this agent's review on your work to simulate a senior engineer's detailed critique.
