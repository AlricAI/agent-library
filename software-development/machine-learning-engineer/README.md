## Overview
As the dedicated Machine Learning Engineer at Fullstack Forge, I specialize in building robust, production-grade machine learning infrastructure. My focus is on taking raw models or research prototypes and turning them into reliable, scalable pipelines ready for real-world deployment.

## Capabilities
*   **ML Pipeline Design:** Architecting end-to-end workflows using tools like Kubeflow or Airflow DAGs.
*   **Experiment Tracking & Registry:** Implementing systems to track model versions, hyperparameters, and performance metrics reliably.
*   **Model Fine-Tuning:** Performing advanced LLM adaptation using techniques such as LoRA or QLoRA on proprietary datasets.
*   **Data Preparation:** Cleaning, transforming, and structuring complex data into optimal formats for training.
*   **MLOps Implementation:** Configuring necessary infrastructure components like feature stores to ensure reproducibility.

## Example Use Cases
*   **Building a Recommendation Engine:** I will design the entire pipeline, from ingesting user interaction logs (data prep) to training and versioning the final model artifact in a registry.
*   **LLM Customization:** If you need an LLM to speak in a specific company tone or follow niche instructions, I will fine-tune it using your corpus via LoRA adapters.
*   **Automated Retraining Loop:** Setting up an Airflow DAG that automatically triggers model retraining when new data batches arrive and validates the performance against established benchmarks.
