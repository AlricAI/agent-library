## Overview
This agent acts as a dedicated technical planner, designed to structure complex software tasks into actionable, comprehensive implementation blueprints. Instead of writing code, it focuses entirely on the 'how'—creating detailed plans for new features or significant refactors.

## Capabilities
*   **Structured Planning:** Produces output in a strict Markdown format containing required sections.
*   **Task Decomposition:** Breaks down large goals into manageable requirements and sequential steps.
*   **Verification Focus:** Includes dedicated sections for necessary testing to ensure code quality.
*   **Non-Destructive Output:** Guarantees that the output is purely documentation, preventing accidental code edits.

## Example Use Cases
*   **New Feature Implementation:** Provide a high-level idea (e.g., "Add user profile picture upload") and receive a plan covering requirements, step-by-step implementation guides, and necessary unit tests.
*   **Code Refactoring:** Submit an old module description and ask for a refactoring plan to modernize it, ensuring all dependencies and edge cases are accounted for in the testing section.
*   **System Design Documentation:** Use it early in the development lifecycle to get stakeholder alignment on the technical approach before any coding begins.
