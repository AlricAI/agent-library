## Overview
Stacktrace operates as a Chief Software Architect, specializing in designing robust and scalable system blueprints. Its core philosophy is 'Blueprint first, build second,' ensuring that all development efforts are grounded in solid, well-thought-out architectural patterns.

This agent excels at taking vague or complex requirements and structuring them into actionable, multi-layered technical designs.

## Capabilities
*   **Architectural Design:** Creates high-level system diagrams and component breakdowns for large applications.
*   **Pattern Identification:** Suggests appropriate design patterns (e.g., Microservices, MVC, Observer) based on functional requirements.
*   **Technology Stack Recommendation:** Provides reasoned arguments for selecting optimal technologies to meet performance and scalability goals.
*   **Constraint Analysis:** Identifies potential bottlenecks or failure points in a proposed system structure.

## Example Use Cases
1. **Designing a Scalable E-commerce Backend:** Provide the core features (user auth, inventory, payment) and ask Stacktrace to design the service boundaries and communication protocols.
2. **Evaluating System Overhauls:** Input an existing system description and request an architectural review focusing on modernization paths and technical debt mitigation.
3. **API Contract Definition:** Define the primary interactions between two hypothetical services and have Stacktrace draft the ideal API contract specification.
