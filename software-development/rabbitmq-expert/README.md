## Overview
This agent serves as a comprehensive expert for all aspects of RabbitMQ implementation. It guides users through setting up resilient, high-throughput messaging architectures, ensuring data integrity and fault tolerance across distributed systems.

## Capabilities
*   **Architecture Design:** Designs scalable patterns like Publish/Subscribe and Request/Reply using advanced clustering techniques.
*   **Configuration Mastery:** Provides expert configuration for exchanges, queues, bindings, message TTLs, and dead-letter exchanges (DLX).
*   **Reliability & HA:** Implements best practices for high availability, including mirroring and robust failure handling.
*   **Optimization & Tuning:** Offers strategies to optimize throughput, manage resource utilization, and troubleshoot performance bottlenecks.
*   **Security Implementation:** Guides the setup of secure connections using TLS and proper authentication mechanisms.

## Example Use Cases
*   **Building an Event Bus:** Need to set up a central, durable event bus for microservices communication? Ask this agent to design the full topology, including necessary exchanges and bindings.
*   **Implementing Background Jobs:** If you have asynchronous tasks that require guaranteed delivery (e.g., sending emails), use it to configure reliable queues with acknowledgments and retry logic.
*   **Troubleshooting Latency:** When message flow is slow or messages are being lost intermittently, this agent can help diagnose the root cause by reviewing your current setup against best practices.
