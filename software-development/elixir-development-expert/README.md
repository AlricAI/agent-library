## Overview
This agent embodies the knowledge of a senior Elixir developer, specializing in building highly reliable, concurrent, and fault-tolerant systems. It deeply understands the OTP ecosystem, making it ideal for architecting complex, scalable applications using Phoenix and LiveView.

## Capabilities
*   **OTP Mastery:** Designs robust supervision trees, implements GenServer state management, and utilizes Agents/Tasks correctly.
*   **Concurrency Patterns:** Implements message passing, process linking, and advanced flow control (e.g., GenStage).
*   **Idiomatic Elixir:** Ensures all generated code follows the latest Elixir style guide, emphasizing immutability and pattern matching.
*   **Testing & Quality:** Focuses on generating comprehensive ExUnit tests, including doctests, and ensuring proper documentation via ExDoc.
*   **System Architecture:** Reviews `mix.exs` and process architecture to ensure scalability and fault tolerance.

## Example Use Cases
*   **Building Real-Time Features:** Developing complex state management for a dashboard using LiveView backed by an OTP GenServer.
*   **Designing Microservices:** Architecting inter-process communication between different application components using message passing and supervisors.
*   **Refactoring Legacy Code:** Reviewing existing modules to apply modern Elixir idioms, improve concurrency handling, and implement proper error boundaries (e.g., using `with` statements).

When you need an application component that must remain available even when parts of it fail, this agent is your go-to resource.
