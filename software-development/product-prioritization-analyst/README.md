## Overview
The Product Prioritization Analyst acts as a specialized consultant for product execution teams. Its core function is to guide teams through the complex process of deciding what features or initiatives should be built next by applying established, rigorous prioritization methodologies.

When faced with an overwhelming backlog or conflicting feature requests, this agent ensures that decisions are data-driven and systematic rather than based on gut feeling alone.

## Capabilities
*   **Framework Selection:** Determines the most appropriate prioritization model (e.g., RICE, ICE, Kano, MoSCoW) based on the input context and business goals.
*   **Scoring & Analysis:** Applies the chosen framework to provided features, generating clear scores, weights, and comparative rankings.
*   **Structured Output:** Produces comprehensive analyses that clearly justify the recommended order of development.
*   **Workflow Handoff:** Designed to hand off finalized priorities smoothly to subsequent roles like Roadmap Specialists or Sprint Managers for execution.

## Example Use Cases
*   **Backlog Comparison:** A team provides 20 potential features, and you need to determine which 5 offer the highest immediate ROI. The agent can run an RICE analysis.
*   **Methodology Selection:** When a client states they are unsure whether to use MoSCoW or Opportunity Score, the agent analyzes their stated constraints (e.g., regulatory needs vs. market potential) and recommends the best fit.
*   **Feature Triage:** After initial discovery, you receive a list of ideas; the agent structures them into a ranked backlog ready for sprint planning.
