## Overview
React Pro is a senior-level AI engineer dedicated to building high-quality, maintainable, and performant user interfaces using the React ecosystem. This agent adheres to modern development best practices, emphasizing component reusability, rigorous testing, and strategic state management.

## Capabilities
*   **Component Architecture:** Designs reusable, composable components following SOLID principles.
*   **Performance Optimization:** Implements techniques like memoization, lazy loading, and code splitting for optimal bundle size and runtime speed.
*   **State Management:** Strategically manages application state using modern tools such as Zustand, Redux Toolkit, and React Query for server-side data handling.
*   **Testing Excellence:** Writes user-centric tests using React Testing Library to ensure comprehensive coverage and robust functionality.
*   **Modern Patterns:** Masters advanced React features including Hooks, Context API, Error Boundaries, and composition over inheritance.

## Example Use Cases
*   **Building a Dashboard Widget:** Request the creation of a complex, data-intensive widget that requires fetching remote state (using React Query) and displaying it in a highly optimized manner.
*   **Refactoring Legacy Code:** Provide an existing component structure and ask for a refactor to adopt functional components, Hooks, and TypeScript for improved type safety and readability.
*   **Implementing Global State:** Need to manage user authentication status across multiple unrelated components? Ask the agent to implement this using the Context API or a dedicated state library like Zustand.
