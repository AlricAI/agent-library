## Overview
This agent acts as an expert Product Manager, specializing in structuring complex product initiatives using industry-standard agile methodologies. It transforms vague ideas or high-level feature requests into fully fleshed-out Project Epics and granular User Stories.

The goal is to provide development teams with unambiguous requirements that include business justification, success metrics, and clear acceptance criteria from the outset.

## Capabilities
*   **Epic Creation:** Generates comprehensive epics covering strategic context, target personas, key performance indicators (KPIs), and overall scope.
*   **User Story Detailing:** Structures individual user stories using the standard 'As a [persona], I want to [action], so that [benefit]' format.
*   **Structured Acceptance Criteria:** Includes detailed acceptance criteria, technical considerations, definition of done (DoD), and dependency mapping for each story.
*   **Methodology Adherence:** Ensures all outputs align with best practices in agile product development.

## Example Use Cases
1. **Feature Definition:** Provide a high-level feature idea (e.g., "We need to improve user onboarding") and receive a structured Epic that breaks it down into multiple, testable User Stories.
2. **Scope Planning:** When starting a new project phase, use this agent to establish the foundational Epics needed before development sprints begin.
3. **Requirement Documentation:** Use it as a living document generator to ensure all technical specifications are traceable back to defined business value and user needs.
