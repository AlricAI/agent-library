## Overview
This agent acts as a specialized prompt engineer dedicated to creating system prompts for AI agents built with Pydantic. Its core philosophy is that "Clarity beats complexity," ensuring the generated prompts are concise, direct, and highly focused on the agent's essential function.

It streamlines the often complex process of defining agent behavior by adhering to strict principles of brevity and simplicity, preventing over-instruction while maximizing model performance.

## Capabilities
*   **System Prompt Generation**: Creates comprehensive system prompts tailored for Pydantic agents.
*   **Simplicity Enforcement**: Adheres to rules like keeping prompts under 300 words and avoiding redundancy.
*   **Structured Design**: Generates clear Role/Identity sections, defined Core Competencies, and essential Behavioral Guidelines.
*   **Focus Optimization**: Ensures the prompt only contains information critical for the agent's core task.

## Example Use Cases
1. **Agent Onboarding**: After defining a new agent's requirements (e.g., 'a financial data summarizer'), feed those requirements to this agent to receive a ready-to-use, optimized system prompt.
2. **Prompt Refinement**: If an existing agent is performing poorly due to vague instructions, use this agent to simplify and tighten the current system prompt for better adherence.
3. **Role Definition**: Use it to establish a precise persona (e.g., 'Socratic Tutor' or 'Technical Documentation Reviewer') by providing the high-level goals.
