## Overview
This agent suite automates the entire process of building a high-converting landing page, simulating a small, specialized development team. It coordinates four distinct AI roles—Content Creator, UI Designer, Frontend Developer, and Growth Hacker—to move from initial product concept to fully coded, optimized webpage in one cohesive workflow.

## Capabilities
*   **Multi-Stage Orchestration:** Manages sequential and parallel tasks across multiple specialized agents.
*   **Copy Generation:** Creates targeted, scannable marketing copy based on defined personas and tones.
*   **Design Specification:** Outputs detailed UI/UX specs including wireframes, color palettes, typography, and component requirements (e.g., Linear aesthetic).
*   **Code Implementation:** Generates production-ready frontend code using modern stacks like HTML and Tailwind CSS.
*   **Conversion Optimization:** Integrates growth hacking principles to ensure the final output maximizes visitor action rates.

## Example Use Cases
1. **Product Launch:** Given a new SaaS API, run this workflow to generate all necessary marketing copy, design specs, and functional code for a launch day landing page.
2. **Website Redesign:** Feed in existing brand guidelines and target goals to rapidly prototype and build out key conversion sections of an outdated website.
3. **A/B Testing Assets:** Quickly generate multiple versions (e.g., one focused on 'Speed' copy, another on 'Integration Depth') by re-running the content creation step with different prompts.
