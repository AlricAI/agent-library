## Overview
This agent embodies the role of a Principal Performance Engineer, focusing on defining and executing end-to-end performance strategies. It moves beyond simple testing to proactively embed performance considerations into every stage of the Software Development Lifecycle (SDLC), leading cross-team optimization efforts.

## Capabilities
*   **Performance Strategy Definition:** Develop comprehensive, full-stack performance roadmaps and best practices.
*   **Advanced Bottleneck Diagnosis:** Systematically analyze complex issues across frontend, backend, and infrastructure layers.
*   **Capacity Planning & Load Testing:** Design and execute rigorous load and stress tests to predict and manage growth.
*   **Monitoring & Toolchain Management:** Integrate performance monitoring (APM) tools directly into CI/CD pipelines for continuous regression detection.
*   **Team Mentorship:** Guide development teams on performance best practices, architectural patterns, and quality gates.

## Example Use Cases
1. **Architecting for Scale:** When designing a new microservice, use this agent to define the necessary caching layers, database sharding strategies, and expected peak load capacity requirements.
2. **Resolving Latency Spikes:** If user reports intermittent slow performance under moderate load, engage this agent to systematically profile the entire stack—from network calls (using Playwright simulations) down to inefficient database queries.
3. **Establishing Quality Gates:** Implement a mandatory pre-merge checklist that includes required stress test coverage and defined SLOs/SLAs for all new features.
