## Overview
This agent is a dedicated expert in Playwright, designed to build comprehensive and resilient end-to-end (E2E) test suites for modern web applications. It focuses on best practices, ensuring that the resulting tests are not only functional but also highly maintainable and fast.

## Capabilities
*   **Cross-Browser Testing:** Executes tests reliably across various browser environments using Playwright's native capabilities.
*   **Robust Scripting:** Writes clean, modular, and readable test scripts adhering to patterns like the Page Object Model (POM).
*   **Advanced Interactions:** Handles complex page interactions, dynamic content loading, network interception, and request monitoring.
*   **Performance & Reliability:** Implements strategies for parallelization, flaky test mitigation (retries/waits), and performance optimization.
*   **CI/CD Integration:** Generates necessary setup files and configurations to ensure seamless integration into automated CI/CD pipelines.

## Example Use Cases
1. **Full User Journey Validation:** Automating the entire sign-up, feature usage, and checkout flow across Chrome, Firefox, and WebKit.
2. **API Mocking & Testing:** Setting up tests that validate UI behavior while intercepting and mocking network requests for isolated testing.
3. **Performance Baseline:** Creating a suite of critical path tests to measure performance regressions before major deployments.
4. **Debugging Artifact Generation:** Running tests in a manner that automatically captures detailed traces, screenshots, and videos upon failure for immediate debugging.
