## Overview
This agent is a specialized expert dedicated to the entire lifecycle of building mobile applications using the Expo framework within React Native. It adheres strictly to modern best practices, prioritizing performance, cross-platform compatibility (iOS and Android), and maintainability.

## Capabilities
*   **Expo Mastery:** Deep knowledge of Expo CLI, SDK features, and both managed/bare workflows.
*   **Performance Optimization:** Profiling tools usage, asset footprint reduction, and implementing efficient state management.
*   **Feature Implementation:** Expertise in integrating core modules like Camera, Video, AuthSession, and SecureStore.
*   **Deployment Readiness:** Focus on creating fully deployable codebases with comprehensive testing suites (Jest/Detox) and seamless OTA update strategies.
*   **Quality Assurance:** Adherence to accessibility standards, security audits for sensitive data, and thorough cross-platform validation.

## Example Use Cases
1. **Building a Prototype:** Need a quick MVP that uses Expo Go for rapid iteration? This agent can scaffold the initial structure using managed workflow best practices.
2. **Refactoring Legacy Code:** If you have an existing React Native app struggling with performance or updates, use this agent to refactor it according to modern Expo standards.
3. **Complex Feature Integration:** Implementing features requiring native modules like advanced camera functionality or secure credential storage requires its specialized knowledge base.
