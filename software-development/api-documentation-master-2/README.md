## Overview
This agent functions as an expert API Documentation Specialist, dedicated to creating world-class developer experiences. It masters modern documentation standards like OpenAPI 3.1+ and integrates AI-powered workflows to ensure your API documentation is comprehensive, interactive, and highly accessible.

Its primary goal is not just to document endpoints, but to actively drive API adoption by minimizing the time developers take to integrate with your service.

## Capabilities
*   **Modern Specification Mastery:** Authoring using OpenAPI 3.1+ and supporting AsyncAPI for event-driven architectures.
*   **AI Content Generation:** Utilizes AI assistance for generating developer-friendly explanations, updating content from code comments, and creating multi-language code examples.
*   **Interactive Portals:** Builds documentation interfaces compatible with Swagger UI, Redoc, and custom portals (e.g., Docusaurus) featuring 'Try-it-now' functionality.
*   **Schema Support:** Handles complex schema types including JSON Schema validation and GraphQL SDL best practices.
*   **Lifecycle Management:** Documents the entire API lifecycle, from initial design through to deprecation notices.

## Example Use Cases
1. **New Product Launch:** Provide a set of endpoints and let the agent generate a complete developer portal, including interactive testing examples for every resource.
2. **Updating Specs:** Feed it an existing OpenAPI YAML file and ask it to update all example payloads to reflect recent changes in your backend service.
3. **Onboarding Improvement:** Use it to take raw technical notes and transform them into a structured, tutorial-style guide suitable for new developers.
