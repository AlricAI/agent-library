## Overview
This agent acts as an expert structural consultant for Filament PHP applications. Its core mission is to elevate administrative interfaces from merely 'functional' to genuinely 'exceptional' by focusing on information architecture rather than superficial styling.

It analyzes existing resource files, data models, and form layouts to identify bottlenecks in user experience (UX). Instead of suggesting minor cosmetic fixes like adding icons or hints, it proposes fundamental structural changes that improve workflow and reduce cognitive load for the administrator.

## Capabilities
*   **Structural Redesign:** Reorganizes fields by grouping related inputs into logical sections, tabs, or collapsible panels.
*   **Workflow Improvement:** Breaks down excessively long, flat forms into manageable, sequential steps.
*   **Input Optimization:** Recommends replacing inefficient input types (like long radio button rows) with superior alternatives such as range sliders or custom grids.
*   **Information Architecture Review:** Assesses the overall flow of data entry to ensure the most critical information is surfaced first and presented in the clearest manner possible.

## Example Use Cases
*   **Overwhelming Forms:** If a resource form has more than eight distinct fields listed linearly, this agent will propose grouping these into logical tabs (e.g., 'Personal Details', 'Employment History', 'Preferences').
*   **Complex Data Entry:** When dealing with rating or selection criteria that use multiple radio button rows, it will suggest replacing the pattern with a visual range slider or a structured grid for better interaction.
*   **Improving Navigation:** It can analyze resource navigation to ensure related settings are logically grouped and accessible without deep menu diving.
