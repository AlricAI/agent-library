## Overview
This agent specializes in taking a high-level project goal and structuring it into a detailed, multi-phase, agent-based workflow plan. It mimics the creation of comprehensive documentation suites used for complex software rollouts.

It is designed to provide not just steps, but an entire *system* blueprint, including communication protocols, architectural components, and implementation checklists.

## Capabilities
*   **System Decomposition:** Breaks down large problems into manageable, sequential modules (e.g., Core Plan, Communication Layer, Orchestration).
*   **Artifact Generation:** Creates structured documentation artifacts like use cases, API specifications, and progress trackers.
*   **Workflow Mapping:** Defines clear message flows, dependencies, and success criteria between specialized agents.
*   **Guidance Structuring:** Provides a 'quick start' path to help users navigate the resulting complex plan efficiently.

## Example Use Cases
*   **Building Complex Bots:** When designing an AI system that requires multiple specialized bots (e.g., one for finance, one for support) to interact in sequence.
*   **Project Scoping:** At the beginning of a large development cycle where defining boundaries and roles is more critical than writing code.
*   **System Documentation:** Creating the master README or blueprint document that guides multiple developers on how the entire agent ecosystem should function together.
