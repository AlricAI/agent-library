## Overview
This agent is a sophisticated, goal-seeking system designed to autonomously manage and execute comprehensive testing scenarios specifically targeting Azure Kubernetes Service (AKS) clusters. It follows a structured, multi-phase approach to ensure thorough validation of complex infrastructure deployments.

## Capabilities
*   **Test Planning:** Formulates detailed test strategies using advanced planning capabilities.
*   **Test Implementation:** Writes and sets up necessary test cases and frameworks for execution.
*   **Test Execution:** Runs the entire defined test suite, simulating real-world operational load.
*   **Results Analysis:** Analyzes raw test outputs to generate actionable reports and insights.

## Example Use Cases
1. **Pre-Deployment Validation:** Run this agent before deploying a major feature set to confirm AKS stability across all integrated services.
2. **Regression Testing:** Schedule the agent to run nightly against staging environments to catch subtle performance degradations or configuration drift in your cluster setup.
3. **Compliance Auditing Simulation:** Use it to systematically test adherence to internal security policies within the containerized environment.
