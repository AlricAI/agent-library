## Overview
Pixelpit is an autonomous, self-bootstrapping AI game studio designed to operate continuously, generating a high volume of creative output. It simulates the chaos and energy of a small indie development team, allowing multiple specialized agents (Game Makers) to work on games end-to-end from concept through deployment.

While powered primarily by cost-effective models like Haiku, it utilizes more powerful models for coordination and interface management, mimicking a fully operational studio environment.

## Capabilities
*   **End-to-End Game Development:** Each designated 'Game Maker' owns a game from initial concept through coding, social integration, and final build preparation.
*   **Self-Organization:** The studio can spawn emergent roles (e.g., VP Engineering, Design Lead) as the complexity or needs of the project dictate, simulating organizational growth.
*   **Multi-Agent Coordination:** Features a core structure involving a Partner interface, Creative Head for style review, and dedicated testing agents to ensure build quality across different platforms.
*   **Rapid Iteration:** Designed for 'volume over polish,' making it ideal for quickly generating multiple playable prototypes or game concepts simultaneously.

## Example Use Cases
*   **Concept Prototyping Marathon:** Need 5 distinct, playable mini-games ideas tested in a week? Pixelpit can spin up the necessary roles and start development immediately.
*   **Game Feature Stress Testing:** Test how quickly an entire feature set (e.g., leaderboards, social sharing) can be integrated across multiple disparate game concepts.
*   **Creative Ideation Pipeline:** Use it as a brainstorming partner that doesn't sleep, generating art direction suggestions and core mechanics for diverse genres.
