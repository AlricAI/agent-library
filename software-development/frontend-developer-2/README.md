## Overview
This agent embodies the role of a senior frontend engineer and AI pair programmer, specializing in building scalable, maintainable, and production-ready React applications. It enforces modern development best practices, focusing heavily on clean architecture, performance optimization, and WCAG accessibility standards.

## Capabilities
*   **Component Development:** Creates high-quality React components using TypeScript and modern functional patterns (Hooks).
*   **UI/UX Implementation:** Ensures designs are responsive, mobile-first, and adhere to strict accessibility guidelines.
*   **Performance Optimization:** Implements techniques like code splitting, lazy loading, and memoization for optimal bundle size and runtime speed.
*   **State Management:** Selects and implements appropriate state solutions (Context API, Zustand, Redux) based on the application's complexity.
*   **Testing Strategy:** Integrates unit, integration, and end-to-end testing practices to ensure comprehensive code coverage.

## Example Use Cases
1. **Building a New Feature:** Requesting a complex dashboard widget that requires state management, API integration, and adherence to the existing design system.
2. **Refactoring Legacy Code:** Providing an older component and asking the agent to refactor it using modern React Hooks and TypeScript for improved readability and type safety.
3. **Accessibility Audit:** Asking the agent to review a set of components and specifically identify and fix any WCAG 2.1 AA violations.
