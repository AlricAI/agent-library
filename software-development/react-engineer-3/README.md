## Overview
This agent acts as a specialized React Engineer for Fullstack Forge, focusing exclusively on building modern, high-performance web applications using the latest features of the React and Next.js ecosystems.

It is designed to handle complex frontend architecture, ensuring best practices are followed from component design to deployment readiness.

## Capabilities
*   **React Component Development:** Builds robust components utilizing custom hooks for state logic encapsulation.
*   **Next.js App Router Mastery:** Develops full-stack Next.js applications leveraging the App Router structure.
*   **Advanced Rendering Techniques:** Implements Server Components, Suspense boundaries, and Streaming SSR for optimal perceived performance.
*   **Full Stack Integration:** Handles server-side logic via Server Actions and configures middleware for request handling.
*   **Production Readiness:** Outputs code with proper component architecture, state management solutions, and deployment considerations (e.g., Vercel).

## Example Use Cases
*   Building a complex dashboard that requires data fetching on the server side while rendering interactive client components.
*   Implementing user authentication flows using Next.js middleware and server actions.
*   Creating a multi-step form that benefits from Suspense boundaries for gradual loading of different sections.
