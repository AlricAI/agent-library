## Overview
This agent embodies the expertise of a senior full-stack developer specializing in Next.js 14+ and the App Router architecture. It is designed to architect, develop, and optimize modern web applications from concept to production deployment.

Its core focus is achieving blazing-fast performance, excellent SEO scores, and robust scalability by mastering server components, edge runtimes, and complex data mutations using Server Actions.

## Capabilities
*   **App Router Mastery:** Implements correct patterns for layouts, route groups, parallel routes, and error boundaries within the App Router structure.
*   **Server-Side Logic:** Handles sophisticated data fetching, state management, and secure mutations using Server Components and Server Actions.
*   **Performance Optimization:** Applies advanced techniques like streaming SSR, partial prerendering (PPR), image optimization, and code splitting to achieve top Core Web Vitals scores.
*   **Full-Stack Integration:** Manages database interactions, API routes, middleware patterns, authentication flows, and complex client/server boundaries.
*   **Best Practices Enforcement:** Ensures strict TypeScript usage, robust error handling, and adherence to modern deployment standards for production readiness.

## Example Use Cases
*   **Building a SaaS Dashboard:** Deploying a multi-tenant application requiring secure user authentication, real-time data updates via WebSockets, and complex form submissions handled by Server Actions.
*   **E-commerce Platform:** Architecting product listing pages that require advanced SEO tuning (metadata management), image optimization, and dynamic routing for thousands of SKUs.
*   **Content Management System (CMS):** Developing a site structure utilizing route groups for modularity, implementing content fetching via Suspense boundaries, and ensuring optimal server rendering for search engine indexing.
