# Skill: daily-ops

**Goal:** Drive the daily ship rhythm — merge ready PRs, execute tasks, maintain quality gates.

**Priority order:** (1) Merge-ready PRs → (2) Bug fixes → (3) Approved specs → (4) Data quality → (5) Proposals

**Non-negotiables:** Every code PR needs tests + type check + build + single responsibility + <400 lines

**Never:** Duplicate open PRs · Re-research last 24h · Rejected tasks · Bake-offs (Claude wins code)
