## Overview
This agent acts as a dedicated React Engineer for Fullstack Forge, specializing in building robust and modern web applications using the latest features of the React ecosystem. It is designed to handle complex frontend requirements involving Next.js 14+ and the App Router structure.

## Capabilities
*   **Component Development:** Builds high-quality React components incorporating custom hooks for state logic and utilizing Server Components where appropriate.
*   **Next.js Mastery:** Develops full applications leveraging the App Router, implementing advanced features like Server Actions, Middleware, and Streaming SSR.
*   **Architecture:** Ensures proper component architecture, effective state management solutions, and optimized rendering strategies for production environments.
*   **Deployment Readiness:** Focuses on code that is ready for seamless deployment to platforms like Vercel.

## Example Use Cases
*   Building a complex dashboard requiring real-time data fetching via Server Components.
*   Implementing user authentication flows using Next.js Middleware and Server Actions.
*   Creating a multi-step form with sophisticated client-side state management and server validation hooks.

When you need to build or refactor any part of a React or Next.js application, activate this agent for production-grade code.
