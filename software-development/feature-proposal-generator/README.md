## Overview
This agent acts as a structured Product Owner assistant, guiding you through the process of creating actionable feature proposals. Instead of vague ideas, it forces you to think critically about user problems, competitive gaps, and technical feasibility.

It follows a rigorous Standard Operating Procedure (SOP) designed to ensure that any proposed feature is well-scoped, data-backed, and ready for engineering review.

## Capabilities
* **Structured Proposal Generation:** Outputs proposals using a comprehensive markdown template covering Problem, Solution, Criteria, and Risk Assessment.
* **Gap Analysis:** Prompts the user to compare against competitors (e.g., AllTrails, Gaia GPS) to identify unique selling propositions.
* **Feasibility Check:** Guides the proposal toward actionable scopes (Small/Medium/Large effort estimates).
* **Pre-Proposal Research Checklist:** Reminds the user to check existing documentation, open PRs, and underlying data before submitting an idea.

## Example Use Cases
When you notice recurring negative feedback in support tickets regarding filtering options, use this agent. You can prompt it with:

"Generate a feature proposal for improving trail search usability based on recent user complaints about difficulty levels." 

The resulting output will provide a concrete plan, including specific UI elements to change and the necessary technical considerations, making your idea immediately actionable for development.
