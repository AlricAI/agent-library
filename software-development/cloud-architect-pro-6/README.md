## Overview
Cloud Architect Pro is a senior-level AI consultant specializing in designing, reviewing, and optimizing complex cloud infrastructure across major providers (AWS, Azure, GCP). It ensures that proposed architectures adhere to the principles of the Well-Architected Framework while prioritizing business value, security, and cost efficiency.

## Capabilities
*   **Multi-Cloud Strategy:** Develops comprehensive strategies for workload distribution, data sovereignty compliance, and mitigating vendor lock-in across disparate cloud environments.
*   **Cost Optimization (FinOps):** Analyzes resource utilization to recommend specific savings opportunities, including right-sizing, reserved instance planning, and implementing storage lifecycle policies to achieve significant cost reductions (>30%).
*   **Security & Compliance:** Enforces Zero-Trust principles, designs robust network segmentation, and validates architectures against necessary compliance standards.
*   **Resilience Engineering:** Designs for high availability (99.99%) using multi-region failover automation, defining clear Recovery Time Objectives (RTO) and Recovery Point Objectives (RPO).
*   **Best Practice Adherence:** Guides implementation following Infrastructure as Code (IaC) principles and operational excellence patterns.

## Example Use Cases
1. **New System Design:** Provide a set of business requirements, and the agent will output a preliminary multi-cloud architecture diagram concept, detailing necessary services in AWS, Azure, and GCP.
2. **Cost Audit:** Input an existing cloud spending profile, and the agent will generate actionable recommendations for cost optimization, specifying which resources to right-size or shift to reserved instances.
3. **Disaster Recovery Planning:** Define a critical application workload and its RTO/RPO, allowing the agent to design a fully automated failover mechanism across two distinct geographic regions.
