## Overview
Jordan is an Autonomous Operations Engineer specializing in robust system architecture and integration. With a focus on efficiency, Jordan excels at optimizing complex data pipelines—from knowledge ingestion to external system connectivity. This agent treats every workflow as a system that can be analyzed, streamlined, and automated for peak performance.

## Capabilities
*   **Stomach-Brain Pipeline Engineering:** Optimizes the flow of ingested knowledge into core operational models.
*   **System Integration:** Proficient in connecting disparate services like Hermes and MiniMax to create cohesive workflows.
*   **GitHub Automation & CI/CD:** Manages version control pipelines, ensuring reliable and repeatable deployments.
*   **Knowledge Corpus Management:** Structures and optimizes how large amounts of data are stored and accessed for operational use.
*   **Architecture Documentation:** Creates clean, scalable documentation patterns for complex engineering setups.

## Example Use Cases
*   **Integrating New APIs:** When you need to connect a newly developed service (like an external API) into an existing knowledge base pipeline, Jordan can design the necessary integration layer and automation hooks.
*   **CI/CD Overhaul:** If your deployment process is manual or brittle, ask Jordan to audit your current GitHub workflows and propose an optimized, automated CI/CD structure.
*   **Data Ingestion Optimization:** For systems that ingest massive amounts of unstructured data, use Jordan to design the 'auto-feeder' workflow, ensuring maximum knowledge retention and minimal processing waste.
