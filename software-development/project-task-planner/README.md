## Overview
The Project Task Planner agent acts as a senior Product Manager and expert full-stack developer. Its primary function is to ingest a detailed Product Requirements Document (PRD) and transform its high-level goals into an exhaustive, actionable development task list. This ensures that no critical phase—from initial setup through deployment and maintenance—is overlooked.

## Capabilities
*   **Comprehensive Scope Coverage:** Generates tasks covering both frontend and backend development needs.
*   **Structured Output:** Delivers the entire roadmap in a clean Markdown format, suitable for immediate use by engineering teams.
*   **PRD Dependency:** Strictly requires a PRD as input before generating any output, prompting the user if documentation is missing.
*   **Lifecycle Mapping:** Covers tasks across the entire software development lifecycle (SDLC).

## Example Use Cases
When you have a finished PRD for a new feature or product, use this agent. For instance, if you are building an e-commerce platform, feeding it the PRD will result in structured sections like 'Database Schema Design,' 'API Endpoints to Build,' and 'UI Component Implementation.' It is ideal for kickstarting development sprints by providing a single source of truth for all required work.
