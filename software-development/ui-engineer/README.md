## Overview
The UI Engineer agent is an expert resource for all things modern frontend development. It specializes in creating high-quality, maintainable, and production-ready user interfaces using contemporary frameworks like React, Vue, or Angular. Its core focus is on delivering code that adheres to industry best practices, ensuring the resulting components are not only functional but also highly readable, accessible, and performant.

## Capabilities
*   **Framework Mastery:** Generates clean code in modern JavaScript/TypeScript, supporting major frameworks (React, Vue, Angular).
*   **Styling Expertise:** Implements advanced styling using CSS-in-JS or Tailwind CSS, always prioritizing responsive, mobile-first design.
*   **Architecture Design:** Structures solutions using component-driven architecture and adheres to SOLID principles for maximum reusability.
*   **Quality Assurance:** Automatically incorporates best practices like TypeScript typing, proper state management patterns (Zustand, Redux), and WCAG accessibility compliance.
*   **Integration Focus:** Designs API-agnostic components with clear abstraction layers for easy backend integration.

## Example Use Cases
*   **Building Components:** Need a complex, responsive dashboard widget? Ask it to build the entire component structure from scratch, complete with state handling and styling.
*   **Code Review & Refactoring:** Paste in an existing React component that feels messy. The agent will review it for performance bottlenecks, suggest TypeScript improvements, and refactor it to modern standards.
*   **Implementing Patterns:** Require a sophisticated data-fetching mechanism? Use this agent to implement the correct pattern (e.g., using React Query or Context API) for optimal state management.
