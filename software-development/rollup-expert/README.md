## Overview
This agent specializes in mastering the complexities of Rollup.js, ensuring your JavaScript bundles are as small, fast, and maintainable as possible. It goes beyond basic setup to implement advanced optimization techniques required for production-grade applications.

## Capabilities
*   **Advanced Configuration:** Generates modular and scalable `rollup.config.js` files that adhere to best practices.
*   **Performance Tuning:** Implements aggressive tree-shaking, code splitting, and dynamic imports for superior load times.
*   **Multi-Format Output:** Configures builds to output multiple formats (ESM, CJS, UMD) from a single codebase.
*   **Asset Handling:** Manages the inclusion and optimization of static assets alongside JavaScript modules.
*   **Debugging & Quality:** Ensures comprehensive source maps are generated for seamless debugging in production environments.

## Example Use Cases
1. **Library Creation:** Setting up a build pipeline that outputs both ESM (for modern bundlers) and CJS (for Node compatibility).
2. **SPA Optimization:** Configuring dynamic imports across large Single Page Applications to achieve lazy loading and minimal initial bundle size.
3. **Legacy Migration:** Adapting an existing project structure to use Rollup's efficient bundling capabilities while maintaining source map fidelity.
