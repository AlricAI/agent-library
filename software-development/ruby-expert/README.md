## Overview
This agent is a deep expert in the Ruby programming language, designed to generate code that is not only functional but also idiomatic, performant, and maintainable. It adheres strictly to community best practices, favoring composition over complex inheritance structures.

## Capabilities
*   **Idiomatic Code Generation:** Writes clean, readable Ruby following established conventions (e.g., RuboCop compliance).
*   **Performance Optimization:** Identifies potential bottlenecks and suggests algorithmic improvements, including memory management strategies.
*   **Advanced Techniques:** Proficient in metaprogramming, DSL creation, and effective use of blocks/procs.
*   **Robust Testing:** Generates comprehensive unit and integration tests using RSpec or Minitest.
*   **Error Handling:** Implements thorough exception handling with clear context and meaningful messages.

## Example Use Cases
1.  **Building a Domain-Specific Language (DSL):** Need to model complex business rules in Ruby? This agent can generate the necessary metaprogramming structure while ensuring readability.
2.  **Performance Review:** Provide an existing chunk of slow Ruby code, and it will refactor it, providing benchmarks and explanations for the performance gains.
3.  **API Service Layer:** Developing a new service object that requires strict adherence to OOP principles and thorough testing coverage.
