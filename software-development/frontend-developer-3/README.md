## Overview
This expert agent specializes in building cutting-edge, high-performance frontend applications using the latest features of React (v19+) and Next.js (v15+). It masters modern web architecture patterns, ensuring components are not only functional but also highly optimized for speed, responsiveness, and accessibility.

## Capabilities
*   **React Mastery:** Implements advanced patterns including Server Components (RSC), Suspense, Concurrent Rendering, and utilizing hooks like `useActionState` and `useOptimistic`.
*   **Next.js Full-Stack Integration:** Proficient in the App Router, leveraging Server Actions for seamless data mutations and optimizing rendering strategies (ISR, dynamic rendering).
*   **Architecture & Performance:** Applies component-driven development, micro-frontends concepts, and advanced build optimizations like code splitting and bundle analysis.
*   **State Management:** Implements modern state solutions using libraries such as Zustand or TanStack Query for robust server and client state handling.

## Example Use Cases
*   **Building a Dashboard:** Generating a complex dashboard layout featuring real-time data fetching, optimistic UI updates via Server Actions, and responsive design across multiple viewports.
*   **Implementing a Form Workflow:** Developing multi-step forms that utilize React's built-in state management alongside Next.js server actions for secure, atomic data submission.
*   **Optimizing Load Times:** Analyzing existing components to suggest performance improvements, such as implementing `React.memo` or optimizing image loading via Next/Image.
