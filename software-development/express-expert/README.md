## Overview
This agent is a dedicated expert in building robust backend services using Express.js. It adheres strictly to modern best practices, focusing on maintainability, security, and performance. Whether you need a simple CRUD API or a complex microservice structure, this agent provides the necessary scaffolding.

## Capabilities
*   **Structured Architecture:** Implements clean project layouts with modular routing using `express.Router`.
*   **Security Implementation:** Automatically integrates best practices like setting security headers (e.g., Helmet) and handling authentication/authorization flows.
*   **Middleware Pipeline Management:** Designs and implements middleware for cross-cutting concerns such as logging, request validation, and error handling.
*   **Performance Optimization:** Includes strategies for caching, compression, and efficient asynchronous operations using `async/await`.
*   **Configuration & Resilience:** Manages environment variables centrally and provides comprehensive, graceful error handling middleware.

## Example Use Cases
*   **Building a REST API:** Generate the complete boilerplate for an API that manages users and products, including validation and CRUD endpoints.
*   **Implementing Authentication:** Scaffold out user login/registration flows with JWT or session management.
*   **Adding Observability:** Integrate Winston or Morgan logging middleware to track all incoming requests and errors automatically.
*   **Creating a Secure Template:** Start a new project that is pre-configured with security headers, environment variable loading, and basic error handling.
