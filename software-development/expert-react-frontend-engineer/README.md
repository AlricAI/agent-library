## Overview
This agent embodies the expertise of a world-class React Frontend Engineer, specializing in the latest advancements up to and including React 19.2. It provides deep knowledge across modern hooks, Server Components (RSC), Actions API, and performance optimization techniques.

Whether you are building a complex Single Page Application (SPA) or need to manage intricate client-server boundaries, this agent ensures adherence to best practices for type safety, concurrency, and optimal user experience.

## Capabilities
*   **React 19 Mastery**: Expert implementation using `<Activity>`, `use()`, `useFormStatus`, and Actions API.
*   **Architecture Design**: Proficient in structuring applications around Server Components, managing data fetching boundaries, and implementing streaming UI.
*   **Performance Tuning**: Advanced application of memoization, code splitting, Suspense boundaries, and adherence to Core Web Vitals.
*   **State Management**: Advises on the optimal state solution (Zustand, Context, Redux Toolkit) for any given complexity level.
*   **Quality Assurance**: Ensures WCAG compliance through semantic HTML and ARIA attributes while providing robust testing strategies using RTL/Vitest.

## Example Use Cases
*   **Building a Dashboard**: Generate the full component structure for a data-intensive dashboard, correctly implementing Server Components for initial load data and Client Components for interactive widgets.
*   **Implementing Complex Forms**: Develop a multi-step form utilizing React Actions for server-side validation and `useOptimistic` for instant UI feedback during submission.
*   **Performance Review**: Analyze existing component structures to identify potential rendering bottlenecks, suggesting migrations from manual memoization to compiler optimizations or proper Suspense wrapping.
