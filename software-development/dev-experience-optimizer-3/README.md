## Overview
The Dev Experience Optimizer acts as a dedicated consultant to enhance the entire software development lifecycle (SDLC) experience. Its core mission is to make development joyful and highly productive by proactively identifying and eliminating points of friction, from initial setup to daily coding tasks.

This agent focuses on making complex processes feel seamless—the ideal state where great tooling works so well it becomes invisible.

## Capabilities
*   **Environment Setup:** Automates dependency installation, simplifies onboarding guides (aiming for under 5 minutes), and establishes intelligent default configurations.
*   **Workflow Optimization:** Identifies repetitive tasks suitable for automation, suggesting useful aliases, shortcuts, and optimizing build/test pipelines.
*   **Tooling Enhancement:** Configures IDE settings, sets up necessary Git hooks, and creates project-specific CLI commands to standardize development practices.
*   **Documentation Improvement:** Generates actionable setup guides, interactive examples, and comprehensive troubleshooting documentation that remains current with the codebase.

## Example Use Cases
1. **New Project Onboarding:** When starting a new repository, use this agent to generate a complete `README` and initial configuration files (`Makefile`, `.git/hooks`) so a new developer can clone and run the application in minutes.
2. **Performance Bottleneck Identification:** If team members complain about slow builds or complex setup steps, feed the agent the existing scripts and error logs; it will profile the workflow to suggest targeted optimizations for build times or dependency management.
3. **Tooling Standardization:** Use it to audit existing project configurations (e.g., `package.json` scripts) and enforce best practices across all development environments.
