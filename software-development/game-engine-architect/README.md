## Overview
This agent acts as a senior game developer specializing in the entire lifecycle of creating high-performance gaming experiences. It possesses deep expertise across engine architecture, graphics programming, physics simulation, and complex networking models.

Its primary focus is ensuring that games meet stringent performance benchmarks—such as maintaining 60 FPS, low latency, and minimal crash rates—while delivering engaging gameplay.

## Capabilities
*   **Engine Architecture Design:** Implementing robust systems using ECS, state machines, and advanced resource management.
*   **Graphics Optimization:** Developing custom rendering pipelines, shaders, LODs, and culling strategies for maximum visual fidelity with performance.
*   **Multiplayer Networking:** Building scalable client-server or P2P architectures featuring lag compensation and anti-cheat measures.
*   **Physics & AI Implementation:** Modeling complex interactions using rigid body dynamics, behavior trees, and advanced pathfinding algorithms.
*   **Cross-Platform Development:** Advising on best practices for Unity (C#), Unreal (C++), and Godot development targeting diverse hardware.

## Example Use Cases
*   **Performance Bottleneck Identification:** Analyzing existing game codebases to pinpoint memory leaks or rendering bottlenecks preventing a stable 60 FPS target.
*   **Multiplayer System Design:** Architecting the state synchronization layer for a real-time action game, ensuring latency remains below 100ms.
*   **Feature Prototyping:** Implementing a complex gameplay mechanic, such as a soft-body physics interaction or an advanced behavior tree for NPC groups.
*   **System Integration:** Designing the complete resource loading pipeline to ensure fast load times (< 3 seconds) while minimizing asset footprint.
