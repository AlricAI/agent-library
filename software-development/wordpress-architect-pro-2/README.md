## Overview
This agent embodies the expertise of a senior WordPress architect with over 15 years in the field. It is designed to treat WordPress not just as a CMS, but as a robust, scalable application framework capable of handling enterprise-level traffic and complexity.

It systematically audits existing installations against industry best practices, focusing on performance benchmarks (like sub-1.5s load times) and security hardening.

## Capabilities
* **Full-Stack Development:** Mastery over PHP 8.x optimization, MySQL tuning, and modern JavaScript frameworks like React/Vue within the WordPress ecosystem.
* **Custom Code Generation:** Expert creation of custom themes, complex plugins using OOP principles, and advanced Gutenberg blocks (including dynamic and pattern support).
* **Performance Engineering:** Implementing object caching (Redis), optimizing database queries, managing transients, and ensuring Core Web Vitals compliance.
* **Security & Scalability:** Auditing for vulnerabilities, enforcing PSR-12 standards, and architecting solutions for high uptime (>99.99%).

## Example Use Cases
* **Enterprise Migration:** Analyzing a legacy site structure and rebuilding it into a modern, scalable multisite network using custom block patterns.
* **Performance Overhaul:** Given slow load times, the agent will identify database bottlenecks, implement object caching layers, and refactor inefficient WP_Query calls to meet sub-1.5s benchmarks.
* **Custom Feature Buildout:** Developing a complex membership plugin that requires secure REST API endpoints for user data management and background processing via queue systems.
