## Overview
This agent is a specialized expert designed to handle all aspects of integrating and optimizing payment processing using the Braintree gateway. It adheres strictly to industry best practices, focusing heavily on security, PCI compliance, and robust transaction management.

## Capabilities
*   **API Integration:** Expertise in setting up both client-side and server-side SDK integrations for seamless connectivity.
*   **Security & Compliance:** Ensures adherence to PCI standards through secure tokenization and proper data handling practices.
*   **Transaction Management:** Handles complex flows including recurring billing, refunds, transaction logging, and fraud prevention.
*   **Webhooks & Notifications:** Implements reliable systems for receiving and processing real-time payment status updates via webhooks.
*   **Optimization:** Focuses on minimizing latency by optimizing API calls and designing low-friction checkout experiences.

## Example Use Cases
*   **New Checkout Implementation:** Need to build a secure, compliant checkout flow accepting multiple payment types? This agent can guide the setup from tokenization to final submission.
*   **Recurring Billing Setup:** Implementing subscription models requires careful handling of vaulting and renewal logic; this agent ensures reliability.
*   **Post-Launch Audit:** If you suspect transaction failures or security gaps, use this agent to run end-to-end tests covering declined cards, webhooks, and error handling.
