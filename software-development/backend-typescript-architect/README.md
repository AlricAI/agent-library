## Overview
This agent embodies the expertise of a Senior Backend TypeScript Architect, specializing in building high-performance, scalable server-side applications using TypeScript and the Bun runtime. It adheres to modern engineering principles like SOLID, clean architecture, and rigorous type safety.

## Capabilities
*   **API Design:** Designing robust RESTful or GraphQL endpoints with proper versioning and contracts.
*   **Database Integration:** Architecting schemas, optimizing queries, and implementing ORM/query builder logic for various databases.
*   **Security Implementation:** Applying best practices for authentication (JWT), authorization, and mitigating common vulnerabilities (OWASP).
*   **Architecture Patterning:** Guiding the implementation of microservices, event-driven patterns, and scalable system designs.
*   **Code Quality Assurance:** Writing self-documenting, highly type-safe code with strategic comments explaining *why* decisions were made.

## Example Use Cases
When you need to build a new service endpoint, optimize an existing slow query, or structure a complex multi-service backend, this agent is ideal. For instance, if you are tasked with creating a user profile management system that requires secure endpoints, database interaction, and validation layers, the Backend TypeScript Architect will guide you through the entire process—from initial architectural diagrams to unit test scaffolding.

It excels in tasks demanding deep technical rigor, ensuring the resulting code is not just functional, but maintainable, performant, and production-ready.
