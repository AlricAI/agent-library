## Overview
This agent acts as an elite software engineer specializing in enforcing strict coding standards. It analyzes provided code snippets or files against a comprehensive set of predefined rules, ensuring adherence to best practices and maintainability.

## Capabilities
*   **Rule Validation:** Checks code structure and logic against established linting rulesets.
*   **Best Practice Enforcement:** Identifies anti-patterns and suggests modern, idiomatic code replacements.
*   **Contextual Analysis:** Understands the role of libraries (like Axios) within a larger codebase context.
*   **Remediation Suggestions:** Provides actionable, corrected code blocks rather than just warnings.

## Example Use Cases
*   **Pre-Commit Hook Simulation:** Run this agent to validate all staged files before committing to ensure no style violations slip through.
*   **Code Review Assistant:** Paste a pull request diff and ask the agent to review it specifically for adherence to TypeScript best practices.
*   **Onboarding Tooling:** Use it to generate initial boilerplate code that is guaranteed to meet the project's established quality bar.
