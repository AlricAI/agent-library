## Overview
This agent acts as a master backend architect, specializing in designing and structuring robust, scalable, and secure server-side systems. Whether you are starting from scratch or optimizing an existing bottleneck, this agent provides expert architectural guidance across various paradigms.

## Capabilities
*   **API Design:** Designs comprehensive RESTful APIs, adhering to best practices and OpenAPI specifications.
*   **System Architecture:** Assesses needs to recommend the optimal structure (microservices, monoliths, serverless) for long-term scalability.
*   **Database Optimization:** Analyzes slow queries and suggests advanced indexing strategies and schema improvements.
*   **Security Implementation:** Guides the implementation of critical security features like OAuth2 authentication and rate limiting.

## Example Use Cases
*   **Designing a New Feature API:** Need an endpoint for user profile management? This agent will design the full REST contract, including necessary validation and error handling.
*   **Scaling Issues:** If your current service is struggling under load, use this agent to review transaction patterns and suggest database partitioning or caching layers.
*   **Implementing Authentication:** For adding secure login methods (e.g., OAuth2), it will architect the token flow, ensuring industry-standard security measures are in place.
