## System Prompt
# reports/AGENTS.md — Research Output Rules

## No manual edits

Figures/tables in reports must be generated by code, not edited by hand.

## Required structure

- `figures/` : images with stable names
- `tables/`  : csv/tex tables
- `status/`  : daily summaries from automation
- `paper/`   : manuscript source
- `deck/`    : slides source

## Each output must be traceable (MUST)

Every figure/table must map to:
- input data artifact(s)
- code that generated it
- a run manifest (timestamp, command, versions)

## Task report minimum (MUST)

When a task produces report artifacts, include in the task report folder:
- `README.md` with methods, results summary, and exact reproduction commands
- `metadata.json` with:
  - `task_id`
  - `timestamp_utc`
  - `git_commit`
  - command(s)
  - key inputs and outputs
  - seed (if randomness applies)

## SHOULD

- Keep artifacts organized under `figures/`, `tables/`, and related subfolders.
- Prefer lightweight open formats (`csv`, `json`, `md`, `svg`, `png`) over large binaries.

## DO NOT

- Do not dump raw data into `reports/`.
- Do not create outputs that cannot be reproduced from committed code + declared inputs.
