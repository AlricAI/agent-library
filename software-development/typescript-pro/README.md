## Overview
TypeScript Pro is a professional-grade AI agent designed to function as a senior TypeScript engineer. Its core focus is building robust, highly scalable, and maintainable applications for both Node.js backends and browser frontends by rigorously enforcing type safety at every stage.

This agent doesn't just write code; it architects solutions, ensuring that the resulting codebase adheres to modern software engineering best practices, including comprehensive testing and clear architectural patterns.

## Capabilities
*   **Advanced Type System Mastery:** Implements complex features using generics, conditional types, mapped types, and precise domain modeling.
*   **Architectural Design:** Proposes scalable patterns for large systems, covering dependency injection, module federation, and overall system structure.
*   **Type-Safe Development:** Enforces strict type checking throughout the development lifecycle to prevent runtime errors at compile time.
*   **Testing Excellence:** Writes comprehensive unit and integration tests (Jest/Vitest) alongside implementation, advocating for test-driven development.
*   **Tooling & Optimization:** Assists with build system configuration (`tsconfig`, bundlers) and performance tuning for optimal execution parity.

## Example Use Cases
*   **Designing a Microservice API:** Ask it to design the full TypeScript structure (interfaces, routes, service layers) for a new RESTful endpoint, including necessary validation schemas.
*   **Refactoring Legacy Code:** Provide a large, untyped codebase and ask it to refactor it into a modular, type-safe structure, detailing the architectural changes required.
*   **Implementing Complex Logic:** Request a utility function that requires advanced type manipulation, such as creating a generic data mapper with conditional return types based on input parameters.
