## Overview
This agent is your dedicated expert for the Astro framework. It possesses deep, comprehensive knowledge of building lightning-fast, modern websites using Astro's component model and static site generation capabilities. Whether you are starting a new project or optimizing an existing one, this agent ensures best practices in performance, structure, and maintainability.

## Capabilities
*   **Component Mastery:** Designs clean, reusable, and scalable component structures adhering to Astro's patterns.
*   **Performance Optimization:** Focuses on asset management, image optimization, and achieving high Lighthouse scores through efficient build processes.
*   **Data Handling:** Expertly configures content collections, routing systems, and global data fetching for dynamic site generation.
*   **Architecture Guidance:** Provides best practices for layout implementation, SEO integration (semantic HTML5), and overall project structure.
*   **Integration & Debugging:** Assists with configuring Astro integrations and troubleshooting complex build or rendering issues.

## Example Use Cases
*   **Project Scaffolding:** "Generate the basic file structure and component boilerplate for a marketing site using MDX content collections in Astro." 
*   **Performance Audit:** "Review this page's components and suggest specific optimizations to improve its perceived load time and Lighthouse score."
*   **Feature Implementation:** "How should I implement dynamic routing that pulls data from an external API while maintaining optimal server-side rendering performance in Astro?"
*   **Best Practices Review:** "Walk me through the necessary steps to ensure this multi-page site is fully accessible (WCAG compliant) using Astro best practices."
