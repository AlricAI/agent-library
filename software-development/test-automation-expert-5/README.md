## Overview
This agent embodies the expertise of a senior test automation engineer, specializing in building intelligent and scalable testing ecosystems. It moves beyond simple script execution by integrating advanced concepts like AI-powered generation, self-healing capabilities, and adherence to rigorous quality engineering principles.

## Capabilities
*   **Test-Driven Development (TDD) Mastery:** Facilitates the entire TDD cycle, from generating failing tests to supporting refactoring with regression safety checks. It supports various methodologies including Property-based TDD and BDD integration.
*   **AI-Powered Testing Frameworks:** Implements self-healing automation using industry tools, leverages NLP for natural language test case generation, and uses ML for failure prediction and optimization.
*   **Comprehensive Coverage:** Supports advanced techniques like Test Triangulation and visual AI testing for robust UI validation.
*   **Process Optimization:** Provides support for fast feedback loops, TDD compliance monitoring, and structured development practices (e.g., Baby Steps methodology).

## Example Use Cases
1. **Implementing a New Feature Test Suite:** Provide the core logic or requirements, and the agent will generate an initial set of failing tests following TDD principles, guiding you through the 'red-green-refactor' cycle.
2. **Maintaining Legacy Tests:** Feed in existing test scripts that fail intermittently; the agent can analyze them to suggest self-healing patches or identify underlying structural weaknesses.
3. **Establishing Quality Gates:** Use it to audit a project structure and recommend improvements for test coverage, suggesting where property-based testing or visual regression checks should be added.
