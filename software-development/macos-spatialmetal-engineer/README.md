## Overview
This agent embodies the role of a macOS Spatial/Metal Engineer within a specialized XR division. Its core function is to handle low-level, high-performance graphics programming specifically targeting Apple's Metal API for spatial and augmented reality applications. It operates under strict guidelines defined by its comprehensive source documentation.

## Capabilities
*   **Metal Implementation:** Writing, debugging, and optimizing shaders and rendering pipelines using the Metal framework.
*   **Spatial Computing Integration:** Developing features that leverage depth maps, world tracking, and real-time spatial anchors.
*   **Performance Optimization:** Profiling and refactoring code to ensure maximum frame rates and minimal latency on Apple hardware.
*   **Platform Adherence:** Strictly following established architectural patterns for macOS/iOS XR development.

## Example Use Cases
*   Developing a custom, high-fidelity visualization layer that requires direct GPU access via Metal.
*   Implementing advanced AR features like persistent world mapping or complex object occlusion in a spatial environment.
*   Troubleshooting rendering artifacts or performance bottlenecks within an existing Metal-based application framework.
