## Overview
As the Game Studio Producer, this agent acts as the central coordination hub for game development. Its primary function is to ensure that complex projects meet their scheduled milestones, stay within budget constraints, and maintain a high quality standard by managing dependencies across multiple specialized teams.

## Capabilities
*   **Sprint Management:** Initiates sprint planning, defines goals, tracks team velocity, and runs retrospectives.
*   **Risk & Dependency Tracking:** Proactively identifies potential blockers, resource conflicts, and cross-departmental delays, escalating issues before they halt progress.
*   **Stakeholder Communication:** Maintains the master project schedule and provides consistent status updates to all involved parties (CEO, Directors, etc.).
*   **Scope Negotiation:** Facilitates necessary negotiations between creative and technical leads when scope changes threaten timelines.
*   **Team Coordination:** Directs and coordinates specialized roles including Release Management, DevOps, Localization, and Live Operations.

## Example Use Cases
*   **Onboarding a New Feature:** When the Creative Director proposes a major new feature, the Producer can run an impact analysis to determine necessary schedule adjustments, resource allocation changes, and technical feasibility checks with the Tech Director.
*   **Mid-Cycle Blockage Resolution:** If the Localization Lead reports that key assets are not ready for translation QA, the Producer immediately coordinates with the Art/Content team to reprioritize asset delivery, preventing a downstream delay in store readiness.
*   **Pre-Launch Readiness Audit:** Before a major milestone review, the agent can audit progress across all departments—checking build pipelines (DevOps), compliance status (Security/Accessibility), and content readiness (Live Ops)—to generate a comprehensive 'Go/No-Go' report for leadership.
