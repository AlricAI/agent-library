## Overview
This agent is designed to act as a highly structured technical pair programmer, specializing in guiding users through complex software development lifecycles. It excels at projects involving multiple interconnected technologies, such as virtual assistants built with Docker, APIs, and messaging platforms.

It adheres to a strict, methodical troubleshooting framework, ensuring the user understands not just *what* to do, but *why*, *how* it works, and *how to verify* success.

## Capabilities
*   **Structured Problem Solving:** Follows a mandatory four-step process (Explain $\rightarrow$ Mean $\rightarrow$ Next Step $\rightarrow$ Verify) for all technical guidance.
*   **Multi-Technology Support:** Proficient in guiding setups involving Docker, APIs, Telegram bots, and general system configurations.
*   **Context Retention:** Maintains awareness of the user's project context (e.g., Nanobot Ponce), ensuring advice is relevant to the current architecture.
*   **Workflow Optimization:** Focuses on practical, command-line oriented solutions (CMD/Terminal) over abstract theory.

## Example Use Cases
*   **Docker Deployment Debugging:** If a container fails to start, this agent will guide you through checking logs, understanding networking issues, and verifying port mappings step-by-step.
*   **API Integration Walkthrough:** When connecting two services via an API, it will explain the required authentication flow, provide example calls, outline the next steps for error handling, and suggest verification endpoints.
*   **Project Architecture Review:** You can ask it to review your current setup (e.g., 'How should I structure my Python code alongside a Docker Compose file?'), and it will provide an organized, actionable plan.
