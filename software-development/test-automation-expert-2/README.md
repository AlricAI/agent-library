## Overview
This agent embodies the expertise of a senior test automation engineer, specializing in building intelligent, resilient, and scalable testing ecosystems. It moves beyond simple script execution by integrating advanced concepts like self-healing mechanisms, AI-driven generation, and comprehensive quality engineering principles.

## Capabilities
*   **Test-Driven Development (TDD) Mastery:** Facilitates the entire TDD lifecycle, supporting both state-based (Chicago School) and interaction-based (London School) approaches. It automates test naming conventions and tracks TDD compliance metrics.
*   **AI-Powered Frameworks:** Implements self-healing automation using industry tools and leverages NLP for generating and maintaining test cases from natural language specifications.
*   **Advanced Validation:** Provides visual AI testing capabilities for accurate UI regression detection and uses predictive analytics to optimize test execution paths.
*   **Comprehensive Coverage:** Supports advanced techniques like Property-Based TDD, Behavior-Driven Development (BDD) integration, and Test Triangulation for maximum code coverage.

## Example Use Cases
1. **Implementing a New Feature Test Suite:** Provide the core business logic, and this agent will generate initial test cases using BDD syntax, then structure them into a self-healing framework ready for CI/CD pipeline integration.
2. **Maintaining Legacy Tests:** Feed it existing brittle UI tests; the agent will analyze failure patterns and suggest refactoring paths or update selectors to ensure the tests remain robust against minor UI changes.
3. **Establishing TDD Practices:** Guide a development team through a new module by enforcing a strict red-green-refactor cycle, providing immediate feedback on test coverage gaps and adherence to best practices.
