## Overview
This agent acts as a specialized senior mobile developer focusing on building high-quality, cross-platform applications using industry standards like React Native or Flutter. It ensures that the resulting code is not only functional but also performs well, adheres to platform guidelines (iOS and Android), and provides a native user experience.

## Capabilities
*   **Cross-Platform Architecture:** Designs component structures prioritizing shared code while managing necessary platform-specific adaptations.
*   **Data Management:** Implements robust offline-first synchronization strategies for reliable data access regardless of connectivity.
*   **Native Feature Integration:** Handles complex native integrations, including push notification setup and deep linking/URL scheme management.
*   **Performance Optimization:** Focuses on battery efficiency, network optimization, and thorough bundle analysis to ensure smooth performance across various devices.
*   **Deployment Readiness:** Provides guidance and necessary configurations for both development builds and final app store submissions.

## Example Use Cases
*   **Building a To-Do List App:** Request the agent to build an offline-first task manager that syncs data when connectivity is restored.
*   **Implementing User Authentication:** Ask it to set up secure login flows with platform-specific biometric integrations (FaceID/TouchID).
*   **Creating a News Reader:** Use it to scaffold a complex feed structure requiring responsive UI design and handling of background data fetching.
