## Overview
This agent specializes in mastering the Auth0 platform, acting as an expert consultant for all aspects of identity and access management (IAM). It is designed to help developers and security engineers implement robust, scalable, and secure authentication systems using industry best practices.

## Capabilities
*   **Protocol Mastery:** Deep understanding and correct implementation of OAuth 2.0 and OpenID Connect flows.
*   **Integration Support:** Expertise in configuring social logins (Google, Facebook, etc.) and integrating with various third-party Identity Providers (IdPs).
*   **Advanced Logic:** Proficiency in utilizing Auth0 Rules and Hooks to implement custom authentication logic beyond standard configurations.
*   **Security Hardening:** Ability to configure Multi-Factor Authentication (MFA), manage user roles/permissions, and enforce least privilege access.
*   **Testing & Validation:** Generates comprehensive test reports and configuration files, ensuring all implemented flows are verified using tools like Postman.

## Example Use Cases
1. **New Application Onboarding:** Need to secure a new web application with social sign-in? This agent will guide you through setting up the correct OAuth flow and linking necessary identity providers.
2. **Security Audit & Hardening:** If your current Auth0 setup lacks MFA enforcement or has outdated security policies, this agent can audit your configuration against best practices and provide remediation steps.
3. **Custom Authentication Flow:** Implementing a complex business rule that requires custom logic before token issuance? Use this agent to write and test the necessary Auth0 Rules or Hooks.
