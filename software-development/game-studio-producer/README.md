## Overview
As the Producer at a game studio, this agent acts as the central coordination hub for all development efforts. Its primary function is to ensure that complex projects—like video games—ship on time, within budget, and meeting predefined quality standards by managing dependencies across multiple specialized teams.

## Capabilities
*   **Sprint Management:** Initiates sprint planning, defines goals, tracks team velocity, and facilitates retrospectives.
*   **Risk & Dependency Tracking:** Proactively identifies potential blockers, resource conflicts, and cross-departmental bottlenecks before they halt progress.
*   **Scope Negotiation:** Facilitates necessary discussions between creative and technical leads to analyze the schedule impact of any requested scope changes.
*   **Stakeholder Communication:** Runs daily standups, sprint reviews, and maintains a master project schedule for all stakeholders.
*   **Team Coordination:** Delegates tasks and coordinates efforts across specialized roles such as Release Management, DevOps, Localization, and Live Operations.

## Example Use Cases
*   **Project Kickoff:** When starting a new development cycle, use this agent to generate the initial sprint plan, assign capacity estimates, and establish key milestones for all departments.
*   **Blocker Resolution:** If the QA team reports that they are blocked because the latest build from DevOps is unstable, the Producer can coordinate with the DevOps Engineer and Technical Director to prioritize a fix and keep the schedule moving.
*   **Scope Creep Management:** When the Creative Director proposes adding a major new feature late in development, this agent will run an impact analysis against the current roadmap, presenting the necessary trade-offs (time, budget, or quality) to the leadership team.
