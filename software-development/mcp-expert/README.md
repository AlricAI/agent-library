## Overview
This agent acts as a dedicated Model Context Protocol (MCP) expert, specializing in the end-to-end design and implementation of complex protocol integrations. It ensures that resulting configurations are production-ready, secure, and highly performant.

## Capabilities
*   **Integration Analysis:** Thoroughly analyzes target service requirements to define necessary MCP structures.
*   **Configuration Generation:** Produces complete, standard JSON configuration files for MCP servers.
*   **Security Hardening:** Implements best practices for authentication, environment variable management, and secure coding patterns.
*   **Performance Tuning:** Optimizes settings by incorporating rate limiting, timeouts, and resource usage considerations.
*   **Deployment Guidance:** Provides actionable installation commands (`npx`) and comprehensive testing/troubleshooting guides.

## Example Use Cases
*   **Building a New Service Connector:** Need to connect an internal microservice using the MCP standard? This agent will provide the full JSON config, required environment variables, and deployment steps.
*   **Auditing Existing Setup:** If you have an existing integration that is slow or insecure, use this agent to review it against best practices for performance and security.
*   **Protocol Implementation Design:** When designing a new feature that requires adherence to strict protocol standards, leverage this expert to structure the initial framework.
