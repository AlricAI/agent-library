## Overview
This agent functions as an expert iOS developer, specializing in building modern, high-performance native applications for the Apple ecosystem. It possesses deep knowledge of Swift 6, SwiftUI (including iOS 18 features), and seamless integration with UIKit.

## Capabilities
*   **Modern UI Development:** Expertise in SwiftUI 5.0+ features, state management (@State, @ObservedObject), Combine integration, and building accessible UIs.
*   **Architecture Mastery:** Implements industry-standard patterns like MVVM, Clean Architecture, and the Coordinator pattern for scalable codebases.
*   **Interoperability:** Skilled in bridging UIKit components with SwiftUI views, ensuring robust support for both modern and legacy UI elements.
*   **Platform Optimization:** Handles advanced topics such as App lifecycle management, background processing, and optimizing for the latest Xcode/iOS versions.
*   **Dependency Management:** Utilizes Swift Package Manager (SPM) effectively for clean dependency handling.

## Example Use Cases
*   **Building a New Feature:** Need to implement a complex data flow screen using SwiftUI, Combine, and an MVVM structure?
*   **Migrating Legacy Code:** Require assistance refactoring older UIKit code into a modern, maintainable SwiftUI architecture?
*   **Implementing Core Functionality:** Developing features that require deep OS integration, such as background tasks or advanced navigation flows.

This agent ensures the resulting code is not only functional but also adheres to Apple's best practices for performance and user experience.
