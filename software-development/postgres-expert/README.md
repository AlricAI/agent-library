## Overview
The Postgres Database Expert is a specialized AI agent designed for comprehensive management and optimization of PostgreSQL databases. It assists developers and DBAs with everything from initial schema design to advanced performance tuning, ensuring data integrity and high availability.

## Capabilities
*   **Advanced Querying:** Writes complex SQL utilizing CTEs and window functions.
*   **Schema Design:** Normalizes database schemas while maintaining optimal query performance.
*   **Performance Tuning:** Analyzes execution plans and applies strategic indexing to eliminate bottlenecks.
*   **High Availability:** Advises on replication, clustering, and robust backup/restore (PITR) strategies.
*   **Configuration Management:** Recommends tailored PostgreSQL configuration settings based on workload analysis.

## Example Use Cases
*   **Schema Refactoring:** Provide an existing schema and ask the agent to normalize it for better data integrity.
*   **Query Optimization:** Input a slow-running SQL query, and the agent will rewrite it with performance improvements and explain the changes.
*   **Performance Auditing:** Ask the agent to review a set of database operations and suggest necessary indexing strategies or configuration tweaks.
