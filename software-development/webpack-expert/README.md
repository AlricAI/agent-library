## Overview
This agent specializes in the intricacies of Webpack, acting as an expert consultant for setting up robust and highly optimized build pipelines. It covers everything from basic configuration to advanced techniques like code splitting and dependency tree-shaking.

## Capabilities
*   **Configuration Generation:** Creates modular and maintainable `webpack.config.js` files based on project needs.
*   **Asset Handling:** Implements correct loaders and plugins for CSS, images, and various assets.
*   **Performance Optimization:** Configures code splitting using dynamic imports to minimize initial load times.
*   **Path Management:** Sets up module resolution aliases and precise output path configurations.
*   **Development Setup:** Provides best practices for `webpack-dev-server`, including Hot Module Replacement (HMR).
*   **Environment Control:** Correctly utilizes `DefinePlugin` and mode settings for environment-specific builds.

## Example Use Cases
1. **Initial Setup:** "I'm starting a React project; generate a basic, modern Webpack config that handles Babel, CSS imports, and sets up HMR." 
2. **Optimization:** "My bundle size is too large. How can I implement aggressive tree-shaking and chunking for my vendor libraries?"
3. **Troubleshooting:** "I'm getting an error resolving components in a subdirectory; help me configure module aliases correctly."

By following its quality checklist, this agent ensures your build process is not only functional but also performant and scalable.
