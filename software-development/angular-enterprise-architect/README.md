## Overview
This agent acts as a senior Angular Architect specializing in building robust, scalable enterprise applications using modern Angular (v15+). It enforces industry best practices across architecture, state management, and performance optimization to ensure the resulting codebase is maintainable for large teams.

## Capabilities
*   **Architecture Review:** Analyzes module structure, lazy loading implementation, and adherence to core/feature module separation.
*   **State Management Implementation:** Guides the setup of NgRx patterns, including optimized selectors, effects, and store design.
*   **Performance Tuning:** Implements strategies like `OnPush` change detection, virtual scrolling, and bundle optimization checks.
*   **Advanced RxJS Patterns:** Ensures correct usage of complex Observable chains, Subjects, and memory management techniques.
*   **Micro-frontend Setup:** Assists with module federation patterns for building distributed applications.

## Example Use Cases
1. **Initial Project Scaffolding:** When starting a new large application, use this agent to establish the foundational structure (Core/Shared modules, routing guards) ensuring scalability from day one.
2. **Performance Bottleneck Resolution:** If performance profiling reveals slow rendering times, invoke the agent to review component strategies and suggest `OnPush` or virtual scrolling implementations.
3. **State Logic Refactoring:** When complex business logic accumulates in components, use this agent to refactor state handling into optimized NgRx effects and services.
