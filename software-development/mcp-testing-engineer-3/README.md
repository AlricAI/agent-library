## Overview
This agent functions as an elite Model Context Protocol (MCP) Testing Engineer, providing comprehensive quality assurance for MCP server implementations. It moves beyond basic unit testing to simulate real-world stress, security threats, and protocol deviations.

When you need absolute confidence in an MCP server's stability, compliance, and security posture, this agent generates detailed, actionable reports covering every critical aspect of the service.

## Capabilities
*   **Schema & Protocol Validation:** Ensures 100% adherence to official MCP specifications, including JSON-RPC batching and Streamable HTTP semantics.
*   **Security Auditing:** Executes penetration tests for common vulnerabilities like confused deputy attacks and injection flaws, providing CVSS scoring.
*   **Performance Testing:** Measures latency under concurrent connections, testing rate limiting and resource exhaustion scenarios.
*   **Comprehensive Debugging:** Validates completion endpoints with large datasets and handles complex error response pathways.
*   **Test Plan Generation:** Creates structured test plans covering functional, safety, and performance dimensions for any given server implementation.

## Example Use Cases
1. **Pre-Release QA:** Before deploying a new MCP service, run this agent to generate a full compliance report ensuring zero critical vulnerabilities are present.
2. **Integration Testing:** When integrating two different services via an MCP layer, use it to validate the data exchange integrity and protocol handshakes.
3. **Security Hardening:** Simulate a penetration test against an existing endpoint to proactively discover potential injection points or authentication boundary weaknesses.
