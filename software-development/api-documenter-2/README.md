## Overview
Api Documenter is an expert agent specializing in transforming raw API knowledge into world-class, developer-first documentation. It treats the documentation as the primary contract for your service, ensuring clarity and completeness.

This agent excels at producing industry-standard artifacts that developers can immediately use to integrate with your endpoints, saving significant onboarding time.

## Capabilities
*   **OpenAPI 3.0 Generation:** Creates fully validated OpenAPI specifications from API descriptions or endpoint details.
*   **Multi-Language Examples:** Provides ready-to-use code snippets in popular languages like Python, JavaScript, Java, and `curl`.
*   **Testing Collections:** Builds structured Postman collections to allow immediate testing of endpoints without manual setup.
*   **Comprehensive Guides:** Designs clear sections for authentication flows, detailed error handling (including status codes), and request/response body schemas.
*   **Developer Focus:** Prioritizes actionable, copy-paste-ready examples over abstract explanations.

## Example Use Cases
*   **Onboarding New Developers:** Feed it your API endpoints and receive a complete documentation package ready for internal wikis or developer portals.
*   **API Versioning:** Update existing specs to reflect changes in endpoints, ensuring backward compatibility is documented.
*   **Client Integration:** Generate a full Postman collection so that frontend or mobile teams can begin testing against the defined contract immediately.
