## Overview
This agent acts as a senior Kubernetes Architect, providing deep expertise across the entire cloud-native stack. It specializes in moving organizations toward modern, declarative infrastructure management using GitOps principles.

Whether you are deploying to EKS, AKS, or GKE, or managing complex multi-cluster setups, this agent ensures your architecture is scalable, secure, and developer-friendly.

## Capabilities
*   **Cloud Provider Mastery**: Deep knowledge of managed services (EKS, AKS, GKE) and self-managed deployments.
*   **GitOps Implementation**: Expert guidance on ArgoCD and Flux v2 for declarative infrastructure management.
*   **Progressive Delivery**: Designing robust CI/CD pipelines using canary or blue/green strategies.
*   **Infrastructure as Code (IaC)**: Proficiency in Terraform, Helm 3.x, Kustomize, and Pulumi for provisioning resources.
*   **Security & Observability**: Advising on service mesh integration (Istio), policy enforcement, and cost optimization patterns.

## Example Use Cases
1. **Designing a Multi-Cloud Strategy**: Ask the agent to compare implementing a service mesh across EKS and AKS using Istio versus Linkerd, detailing the necessary GitOps repository structure.
2. **Implementing GitOps for Canary Releases**: Provide your application manifest and ask the agent to outline the required Argo Rollouts configuration and associated CI/CD steps for a zero-downtime canary deployment.
3. **Platform Blueprinting**: Request an architectural blueprint for a multi-tenant platform on GKE, covering networking segmentation (VPC/NetworkPolicy), identity management, and recommended tooling stack.
