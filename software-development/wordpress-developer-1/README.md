## Overview
This specialized AI agent acts as an expert WordPress developer, capable of handling everything from modern theme creation to complex, secure plugin development. It adheres strictly to current WordPress coding standards and best practices, ensuring the resulting code is scalable, performant, and maintainable.

## Capabilities
*   **Theme Development:** Creating custom themes with modern block editor (Gutenberg) integration and optimizing template hierarchies.
*   **Plugin Creation:** Building secure, architecturally sound plugins using proper WordPress hooks, APIs, and object-oriented PHP.
*   **Headless Solutions:** Implementing integrations for headless setups using REST API or GraphQL.
*   **Performance & Security:** Applying advanced caching strategies, database optimization, and comprehensive security hardening measures (sanitization, escaping).
*   **Architecture:** Setting up complex structures like multisite networks with custom management tools.

## Example Use Cases
1. **Building a Custom Theme:** Requesting a theme structure that uses `theme.json` for block-based design while maintaining full responsiveness and WCAG accessibility.
2. **Developing a Plugin:** Needing a custom plugin to manage specialized data via CPTs, requiring secure database interaction using `$wpdb` and custom meta boxes.
3. **API Integration:** Implementing the backend logic for a headless setup where content needs to be fetched securely via the WordPress REST API.
