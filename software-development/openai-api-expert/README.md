## Overview
This expert agent is designed to be your dedicated resource for any project involving the OpenAI API. It guides development from initial documentation review through to deployment, focusing heavily on robustness, security, and cost optimization.

## Capabilities
*   **Comprehensive Documentation Review:** Starts every engagement by thoroughly analyzing relevant OpenAI API documentation.
*   **Robust Error Handling:** Implements comprehensive try/catch blocks and specific error handling for all API calls.
*   **Rate Limit Management:** Develops strategies, including caching and request throttling, to prevent hitting rate limits.
*   **Security & Compliance:** Ensures secure storage of API keys and adherence to OpenAI's data privacy guidelines.
*   **Optimization & Cost Control:** Analyzes usage patterns to suggest cost-effective batching and call frequency adjustments.
*   **Testing Infrastructure:** Builds comprehensive test suites, including mock server setups, for reliable development cycles.

## Example Use Cases
*   **Building a Production Chatbot:** Integrate the API with advanced streaming capabilities while implementing fallback logic for high availability.
*   **Data Processing Pipeline:** Develop a secure batch processing system that handles large volumes of data submissions, complete with detailed audit logs and cost analysis reports.
*   **Version Migration:** Update an existing application to utilize the latest OpenAI model features while maintaining backward compatibility through versioning strategies.
