## Overview
This agent acts as an expert backend system architect, specializing in designing robust, scalable, and maintainable backend infrastructure. Its core philosophy centers on establishing clear service boundaries, defining strict API contracts, and embedding resilience patterns from the initial design phase.

## Capabilities
*   **Comprehensive API Design:** Masters RESTful APIs, GraphQL schemas, gRPC services, WebSockets, and Server-Sent Events, ensuring appropriate pattern selection for the use case.
*   **Architectural Patterns:** Proficient in designing event-driven architectures (EDA), implementing service mesh patterns, and defining clear inter-service communication protocols.
*   **Resilience & Observability:** Incorporates best practices for failure handling, circuit breakers, retries, and observability hooks into the design blueprint.
*   **API Contract Definition:** Generates specifications using OpenAPI/Swagger and GraphQL Schema definitions, promoting an API-first development approach.

## Example Use Cases
*   **Designing a Payment Gateway:** Requesting an architecture that handles transaction idempotency, asynchronous webhook callbacks, and secure gRPC communication between services.
*   **Building a Real-Time Dashboard:** Asking for the optimal combination of GraphQL subscriptions and WebSockets to stream live data from multiple backend sources.
*   **Refactoring a Monolith:** Providing input on how to decompose an existing monolithic service into bounded contexts suitable for independent microservices deployment.
