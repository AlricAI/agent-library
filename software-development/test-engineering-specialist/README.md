## Overview
This Test Engineering Specialist agent provides deep expertise in modern software quality assurance practices. It guides users through the entire testing lifecycle, from initial requirement analysis to final coverage reporting, ensuring that applications are reliable and robust.

## Capabilities
*   **Test Strategy Design:** Develop comprehensive test plans covering unit, integration, and end-to-end (E2E) scenarios.
*   **Framework Implementation:** Write high-quality tests using industry standards like pytest, Jest, or Playwright.
*   **Best Practices Adherence:** Ensures tests follow patterns like AAA (Arrange-Act-Assert) and maintain DRY principles for readability and longevity.
*   **Coverage Analysis:** Focuses on achieving high code coverage (>80%) by systematically identifying and testing edge cases and error paths.
*   **CI/CD Integration:** Designs test suites suitable for automated execution within continuous integration pipelines.

## Example Use Cases
1. **Feature Validation:** Provide a new feature's requirements, and the agent will generate a multi-layered test suite (unit tests for logic, E2E tests for user flow).
2. **Refactoring Safety Net:** When updating legacy code, use this agent to write comprehensive regression tests that guarantee existing functionality remains unbroken.
3. **Performance Check:** Request load testing scenarios and basic performance assertions to identify potential bottlenecks early in the development cycle.
