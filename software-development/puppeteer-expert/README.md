## Overview
This agent specializes in mastering the Puppeteer library to automate interactions with modern web browsers. It is designed for developers needing robust, reliable solutions for tasks that require a full browser environment, such as scraping dynamic content or performing end-to-end testing.

## Capabilities
*   **Headless Automation:** Executes scripts reliably in headless mode while maintaining functionality equivalent to visible browsing.
*   **Advanced Scraping:** Handles complex Single Page Applications (SPAs) and AJAX-loaded content by managing waits and network requests.
*   **Interaction Simulation:** Automates user flows, including form submissions, clicks, and navigation across multi-step websites.
*   **Data Capture & Verification:** Capable of capturing screenshots at critical points and generating structured outputs like JSON or CSV from scraped data.
*   **Robust Scripting:** Implements best practices such as error handling, session management, and resource efficiency for maintainable code.

## Example Use Cases
1. **E-commerce Data Collection:** Scrape product listings from a site that loads prices via JavaScript, ensuring all pagination steps are covered.
2. **Automated Testing:** Simulate a full user journey (login -> search -> add to cart) and generate visual proof via screenshots at each stage.
3. **API Proxying/Scraping:** Intercept network requests to extract data that is not directly visible in the HTML source, logging it into a structured format.
