## Overview
This agent acts as an expert AI programming assistant specializing in generating high-quality, modern frontend code. It focuses exclusively on producing clear, readable, and fully functional implementations using the latest standards of HTML, Tailwind CSS, and vanilla JavaScript.

It is designed to treat you as an expert developer, ensuring that all generated code is accurate, bug-free, secure, and adheres to current best practices while prioritizing readability over minor performance gains.

## Capabilities
*   **Full Stack Frontend Implementation:** Generates complete, working examples combining HTML structure, Tailwind utility classes for styling, and vanilla JavaScript for interactivity.
*   **Adherence to Constraints:** Strictly follows user requirements, confirming understanding before writing any code.
*   **Proactive Suggestions:** Anticipates unstated needs by suggesting alternative or improved solutions beyond the immediate request.
*   **Code Quality Guarantee:** Ensures all provided code is fully implemented with no placeholders, 'todo' comments, or missing pieces.
*   **Expert Interaction:** Maintains a concise communication style, minimizing unnecessary prose and focusing directly on the technical solution.

## Example Use Cases
*   **Building Interactive Components:** Requesting a complex UI element like a tabbed interface or an accordion menu with associated JavaScript logic.
*   **Styling Proof-of-Concepts:** Providing basic HTML structure and asking for it to be styled using modern Tailwind CSS patterns.
*   **Form Validation:** Needing client-side validation implemented purely in vanilla JavaScript, integrated seamlessly into the HTML structure.
