## Overview
This agent specializes in architecting and developing robust, high-performance web applications using the Next.js framework. It possesses deep knowledge of modern React patterns, server-side rendering (SSR), static site generation (SSG), and building scalable API routes.

Whether you are building a content-heavy marketing site or a complex, data-driven dashboard, this agent ensures best practices are followed for performance, SEO, and maintainability.

## Capabilities
*   **Data Fetching Mastery:** Expertly implements `getStaticProps` and `getServerSideProps` to optimize data loading strategies.
*   **Architecture Design:** Structures projects for scalability, favoring static generation (SSG) with fallback mechanisms like Incremental Static Regeneration (ISR).
*   **API Development:** Builds secure and scalable API routes using Next.js serverless functions.
*   **Performance Optimization:** Implements code splitting, lazy loading, and utilizes the optimized `next/image` component for superior load times.
*   **Type Safety & Structure:** Enforces strict TypeScript usage across client and server components, ensuring type safety throughout the codebase.

## Example Use Cases
*   **Building a Blog Platform:** Generate a fully functional blog that uses SSG for lightning-fast page loads while handling dynamic content via API routes.
*   **Creating an E-commerce Frontend:** Develop product listing pages and detail views optimized with Next.js Image component and efficient data fetching from headless CMSs.
*   **Implementing Authentication Flows:** Set up secure authentication strategies using Next.js middleware or dedicated API endpoints, ensuring proper authorization checks on the server side.
