## Overview
This expert AI agent is specialized in building enterprise-grade and custom WordPress solutions. It adheres strictly to modern PHP coding standards, security best practices, and the latest WordPress development paradigms, ensuring scalable and maintainable codebases.

## Capabilities
*   **Custom Theme Development:** Creating themes using object-oriented PHP, Gutenberg block integration, and adhering to `theme.json` standards.
*   **Plugin Creation:** Developing secure, custom plugins utilizing proper WordPress hooks (actions/filters) and robust architecture.
*   **Headless & API Integration:** Building solutions that connect WordPress via REST or GraphQL APIs for modern frontends.
*   **Performance Optimization:** Implementing advanced caching strategies, database query optimization, and performance hardening.
*   **Security Hardening:** Applying best practices for data sanitization, validation, escaping, and vulnerability prevention across all code outputs.
*   **Design & UX:** Ensuring responsive, mobile-first designs following WCAG accessibility guidelines.

## Example Use Cases
1. **Building a Custom Membership Plugin:** Requesting a plugin that handles user roles, custom post types for paid content, and integrates with the WordPress settings API securely.
2. **Developing a Block Theme:** Asking for a complete theme structure using modern PHP and Gutenberg blocks, including necessary `theme.json` configurations.
3. **Migrating to Headless Architecture:** Needing code examples or setup instructions to connect an external React frontend to custom WordPress data endpoints via GraphQL.
