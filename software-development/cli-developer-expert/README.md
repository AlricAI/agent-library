## Overview
This agent acts as a senior CLI developer specializing in crafting best-in-class command-line interfaces and developer utilities. Its expertise covers the entire lifecycle of building terminal tools, from initial workflow analysis to final deployment considerations.

The core focus is on achieving superior Developer Experience (DX), ensuring that the resulting tool is fast, reliable, intuitive, and integrates seamlessly into existing developer workflows.

## Capabilities
*   **Architecture Design:** Planning command hierarchies, subcommand organization, flag/option design, and plugin architectures.
*   **Advanced Argument Parsing:** Implementing complex positional arguments, optional flags, type coercion, and robust validation rules.
*   **Interactive Prompting:** Building sophisticated user interactions including multi-select lists, confirmation dialogs, progress indicators, and autocomplete support.
*   **Performance & Reliability:** Ensuring strict performance benchmarks (e.g., startup time < 50ms) and comprehensive cross-platform compatibility.
*   **Error Handling:** Implementing graceful failure modes with helpful messages, clear error codes, and built-in debugging aids.

## Example Use Cases
*   **Building a Project Scaffolder:** Designing a CLI that scaffolds new projects by prompting the user for necessary details (e.g., framework choice, name) while validating inputs against predefined schemas.
*   **Creating a Workflow Orchestrator:** Developing a tool that manages multi-step processes, showing clear progress bars and handling intermediate failures with recovery suggestions.
*   **Developing Internal Developer Tools:** Architecting complex internal utilities that require reading configuration from multiple sources (environment variables, config files, CLI flags) while maintaining high performance.
