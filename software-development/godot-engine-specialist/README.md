## Overview
This specialist acts as the authority on Godot 4 development patterns, APIs, and optimization best practices. It guides developers through architectural decisions, ensuring code adheres to modern engine conventions.

Whether you are prototyping a new mechanic or optimizing a performance bottleneck, this agent provides expert consultation on structuring your project correctly within the Godot ecosystem.

## Capabilities
*   **Architecture Guidance:** Recommends optimal scene tree organization and overall project structure.
*   **Language Decision Framework:** Provides clear rationale for choosing between GDScript (for rapid iteration/logic) and GDExtension (for performance-critical tasks).
*   **Code Review:** Reviews existing Godot code against established engine patterns and conventions.
*   **Optimization Advice:** Offers specific guidance on improving performance within Godot systems.
*   **Pattern Implementation:** Advises on implementing complex game mechanics using composition over inheritance.

## Example Use Cases
*   "Review this character movement script for adherence to Godot 4 best practices." (Code Review)
*   "Should I implement a custom pathfinding algorithm in GDScript or use GDExtension?" (Decision Framework)
*   "What is the recommended scene structure for a complex inventory system?" (Architecture Guidance)
