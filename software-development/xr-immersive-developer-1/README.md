## Overview
This agent embodies the expertise of a senior WebXR engineer, capable of building robust, performant, and cross-platform immersive 3D applications directly in the browser. It specializes in bridging cutting-edge spatial computing APIs with practical, production-ready code.

## Capabilities
*   **WebXR Implementation:** Scaffolding projects utilizing core WebXR Device APIs for full immersion support (hand tracking, gaze, controllers).
*   **3D Engine Proficiency:** Expertise across major frameworks including Three.js, A-Frame, and Babylon.js.
*   **Interaction Design:** Implementing complex spatial interactions such as raycasting, hit testing, and physics integration within virtual environments.
*   **Performance Optimization:** Applying advanced techniques like occlusion culling, shader tuning, and Level of Detail (LOD) management to ensure smooth frame rates across diverse hardware.
*   **Compatibility Layering:** Designing experiences with built-in fallback mechanisms to gracefully degrade functionality when targeting non-XR or limited-capability devices.

## Example Use Cases
*   **Developing a Virtual Training Module:** Request code structure for an interactive, multi-stage VR simulation that requires precise hand tracking inputs and physics collision detection.
*   **Debugging Cross-Platform Input:** Provide a scenario where input fails on mobile AR versus desktop WebXR, and ask the agent to diagnose potential compatibility layers or API misuses.
*   **Building an Immersive Data Visualization:** Ask for a boilerplate setup using Three.js that visualizes complex datasets in 3D space, ensuring performance optimization is prioritized from the start.
