## Overview
This collection serves as a master index and summary for 85 highly specialized Claude Code sub-agents. It meticulously organizes agents across core software development pillars, ensuring comprehensive coverage from initial coding to deployment and security auditing.

The structure is designed for maximum utility, grouping experts into distinct functional areas so you can quickly locate the precise agent needed for any stage of the software lifecycle.

## Capabilities
* **Comprehensive Indexing:** Provides a categorized breakdown of all 85 available agents.
* **Domain Specialization:** Agents are segmented into Development & Programming, Testing & QA, DevOps & Infrastructure, Security & Compliance, and more.
* **Detailed Documentation:** Each sub-agent includes detailed system prompts defining its expertise and methodology.
* **Tool Configuration Guidance:** Outlines necessary tool configurations and security considerations for optimal agent performance.

## Example Use Cases
* **Full Stack Development Cycle:** Initiate a project by using the 'Development & Programming' agents, then transition to 'DevOps & Infrastructure' for CI/CD setup, followed by 'Security & Compliance' for pre-launch audits.
* **Quality Assurance Deep Dive:** When testing complex features, leverage the dedicated 'Testing & Quality Assurance' agents to build multi-layered test plans automatically.
* **Data Pipeline Implementation:** For ML projects, utilize the specialized agents within 'Data & Analytics' to manage data ingestion, model training, and deployment pipelines efficiently.
