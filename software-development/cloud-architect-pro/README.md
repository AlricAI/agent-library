## Overview
Cloud Architect Pro is a senior-level AI solutions architect specializing in designing robust, resilient, and cost-optimized infrastructure across major cloud providers (AWS, Azure, GCP). It translates high-level business requirements into actionable, production-ready architecture blueprints, with a strong emphasis on Infrastructure as Code (IaC) using Terraform.

## Capabilities
* **Multi-Cloud Architecture:** Designs scalable, resilient architectures supporting multi-region and hybrid cloud deployments across AWS, Azure, and GCP.
* **FinOps & Cost Optimization:** Implements FinOps best practices by recommending resource right-sizing, savings plan strategies, and cost governance controls to minimize cloud spend.
* **Infrastructure as Code (IaC):** Develops clean, modular, and testable infrastructure definitions using Terraform.
* **Security Architecture:** Designs security layers incorporating Zero-Trust models, comprehensive IAM policies, network segmentation, and data encryption at rest and in transit.
* **Migration & Automation:** Provides detailed cloud migration roadmaps, addresses vendor lock-in risks, and integrates architecture into CI/CD pipelines for operational excellence.

## Example Use Cases
1. **New Service Deployment:** Provide a use case (e.g., 'A global e-commerce site needing low latency') and receive a multi-region, serverless architecture plan with associated Terraform modules.
2. **Cost Reduction Audit:** Input existing infrastructure details or cost reports and ask the agent to identify immediate optimization opportunities based on FinOps principles.
3. **Cloud Migration Strategy:** Outline an on-premises system and request a phased migration strategy detailing necessary networking changes, service mappings, and required IaC adjustments for AWS adoption.
