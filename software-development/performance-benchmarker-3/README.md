## Overview
Performance Benchmarker is an expert AI agent dedicated to ensuring that all applications and infrastructure meet rigorous performance Service Level Agreements (SLAs). It operates with a data-driven, optimization-obsessed mindset, treating performance excellence as a core requirement for user experience.

This agent moves beyond simple testing; it establishes comprehensive baselines, predicts bottlenecks, and recommends actionable, measurable improvements across the stack.

## Capabilities
*   **Comprehensive Load Testing:** Executes load, stress, endurance, and scalability assessments to find breaking points.
*   **Web Performance Optimization:** Focuses on Core Web Vitals (LCP < 2.5s, FID < 100ms, CLS < 0.1) using advanced frontend techniques like code splitting and lazy loading.
*   **Bottleneck Identification:** Systematically analyzes performance data to pinpoint resource constraints in backend services or infrastructure.
*   **Capacity Planning:** Forecasts future resource needs based on projected usage growth patterns.
*   **Monitoring Strategy:** Designs monitoring systems with predictive alerting for real-time operational tracking.

## Example Use Cases
1. **Pre-Launch Stress Test:** Run a simulated peak traffic load test against a new e-commerce checkout flow to ensure it handles 5x expected daily volume without degrading response time.
2. **Frontend Audit:** Analyze the current landing page's performance metrics, providing specific code recommendations (e.g., asset bundling, image optimization) to improve LCP and CLS scores.
3. **Scalability Assessment:** Given a projected 30% user growth over the next quarter, model the required database read/write capacity and recommend necessary horizontal scaling adjustments.
