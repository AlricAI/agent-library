## Overview
Sql Pro is an expert AI specialist dedicated to mastering modern SQL dialects and high-performance database architecture. It excels in optimizing complex queries, designing robust data models for cloud-native environments, and handling the intricacies of hybrid OLTP/OLAP systems.

This agent acts as a senior Database Architect, ensuring your data solutions are not only functional but also scalable, efficient, and performant under enterprise load.

## Capabilities
*   **Advanced Querying:** Writes complex SQL utilizing window functions, recursive CTEs, and advanced JOIN techniques for deep analytical insights.
*   **Performance Tuning:** Analyzes query execution plans to identify bottlenecks, suggesting optimal indexing strategies, partitioning schemes, and resource tuning adjustments.
*   **Multi-Database Expertise:** Proficient across major platforms including Snowflake, BigQuery, Aurora, Redshift, and specialized systems like Neo4j (Graph) and TimescaleDB (Time-Series).
*   **Data Modeling:** Assists in designing normalized or denormalized schemas suitable for specific workloads (OLTP vs. OLAP).
*   **Cloud Architecture:** Provides guidance on multi-region deployment, replication strategies, and best practices for cloud database services.

## Example Use Cases
*   **Performance Audit:** Provide a slow query and the schema; Sql Pro will return an optimized version with detailed explanations of the performance improvements.
*   **Schema Design:** Describe your application's data requirements (e.g., 'I need to track user activity over time, including relationships between users and devices'); Sql Pro will generate a recommended, normalized database schema.
*   **Complex Reporting:** Request an analytical report that requires joining transactional data with historical metrics; Sql Pro will construct the necessary multi-stage CTE query.
