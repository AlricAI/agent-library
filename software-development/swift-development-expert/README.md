## Overview
This agent embodies the expertise of a senior Swift developer, specializing in modern Apple development using Swift 5.9+ and SwiftUI. It is designed to guide you through building highly robust, type-safe, and performant applications across iOS, macOS, and server-side targets.

Its core focus areas include mastering structured concurrency with `async/await`, implementing complex logic using Protocol-Oriented Programming (POP), and ensuring adherence to the latest Swift API design guidelines.

## Capabilities
*   **Modern Concurrency:** Implements solutions using Actors, Structured Concurrency, and `AsyncSequence` while rigorously preventing race conditions.
*   **SwiftUI Mastery:** Creates declarative views with advanced state management, custom modifiers, and performance optimizations.
*   **Protocol-Oriented Design (POP):** Architectures codebases using protocol composition, associated types, and type erasure for maximum flexibility and testability.
*   **Code Quality Enforcement:** Adheres to strict standards including SwiftLint compliance, high test coverage (>80%), and comprehensive API documentation.
*   **Platform Analysis:** Can analyze existing project structures, `Package.swift`, and dependency configurations to ensure seamless integration of new features.

## Example Use Cases
*   **Building a Complex Data Flow:** Need to create an MVVM-C architecture for a multi-screen app? Ask it to structure the ViewModels using Actors for state isolation and implement data fetching with `AsyncStream`.
*   **Implementing Custom UI Components:** Requires a reusable, highly performant custom chart view in SwiftUI? Use this agent to define the necessary protocols and build the declarative composition.
*   **Refactoring Legacy Code:** Have an older codebase struggling with thread safety or memory leaks? Provide the files, and it will refactor them using modern concurrency patterns and ARC best practices.
