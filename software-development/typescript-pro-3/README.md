## Overview
TypeScript Pro is an elite AI agent designed for developers tackling the most complex aspects of TypeScript development. It operates as a senior architect, ensuring that all generated code adheres to best practices for type safety, maintainability, and scalability in enterprise environments.

This agent specializes not just in writing TypeScript, but in *thinking* in TypeScript—understanding compiler constraints, optimizing inference, and implementing advanced patterns like mapped types and conditional logic.

## Capabilities
*   **Advanced Type System Implementation:** Generates complex utility types, conditional types, and constrained generics for maximum type safety.
*   **Architecture Design:** Structures large codebases using robust interfaces, abstract classes, and well-defined module boundaries.
*   **Type Optimization:** Analyzes existing code to suggest improvements in type inference and build performance (e.g., optimizing `tsconfig.json`).
*   **Metadata & Decorators:** Implements decorators and metadata programming patterns common in frameworks like NestJS or TypeORM.
*   **Testing Integration:** Provides accompanying unit tests using Jest/Vitest, complete with proper type assertions to validate complex logic.

## Example Use Cases
1. **Building a Domain Model:** Need to define a highly polymorphic data structure that must enforce specific relationships between several interconnected entities? Ask for it with advanced generics and union types.
2. **Framework Integration:** Developing a service layer in Node.js/Express that requires strict input validation and guaranteed return types across multiple endpoints?
3. **Utility Library Creation:** Creating a generic utility function that needs to operate safely on various object shapes while maintaining compile-time guarantees? TypeScript Pro will provide the necessary type guards and constraints.
