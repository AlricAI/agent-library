## Overview
This agent acts as a senior-level React specialist, focusing on implementing best practices for modern web development using React 18 and beyond. It is designed to elevate component architecture from functional prototypes to production-grade, highly optimized, and maintainable systems.

## Capabilities
*   **Advanced Architecture:** Implements complex patterns like Compound Components, Render Props, and Context optimization.
*   **Performance Tuning:** Proficient in optimizing rendering cycles using `React.memo`, `useMemo`, `useCallback`, code splitting, and selective hydration.
*   **State Management Mastery:** Can architect solutions using modern libraries such as Zustand, Redux Toolkit, Jotai, or leverage server state management via React Query.
*   **Full Stack Integration:** Deep knowledge of Next.js/Remix patterns, including Server Components, Streaming SSR, and robust data fetching strategies.
*   **Quality Assurance Focus:** Adheres to best practices by ensuring high test coverage (RTL/Jest) and consistent accessibility compliance.

## Example Use Cases
1. **Building a Dashboard:** Request the agent to scaffold a complex dashboard component that requires real-time data fetching, virtual scrolling for large datasets, and optimized state handling across multiple widgets.
2. **Refactoring Legacy Code:** Provide an existing component structure and ask the agent to refactor it using functional components, custom hooks, and TypeScript strict mode to improve reusability and performance score.
3. **Implementing Complex UI:** Need a multi-step form? Ask for implementation using React Hook Form integrated with server actions in Next.js, ensuring robust validation and progressive enhancement.
