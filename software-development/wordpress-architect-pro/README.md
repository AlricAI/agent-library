## Overview
This agent embodies the knowledge of a senior WordPress architect with over 15 years of experience. It is designed to treat WordPress not merely as a CMS, but as a robust, scalable application framework capable of handling enterprise-level traffic and complexity.

It specializes in auditing existing sites against industry best practices (like achieving sub-1.5s load times and perfect security scores) and developing custom solutions using modern PHP, JavaScript frameworks (React/Vue), and advanced Gutenberg block development techniques.

## Capabilities
*   **Full-Stack Development:** Expertise in core WordPress functions, Custom Post Types, Taxonomies, and robust OOP plugin architecture.
*   **Performance Engineering:** Implements object caching (Redis), optimizes complex MySQL queries, manages transients, and ensures Core Web Vitals compliance.
*   **Modern Frontend Build:** Masters Block Theme implementation, Full Site Editing (FSE), and advanced Gutenberg block creation with Server-Side Rendering.
*   **Security Hardening:** Audits for vulnerabilities and enforces best practices to maintain high uptime and security scores.
*   **API Integration:** Builds secure REST API endpoints and handles complex background processing via queue management.

## Example Use Cases
1. **Performance Audit:** Provide a site's current metrics, and the agent will generate an actionable plan to reduce page load times below 1.5 seconds by optimizing database queries and implementing advanced caching layers.
2. **Custom Plugin Build:** Request a complex feature (e.g., multi-step checkout with custom logic), and the agent will architect the solution using OOP principles, namespaces, and secure AJAX handling.
3. **Theme Modernization:** If given an old theme structure, it can guide the migration process to a modern Block Theme framework, ensuring WCAG 2.1 accessibility compliance throughout.
