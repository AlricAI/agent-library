## Overview
Marshal is a specialized AI agent designed to act as a super-gremlin expert in product management and project coordination. Its core function is to keep complex development missions on track by ensuring documentation is unambiguous, roadmaps are prioritized, and communication is crystal clear.

It operates using a structured heartbeat protocol, loading its full persona and learnings from Letta Memory at the start of every session to maintain continuity.

## Capabilities
*   **Product Requirement Document (PRD) Generation:** Writes detailed feature specifications that leave no room for ambiguity during implementation.
*   **Roadmap Management:** Organizes and prioritizes features using industry-standard frameworks like RICE or MoSCoW.
*   **Sprint Planning:** Leads the process of defining clear, achievable sprint goals for development teams.
*   **Stakeholder Communication:** Drafts honest, highly scannable updates tailored for various stakeholders.
*   **Progress Tracking:** Monitors success metrics and proactively flags any project goals that are at risk of slipping.

## Example Use Cases
*   **New Feature Launch:** Provide a high-level concept, and Marshal will generate a comprehensive PRD, including acceptance criteria and technical considerations.
*   **Quarterly Planning:** Feed it a list of desired features; Marshal will structure them into a prioritized roadmap using MoSCoW analysis.
*   **Sprint Kickoff:** Give it the product goal for the next cycle, and it will draft actionable sprint goals and assign preliminary tasks via its integrated task management skills.
