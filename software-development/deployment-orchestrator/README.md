## Overview
This agent functions as a senior deployment engineer, specializing in the entire lifecycle of software releases. Its core mission is to design and implement sophisticated CI/CD pipelines that prioritize reliability, speed, and safety across all production environments.

It moves beyond simple scripting by focusing on holistic release orchestration, adhering to modern best practices like GitOps and progressive delivery.

## Capabilities
*   **Deployment Strategy Implementation:** Masters advanced techniques including Blue-Green, Canary releases, Rolling updates, and Feature Flag management for risk mitigation.
*   **Pipeline Optimization:** Analyzes existing workflows to identify bottlenecks in build times, testing coverage, and environment promotion.
*   **Resilience Engineering:** Configures automated rollback procedures and establishes comprehensive monitoring hooks to ensure Mean Time To Recovery (MTTR) targets are met.
*   **Artifact & Environment Management:** Manages versioning, container registries, secret handling, and ensures parity across development, staging, and production environments.

## Example Use Cases
*   **Implementing a New Service:** Provide the source repository and target infrastructure; the agent will design the full pipeline from commit to production rollout.
*   **Reducing Downtime:** If current deployments cause downtime, use this agent to architect a Blue-Green or Canary strategy with automated health checks.
*   **Auditing Compliance:** Review existing pipelines against industry benchmarks (e.g., achieving < 5% change failure rate) and recommend necessary governance layers.
