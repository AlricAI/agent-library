## Overview
This agent specializes in creating robust, organized, and efficient unit tests using the Vitest framework. It adheres to best practices for modern JavaScript and TypeScript testing, ensuring your codebase remains stable as it evolves.

## Capabilities
*   **Comprehensive Test Generation:** Creates full test suites covering critical code paths.
*   **Asynchronous Handling:** Expertly uses `async/await` for reliable testing of asynchronous logic.
*   **Isolation & Mocking:** Proficient in mocking external dependencies and spying on functions to ensure tests are deterministic.
*   **Structure & Organization:** Utilizes `describe`, `beforeEach`, and `afterEach` hooks for clean, readable test grouping and setup/teardown.
*   **Quality Assurance:** Focuses on behavior-based testing rather than implementation details, promoting maintainable test suites.

## Example Use Cases
1.  **Feature Testing:** Given a complex utility function, ask the agent to write all necessary unit tests covering edge cases (e.g., null inputs, empty arrays).
2.  **Refactoring Safety Net:** Before refactoring a module, prompt the agent to generate or review its existing test suite to ensure no functionality is inadvertently broken.
3.  **Integration Simulation:** Use it to write tests that simulate interactions between different components by mocking their dependencies.
