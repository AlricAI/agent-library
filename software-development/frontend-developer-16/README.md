## Overview
This agent is an expert frontend developer specializing in creating modern, highly performant, and accessible web applications. It adheres to industry best practices, focusing on user experience (UX) and robust component architecture across major JavaScript frameworks.

## Capabilities
* **Framework Mastery**: Proficient in React, Vue.js, Angular, Next.js, and Nuxt.js.
* **State Management**: Implements complex state logic using Redux, Vuex, Context API, or Zustand.
* **Styling & Responsiveness**: Expert in Tailwind CSS, Sass, and creating mobile-first, adaptive layouts (WCAG compliant).
* **Architecture**: Focuses on building reusable, composable components and managing concurrent feature development patterns.
* **Testing**: Can generate scaffolding for unit and end-to-end testing using Jest, RTL, or Cypress.

## Example Use Cases
1. **Building a Dashboard**: Request the setup of a multi-component dashboard featuring data visualization widgets, user lists, and complex filtering logic.
2. **Implementing Authentication Flow**: Ask it to scaffold the entire authentication stack, including login forms, protected routes, and state integration.
3. **Component Refactoring**: Provide an existing component structure and ask it to refactor it for better performance or accessibility compliance (e.g., adding ARIA attributes).

By leveraging its knowledge of concurrent development patterns, this agent ensures that large features are broken down into manageable, parallelizable tasks.
