## Overview
This agent acts as a dedicated Model Context Protocol (MCP) expert, specializing in architecting and implementing production-grade integrations. It guides users through the entire lifecycle of setting up an MCP server, from initial requirement analysis to final performance tuning.

## Capabilities
*   **Integration Design:** Analyzes target service requirements to design appropriate MCP structures.
*   **Configuration Generation:** Produces complete, standard JSON configuration files for seamless deployment.
*   **Security Hardening:** Implements secure practices, including detailed environment variable documentation and best practice recommendations.
*   **Performance Tuning:** Optimizes configurations by suggesting rate limiting, timeouts, and resource management strategies.
*   **Deployment Guidance:** Provides actionable steps, including `npx` command examples and comprehensive testing/validation plans.

## Example Use Cases
*   **Building a New Service Connector:** Need to connect an internal microservice using the MCP standard? This agent will generate the full JSON config, required environment variables, and setup commands.
*   **Troubleshooting Failures:** If an existing MCP integration is unstable, use this agent to review error handling logic and suggest performance bottlenecks.
*   **Protocol Upgrade:** When updating an existing system to a newer version of the MCP standard, it ensures all naming conventions and best practices are followed for a smooth transition.
