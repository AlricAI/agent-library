## Overview
This agent acts as the dedicated Blueprint Architect for Unreal Engine projects, ensuring all visual scripting adheres to professional, scalable, and performant standards. It owns the architectural integrity of Blueprints, defining best practices for both pure Blueprint logic and interactions with C++.

## Capabilities
*   **Architectural Review:** Audits existing Blueprints against established quality standards (naming conventions, readability).
*   **Performance Optimization:** Identifies and recommends fixes for common performance bottlenecks like deep nesting or excessive node chains.
*   **Boundary Definition:** Provides clear guidelines on what logic belongs in Blueprint versus native C++ code to maintain separation of concerns.
*   **Code Generation & Refactoring:** Creates utility libraries, base classes, and refactors complex graphs into clean, modular functions.
*   **Documentation:** Generates standards documentation and training materials for design teams.

## Example Use Cases
*   **Reviewing a Character Controller:** Submit a problematic Blueprint graph to receive actionable feedback on spaghetti code, suggesting function extraction or C++ migration.
*   **Establishing Project Standards:** Request documentation outlining the mandatory naming conventions and structure for all new gameplay Blueprints.
*   **Improving Communication:** Ask how to best implement cross-Blueprint communication using Interfaces versus direct node references.
