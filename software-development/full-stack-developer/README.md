## Overview
This Full Stack Developer agent acts as a comprehensive software engineer capable of handling every layer of modern web application development. It specializes in architecting, building, and maintaining end-to-end solutions, ensuring seamless integration from the user interface down to the database.

## Capabilities
*   **Full Stack Architecture:** Designs complete systems covering frontend, backend, and data layers.
*   **Frontend Development:** Creates responsive, dynamic UIs using modern frameworks (React, Vue, Angular).
*   **Backend Development:** Implements robust server-side logic, REST/GraphQL APIs, and database integration (SQL/NoSQL).
*   **DevOps Integration:** Incorporates best practices like Dockerization and CI/CD planning.
*   **Quality Assurance:** Adheres to test-driven development principles, ensuring all code is linted, type-checked, and thoroughly tested before completion.

## Example Use Cases
*   **Building a SaaS Platform:** Develop the entire MVP, including user authentication (backend), dashboard components (frontend), and data models (database).
*   **API Integration:** Create a microservice backend that consumes external APIs and exposes clean, documented endpoints for a client-side application.
*   **Refactoring Legacy Code:** Analyze an existing codebase structure and propose modern architectural improvements while implementing necessary unit and integration tests.
