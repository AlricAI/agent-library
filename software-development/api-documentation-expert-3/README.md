## Overview
This agent acts as a senior API documenter, specializing in creating world-class, developer-friendly documentation. It ensures that your APIs are not just functional, but also exceptionally easy to understand and integrate for any consuming developer.

The core focus is on achieving OpenAPI 3.1 compliance while providing interactive elements that minimize the learning curve for new users.

## Capabilities
*   **Specification Generation:** Creates complete documentation adhering strictly to OpenAPI specifications (including schemas, endpoints, parameters, and security schemes).
*   **Interactive Portals:** Implements 'Try-it-out' functionality, request builders, and API explorers directly within the documentation.
*   **Comprehensive Coverage:** Ensures 100% endpoint coverage, detailed error handling guides, and clear authentication walkthroughs (OAuth 2.0, API Keys, etc.).
*   **Multi-Format Support:** Can document various protocols beyond standard REST, including GraphQL schemas, WebSockets, gRPC, and CLI usage.
*   **Code Example Generation:** Provides diverse, ready-to-use code examples across multiple languages for common use cases like pagination, filtering, and batch operations.

## Example Use Cases
*   **Onboarding New Users:** Feed it raw API endpoints and requirements; it will output a complete, interactive guide that new developers can immediately start testing against.
*   **API Version Updates:** Provide existing documentation and the changes; it will analyze gaps and update the OpenAPI spec to reflect versioning changes accurately.
*   **Improving Developer Experience (DX):** When users report confusion about error codes or authentication flows, use this agent to generate detailed resolution steps and best practice guides for those specific pain points.
