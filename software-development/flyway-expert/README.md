## Overview
This AI agent specializes in mastering Flyway for comprehensive database version control. It guides users through establishing robust, repeatable, and safe schema migration strategies, ensuring that database changes are managed systematically from development to production.

## Capabilities
*   **Script Generation:** Creates well-structured, idempotent, and reversible SQL migration scripts following best practices.
*   **Strategy Development:** Helps define clear database versioning strategies suitable for complex, large-scale applications.
*   **Automation & CI/CD:** Provides guidance and setup code for integrating Flyway into automated deployment pipelines.
*   **Validation & Testing:** Implements quality checks, including suggesting testing frameworks and pre-flight validation steps.
*   **Recovery Planning:** Develops detailed rollback procedures and recovery guides to mitigate risks during deployments.

## Example Use Cases
1. **New Feature Rollout:** Need to add a new table and modify an existing column across staging and production databases? This agent will generate the necessary, versioned scripts and the deployment plan.
2. **Database Upgrade:** Migrating from an old schema structure to a significantly updated one requires careful planning. Use this agent to map out the entire upgrade path with validation checkpoints.
3. **CI/CD Integration:** Integrating database changes into Jenkins or GitHub Actions? The agent will provide optimized configuration snippets and workflow steps for automated execution.
