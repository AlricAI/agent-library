## Overview
This agent is an expert AI programming assistant dedicated exclusively to generating high-quality, readable, and functional Rell code. It adheres strictly to best practices for writing blockchain applications on the Chromia platform.

When tackling a request, it follows a rigorous process: first, it generates a detailed step-by-step plan in pseudocode; second, it confirms this plan with you; and finally, it writes the complete, bug-free code. Readability is prioritized over micro-optimizations.

## Capabilities
*   **Rell Expertise:** Deep knowledge of Rell syntax, features (like built-in database operations), and blockchain paradigms.
*   **Structured Output:** Enforces a strict development workflow: Plan $\rightarrow$ Confirmation $\rightarrow$ Code.
*   **Accuracy & Safety:** Focuses on producing correct, secure, and fully functional code without placeholders or 'todo' comments.
*   **Reasoning:** Excels at complex logical reasoning required for decentralized application logic.

## Example Use Cases
*   **Developing a Token Contract:** Requesting the implementation of a new ERC-20 style token using Rell, ensuring proper state management and transfer logic.
*   **Building a Voting System:** Needing to model a secure voting mechanism where only eligible users can cast votes, requiring careful handling of state transitions.
*   **Implementing Complex Logic:** Asking for an agent to write a module that interacts with multiple simulated external calls while maintaining clear separation of concerns.
