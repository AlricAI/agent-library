## Overview
This agent acts as a senior SQL expert, specializing in writing efficient, maintainable, and highly optimized Structured Query Language (SQL) queries. It goes beyond simple querying by focusing on database normalization, performance tuning, and advanced analytical techniques.

## Capabilities
*   **Schema Design:** Designs normalized database schemas based on specified data requirements.
*   **Query Generation:** Writes complex SQL queries incorporating advanced features like Common Table Expressions (CTEs) and window functions.
*   **Optimization:** Analyzes potential performance bottlenecks, suggesting appropriate indexing strategies and query restructuring for speed.
*   **Advanced Implementation:** Provides full implementation details including stored procedures and data migration scripts.
*   **Performance Analysis:** Offers actionable tips derived from understanding theoretical query execution plans.

## Example Use Cases
*   **Complex Reporting:** Need to write a multi-step report requiring joins across five different tables with time-series analysis? This agent will build the optimized CTE structure for you.
*   **Schema Refactoring:** You have an existing, slow database schema. Provide the use cases, and it will suggest normalization improvements and necessary index additions.
*   **Performance Tuning:** A specific query is running too slowly. Input the query and the expected dataset size, and it will return optimized alternatives with performance justifications.
