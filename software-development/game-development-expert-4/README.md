## Overview
This agent acts as an expert game developer, guiding you through the entire lifecycle of multi-platform game creation. Its core philosophy is that technology must serve the experience, ensuring your focus remains on gameplay rather than technical trends.

Whether you are building a hyper-casual mobile title or a complex AAA VR experience, this agent helps select the optimal engine and architecture based on your goals.

## Capabilities
*   **Platform Selection:** Determines the best technology stack (Phaser, Godot, Unity, Unreal) based on target platforms (Web, Mobile, PC, VR/AR).
*   **Engine Comparison:** Provides detailed comparisons of major engines regarding 2D support, 3D quality, learning curve, and cost.
*   **Architectural Guidance:** Advises on core game loop implementation, multiplayer networking models, and performance optimization strategies (e.g., profiling before optimizing).
*   **Design Pattern Application:** Assists in implementing established game design patterns to ensure robust gameplay logic.

## Example Use Cases
*   **Project Scoping:** "I want to make a 2D platformer for both the web and mobile. Which engine should I use, and what are the initial steps?"
*   **Performance Review:** "My character movement feels laggy on low-end Android devices. Can you review my update loop structure for performance bottlenecks?"
*   **Technology Choice:** "We have a team of four with moderate experience, aiming for realistic 3D graphics on PC and console. Should we use Unity or Unreal?"

By following the principles of 'Gameplay First' and maintaining a focus on achieving consistent frame rates (60fps baseline), you can build professional-grade games efficiently.
