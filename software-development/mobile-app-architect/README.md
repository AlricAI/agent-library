## Overview
This agent acts as a senior mobile app developer, specializing in creating robust, high-performance applications for both native (iOS/Android) and cross-platform environments. It ensures the final product delights users while meeting strict technical benchmarks like low startup time and minimal battery drain.

## Capabilities
*   **Platform Mastery:** Proficient in Swift/SwiftUI, Kotlin/Jetpack Compose, React Native, and Flutter.
*   **Performance Optimization:** Implements strategies for reducing launch times, optimizing memory usage, and ensuring battery efficiency.
*   **Feature Implementation:** Handles complex requirements including offline capabilities (sync mechanisms, local storage), ARKit integration, and advanced UI/UX patterns.
*   **Compliance & Quality:** Ensures adherence to AAA accessibility standards and platform-specific guidelines (e.g., Material Design 3).

## Example Use Cases
*   **Building a complex e-commerce app:** The agent can architect the entire stack, from local product catalog caching for offline browsing to integrating payment gateways.
*   **Developing a fitness tracker:** It will optimize background processing using WorkManager/background tasks and ensure efficient battery usage while tracking data.
*   **Creating a cross-platform utility:** If you need an app running on both platforms with consistent UX, it will select the optimal framework (Flutter or RN) and implement platform-specific optimizations where necessary.
