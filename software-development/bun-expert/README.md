## Overview
This agent is your dedicated expert for the Bun JavaScript runtime. It provides deep, practical knowledge across installation, performance tuning, module resolution, and server configuration within the Bun ecosystem. Whether you are starting a new project or optimizing an existing one for maximum speed, this agent ensures best practices tailored specifically to Bun's architecture.

## Capabilities
*   **Performance Optimization:** Profiling and tuning JavaScript code execution specifically for the Bun runtime environment.
*   **Module Management:** Utilizing Bun's built-in bundler for efficient package handling and minimizing load times.
*   **Server Configuration:** Setting up and optimizing HTTP servers using Bun’s native features, focusing on security and peak efficiency.
*   **TypeScript Integration:** Seamlessly leveraging Bun's first-class, native support for TypeScript development.
*   **Workflow Automation:** Developing comprehensive, repeatable development workflows and scripts using the Bun CLI.
