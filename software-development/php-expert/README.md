## Overview
This agent is a specialized expert for crafting robust, modern, and secure applications using PHP 8 and newer features. It adheres strictly to industry best practices, focusing on clean architecture, performance optimization, and comprehensive security measures.

## Capabilities
*   **Modern PHP Implementation:** Leverages advanced features like match expressions and attributes for cleaner syntax.
*   **Architectural Design:** Employs established Design Patterns while maintaining a strong separation of concerns (SoC).
*   **Security Hardening:** Routinely validates inputs, handles sessions securely, and prevents common vulnerabilities like XSS and SQL injection using PDO.
*   **Code Quality Assurance:** Enforces PSR standards, generates comprehensive PHPDoc blocks, and integrates static analysis checks (PHPCS/PHPMD).
*   **Testing & Reliability:** Prioritizes unit testing with PHPUnit, aiming for high code coverage (>90%) and thorough error handling.

## Example Use Cases
1. **Building a REST API Endpoint:** Request the agent to build an endpoint that handles user authentication, ensuring secure session management and proper input validation.
2. **Refactoring Legacy Code:** Provide a block of old PHP code and ask it to refactor it using modern OOP principles, adding necessary PHPDoc blocks and unit tests.
3. **Database Integration:** Task the agent with creating a data access layer (DAL) for a specific database schema, ensuring all queries use prepared statements via PDO.
