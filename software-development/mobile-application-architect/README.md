## Overview
This AI agent emulates a senior mobile developer with deep expertise in creating high-performance, cross-platform applications. Its core mission is to maximize code reuse while ensuring the final product feels perfectly native on both iOS and Android.

It enforces rigorous development standards covering everything from cold start times and memory usage to advanced features like offline synchronization and biometric integration.

## Capabilities
*   **Performance Profiling:** Analyzes benchmarks for battery impact, cold start time (<1.5s), and frame rates (targeting 120 FPS).
*   **Cross-Platform Strategy:** Ensures over 80% code sharing while implementing platform-specific UI adhering to the latest OS guidelines (iOS 18+, Android 15+).
*   **Data Architecture:** Implements robust offline-first patterns using local databases, conflict resolution strategies, and delta synchronization.
*   **Native Integration:** Handles complex device interactions including BLE, biometrics (Face ID/Touch ID), GPS, and platform-specific APIs like HealthKit.
*   **Optimization Checklist Adherence:** Automatically verifies compliance with best practices such as efficient image caching (WebP/AVIF) and background task management.

## Example Use Cases
*   **Building a Field Service App:** Need to design an app that works reliably offline, syncs data when connectivity returns, and uses the camera/GPS extensively. 
*   **Performance Audit:** Reviewing an existing codebase to identify bottlenecks causing high battery drain or slow initial load times.
*   **Feature Implementation:** Integrating complex features like background location tracking with robust error handling and retry logic.
