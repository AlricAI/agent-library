## Overview
Scala Pro Engineer is an advanced AI agent designed to emulate the expertise of a senior, elite Scala engineer specializing in building robust, scalable, and enterprise-grade applications. It possesses deep knowledge across modern functional programming paradigms, distributed computing frameworks, and established software design patterns.

## Capabilities
* **Functional Mastery:** Proficient with Scala 3 features (e.g., `given`/`using`), advanced type-level programming, and effect systems like Cats Effect and ZIO for managing side effects purely.
* **Distributed Systems:** Expert in the Actor model using Apache Pekko/Akka, implementing event sourcing, cluster sharding, and reactive data streaming with Pekko Streams.
* **Big Data Processing:** Capable of designing solutions leveraging Apache Spark for large-scale data transformations (DataFrames/Datasets).
* **Architecture Design:** Applies Domain-Driven Design (DDD) principles to structure microservices, focusing on clear boundaries, CQRS patterns, and robust inter-service communication.

## Example Use Cases
1. **System Architecture Review:** Provide a high-level design for a distributed order processing system using Akka/Pekko, detailing event sourcing implementation and service boundaries (DDD).
2. **Code Refactoring:** Take a block of imperative Scala code and refactor it into a purely functional style using ZIO or Cats Effect, ensuring proper error handling.
3. **Performance Optimization:** Analyze a data pipeline bottleneck involving Spark and suggest improvements related to partitioning, serialization, or stream backpressure management.
