## Overview
This agent acts as an expert code reviewer and formatter, specializing in modern JavaScript/TypeScript frameworks. It enforces strict coding conventions across popular stacks including Next.js App Router, React, Node.js, and utility libraries like tRPC, Shadcn UI, and Tailwind CSS.

Its primary goal is to elevate code quality by ensuring adherence to established patterns for naming, structure, error handling, and performance optimization.

## Capabilities
*   **TypeScript Expertise:** Enforces strict typing and best practices within TypeScript codebases.
*   **Framework Adherence:** Provides specific guidance for Next.js (App Router) and Expo environments regarding data fetching and rendering lifecycles.
*   **UI/Styling Consistency:** Validates component structure using React, Radix UI, and Tailwind CSS conventions.
*   **Structural Review:** Checks naming conventions, error handling patterns, and overall code architecture for maintainability.
*   **Optimization Suggestions:** Identifies potential performance bottlenecks related to rendering or data fetching.

## Example Use Cases
*   **Code Refactoring:** Paste a module and ask the agent to refactor it to comply with Next.js Server Component best practices.
*   **Convention Check:** Submit a set of components and request an audit against established React/TypeScript naming conventions.
*   **Error Handling Review:** Provide a data fetching hook and ask for validation on robust error handling mechanisms using tRPC patterns.
