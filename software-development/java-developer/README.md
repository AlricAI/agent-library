## Overview
This agent acts as a senior Java developer specializing in modern Java features and robust enterprise architecture. It ensures that all generated code adheres to the latest best practices, focusing on readability, performance, and maintainability.

When tackling a project, it systematically analyzes existing structures, determines necessary dependencies (e.g., Spring Boot versions), and implements solutions using contemporary Java paradigms.

## Capabilities
*   **Modern Java Implementation:** Utilizes features like Records, sealed classes, pattern matching, streams, and functional programming patterns.
*   **Concurrency Handling:** Implements thread-safe concurrent logic, including structured concurrency and `CompletableFuture` usage.
*   **Framework Expertise:** Proficient in Spring Boot, dependency injection, and building reactive components.
*   **Quality Assurance:** Provides comprehensive JUnit 5 tests (including parameterized tests) and can generate JMH benchmarks for performance validation.
*   **Code Quality:** Ensures proper Javadoc generation, robust exception handling (`try-with-resources`), and adherence to strict Java coding standards.

## Example Use Cases
*   **Building a Microservice:** Need to scaffold a new RESTful microservice using Spring Boot 3+ with reactive endpoints? This agent will provide the necessary configuration and service layers.
*   **Performance Tuning:** Identifying bottlenecks in a data processing pipeline that requires advanced stream collectors or virtual thread utilization for optimization.
*   **Complex Data Transformation:** Implementing a multi-step ETL process where data must be processed concurrently while maintaining transactional integrity.
