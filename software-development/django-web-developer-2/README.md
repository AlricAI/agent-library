## Overview
This agent embodies the expertise of a senior Django developer specializing in modern Python web development using Django 4+. It adheres to best practices for building secure, scalable, and high-performance applications following the Model-View-Template (MVT) pattern.

## Capabilities
*   **Architecture Design:** Assesses project requirements, reviews application structure, and advises on optimal database design and scalability patterns.
*   **API Development:** Masters Django REST Framework for building robust APIs, covering serializers, viewsets, authentication, and throttling.
*   **Modern Python & Async:** Implements solutions using Python 3.11+ syntax, fully leveraging async views and ASGI capabilities where performance gains are necessary.
*   **Optimization & Security:** Focuses rigorously on ORM optimization (select_related, prefetch), implementing comprehensive security hardening (CSRF, XSS prevention), and ensuring high test coverage (>90%).
*   **Development Lifecycle:** Provides guidance across the entire stack, from initial setup (settings management, middleware) to deployment readiness.

## Example Use Cases
*   **Building a Scalable Backend:** Need to implement a complex e-commerce platform API? This agent will structure the models, define optimized serializers, and set up necessary permissions.
*   **Migrating to Async:** If your application involves heavy I/O (e.g., external API calls or WebSockets), use this agent to refactor views into async functions for maximum throughput.
*   **Code Review & Hardening:** Submit existing Django code for a security and performance audit, receiving actionable advice on ORM inefficiencies, missing middleware, or potential vulnerabilities.
