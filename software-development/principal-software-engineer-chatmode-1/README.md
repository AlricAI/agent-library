## Overview
This agent emulates the role of a Principal Software Engineer, offering expert technical guidance that balances theoretical engineering excellence with pragmatic, deliverable solutions. It adopts a thoughtful, senior perspective akin to renowned software thought leaders.

## Capabilities
*   **Design Pattern Application**: Provides advice on established patterns (e.g., Gang of Four) while adhering to SOLID principles and KISS/YAGNI.
*   **Code Quality Assessment**: Reviews code for readability, maintainability, and adherence to clean code practices.
*   **Testing Strategy**: Develops comprehensive testing plans, covering unit, integration, and end-to-end levels.
*   **Technical Debt Management**: Identifies technical debt, documents its consequences, and proactively suggests remediation via issue tracking.
*   **Risk & Requirements Analysis**: Critically reviews requirements to document assumptions, identify edge cases, and assess architectural risks.

## Example Use Cases
1. **Code Review**: Paste a function or class and ask for a review focusing on SOLID compliance and testability.
2. **Design Consultation**: Present a high-level feature requirement and ask the agent to propose an architecture, detailing trade-offs between scalability and implementation time.
3. **Refactoring Plan**: Provide a section of legacy code and request a phased plan to address technical debt while maintaining current functionality.
