## Overview
This agent is a specialized expert focused entirely on CockroachDB, providing deep knowledge across the entire lifecycle of a distributed SQL database. It assists with everything from initial cluster setup and schema design to advanced query optimization and multi-region deployment strategies.

## Capabilities
*   **Schema Design:** Creates scalable and resilient database schemas that avoid common pitfalls in distributed environments.
*   **Query Optimization:** Analyzes and rewrites complex queries to ensure efficient execution across the cluster, utilizing appropriate indexing.
*   **Cluster Management:** Provides guidance on deployment best practices, including multi-region setups and load balancing.
*   **Performance Tuning:** Offers recommendations for tuning CockroachDB's internal settings and parameters for optimal throughput.
*   **Resilience & Backup:** Develops comprehensive backup, restore, and high-availability procedures.

## Example Use Cases
*   **Schema Review:** "Review this application schema for scalability issues when deployed across three geographic regions." 
*   **Query Tuning:** "Optimize this slow transaction query to minimize latency in a CockroachDB cluster." 
*   **Deployment Plan:** "Generate a step-by-step deployment script for a highly available, multi-node CockroachDB setup."

By adhering to best practices and rigorous quality checks, this agent ensures your data layer is robust, performant, and ready for enterprise scale.
