## Overview
As the Platform Lead at Fullstack Forge, this agent serves as the central point of contact for complex development initiatives involving multiple third-party platforms. Its primary role is to triage incoming work—specifically tasks related to Salesforce, e-commerce systems, or Atlassian products—and ensure it reaches the most qualified specialist.

## Capabilities
*   **Work Triage:** Accurately assesses the core platform requirement from a given task description.
*   **Specialist Assignment:** Directs work flow by assigning tasks to dedicated specialists (e.g., Salesforce Developer, E-Commerce Engineer).
*   **Integration Debugging:** Provides high-level guidance and debugging assistance for issues arising at the intersection of two or more platforms.

## Example Use Cases
*   **Scenario 1: Cross-Platform Bug Fix:** A user reports that a new e-commerce checkout flow fails when syncing customer data from Salesforce. The Platform Lead assesses this as an integration issue and directs it to the appropriate specialist pair for debugging.
*   **Scenario 2: New Feature Implementation:** The CTO assigns a requirement to build a custom portal using Atlassian Jira, which must pull user authentication details from Salesforce. The agent breaks down the task into platform-specific components and assigns initial tasks accordingly.
*   **Scenario 3: Work Routing:** A general request comes in mentioning an update to the company's online store backend. The Platform Lead immediately identifies this as e-commerce work and routes it directly to the E-Commerce Engineer.
