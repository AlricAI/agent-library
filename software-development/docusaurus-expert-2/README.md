## Overview
This agent acts as a dedicated expert for Docusaurus documentation sites, providing deep knowledge across configuration, theming, and build processes. It is designed to help developers maintain consistency and resolve complex issues in modern Docusaurus v2/v3 setups.

## Capabilities
*   **Configuration Analysis:** Thoroughly examines `docusaurus.config.js` and `sidebars.js` for syntax errors or structural inconsistencies.
*   **Troubleshooting:** Provides step-by-step debugging guides for common build failures, referencing specific file paths relative to the project root.
*   **Theming & Styling:** Offers solutions for custom CSS integration, component customization (MDX), and theme maintenance.
*   **Dependency Management:** Checks `package.json` dependencies and validates necessary build scripts.
*   **Best Practices:** Recommends performance optimizations and deployment configurations for various hosting platforms.

## Example Use Cases
*   **Initial Setup:** Need to structure a new multi-site documentation portal using Docusaurus best practices.
*   **Build Failure:** The site fails to build with an error related to sidebar navigation or version compatibility; the agent will pinpoint the faulty configuration block.
*   **Theming Issues:** Custom components are not rendering correctly, requiring guidance on MDX integration and CSS scoping within the theme.
