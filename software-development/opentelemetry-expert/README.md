## Overview
This expert agent provides deep, actionable knowledge across the entire OpenTelemetry (OTel) stack. It guides users through best practices for instrumenting applications to generate high-quality, standardized telemetry data—including traces, metrics, and logs. Whether you are starting a new microservice or optimizing an existing observability pipeline, this agent ensures your system meets industry standards for monitoring.

## Capabilities
*   **Instrumentation Guidance:** Assisting with the correct setup of application code to emit OTel spans and attributes.
*   **Context Propagation:** Ensuring trace context is correctly passed across service boundaries (e.g., HTTP headers).
*   **Metrics & Tracing:** Implementing efficient collection, aggregation, and correlation between metrics and traces.
*   **Best Practices Auditing:** Reviewing existing setups against semantic conventions and performance overhead.
*   **Strategy Implementation:** Developing tailored sampling strategies and data retention policies for cost management.

## Example Use Cases
*   **New Service Onboarding:** Instrument a brand-new service to ensure all critical business paths are traceable from day one.
*   **Performance Bottleneck Identification:** Review existing traces to pinpoint latency hotspots by analyzing span duration and context flow across multiple services.
*   **Observability Gap Analysis:** Audit an application suite to identify areas lacking proper instrumentation or consistent tagging, ensuring comprehensive coverage for root cause analysis.
