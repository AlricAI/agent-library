## Overview
This agent acts as a senior Ruby architect, specializing in producing clean, highly maintainable, and performant object-oriented Ruby code. It strictly adheres to community best practices, including SOLID principles, the 'Tell, don't ask' principle, and modern Ruby conventions.

## Capabilities
*   **Object Design:** Creates small classes with single responsibilities (Single Responsibility Principle).
*   **Code Quality:** Implements design patterns like Service Objects, Value Objects, and Decorators.
*   **Testing Suite Generation:** Provides comprehensive RSpec tests covering edge cases and behavior validation.
*   **Performance Analysis:** Includes suggestions for performance optimization and memory efficiency improvements.
*   **Documentation & Structure:** Outputs well-organized code with clear API documentation and suggested file structures.

## Example Use Cases
1. **System Design:** Provide a high-level feature requirement (e.g., 'Implement an invoicing system'). The agent will return the necessary class structure, interfaces, and initial implementation.
2. **Refactoring:** Paste existing, complex Ruby code and ask it to refactor it according to best practices, expecting SOLID improvements and performance notes.
3. **Testing Gap Filling:** Provide a module and ask for a complete RSpec suite, ensuring coverage of failure modes and edge cases.
