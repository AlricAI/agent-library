## Overview
This agent functions as an expert API documentation specialist, dedicated to creating world-class developer experiences. It masters modern standards like OpenAPI 3.1+ and integrates AI tools to ensure the resulting documentation is not only accurate but also highly interactive and easy for developers to adopt.

Its primary goal is to reduce developer integration time by providing comprehensive, contract-driven documentation from design through deprecation.

## Capabilities
*   **Modern Standards Mastery:** Authoring specifications using OpenAPI 3.1+, AsyncAPI for event streams, and documenting GraphQL schemas.
*   **AI Content Generation:** Utilizing AI assistance to generate natural language explanations, code examples across multiple languages, and maintain content consistency.
*   **Interactive Portals:** Generating documentation compatible with industry standards like Swagger UI and Redoc, often deploying within custom developer portal frameworks (e.g., Docusaurus).
*   **Testing & Validation:** Includes capabilities for automated testing of documentation examples and ensuring payload validation via JSON Schema integration.
*   **Workflow Support:** Manages the entire API lifecycle documentation process, from initial design to final deprecation notices.

## Example Use Cases
1. **New API Onboarding:** Provide a set of endpoints and models; the agent will generate a complete, interactive developer portal draft including OpenAPI specs and usage guides.
2. **Updating Legacy Docs:** Feed in existing code comments or an older specification; the agent will modernize it to OpenAPI 3.1+ and suggest AI-powered improvements for clarity.
3. **Event API Documentation:** Documenting real-time event streams by generating both REST endpoints documentation and corresponding AsyncAPI specifications.
