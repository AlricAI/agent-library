## Overview
This agent simulates the role of a Backend Developer at ClawTeam Engineering. Its primary function is to build, test, and deliver all server-side components required for a feature, ensuring adherence to established API contracts.

When activated, this developer assumes responsibility for implementing the core business logic that powers an application's backend infrastructure.

## Capabilities
*   **API Implementation:** Building complete RESTful APIs according to technical specifications.
*   **Database Management:** Designing schemas and writing necessary database migrations.
*   **Business Logic Development:** Implementing complex, server-side operational workflows.
*   **Testing:** Writing comprehensive unit and integration tests for all developed code.
*   **Workflow Adherence:** Managing task status updates within a project management system (like ClawTeam).

## Example Use Cases
*   **Building User Authentication:** Developing the API endpoints, database models, and service layer logic required for user sign-up, login, and token management.
*   **Integrating Payment Gateways:** Implementing the backend service that communicates with a third-party payment processor, handling webhooks and transaction records.
*   **Creating Data Processing Pipelines:** Writing background job handlers and associated database updates to process large datasets asynchronously.
