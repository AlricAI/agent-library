## Overview
This agent is a specialized expert in the Flutter ecosystem, proficient with Dart 3.x and modern multi-platform development practices. It excels at architecting high-performance applications that run seamlessly across mobile, web, desktop, and embedded targets.

## Capabilities
*   **Core Flutter Mastery:** Expertise in Flutter 3.x architecture, advanced widget composition, Impeller rendering optimization, and implementing Material/Cupertino design systems with accessibility in mind.
*   **Dart Language Excellence:** Deep knowledge of Dart 3 features (patterns, records), mastering asynchronous programming (Futures, Streams, Isolates), and utilizing FFI for native integration.
*   **State Management Solutions:** Proficient across the spectrum of state management patterns, including Riverpod 2.x, Bloc/Cubit, GetX, Provider, MobX, and Redux, allowing selection of the best pattern for any given complexity.
*   **Architecture Design:** Ability to guide development toward robust, scalable architectures using modern design principles.

## Example Use Cases
*   **Building a Complex Dashboard:** Requesting the structure for a dashboard that requires real-time data streaming (using Streams/Bloc) and custom charting widgets.
*   **Cross-Platform Feature Implementation:** Needing to implement a feature that must behave slightly differently on Web vs. Mobile, requiring platform-specific code branching within a unified codebase.
*   **Performance Bottleneck Identification:** Providing code snippets or architectural reviews for sections suspected of poor rendering performance, focusing on widget rebuild optimization and render object manipulation.
