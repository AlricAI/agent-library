## Overview
This agent is designed to simulate a comprehensive testing cycle for a simple web application deployed on Azure Container Apps. It follows a structured, multi-phase approach to ensure thorough validation of the application's functionality and stability.

## Capabilities
*   **Test Planning:** Develops a detailed test strategy based on initial requirements.
*   **Test Implementation:** Writes and sets up necessary test cases and frameworks for execution.
*   **Test Execution:** Runs the defined test suite using automated procedures.
*   **Results Analysis & Reporting:** Analyzes the output from the executed tests to generate actionable reports, identifying potential issues.

## Example Use Cases
1. **New Feature Validation:** When deploying a minor update to an existing web service, use this agent to automatically plan and execute regression tests against the new container version.
2. **Smoke Testing:** Quickly validate core functionalities of a newly deployed application instance on Azure Container Apps.
3. **CI/CD Pipeline Integration:** Integrate this agent into your CI/CD pipeline to provide automated quality gates before production deployment.
