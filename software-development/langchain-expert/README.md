## Overview
This agent specializes in the end-to-end development of sophisticated, production-ready pipelines using LangChain. It acts as a senior software architect for LLM applications, focusing heavily on modularity, performance optimization, and robust error handling.

## Capabilities
*   **Complex Pipeline Construction:** Designs multi-step workflows by chaining together various LangChain components (loaders, parsers, chains).
*   **Data Ingestion Mastery:** Expertly handles document loading and parsing from diverse sources using specialized loaders.
*   **Optimization & Profiling:** Identifies bottlenecks within existing pipelines and suggests performance improvements for efficiency and scalability.
*   **Advanced Integration:** Seamlessly integrates custom components and utilizes advanced text embedding techniques within the chain structure.
*   **Best Practices Adherence:** Ensures all generated code follows LangChain's latest best practices, prioritizing readability and maintainability.

## Example Use Cases
1. **Building a Custom RAG System:** Need to ingest PDFs from multiple sources, chunk them optimally, embed them using a specific model, and build a retrieval-augmented generation chain for Q&A.
2. **Workflow Debugging:** You have an existing LangChain pipeline that fails intermittently; this agent can audit the error handling and pinpoint the unstable component.
3. **Data Transformation Pipeline:** Constructing a system that takes raw unstructured text, processes it through multiple filters (e.g., entity extraction $\rightarrow$ summarization $\rightarrow$ categorization), and outputs structured JSON.
