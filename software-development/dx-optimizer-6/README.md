## Overview
This agent acts as a dedicated Developer Experience (DX) specialist, designed to eliminate friction points in your software development lifecycle. Its goal is to make the entire process of building and maintaining code feel joyful and highly productive by optimizing tooling, streamlining setups, and automating tedious tasks.

When you feel setup is slow, or if team feedback indicates workflow bottlenecks, invoke this agent to diagnose and resolve systemic inefficiencies.

## Capabilities
*   **Workflow Profiling:** Analyzes existing development routines to pinpoint time sinks and pain points.
*   **Tooling Research & Implementation:** Researches and recommends best-in-class tools and configurations for your stack.
*   **Rapid Setup Optimization:** Can simplify complex environment setups, aiming for completion in under five minutes.
*   **Automation Generation:** Creates useful shortcuts, custom CLI commands (e.g., `.claude/commands`), and task runners (`Makefile`).
*   **Quality & Documentation:** Configures essential elements like Git hooks and improves `README` files with accurate setup guides.
*   **Performance Tuning:** Optimizes build times, test suites, and overall feedback loops.

## Example Use Cases
1. **New Project Onboarding:** "Optimize the initial setup for a new microservice written in Python and TypeScript." (Expect optimized `package.json` scripts and basic linting hooks).
2. **Team Bottleneck Resolution:** "Our CI/CD pipeline feels slow; analyze our current build process and suggest improvements."
3. **Toolchain Integration:** "We are adding GraphQL to an existing REST API. Guide me through the necessary tooling, client setup, and required aliases."
