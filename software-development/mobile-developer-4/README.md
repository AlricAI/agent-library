## Overview
This specialized agent acts as an expert mobile developer, focusing on building high-quality, cross-platform applications using leading frameworks like React Native or Flutter. It handles the complexities of modern mobile development, ensuring code reusability while maintaining a native feel across iOS and Android.

## Capabilities
*   **Cross-Platform Architecture:** Designs component structures prioritizing shared logic while accommodating platform-specific UI/UX needs.
*   **Offline Synchronization:** Implements robust offline-first data strategies for reliable user experiences regardless of connectivity.
*   **Native Integration:** Handles the implementation of platform-specific features, including push notifications and deep linking.
*   **Performance Optimization:** Focuses on battery efficiency, network optimization, and thorough performance profiling.
*   **Deployment Readiness:** Provides guidance and necessary code structures for both development builds and final App Store/Play Store submissions.

## Example Use Cases
*   **Building a To-Do List App:** Request the agent to build a cross-platform task manager that syncs data locally when offline and pushes notifications for due items.
*   **E-commerce Prototype:** Ask it to scaffold a product browsing screen with platform-specific image loading optimizations and deep linking capabilities for product pages.
*   **Complex Data Entry Form:** Use it to design a multi-step form requiring local state management and eventual synchronization with a backend service.
