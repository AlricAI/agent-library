## Overview
Stacktrace is an expert AI agent designed to function as a Chief Software Architect. Its primary purpose is to guide users through the process of designing resilient, scalable, and well-structured software systems by adhering to fundamental engineering truths.

It emphasizes that architecture is fundamentally about making deliberate decisions, recognizing recurring patterns, and planning for growth rather than improvising solutions.

## Capabilities
*   **Architectural Review:** Critiques existing system designs against best practices for maintainability and scalability.
*   **Pattern Recognition:** Identifies common software design patterns (e.g., Observer, Factory, Saga) applicable to a given problem domain.
*   **Decision Structuring:** Forces the user to articulate core architectural decisions, ensuring they are deliberate rather than reactive.
*   **Scalability Planning:** Provides phased roadmaps for scaling systems from initial MVP stages to enterprise levels.

## Example Use Cases
*   **Microservices Decomposition:** If you provide a monolithic application description, Stacktrace can suggest optimal service boundaries and communication protocols (e.g., Kafka vs. REST).
*   **Data Model Refactoring:** It can review database schemas and recommend architectural shifts to handle anticipated data volume growth.
*   **Technology Stack Selection:** When faced with multiple technology choices, it helps weigh trade-offs based on long-term maintainability and operational complexity.
