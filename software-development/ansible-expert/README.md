## Overview
This agent acts as an expert consultant for Ansible automation. It specializes in creating, optimizing, and validating infrastructure-as-code solutions using best practices to ensure reliable and idempotent deployments across diverse environments.

## Capabilities
*   **Playbook Generation:** Creates well-structured, modular, and reusable playbooks and roles adhering to industry standards.
*   **Configuration Management:** Handles complex state management for multiple platforms, ensuring consistent configurations.
*   **Security & Secrets:** Implements secure credential handling using Ansible Vault best practices.
*   **Dynamic Inventory:** Develops solutions utilizing dynamic inventory sources for flexible infrastructure scaling.
*   **Optimization & Best Practices:** Focuses on idempotency, error handling, performance tuning, and adherence to community guidelines.

## Example Use Cases
*   **Multi-Tier Deployment:** Orchestrate the deployment of a complex application stack (web, app, db) across staging and production environments using roles.
*   **System Hardening:** Write playbooks to enforce security baselines across a fleet of servers, including user management and service configuration.
*   **CI/CD Integration:** Develop Ansible tasks designed to integrate seamlessly into Jenkins or GitLab pipelines for automated testing and deployment.
*   **Template Rendering:** Generate dynamic configuration files using Jinja2 templates based on variable inputs.
