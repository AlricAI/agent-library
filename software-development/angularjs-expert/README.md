## Overview
This agent is a specialized expert in the AngularJS framework. It provides deep knowledge across the entire lifecycle of an AngularJS application, from initial structure design to performance tuning and comprehensive testing. Its primary goal is to produce highly maintainable, efficient, and robust codebases that adhere strictly to established AngularJS best practices.

## Capabilities
*   **Performance Optimization:** Identifies and resolves bottlenecks related to digest cycles, watchers, and rendering speed (e.g., suggesting `track by` in `ng-repeat`).
*   **Architectural Guidance:** Advises on structuring large applications using feature modules, services, and dependency injection.
*   **Component Development:** Creates reusable custom directives and components with isolated scopes for modularity.
*   **State Management:** Implements clean patterns for managing application state via controllers and services.
*   **Quality Assurance:** Ensures code quality by validating forms, implementing thorough testing scaffolds (Jasmine/Karma), and adhering to MVC separation of concerns.

## Example Use Cases
*   **Refactoring Legacy Code:** Provide an existing AngularJS module that is slow or poorly structured; the agent will refactor it for better performance and readability.
*   **Building New Features:** Request a new feature component, specifying data requirements; the agent will build it using best practices like lazy loading and proper service usage.
*   **Performance Auditing:** Submit a section of template code that is causing rendering issues; the agent will pinpoint the inefficient pattern (e.g., excessive watchers) and provide optimized alternatives.
