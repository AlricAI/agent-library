## Overview
The Project Planner agent is designed to take a high-level, ambiguous project goal and transform it into a structured, executable development plan. It excels at breaking down monolithic tasks into logical, sequential sub-tasks, ensuring that all necessary components are accounted for before coding begins.

This agent acts as your initial architect, providing the blueprint needed to guide development teams from concept to completion with minimal guesswork.

## Capabilities
*   **Task Decomposition:** Breaks large goals into granular, actionable steps.
*   **Dependency Mapping:** Identifies which tasks must be completed before others can start.
*   **Timeline Generation:** Suggests realistic timelines and sequencing for the entire project lifecycle.
*   **Milestone Definition:** Establishes clear checkpoints and measurable deliverables for progress tracking.

## Example Use Cases
*   **New Feature Implementation:** Provide a description of a new feature (e.g., "Add user profile picture upload to the dashboard"), and receive a step-by-step plan covering frontend, backend API changes, and database updates.
*   **Full Project Scoping:** Input a vague idea like "Build a small e-commerce site," and get a phased roadmap including setup, core functionality (cart/checkout), and deployment steps.
*   **Refactoring Large Modules:** If you need to overhaul an existing system component, this agent can map out the necessary refactoring stages while minimizing downtime risk.
