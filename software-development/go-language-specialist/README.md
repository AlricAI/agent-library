## Overview
As the dedicated Go Engineer for Fullstack Forge, this agent is your specialist resource for all things Go. It focuses on implementing idiomatic, high-performance, and robust backend services.

When you need to build anything involving concurrency, microservices architecture, or require peak performance in Go, activate this agent. It ensures the resulting code adheres to best practices, leveraging modern Go features like generics and interfaces.

## Capabilities
*   **Concurrency Implementation:** Designs and implements complex concurrent patterns using goroutines and channels for efficient resource management.
*   **Microservices Design:** Builds service boundaries using industry standards such as gRPC or RESTful APIs.
*   **Performance Optimization:** Includes profiling suggestions and optimizations, capable of utilizing tools like pprof where necessary.
*   **Code Quality Enforcement:** Writes idiomatic Go code, ensuring robust error handling and leveraging generics and interfaces for type safety.
*   **Testing Suite Generation:** Produces comprehensive test coverage, including table-driven tests and benchmarks.

## Example Use Cases
*   **Building a Real-time Data Stream Processor:** Requesting a service that ingests data concurrently from multiple sources using buffered channels to manage backpressure.
*   **Designing a Service Mesh Component:** Needing a boilerplate microservice skeleton communicating between services via gRPC endpoints.
*   **Optimizing Latency Hotspots:** Providing existing Go code and asking for performance bottlenecks to be identified and optimized with profiling insights.
