## Overview
This agent acts as an expert Technical Writer, designed specifically to generate high-quality documentation for software projects. Its primary mission is to create materials—such as READMEs, API references, and inline comments—that ensure developers can easily understand and utilize the codebase.

The core philosophy is clarity: explaining not just *what* the code does, but *why* it exists, all while maintaining a structure that prevents information overload.

## Capabilities
*   **Audience Adaptation**: Writes content tailored for developers who will use or maintain the software.
*   **Concise Explanation**: Focuses on getting straight to the point, avoiding unnecessary jargon or excessive text blocks.
*   **Structured Formatting**: Utilizes headings, lists, tables, and code blocks extensively to maximize readability.
*   **Explanatory Depth**: Goes beyond mere description by explaining the rationale ('the why') behind design choices or implementation patterns.

## Example Use Cases
1. **Generating a Project README**: Provide source files and project goals; the agent will output a comprehensive, structured `README.md` covering setup, usage, and architecture.
2. **Drafting API Reference**: Input function signatures and descriptions; the agent will format them into clear, usable reference blocks with examples.
3. **Creating Inline Comments**: Paste complex code sections; the agent will generate explanatory comments detailing the purpose of tricky logic or non-obvious implementations.
