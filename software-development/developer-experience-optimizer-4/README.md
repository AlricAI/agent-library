## Overview
The Developer Experience (DX) Optimizer acts as a dedicated specialist to analyze, refine, and streamline your entire software development lifecycle. Its core mission is to make the process of writing code as joyful and productive as possible by eliminating friction points.

This agent doesn't just suggest fixes; it implements tangible improvements across your project structure, tooling, and local environment so that great DX becomes invisible when everything works correctly.

## Capabilities
*   **Workflow Profiling:** Analyzes existing development routines to pinpoint time sinks and bottlenecks.
*   **Tooling Research & Integration:** Researches modern best practices and integrates valuable, relevant developer tools.
*   **Setup Simplification:** Can reduce complex environment setups to under five minutes with clear instructions.
*   **Automation Implementation:** Creates useful shortcuts, custom CLI commands (`.claude/commands`), and automated task runners (e.g., `Makefile`).
*   **Quality & Documentation:** Configures Git hooks for mandatory quality checks and generates comprehensive setup guides.
*   **Optimization Focus:** Targets improvements in build times, test feedback loops, and overall developer satisfaction metrics.

## Example Use Cases
*   **New Project Onboarding:** When starting a greenfield project, use this agent to generate the complete initial scaffolding, including IDE settings, basic scripts, and README documentation.
*   **Team Feedback Loop:** After receiving critical feedback like, "Setting up local services takes too long," invoke the agent to overhaul the setup process.
*   **Performance Bottleneck Identification:** If build times are slow or developers report repetitive manual steps, use this agent to implement task automation and optimized scripts.
