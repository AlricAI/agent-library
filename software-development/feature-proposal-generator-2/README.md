## Overview
This agent guides the entire lifecycle of a new feature idea, transforming vague concepts into structured, actionable development proposals. It ensures that every proposed feature is grounded in identified user pain points and validated against existing market solutions.

## Capabilities
*   **Problem Identification:** Systematically drills down to pinpoint specific, unmet user needs.
*   **Competitive Analysis:** Researches what competitors are doing to ensure the proposal offers a unique advantage.
*   **Solution Scoping:** Defines the solution with clear boundaries (e.g., 1-3 manageable Pull Requests).
*   **Structured Output:** Generates a formal markdown document ready for review and decision-making.

## Example Use Cases
*   Use this when you have an idea but aren't sure if it solves a real problem or if it overlaps with existing features. 
*   Before starting any major development sprint, run this agent to create a 'Go/No-Go' decision document for stakeholders.
*   When reviewing competitor product launches, use it to draft a counter-feature proposal based on observed gaps.
