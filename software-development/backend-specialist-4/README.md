## Overview
This agent embodies the expertise of a Senior Principal Backend Engineer and Systems Architect. Its core function is to move beyond mere functionality, focusing instead on building systems that are resilient, idempotent, scalable, and maintainable. It enforces best practices across API design, database interaction, and overall system architecture.

## Capabilities
*   **Deep Requirement Analysis:** Forces mandatory deep thinking before proposing any architectural solution.
*   **API Decision Matrix:** Guides the selection of appropriate communication patterns and standards for APIs.
*   **Scale-Aware Strategy:** Adjusts technical rigor based on whether the project is an MVP, growing, or enterprise-scale.
*   **Infrastructure Checkpoints:** Ensures critical components (security, performance) are addressed upfront.
*   **Root Cause Analysis (RCA):** Provides structured troubleshooting methodologies for debugging complex backend failures.

## Example Use Cases
*   **Designing a New Service:** Ask it to architect a new payment processing microservice, ensuring adherence to defined API standards and security rules.
*   **Performance Review:** Provide existing code or architecture diagrams and ask it to profile potential bottlenecks using its performance mindset.
*   **Refactoring Strategy:** Use it to review an existing monolithic structure and generate a phased migration plan toward scalable, decoupled services.
