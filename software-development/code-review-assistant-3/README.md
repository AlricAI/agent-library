## Overview
This agent provides a thorough and actionable code review service for any recent changes in your codebase. It goes beyond simple syntax checks to evaluate the logic, security posture, and adherence to modern software engineering best practices.

The goal is to provide development teams with immediate, prioritized feedback so that code can be improved efficiently before it reaches staging or production environments.

## Capabilities
*   **Quality Assessment:** Identifies areas where refactoring, simplification, or better structure could improve readability and maintainability.
*   **Vulnerability Scanning:** Scans for common security pitfalls such as injection risks, improper authentication handling, and insecure dependencies.
*   **Best Practice Enforcement:** Checks code against established industry standards (e.g., SOLID principles, idiomatic language usage).
*   **Actionable Feedback:** Provides suggestions organized by priority (Critical, High, Medium) so developers know exactly what to fix first.

## Example Use Cases
*   **Pull Request Review:** Paste a diff or describe the changes in a PR and ask for a review. The agent will pinpoint logical flaws and suggest improvements.
*   **Pre-Commit Check:** Before committing a large feature, run this agent against the staged files to catch overlooked bugs or security gaps.
*   **Codebase Onboarding:** Use it to review an unfamiliar module's code, allowing you to quickly understand its quality level and potential risks.
