## Overview
This agent functions as an expert API documentation specialist, dedicated to creating world-class developer experiences. It masters modern standards like OpenAPI 3.1+ and integrates AI-powered tools to ensure your documentation is not just accurate, but highly usable and adoption-driving.

## Capabilities
*   **Modern Standards Mastery:** Authoring specifications using OpenAPI 3.1+, supporting AsyncAPI for event streams, and documenting GraphQL schemas.
*   **AI Content Generation:** Utilizes AI assistance to generate natural language explanations, auto-update docs from code comments, and create multi-language code examples.
*   **Interactive Platforms:** Generates optimized documentation views using Swagger UI and Redoc, and can structure content for custom portals (e.g., Docusaurus).
*   **Developer Portal Architecture:** Assists in building entire developer ecosystems, including live testing interfaces and comprehensive onboarding tutorials.

## Example Use Cases
1.  **New API Launch:** Provide source code or initial specs; the agent will generate a complete, interactive documentation set ready for immediate consumption by developers.
2.  **Updating Legacy Docs:** Feed in outdated documentation and associated code changes; the agent will modernize the OpenAPI spec and update all corresponding examples and explanations.
3.  **Building Portals:** Outline the required sections (Getting Started, Authentication, Endpoints); the agent structures the entire developer portal framework, ensuring consistency across all guides.
