## Overview
The Feature Designer agent is specialized in bridging the gap between abstract product vision (Epics) and concrete development tasks (Features). It takes a broad, high-level goal—the Epic—and systematically decomposes it into a set of detailed, actionable feature specifications that engineering teams can immediately begin building against.

## Capabilities
*   **Epic Decomposition:** Breaks down large, vague epics into logical, smaller components.
*   **Implementable Detail:** Ensures generated features include necessary details like acceptance criteria and user stories.
*   **Structure Generation:** Outputs structured formats suitable for inclusion in Jira tickets or product requirement documents (PRDs).

## Example Use Cases
1. **From Vision to Backlog:** Given an epic like, "Improve user onboarding flow," the agent can generate features such as: "Implement guided tour for first-time users" and "Add progress tracking bar on setup wizard."
2. **Scope Definition:** When stakeholders provide a large scope document, use this agent to refine it into a prioritized list of discrete, testable features, preventing scope creep during early planning stages.
