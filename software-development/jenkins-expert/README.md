## Overview
This agent specializes in mastering the entire lifecycle of CI/CD automation within Jenkins. It moves beyond simple scripting to architect resilient, scalable, and secure build pipelines using industry best practices.

Whether you are starting a new project or refactoring legacy jobs, this expert ensures your Jenkins setup adheres to modern standards, emphasizing declarative pipelines and modularity.

## Capabilities
*   **Jenkinsfile Generation:** Creates clean, readable, and optimized `Jenkinsfile` scripts using Declarative Pipeline syntax.
*   **Workflow Automation:** Designs complex CI/CD workflows incorporating parallel stages, webhooks, and external service integrations.
*   **Security Hardening:** Implements best practices for secret management using Jenkins credentials and advises on access control policies.
*   **Infrastructure Guidance:** Provides setup guides for new agents, node configuration, and artifact management strategies.
*   **Optimization & Auditing:** Analyzes existing jobs to suggest performance improvements, identify bottlenecks, and generate compliance reports.

## Example Use Cases
1. **Building a Microservice Pipeline:** Provide the service repository details, and this agent will output a complete `Jenkinsfile` that builds, tests, scans (security check), and deploys the code to staging upon merge to main.
2. **Implementing Blue Ocean Visualization:** Request an upgrade plan for visualizing build history and performance metrics in Jenkins Blue Ocean.
3. **Troubleshooting Build Failures:** Feed it a stack trace or error log from a failed job, and it will diagnose the root cause, suggesting specific syntax corrections or necessary plugin updates.
