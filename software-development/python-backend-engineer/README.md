## Overview
This agent embodies the expertise of a Senior Python Backend Engineer. It is designed to architect, build, and optimize complex, scalable backend services using modern Python best practices. Whether you are starting a new microservice or refactoring legacy code, this agent ensures adherence to SOLID principles, clean architecture, and industry-leading tooling.

## Capabilities
*   **API Development:** Creating robust RESTful APIs with FastAPI, complete with Pydantic validation and OpenAPI documentation.
*   **Database Integration:** Designing efficient schemas and implementing optimized data access layers using SQLAlchemy or similar ORMs.
*   **Architecture & Design:** Adhering to clean architecture patterns and writing modular, maintainable codebases.
*   **Tooling Proficiency:** Utilizing modern tools like `uv` for dependency management and project bootstrapping.
*   **Quality Assurance:** Writing comprehensive unit and integration tests using pytest, alongside implementing type hinting and linting standards (black, isort).

## Example Use Cases
*   **New Service Build:** "I need a new task management API that connects to PostgreSQL. Please architect the entire system, including models, endpoints, and basic CRUD operations."
*   **Code Refactoring:** "This existing Python service handles user authentication but is slow and lacks structure. Can you refactor it for better performance and modularity?"
*   **Performance Optimization:** "We have a background job processing queue that times out frequently. Please review the async logic and suggest performance improvements using caching strategies."
