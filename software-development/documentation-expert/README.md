## Overview
The Documentation Expert acts as a professional technical writer and information architect, bridging the gap between complex software functionality and user comprehension. This agent specializes in developing complete documentation strategies, ensuring that all created materials—from API references to end-user tutorials—are clear, consistent, and highly navigable.

## Capabilities
*   **Strategy Development:** Designs full documentation roadmaps by analyzing target audiences (developers, end-users, admins) and defining content scope.
*   **Multi-Format Content Creation:** Generates diverse assets including user manuals, detailed API specifications, step-by-step tutorials, and release notes.
*   **Information Architecture:** Structures documentation logically with clear hierarchies, indexes, and navigation paths for optimal usability.
*   **Style Guide Enforcement:** Creates and maintains rigorous style guides to ensure absolute consistency in tone, terminology, and formatting across all project materials.

## Example Use Cases
1. **New Feature Onboarding:** Provide the agent with a new software feature description; it will return a structured plan including an 'Overview' section, three 'How-To' tutorials for different user roles, and draft API endpoints documentation.
2. **Knowledge Base Overhaul:** Feed the agent existing disparate documents; it will propose a unified information architecture map and suggest standardized terminology to create a cohesive knowledge base.
3. **API Documentation Generation:** Input raw endpoint specifications (e.g., OpenAPI schema); the agent will generate comprehensive, user-friendly documentation including request/response examples and usage guidelines.
