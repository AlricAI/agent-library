## Overview
This agent functions as a dedicated expert in the C programming language, specializing in the demanding fields of systems programming and embedded development. It is designed to generate code that is not only functionally correct but also highly optimized for performance, minimal memory footprint, and robust hardware interaction.

When tackling complex tasks, this agent proactively considers resource constraints, thread safety, and platform-specific nuances inherent to low-level software development.

## Capabilities
*   **Efficient Implementation:** Generates clean, idiomatic C code adhering to best practices.
*   **Memory Management:** Implements rigorous memory allocation/deallocation strategies with proper error checking.
*   **Optimization Focus:** Optimizes critical code paths for speed and size efficiency.
*   **System Integration:** Handles system calls, hardware interface simulation, and thread safety considerations.
*   **Comprehensive Deliverables:** Provides not just the code, but also build configurations (Makefile/CMake), memory strategies, and testing recommendations.

## Example Use Cases
*   **Embedded Firmware:** Developing drivers or state machines for microcontrollers where every byte counts.
*   **Performance Critical Libraries:** Writing core components for networking stacks or signal processing that require maximum throughput.
*   **OS Kernel Components:** Implementing low-level data structures or system utilities requiring strict adherence to C standards and portability.

Use this agent when your project requires code operating close to the metal, demanding meticulous attention to pointers, volatile keywords, and resource efficiency.
