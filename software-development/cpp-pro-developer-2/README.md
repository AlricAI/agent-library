## Overview
Cpp Pro Developer is an advanced AI agent designed for senior-level C++ development. It specializes in architecting and implementing high-performance, memory-safe systems using the latest features of C++20 and C++23. Its core focus is achieving zero-overhead abstractions while maintaining impeccable code clarity.

## Capabilities
*   **Modern C++ Mastery:** Implements solutions leveraging Concepts, Ranges, Coroutines, and Modules.
*   **Performance Optimization:** Focuses on cache-friendly algorithms, SIMD intrinsics, and Profile-Guided Optimization (PGO).
*   **Low-Level Safety:** Enforces RAII, smart pointer best practices, custom allocators, and rigorous memory checks (ASan/UBSan compliant).
*   **Code Quality Enforcement:** Adheres strictly to C++ Core Guidelines, passes static analysis checks (clang-tidy, cppcheck), and ensures comprehensive Doxygen documentation.
*   **Advanced Techniques:** Proficient in complex template metaprogramming patterns like CRTP, variadic templates, and compile-time computation.

## Example Use Cases
1. **System Component Design:** Developing a low-latency networking module requiring precise memory control and concurrency handling.
2. **Algorithm Implementation:** Optimizing a numerical solver to utilize SIMD intrinsics for maximum throughput on specific hardware architectures.
3. **Code Review & Refactoring:** Analyzing an existing codebase to upgrade deprecated patterns (e.g., raw pointers) to modern, safe constructs while ensuring performance parity.
