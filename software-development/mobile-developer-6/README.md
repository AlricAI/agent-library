## Overview
This specialized agent acts as an expert mobile developer focusing on building high-quality, cross-platform applications using industry standards like React Native or Flutter. It handles the complexities of modern mobile development, ensuring code is both shareable across iOS and Android while maintaining a native feel.

## Capabilities
*   **Cross-Platform Architecture:** Designs component structures prioritizing maximum code reuse while accommodating platform-specific UI/UX conventions.
*   **Offline Synchronization:** Implements robust offline-first data strategies, including local caching and synchronization logic for reliable user experiences.
*   **Native Feature Integration:** Handles complex native integrations such as push notifications (FCM/APNS), deep linking, and device hardware access.
*   **Performance Optimization:** Focuses on battery efficiency, network optimization, and thorough bundle analysis to ensure fast, smooth performance across various devices.
*   **Deployment Readiness:** Provides guidance and necessary code structures for both development builds and final app store submissions (iOS/Android).

## Example Use Cases
*   **Building a To-Do List App:** Request the agent to build an offline-first task manager that syncs data when connectivity is restored.
*   **Implementing Push Notifications:** Ask it to set up a system where users receive real-time alerts for new messages, requiring platform-specific handling for both operating systems.
*   **Creating a Social Feed Viewer:** Use it to scaffold the core components of a feed that must adapt its layout and behavior correctly on both iOS and Android screen sizes.
