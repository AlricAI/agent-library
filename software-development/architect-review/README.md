## Overview
This agent acts as an elite software architect, specializing in modern, robust, and scalable system design. It is designed to ensure that your complex distributed systems adhere to best practices across multiple architectural paradigms.

It provides comprehensive reviews covering everything from high-level patterns like Microservices and Event-Driven Architecture (EDA) down to granular principles like SOLID and specific design patterns.

## Capabilities
*   **Modern Patterns Mastery:** Expertise in Clean Architecture, Hexagonal Architecture, Domain-Driven Design (DDD), and implementing robust microservice boundaries.
*   **Distributed Systems Resilience:** Guidance on service meshes (Istio), event streaming (Kafka), and implementing resilience patterns like Circuit Breakers and Sagas.
*   **Design Principle Enforcement:** Deep knowledge of SOLID principles, Repository/Unit of Work patterns, and Dependency Injection best practices to maintain clean codebases.
*   **API Design:** Advises on optimal API strategies using GraphQL, REST, and gRPC for modern service communication.

## Example Use Cases
1. **System Design Review:** Paste a high-level diagram or description of a new system (e.g., an order processing pipeline) and ask the agent to review it against DDD principles and identify potential bottlenecks.
2. **Code Pattern Check:** Provide a snippet of code that implements a complex interaction and ask, "Does this adhere to the Open/Closed Principle?" for immediate feedback on design flaws.
3. **Technology Selection Guidance:** When deciding between an event-sourcing pattern versus traditional database transactions, prompt the agent to compare trade-offs using its knowledge base.
