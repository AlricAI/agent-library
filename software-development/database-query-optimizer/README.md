## Overview
This agent acts as a senior-level SQL developer specializing in the entire lifecycle of database query development. It possesses mastery across PostgreSQL, MySQL, SQL Server, and Oracle, focusing relentlessly on performance, scalability, and adherence to best practices.

Its core function is to take existing or proposed queries and elevate them from functional code to highly optimized, production-ready assets.

## Capabilities
*   **Advanced Query Construction:** Implements complex patterns like CTEs, recursive queries, window functions (ranking, lead/lag), and geospatial operations.
*   **Performance Tuning:** Deep analysis of execution plans, index selection strategies, and join algorithms to meet strict performance targets (<100ms).
*   **Schema & Architecture Review:** Assesses data volume, access patterns, and proposes optimal indexing (covering, composite) and database structure improvements.
*   **Data Integrity Management:** Enforces ANSI SQL compliance, manages transaction isolation levels, and implements deadlock prevention strategies.

## Example Use Cases
1. **Performance Bottleneck Identification:** Provide a slow query and the schema; the agent will analyze the execution plan and suggest specific index additions or query rewrites to improve speed.
2. **Complex Reporting Logic:** Need to calculate running totals, rank users within groups, or perform time-series analysis? Use this agent for robust window function implementation.
3. **Database Schema Refinement:** If you suspect your current indexing strategy is insufficient for high read/write loads, the agent can review the schema and recommend optimal index definitions.
