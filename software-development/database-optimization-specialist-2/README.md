## Overview
The Database Optimization Specialist is an expert AI designed to diagnose and resolve performance bottlenecks within complex database systems. It moves beyond simple query fixes by addressing underlying architectural issues, including poor schema design, inefficient indexing, and suboptimal connection handling.

This agent provides a comprehensive approach to performance tuning, ensuring that your data layer scales efficiently as your application grows.

## Capabilities
*   **Slow Query Analysis:** Identifies bottlenecks using execution plan analysis and query profiling.
*   **Schema Design Review:** Recommends optimal database schemas for read/write patterns.
*   **Indexing Strategy Development:** Creates specific, measurable index recommendations (B-tree, composite, etc.) with implementation scripts.
*   **Connection Management:** Advises on and configures connection pooling best practices to prevent resource exhaustion.
*   **Performance Monitoring Setup:** Provides guidelines for setting up ongoing monitoring and alerting systems.

## Example Use Cases
*   **Query Refactoring:** Provide a slow SQL query, and the agent will return an optimized version with predicted performance gains (e.g., reducing execution time from 5s to 200ms).
*   **Schema Improvement:** Submit your current table structure, and receive recommendations for normalization, partitioning, or adding covering indexes.
*   **Scalability Audit:** Describe your application's expected load (e.g., 10k writes/sec), and the agent will suggest architectural changes to handle peak traffic.
