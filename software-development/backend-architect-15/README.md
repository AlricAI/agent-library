## Overview
The Backend Architect is a master AI specialized in designing, planning, and advising on robust server-side systems. Whether you are starting from scratch or optimizing an existing service, this agent provides expert guidance across modern architectural patterns.

This agent excels at making critical decisions regarding technology stacks, data modeling, API contracts (like OpenAPI), and ensuring the resulting system is scalable, secure, and maintainable over time.

## Capabilities
*   **API Design:** Creates detailed specifications for RESTful APIs, considering endpoints, request/response bodies, and versioning.
*   **Architecture Selection:** Advises on appropriate architectural patterns (e.g., microservices vs. monoliths vs. serverless) based on project requirements.
*   **Database Modeling:** Assists in designing optimal database schemas, including indexing strategies and query optimization plans for performance at scale.
*   **Security Implementation:** Incorporates best practices for security, such as OAuth2 implementation, rate limiting, and input validation into the design phase.

## Example Use Cases
*   **Designing a New Feature:** If you need an API for a complex feature (e.g., real-time chat), use this agent to draft the entire service contract and data flow.
*   **Performance Bottleneck Resolution:** When queries are slow, prompt it with your existing schema and query patterns; it will suggest indexing improvements or architectural shifts.
*   **System Planning:** For a greenfield project, ask it to compare three potential technology stacks (e.g., Python/Django vs. Go/Gin) for a given set of business requirements.
