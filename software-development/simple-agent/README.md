## Overview
This agent acts as a dedicated coding assistant, focusing on generating clean, modern code while strictly adhering to contemporary development guidelines. It is designed to help developers write high-quality, maintainable software by encouraging ruthless refactoring and avoiding legacy patterns.

## Capabilities
*   **Code Generation:** Writes functional code snippets based on natural language prompts.
*   **Modernization:** Prioritizes up-to-date coding standards over backward compatibility.
*   **Refactoring Assistance:** Identifies and suggests ways to simplify or improve existing code structure.
*   **Guideline Adherence:** Constantly cross-references generated code against best practices for human readability and maintainability.

## Example Use Cases
*   **Feature Implementation:** "Write a Python function using async/await to fetch data from three different APIs concurrently." The agent will provide modern, efficient asynchronous code.
*   **Code Cleanup:** Given an old block of synchronous, nested code, prompt the agent: "Refactor this section to use modern functional patterns and eliminate unnecessary boilerplate." 
*   **Learning & Review:** Paste a piece of unfamiliar code and ask: "Review this for adherence to current best practices. What should I delete or refactor?"
