# MCPs recomendados (starter set)

Esta lista es una base para arrancar. Ajusta segun stack y politicas de seguridad.

## Favoritos del workspace

- `Notion MCP`: `https://developers.notion.com/docs/mcp`
- `Todoist MCP`: `https://developer.todoist.com/api/v1/#tag/Todoist-MCP`
- `Context7`: `https://github.com/upstash/context7`
- `GitHub MCP Server`: `https://github.com/github/github-mcp-server`
- `TestSprite`: `https://www.testsprite.com/`

## Basicos

- `filesystem`: lectura/escritura controlada de archivos del proyecto.
- `git`: estado, diff y flujo de cambios.
- `fetch/http`: consulta de APIs y documentacion.

## Productividad

- `postgres` o `mysql`: inspeccion de esquemas y queries guiadas.
- `github`: issues, PRs, reviews y metadata de repos.

## Investigacion y contenido

- `browser/search`: navegacion y extraccion de referencias.
- `notion` o `confluence`: lectura/escritura de documentacion interna.

## Criterio para marcar como recomendado

- Seguridad revisada
- Mantenimiento activo
- Setup reproducible
- Caso de uso concreto en el equipo

## Estado sugerido

- `Aprobado`: listo para uso general.
- `Piloto`: se prueba con 1-2 proyectos.
- `Descartado`: no cumple seguridad, calidad o mantenimiento.
