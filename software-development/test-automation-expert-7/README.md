## Overview
This agent embodies the expertise of a senior test automation engineer, specializing in building intelligent, robust, and scalable testing ecosystems. It moves beyond simple script execution by integrating AI principles, self-healing capabilities, and advanced quality engineering methodologies into every aspect of the testing lifecycle.

## Capabilities
*   **Test-Driven Development (TDD) Mastery:** Facilitates TDD cycles using both state-based (Chicago School) and interaction-based (London School) approaches, providing guidance for property-based testing and regression safety validation.
*   **AI-Powered Test Generation:** Leverages NLP to generate comprehensive test cases from natural language specifications, ensuring high coverage with minimal manual effort.
*   **Self-Healing Automation:** Implements modern frameworks capable of automatically adapting to minor UI changes (e.g., using tools like Testsigma or Applitools), drastically reducing maintenance overhead.
*   **Advanced Coverage Techniques:** Supports test triangulation and tracks TDD compliance metrics, ensuring development teams maintain high quality standards.
*   **Intelligent Optimization:** Utilizes machine learning for failure prediction and optimizes test execution paths for the fastest possible feedback loop.

## Example Use Cases
1. **Building a New Feature Test Suite:** Provide natural language requirements (e.g., "The user must be able to reset their password via email link within 3 attempts.") and have the agent generate initial, comprehensive test cases covering edge cases and failure paths.
2. **Maintaining Legacy Tests:** Feed in an existing flaky test suite; the agent will analyze it for common failure patterns and suggest self-healing logic or refactoring steps to improve stability.
3. **Adopting TDD Practices:** Work with a development team on a new module, guiding them through writing failing tests first, then implementing the minimum code required to pass those tests, thus enforcing best practices from day one.
