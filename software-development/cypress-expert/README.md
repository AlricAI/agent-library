## Overview
This agent specializes in the Cypress testing framework, providing expertise across the entire lifecycle of end-to-end (E2E) web application testing. It adheres to industry best practices, promoting maintainable, reliable, and comprehensive test suites that integrate seamlessly into modern CI/CD pipelines.

## Capabilities
*   **Test Structure & Design:** Implements Behavior-Driven Development (BDD) patterns and enforces the Page Object Model for clean code separation.
*   **Automation Logic:** Writes complex asynchronous tests, manages fixtures, and configures environment variables correctly.
*   **Resilience & Debugging:** Utilizes Cypress retries to mitigate flaky tests, captures detailed video/screenshots on failure, and optimizes selectors for robustness.
*   **Advanced Features:** Handles network mocking (intercepting API calls) and extends functionality using custom commands and plugins.
*   **Quality Assurance:** Focuses on achieving high test coverage across critical user paths and ensuring cross-browser compatibility where necessary.

## Example Use Cases
*   **Building a New Test Suite:** Provide a set of user stories, and the agent will scaffold a complete, best-practice Cypress project structure, including initial setup files and core tests.
*   **Debugging Flaky Tests:** Paste logs or describe intermittent failures; the agent will analyze the pattern and suggest specific code refactors (e.g., improving waits, adjusting retries) to stabilize the test.
*   **Implementing Complex Flows:** Request an automation script for a multi-step process (like checkout or user onboarding), ensuring all necessary asynchronous steps and data setup are correctly handled.
