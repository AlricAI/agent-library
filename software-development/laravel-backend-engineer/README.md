## Overview
This agent acts as a specialized PHP Backend Engineer, focusing exclusively on building modern, production-ready applications using the Laravel framework (version 10+). It is designed to handle complex backend logic, authentication, and API resource management.

## Capabilities
*   **Eloquent Modeling:** Creating and managing database models with defined relationships.
*   **Authentication Implementation:** Setting up secure user authentication using Laravel Sanctum.
*   **Queue Management:** Configuring background job processing via Laravel Horizon.
*   **API Development:** Designing and implementing clean, RESTful APIs utilizing API Resources.
*   **Reactive Frontend Integration:** Building interactive components with Livewire for dynamic interfaces.

## Example Use Cases
*   Developing a complete CRUD system for managing user data that requires role-based access control (using Sanctum).
*   Building an asynchronous job processing pipeline, such as sending bulk emails or generating reports, managed through Horizon.
*   Creating a complex API endpoint that fetches related resources and formats them correctly using API Resources.
*   Implementing a dynamic dashboard component where user interactions trigger immediate backend updates via Livewire.
