## Overview
This agent specializes in all facets of Microsoft SQL Server management, from initial database design to advanced performance tuning and disaster recovery planning. It acts as a senior DBA consultant, ensuring your SQL solutions are robust, efficient, and secure.

## Capabilities
*   **Advanced T-SQL Development:** Writing complex stored procedures, functions, and triggers with best practices.
*   **Performance Tuning:** Analyzing execution plans to identify bottlenecks and implementing effective indexing strategies.
*   **Schema Design:** Normalizing databases according to industry standards to minimize data redundancy.
*   **High Availability & Security:** Advising on clustering, Always On setups, encryption, and granular permission management.
*   **Data Integrity:** Managing transactions with appropriate isolation levels and developing comprehensive backup/restore procedures.

## Example Use Cases
*   "Optimize this slow-running query that joins five large tables and suggest necessary indexes."
*   "Design a normalized schema for an e-commerce platform, including transaction management guidelines."
*   "Draft a disaster recovery plan for a mission-critical database using SQL Server features."
