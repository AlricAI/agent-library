## Overview
The DX Optimizer acts as a dedicated Developer Experience (DX) specialist. Its primary goal is to eliminate friction points in the software development lifecycle, making the process more joyful and productive for engineering teams.

This agent doesn't just write code; it optimizes *how* code gets written by analyzing existing workflows, researching best practices, and implementing systemic improvements across tooling, setup, and automation.

## Capabilities
*   **Workflow Profiling:** Identifies time sinks and pain points within current development processes.
*   **Environment Simplification:** Reduces complex environment setups to under five minutes with actionable steps.
*   **Automation Implementation:** Creates useful shortcuts, custom CLI commands, and task runners (e.g., `Makefile`).
*   **Tooling Configuration:** Generates optimal IDE settings, recommended extensions, and necessary Git hooks for quality gates.
*   **Documentation Generation:** Produces clear, accurate setup guides and comprehensive troubleshooting documentation.
*   **Performance Tuning:** Optimizes build times, test suites, and feedback loops for faster iteration cycles.

## Example Use Cases
*   **Onboarding New Team Members:** Run this agent to generate a complete, documented setup guide that gets a new developer coding within minutes.
*   **Post-Feedback Refinement:** After receiving team complaints about slow builds or confusing local setups, invoke the agent to systematically address and resolve those bottlenecks.
*   **Project Kickoff:** Use it at the start of any greenfield project to establish best-in-class initial tooling configurations (e.g., pre-configured `package.json` scripts and linting hooks).
