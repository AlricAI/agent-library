## Overview
This agent acts as a specialized AI Engineer, capable of designing and implementing robust, production-ready generative AI applications. It focuses on the entire lifecycle, from initial concept analysis to deployment with monitoring.

The core philosophy is building reliable, cost-efficient systems by emphasizing structured outputs, comprehensive error handling, and iterative testing over mere complexity.

## Capabilities
*   **System Design:** Analyzes requirements to select optimal models (LLMs/APIs) and architectural patterns.
*   **RAG Implementation:** Builds Retrieval-Augmented Generation pipelines featuring optimized chunking strategies and advanced vector database integration.
*   **Prompt Engineering:** Develops, tests, and versions prompts using structured formats like JSON mode and function calling.
*   **Orchestration:** Implements complex agent workflows using frameworks like LangChain or LangGraph.
*   **Reliability & Monitoring:** Incorporates robust error handling, retry logic, token usage tracking, and cost monitoring alerts.

## Example Use Cases
*   **Building a Knowledge Bot:** Need to create an internal chatbot that answers questions based on proprietary documents? This agent will set up the chunking, vector store indexing, and retrieval chain.
*   **Complex Workflow Automation:** Developing a multi-step process (e.g., summarize a document, then draft emails based on it)? It will structure this using agent orchestration patterns.
*   **Performance Tuning:** Your initial LLM prototype is slow or expensive? Use this agent to implement A/B testing frameworks and optimize retrieval logic for better cost efficiency.
