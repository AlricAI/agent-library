## Overview
This agent embodies the role of a seasoned AI/ML Engineer and intelligent systems architect. Its core mission is to guide users through the entire lifecycle of building production-ready, scalable, and ethically sound artificial intelligence applications.

It focuses heavily on moving models from theoretical concepts into reliable, real-world business features, emphasizing robust MLOps practices throughout the process.

## Capabilities
*   **Intelligent System Development:** Designing and architecting ML models (NLP, CV, Recommender Systems) for practical business use cases.
*   **Data Pipeline Construction:** Building end-to-end data pipelines necessary to feed and manage model training data reliably.
*   **Production Deployment & MLOps:** Implementing strategies for deploying models via real-time APIs, managing versioning, and setting up monitoring frameworks (including A/B testing).
*   **AI Ethics & Safety Implementation:** Integrating critical safety checks such as bias detection, fairness metrics, privacy preservation techniques, and model interpretability into the design phase.

## Example Use Cases
1. **Building a Fraud Detection System:** The agent can architect the necessary data ingestion pipeline, select appropriate ML models (e.g., anomaly detection), deploy it via a low-latency API endpoint, and ensure bias testing is included before launch.
2. **Developing an Intelligent Content Moderation Tool:** It will outline the NLP model requirements, suggest techniques for handling sensitive data (privacy), and structure the deployment to monitor for adversarial inputs.
3. **Creating a Scalable Recommendation Engine:** The agent can design the architecture covering both batch retraining cycles and real-time inference needs, ensuring scalability is prioritized from day one.
