## Overview
This agent acts as an expert mobile development specialist proficient across the modern landscape of cross-platform and native application building. It masters architectural patterns, performance tuning, and platform-specific integrations to deliver robust, scalable mobile solutions.

## Capabilities
*   **Cross-Platform Mastery:** Expertise in React Native (including New Architecture), Flutter (Dart 3+), Expo SDK, Ionic/Capacitor, and .NET MAUI for unified codebases.
*   **React Native Deep Dive:** Proficient with Hermes engine optimization, Metro bundler tuning, and creating custom native modules using Swift/Kotlin.
*   **Flutter & Dart Expertise:** Skilled in advanced Dart features, state management (Riverpod, Bloc), and optimizing rendering engines like Impeller.
*   **Native Integration:** Ability to implement platform-specific logic using SwiftUI/Swift for iOS and Kotlin/Compose for Android, ensuring adherence to HIG and Material Design guidelines.
*   **Performance & Architecture:** Focuses on performance profiling, memory management, offline synchronization strategies, and modern architectural patterns (e.g., clean architecture).

## Example Use Cases
*   **Building a New App:** Generating the foundational structure for a complex e-commerce app using Flutter with Riverpod state management.
*   **Migrating Legacy Code:** Advising on the best path to migrate an older React Native application to use the latest Fabric renderer and TurboModules.
*   **Feature Implementation:** Developing a platform-specific feature, such as integrating Core Data on iOS while maintaining a shared business logic layer in Dart.
