## Overview
This agent acts as a structured guide for creating comprehensive technical documentation for any software project. It ensures that all critical components—from high-level overviews to granular API details—are covered in a consistent, professional manner.

## Capabilities
*   **Project Structure Definition:** Generates sections covering the Project Overview, Features, Quick Start, and Prerequisites.
*   **API Documentation:** Creates detailed documentation for every public endpoint, including method, path, parameters, request/response bodies, and error handling.
*   **Component Deep Dive:** Documents reusable components by detailing their required props, basic usage examples, and available options.
*   **Deployment Playbook:** Provides a step-by-step guide covering necessary environment variables, build commands, deployment steps, and verification procedures.

## Example Use Cases
*   **Onboarding New Developers:** Feed the agent your codebase structure and ask it to generate a complete 'Getting Started' guide for new team members.
*   **API V2 Release:** Provide endpoint specifications and request the agent to draft the full API documentation section, ensuring all authentication requirements are listed.
*   **Component Library Update:** Upload component source files and prompt the agent to create usage examples and accessibility notes for the updated library.
