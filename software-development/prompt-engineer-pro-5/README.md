## Overview
Prompt Engineer Pro is an expert agent designed to elevate your interaction with Large Language Models (LLMs). It moves beyond simple prompting by implementing state-of-the-art techniques like Chain-of-Thought, Constitutional AI, and Tree-of-Thoughts. Its goal is to generate production-ready, robust, and highly reliable prompts tailored for specific business outcomes.

## Capabilities
*   **Advanced Reasoning:** Implements CoT, Few-Shot CoT, and explores multiple paths using Tree-of-Thoughts (ToT) for complex problem decomposition.
*   **Safety & Alignment:** Incorporates Constitutional AI principles to ensure outputs are ethical, safe, and self-correcting. It includes patterns for bias mitigation and jailbreak defense.
*   **Meta-Prompting:** Capable of optimizing prompts themselves, facilitating iterative refinement, and suggesting A/B testing frameworks for performance improvement.
*   **Structured Output Generation:** Always provides the final, ready-to-use prompt in a single, copy-pasteable block.

## Example Use Cases
1. **Complex Problem Solving:** Need an agent to solve multi-step logic puzzles? Ask it to generate a prompt utilizing 'Least-to-Most' prompting for structured decomposition.
2. **Safety Guardrails:** Building a customer-facing chatbot that must refuse harmful requests? Request a prompt incorporating Constitutional AI principles and red-teaming checks.
3. **Workflow Improvement:** Your current system prompt is vague. Ask the agent to analyze your goal and generate an optimized, comprehensive meta-prompt for immediate deployment.
