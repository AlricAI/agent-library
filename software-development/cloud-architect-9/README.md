## Overview
Cloud Architect Pro is an expert AI agent specializing in designing robust, scalable, and cost-effective cloud infrastructure across major providers (AWS, Azure, GCP). It adheres to modern DevOps principles by prioritizing Infrastructure as Code (IaC) using Terraform, ensuring that all deployments are automated, repeatable, and secure.

This agent moves beyond simple design suggestions; it provides actionable, production-ready artifacts for implementation.

## Capabilities
*   **Architecture Design:** Creates multi-region, highly available architectures optimized for resilience and performance.
*   **Infrastructure as Code (IaC):** Generates complete, modular Terraform code for deployment, including state management best practices.
*   **Cost Optimization:** Performs deep cost analysis, right-sizing resources, and recommending services to minimize monthly cloud expenditure.
*   **Security & Compliance:** Implements security by default using least-privilege IAM roles, network segmentation (VPC/VNet), and compliance checks.
*   **Resiliency Planning:** Develops auto-scaling policies, load balancing strategies, and comprehensive Disaster Recovery (DR) runbooks with defined RTO/RPO objectives.

## Example Use Cases
1. **New Application Deployment:** Provide use case details (e.g., 'A global e-commerce site handling 10k peak users') and receive a full Terraform module structure, architecture diagram (Mermaid), and initial cost estimate.
2. **Cost Reduction Audit:** Input existing infrastructure descriptions or sample costs, and the agent will return specific recommendations for service downgrades, reserved instance purchases, or architectural shifts to save money.
3. **Migration Planning:** Outline an on-premises system needing cloud migration; the agent will map required services, suggest a phased rollout plan, and draft initial networking configurations.
