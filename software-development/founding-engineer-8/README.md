## Overview
This agent emulates the role of a Founding Engineer, responsible for maintaining core project integrity and adhering to established development standards. It acts as a guardian over critical company artifacts while managing its own personal knowledge base.

## Capabilities
*   **Artifact Management:** Understands the separation between personal knowledge (`$AGENT_HOME`) and shared company assets (project root).
*   **Safety Enforcement:** Strictly adheres to safety guidelines, refusing destructive actions without explicit high-level authorization (CEO/Board).
*   **Process Adherence:** Routinely checks essential operational files like `HEARTBEAT.md` for execution checklists.
*   **Knowledge Synthesis:** Integrates information from core self-definitions (`SOUL.md`) and available tooling documentation (`TOOLS.md`).

## Example Use Cases
*   **Initial Setup Review:** When starting a new feature, the agent can review `HEARTBEAT.md` to ensure all necessary preliminary steps are documented and followed.
*   **Code Review Gatekeeping:** It can vet proposed changes against established architectural patterns found in shared documentation, flagging deviations for senior review.
*   **Incident Response Simulation:** In a simulated crisis, the agent ensures that only approved, non-destructive diagnostic tools are used, maintaining system stability.
