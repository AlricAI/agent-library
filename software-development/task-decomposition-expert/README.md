## Overview
The Task Decomposition Expert acts as a master architect for complex system implementations. Its core function is to take a high-level, ambiguous user objective and systematically break it down into a clear, actionable, and dependency-mapped set of smaller tasks. It excels at designing the overall blueprint for a multi-agent or tool-based solution.

## Capabilities
*   **Goal Deconstruction:** Breaks complex objectives into primary goals, secondary tasks, and atomic actions.
*   **Workflow Mapping:** Identifies task dependencies, suggesting optimal sequencing and parallel execution paths.
*   **Tool & Agent Orchestration:** Recommends the precise combination of tools (e.g., ChromaDB for data operations) and specialized agents required for each component.
*   **Implementation Roadmapping:** Provides a structured plan including milestones, dependency mapping, and risk assessment with mitigation strategies.

## Example Use Cases
*   **Building a Research Platform:** Given the goal to 'Create an internal knowledge base that answers complex questions using proprietary documents,' this agent will decompose it into document ingestion (ChromaDB), query formulation, retrieval, and final synthesis steps. 
*   **System Integration Planning:** When integrating multiple disparate APIs for a single business process, it maps out the required sequence of calls, identifying necessary data handoffs between services.
*   **Complex Automation Pipeline Design:** For automating end-to-end workflows—such as 'Monitor stock prices and generate a report if three specific conditions are met'—it designs the monitoring loop, condition checks, and reporting mechanisms.
