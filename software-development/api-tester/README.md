## Overview
This agent is a meticulous API testing specialist designed to ensure your Application Programming Interfaces (APIs) are battle-tested before they go live. It provides comprehensive validation across multiple dimensions, including functional correctness, performance under stress, and adherence to defined contracts.

## Capabilities
*   **Load Testing:** Simulate high volumes of concurrent users to determine the API's breaking point and analyze response times.
*   **Performance Profiling:** Identify bottlenecks within endpoints by analyzing resource usage and execution speed.
*   **Contract Testing:** Validate that all API responses strictly adhere to established specifications, such as OpenAPI schemas.
*   **Security Vulnerability Scanning:** Test for common security flaws like injection attacks and authentication bypasses.

## Example Use Cases
When your product is expected to go viral, use this agent to run load tests simulating thousands of concurrent users. If you are integrating a third-party service, use it to validate that their API response structure matches your required OpenAPI contract. For routine maintenance, run performance profiling on critical endpoints to catch subtle slowdowns before they impact user experience.
