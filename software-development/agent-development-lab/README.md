## Overview
This agent functions as a dedicated, structured development lab environment for advanced AI agents. It provides a formalized workspace where complex technical ideas can be tested, documented, and architected before external deployment or communication. The core philosophy is to 'Test Before Proposing' by maintaining meticulous records of all experiments.

## Capabilities
*   **Structured Memory Management:** Maintains continuity through daily logs (`memory/YYYY-MM-DD.md`) and long-term architectural decisions (`MEMORY.md`).
*   **Sandboxed Experimentation:** Allows for the free testing of new technologies, benchmarking, and proof-of-concept development without risking production systems.
*   **Architectural Coordination:** Designed to interface with specialized companion agents (e.g., STACKTRACE, PIPELINE) to deep-dive into system architecture and technical strategy.
*   **Controlled Communication:** Enforces a protocol for when to speak—reserving output for technical recommendations or major architectural decisions, while remaining silent otherwise.

## Example Use Cases
*   **System Redesign:** When needing to overhaul the backend infrastructure of an application, use this lab to simulate multiple architectural patterns and document trade-offs against performance metrics.
*   **Technology Evaluation:** To compare three different database solutions for a new product feature, run benchmarks in the sandbox and record the findings systematically.
*   **Complex Workflow Mapping:** Before building a multi-agent workflow, map out the handoffs between specialized agents here to identify potential failure points or bottlenecks.
