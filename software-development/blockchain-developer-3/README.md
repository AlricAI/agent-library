## Overview
This expert agent specializes in the end-to-end development of secure, production-grade blockchain applications. It adopts a security-first mindset, ensuring that all generated code adheres to industry best practices for smart contract development and Web3 integration.

## Capabilities
*   **Secure Smart Contract Development:** Designs and writes robust Solidity contracts, strictly following the Checks-Effects-Interactions pattern.
*   **Vulnerability Mitigation:** Implements advanced security patterns like reentrancy guards, access controls, and comprehensive input validation to prevent common exploits (e.g., flash loan attacks).
*   **DeFi Protocol Implementation:** Builds complex decentralized finance components such as Automated Market Makers (AMMs), lending pools, and staking mechanisms.
*   **Optimization & Testing:** Provides gas consumption analysis, optimization recommendations, and generates extensive test suites using frameworks like Hardhat or Foundry.
*   **Architecture Design:** Assists with cross-chain bridge architecture and integrating Web3 functionality with frontend applications.

## Example Use Cases
1. **Building a Lending Protocol:** Request the agent to design a complete lending platform contract suite, including collateralization logic, interest rate models, and associated unit tests.
2. **Security Audit Simulation:** Provide an existing smart contract and ask the agent to perform a security review, detailing potential attack vectors and suggesting remediation code blocks.
3. **Cross-Chain Integration:** Ask for the architectural blueprint and initial implementation steps for a bridge connecting two different blockchain networks.
