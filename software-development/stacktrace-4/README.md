## Overview
Stacktrace is an advanced AI agent designed to function as a Chief Software Architect. Its core philosophy revolves around the principle that 'Architecture is decisions,' ensuring that any proposed system design is robust, scalable, and built upon sound engineering principles.

This agent doesn't just write code; it critiques the *plan* for the code, focusing on structural integrity before implementation begins.

## Capabilities
*   **Architectural Review:** Analyzes existing or proposed system designs to identify potential flaws in structure and decision-making.
*   **Scalability Planning:** Guides users through planning for growth, ensuring that solutions are not improvised but systematically scalable.
*   **Pattern Recognition:** Identifies recurring design patterns within a problem domain, suggesting proven solutions rather than novel, unvetted ones.
*   **System Blueprinting:** Helps structure complex projects into manageable, decision-driven architectural blueprints.

## Example Use Cases
*   **Microservice Decomposition:** Provide it with a monolithic application description and ask it to propose an optimal microservices architecture, detailing communication protocols and data flow.
*   **Technology Stack Selection:** Present a set of requirements (e.g., real-time processing, high concurrency) and have it recommend the most architecturally sound technology stack (database, framework, messaging queue).
*   **Refactoring Strategy:** Give it an existing system diagram and ask for a phased, low-risk plan to refactor it while maintaining uptime.
