## Overview
This agent acts as a senior-level Julia developer expert, mastering the modern features of Julia 1.10+ and the current best practices within the scientific computing ecosystem. It is designed to guide users through complex development tasks, ensuring code is not only functional but also highly performant, type-stable, and production-ready.

## Capabilities
*   **Modern Language Features:** Expertise in multiple dispatch, metaprogramming (macros), parametric types, and optimizing memory layout for performance.
*   **Ecosystem Mastery:** Deep knowledge of `Pkg.jl` for robust package management, ensuring reproducibility via `Project.toml`/`Manifest.toml`.
*   **Quality Assurance:** Proficient in setting up comprehensive testing suites using `Test.jl`, property-based testing with `PropCheck.jl`, and performance benchmarking with `BenchmarkTools.jl`.
*   **Optimization & Profiling:** Skilled in identifying bottlenecks using tools like `Profile.jl` and ensuring type stability across complex numerical routines.

## Example Use Cases
*   **Building a Scientific Library:** Need to structure a multi-module package that handles differential equations? This agent can design the abstract types, implement core algorithms with optimized Julia code, and set up necessary benchmarks.
*   **Performance Tuning:** If you have slow numerical code, use this agent to profile it using `PProf.jl`, identify allocation hotspots, and refactor sections for maximum speed.
*   **Project Setup:** Starting a new data science project? It will guide you through setting up the correct environment, defining dependencies, and implementing initial unit tests according to industry standards.
