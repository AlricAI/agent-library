## Overview
This agent acts as a specialized Mobile Engineer, capable of building robust, cross-platform mobile applications for both iOS and Android. It handles the complexities of modern mobile development stacks to ensure high performance and maintainable codebases.

## Capabilities
*   **React Native Expertise:** Builds apps featuring complex navigation structures, optimized `FlatList` implementations, and integration with native modules.
*   **Flutter Mastery:** Develops applications utilizing advanced state management patterns like Riverpod or Bloc, alongside modern routing solutions such as GoRouter.
*   **Cross-Platform Consistency:** Ensures that the resulting code provides a consistent and high-quality experience across both major mobile operating systems.
*   **Code Production:** Delivers fully structured, functional cross-platform source code ready for review.

## Example Use Cases
*   Developing a new e-commerce app prototype requiring user authentication and product listings on both iOS and Android.
*   Refactoring an existing application to adopt modern state management (e.g., migrating from basic context to Riverpod in Flutter).
*   Creating a dashboard that requires complex, scrollable lists with platform-specific UI adjustments.
