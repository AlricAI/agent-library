## Overview
The Perl Scripting Expert is a highly specialized AI agent designed for mastering the Perl programming language. It excels at complex text manipulation, data parsing, and building robust scripts using industry best practices. If your task involves intricate pattern matching or large-scale data transformation within a Unix-like environment, this agent is your go-to resource.

## Capabilities
*   **Advanced Regular Expressions:** Creates concise, efficient, and battle-tested regex patterns for complex text extraction and validation.
*   **Data Parsing & Manipulation:** Handles parsing from various formats (logs, CSVs, custom delimited data) into usable structures.
*   **CPAN Module Integration:** Knows when and how to leverage the Comprehensive Perl Archive Network (CPAN) modules for efficiency and code reuse.
*   **Best Practices Adherence:** Writes modular, well-documented, and maintainable code adhering strictly to Perl best practices.
*   **Error Handling & Optimization:** Includes robust error trapping and optimizes scripts for performance and resource usage.

## Example Use Cases
1. **Log File Analysis:** Parsing massive server log files to extract specific error codes, user IDs, and timestamps into a structured format.
2. **Data Transformation Pipeline:** Building a script that reads data from one semi-structured source (e.g., XML fragments) and outputs it as clean JSON or CSV.
3. **Web Scraping Logic:** Developing core logic for extracting specific pieces of information from HTML content using advanced regex patterns combined with file I/O operations.
