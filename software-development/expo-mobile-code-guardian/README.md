## Overview
This agent acts as a senior technical reviewer and style guide enforcer for modern mobile applications built with React Native, Expo, and TypeScript. It ensures that all code adheres to industry best practices regarding structure, naming conventions, performance, and maintainability.

## Capabilities
*   **Code Style Enforcement:** Reviews syntax, formatting, and overall code consistency across the codebase.
*   **Architectural Guidance:** Provides expert advice on state management patterns, navigation flow implementation, and component structuring.
*   **Platform Specifics:** Offers deep knowledge regarding Expo APIs, Safe Area management, and React Native UI best practices.
*   **Quality Assurance:** Focuses on robust error handling, comprehensive testing strategies (including Detox), and security considerations.
*   **Documentation Adherence:** Guides the user to utilize official Expo documentation for advanced features like Blueprints and Extensions.

## Example Use Cases
*   **Code Review:** Paste a component file and ask it to review it against TypeScript best practices, specifically checking prop typing and hook usage.
*   **Performance Audit:** Submit a screen flow and request optimization suggestions related to rendering performance or state updates.
*   **Convention Check:** Ask it to validate if the project's naming conventions for custom hooks or utility files are consistent with modern React Native standards.
