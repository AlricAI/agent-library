## Overview
The Game Studio Producer agent acts as the central coordination point for a game development team. Its primary function is to ensure that the project ships on time, within budget, and meets the agreed-upon quality standards by managing dependencies and overseeing all departmental efforts.

## Capabilities
*   **Sprint Management:** Defines sprint goals, tracks team velocity, assigns capacity, and facilitates retrospectives.
*   **Milestone Tracking:** Monitors deliverables across all departments against a master project schedule.
*   **Risk & Dependency Management:** Proactively identifies potential blockers, resource conflicts, and cross-departmental dependencies, escalating issues before they halt progress.
*   **Scope Negotiation:** Facilitates necessary discussions between creative and technical leads to analyze the impact of scope changes on timelines.
*   **Meeting Coordination:** Runs structured standups, sprint reviews, and milestone review meetings.

## Example Use Cases
*   **Initiating a Cycle:** Start by generating a detailed sprint plan based on current roadmap goals and available team capacity.
*   **Handling Delays:** If the Art department reports a delay in asset delivery, use this agent to immediately assess the impact on the next integration milestone and negotiate scope adjustments with the Creative Director.
*   **Pre-Launch Audit:** Before release, run a comprehensive audit checklist covering localization readiness, CI/CD pipeline stability (via DevOps), and compliance checks (Accessibility).

This agent is ideal for project managers or leads overseeing complex, multi-disciplinary software development efforts where coordination overhead is high.
