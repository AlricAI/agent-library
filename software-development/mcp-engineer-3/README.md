## Overview
The MCP Engineer at Fullstack Forge is a specialized AI agent dedicated to the Model Context Protocol (MCP). This agent is your go-to resource for any task involving the architecture, implementation, or debugging of MCP servers and clients. It ensures that all integrations adhere strictly to defined protocols.

## Capabilities
*   **Protocol Implementation:** Builds robust MCP server and client scaffolding from scratch.
*   **Tool Handling:** Implements necessary tool handlers to extend agent functionality.
*   **Transport Layer Setup:** Configures various communication channels, including stdio, HTTP, and Server-Sent Events (SSE).
*   **Schema Validation:** Validates data structures using industry standards like Zod or Pydantic for strict compliance.
*   **Debugging & Compliance:** Debugs protocol deviations and ensures end-to-end system compatibility.

## Example Use Cases
*   **Scaffolding a New Microservice:** Need to set up a new client that communicates with an existing MCP backend? This agent can scaffold the entire project structure, including necessary transport layers.
*   **Integrating External Tools:** If your workflow requires calling external APIs via a defined protocol, use this agent to build and validate the required tool handler.
*   **Debugging Connectivity Issues:** When communication fails between two services, provide the context, and the MCP Engineer will debug the protocol compliance across the transport layer.
