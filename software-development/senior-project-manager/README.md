## Overview
SeniorProjectManager acts as an experienced, detail-oriented Project Manager specializing in translating raw site specifications into concrete, executable development tasks. Unlike general assistants, this agent is rigorously focused on adhering *only* to the provided requirements, preventing scope creep and ensuring developers receive actionable items.

It leverages persistent memory to learn from past projects, making its task breakdowns increasingly accurate and efficient over time.

## Capabilities
*   **Specification Analysis:** Reads source specs to quote exact requirements without embellishment. It proactively identifies gaps or ambiguities in the provided documentation.
*   **Actionable Task Breakdown:** Decomposes large specifications into small, manageable development tasks, each designed to be completed within 30-60 minutes by a developer.
*   **Acceptance Criteria Generation:** Includes clear acceptance criteria for every generated task, leaving no room for guesswork.
*   **Technical Stack Extraction:** Identifies and documents necessary technical details, including CSS frameworks, animation needs, and specific Laravel/Livewire integrations.
*   **Scope Control:** Enforces a strict focus on functional requirements first, resisting the urge to add 'premium' or 'luxury' features not explicitly requested.

## Example Use Cases
1. **New Feature Implementation:** Provide a detailed product brief; the agent will return a prioritized task list ready for a sprint board.
2. **Scope Review:** Feed in an existing, vague project document and ask it to validate the scope against best practices, flagging potential over-scoping.
3. **Onboarding New Team Members:** Use it to generate a comprehensive 'Project Kickoff' document that summarizes all technical dependencies and initial tasks based on the master spec.
