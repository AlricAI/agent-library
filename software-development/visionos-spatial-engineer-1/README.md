## Overview
This specialized agent is designed for developers building cutting-edge, immersive applications specifically for the visionOS platform. It focuses on mastering the unique paradigms of spatial computing, moving beyond traditional 2D screen layouts into true volumetric user interfaces.

## Capabilities
*   **Liquid Glass Implementation**: Expertise in creating translucent, context-aware UI elements using advanced background effects.
*   **Volumetric Layouts**: Structuring applications using SwiftUI's capabilities for managing content within defined 3D spaces (WindowGroups).
*   **Spatial Widget Integration**: Developing widgets that persist and interact naturally with physical surfaces in the user's environment.
*   **RealityKit-SwiftUI Bridging**: Seamlessly integrating complex 3D models, gestures, and observable entities into SwiftUI views.
*   **Advanced Gesture Handling**: Implementing robust gesture recognition tailored for gaze, touch, and multi-finger interactions in 3D space.

## Example Use Cases
1.  **Interactive Product Viewer**: Creating a spatial window where users can manipulate a 3D model of a product by physically moving around it, with UI overlays appearing as floating glass panels.
2.  **Collaborative Whiteboard**: Building a shared, multi-window group where multiple users can simultaneously draw and annotate on virtual surfaces in the room.
3.  **Immersive Dashboard**: Designing a persistent home dashboard composed of several spatially anchored widgets that adapt their appearance based on ambient light or user focus.
