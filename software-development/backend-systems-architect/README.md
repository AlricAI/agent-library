## Overview
This agent embodies the role of a Senior Principal Backend Engineer and Systems Architect. Its purpose is to move beyond mere functionality to ensure that all engineered systems are resilient, idempotent, highly performant, and scalable from the outset. It enforces best practices across API design, database interaction, and overall system structure.

## Capabilities
* **Deep Requirement Analysis**: Forces mandatory deep dives into requirements before proposing any architecture.
* **API Decision Making**: Utilizes a structured framework to guide decisions on REST vs. GraphQL or other protocols.
* **Scale-Aware Strategy**: Adapts architectural rigor based on the project's current scale (MVP, Growth, Enterprise).
* **Performance Profiling**: Incorporates performance guidelines and anti-patterns checks into every design phase.
* **Standards Adherence**: Ensures all generated code concepts align with defined API standards, database schemas, and security rules.

## Example Use Cases
* **Designing a New Service**: Provide the core business logic, and this agent will output a multi-stage plan covering data modeling, necessary endpoints, and scaling considerations.
* **Refactoring Legacy Code**: Feed it existing system documentation or code snippets to identify bottlenecks, outdated patterns, and suggest modern, resilient replacements.
* **API Contract Definition**: Use it when defining the interaction points between two services to ensure strict adherence to industry-best API standards.
