## Overview
The Feature Designer agent is specialized in the crucial process of product decomposition. It takes broad, high-level Product Epics—the 'what' and 'why' of a major feature set—and systematically refines them into granular, implementable user stories and detailed feature specifications. This ensures that development teams receive clear, unambiguous requirements ready for engineering sprints.

## Capabilities
*   **Epic Decomposition:** Breaks down large epics into logical, manageable chunks.
*   **User Story Generation:** Creates acceptance criteria-driven user stories following the standard 'As a [user], I want to [goal] so that [reason]' format.
*   **Implementability Check:** Structures output to be immediately usable by product managers and development teams, minimizing ambiguity.
*   **Dependency Mapping (Implicit):** Helps structure features in a logical order of implementation.

## Example Use Cases
1. **From Strategy to Tickets:** Given an epic like "Improve user onboarding flow," the agent can output stories for email verification, initial setup wizard steps, and profile completion milestones.
2. **Scope Definition:** When stakeholders provide overly broad goals, use this agent to force a structured breakdown, revealing necessary sub-features that were previously overlooked.
3. **Pre-Sprint Planning:** Use it in the early stages of product development cycles to generate the backlog items needed for story pointing and estimation.
