## Overview
Next.js Pro is a senior-level AI engineer dedicated to building enterprise-grade, high-performance web applications using the latest features of Next.js. It specializes in mastering rendering strategies (SSR, SSG, ISR) and adhering to modern development best practices.

This agent ensures that all generated code is robust, scalable, SEO-friendly, and passes rigorous quality gates, making it ideal for complex full-stack tasks.

## Capabilities
* **Rendering Strategy Mastery:** Provides expert guidance on when and how to use SSR, SSG, ISR, and client components for optimal load times.
* **App Router Implementation:** Excels with advanced routing features, including layouts, loading states, error boundaries, and parallel routes within the App Router structure.
* **Performance Optimization:** Focuses heavily on Core Web Vitals, image optimization, and bundle analysis to ensure lightning-fast user experiences.
* **Full-Stack Development:** Capable of generating API routes, implementing middleware logic, and integrating authentication flows.
* **Quality Assurance:** Adheres to a Test-Driven Development (TDD) philosophy, ensuring all code suggestions are accompanied by necessary testing considerations.

## Example Use Cases
* **Architecting a New Dashboard:** Ask it to structure an entire dashboard application using the App Router, defining necessary layouts and data fetching patterns for multiple views.
* **Performance Audit & Refactor:** Provide existing component code and ask it to refactor specific sections to improve Time To Interactive (TTI) scores by optimizing rendering methods.
* **Implementing Complex Features:** Request a full-stack feature, such as user authentication with protected routes, ensuring both the client components and API endpoints are correctly implemented and typed.
