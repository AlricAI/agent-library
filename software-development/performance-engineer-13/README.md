## Overview
This agent embodies the expertise of a senior Performance Engineer. It specializes in taking complex, high-traffic applications and making them fast, reliable, and observable under stress. Whether you are dealing with slow API response times, memory leaks, or poor user experience scores, this agent provides structured methodologies to diagnose and resolve bottlenecks.

## Capabilities
* **Modern Observability Implementation**: Proficient in setting up full-stack monitoring using OpenTelemetry for distributed tracing across microservices.
* **Advanced Profiling**: Can analyze CPU usage (generating flame graphs), detect memory leaks via heap analysis, and profile I/O bottlenecks across various languages (JVM, Python, Node.js).
* **Comprehensive Load Testing**: Skilled in designing and executing rigorous load tests using industry-standard tools like k6, JMeter, and Gatling for APIs, web frontends, and complex user journeys.
* **User Experience Validation**: Focuses on Real User Monitoring (RUM) metrics, ensuring Core Web Vitals are met across diverse user bases.
* **System Resilience Testing**: Capable of simulating failure scenarios through chaos engineering principles to harden system boundaries.

## Example Use Cases
1. **Diagnosing Latency Spikes**: Provide logs and traces from a recent peak load event; the agent will pinpoint the exact service, database query, or external dependency causing latency spikes.
2. **Optimizing Caching Strategy**: If an API endpoint is slow due to repeated database calls, ask the agent to propose multi-tier caching strategies (e.g., Redis integration) and suggest validation tests for the new cache layer.
3. **Pre-Launch Stress Testing**: Before a major product release, use this agent to design a load test plan simulating 150% of expected peak traffic, identifying breaking points in the architecture.
