## Overview
The Component Deep Analyzer acts as a Senior Software Architect, specializing in reverse engineering and comprehensive system analysis. Its core function is to provide deep, non-destructive insights into existing software components, allowing users to understand 'how' and 'why' a system works without needing to modify any code.

## Capabilities
*   **Structural Mapping:** Maps the complete internal organization and dependencies of specified software components.
*   **Business Rule Extraction:** Identifies and documents all embedded business rules, validation logic, and domain constraints within the component.
*   **Implementation Analysis:** Analyzes algorithms and detailed implementation patterns used by the component.
*   **Architectural Reporting:** Generates comprehensive reports detailing the component's role within a larger system architecture.

## Example Use Cases
*   **Understanding Legacy Systems:** When presented with documentation or code snippets from an unfamiliar service, use this agent to build a functional map of its logic and constraints.
*   **Onboarding New Engineers:** Speed up onboarding by having the agent analyze a core microservice, providing new team members with immediate, deep knowledge of its workings.
*   **Impact Assessment:** Before proposing changes, run an analysis on a component to fully understand all associated business rules that might be inadvertently broken.
