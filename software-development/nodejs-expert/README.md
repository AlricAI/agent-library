## Overview
This agent is a specialized expert in the Node.js ecosystem, designed to architect, develop, and optimize server-side applications. It adheres strictly to modern JavaScript standards (ES6+) and industry best practices for building scalable, maintainable, and secure backends.

## Capabilities
*   **Asynchronous Mastery:** Implements clean, readable asynchronous code primarily using `async/await` patterns.
*   **Architecture Design:** Structures applications with clear separation of concerns, promoting modularity.
*   **API Development:** Builds robust RESTful APIs, often utilizing frameworks like Express.js.
*   **Performance Optimization:** Focuses on minimizing blocking operations and efficiently handling data streams.
*   **Quality Assurance:** Ensures code quality through adherence to linting standards (ESLint/Prettier) and comprehensive unit testing (Jest/Mocha).

## Example Use Cases
1.  **Building a Microservice API:** Need a secure, high-throughput REST endpoint that processes user data asynchronously? This agent will provide the scaffold with proper error handling and dependency management.
2.  **Implementing Data Pipelines:** Developing a service that reads large files or streams of data efficiently without memory overflow? It excels at using Node.js Streams for optimal throughput.
3.  **Refactoring Legacy Code:** Taking an existing, poorly structured Node.js module and refactoring it to use modern async patterns and better modularity.
