## Overview
The Error Sentinel is an uncompromising AI agent designed to enforce computational correctness and structural integrity in codebases. It operates under a 'Zero-Tolerance Imperative,' treating all compiler warnings, anti-patterns, and suboptimal constructs as critical errors that must be eradicated preemptively.

Its core mission is not just to fix bugs, but to elevate the entire implementation to a state of optimal, maintainable, and flawless execution, making it ideal for sensitive domains like theoretical physics simulations or high-reliability systems.

## Capabilities
*   **Comprehensive Error Detection:** Identifies syntax, semantic, logical, architectural, and performance errors.
*   **Warning Elimination:** Treats all compiler/interpreter warnings as precursors to failure and resolves them immediately.
*   **Code Smell Remediation:** Actively refactors code to eliminate technical debt and poor structural patterns.
*   **Severity Enforcement:** Systematically addresses issues across a defined hierarchy (Critical $\to$ Low), ensuring maximum robustness.

## Example Use Cases
1. **Physics Simulation Review:** Feed it complex numerical models written in Python or C++ to ensure mathematical stability and optimal resource usage.
2. **Legacy Code Hardening:** Submit old, poorly documented code blocks that have accumulated technical debt for a complete quality overhaul.
3. **Pre-Commit Validation:** Use it as the final gatekeeper before merging critical feature branches to guarantee zero runtime defects.
