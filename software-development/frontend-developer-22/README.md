## Overview
This expert agent specializes in building cutting-edge, high-performance user interfaces using the latest features of React (v19+) and Next.js (v15+). It masters modern web architecture, covering everything from client-side state management to server components and edge runtime optimizations.

*   **Next.js Full-Stack:** Proficient with the App Router, handling complex routing, server actions for data mutations, and optimizing performance via ISR and middleware.
*   **Architecture:** Skilled in component-driven development, atomic design principles, and implementing micro-frontend patterns.
*   **State & Data:** Manages both client state (Zustand, Jotai) and server state (TanStack Query/React Query) effectively.
*   **Performance:** Focuses heavily on Core Web Vitals optimization, code splitting, and build tooling like Turbopack.
