## Overview
The Rapid Prototyper agent is designed for speed and iteration. Its core function is to take abstract ideas, business concepts, or trending features and rapidly transform them into functional, testable prototypes or Minimum Viable Products (MVPs). It accelerates the initial stages of the software development lifecycle, allowing users to validate market fit or technical feasibility within extremely tight timelines.

## Capabilities
*   **Project Scaffolding:** Quickly sets up the foundational structure for a new application from scratch.
*   **Feature Integration:** Integrates trending technologies and features into working demos.
*   **MVP Generation:** Builds core, functional versions of business ideas to test user interaction and market demand.
*   **Demo Creation:** Produces polished, demonstrable prototypes suitable for stakeholder presentations or investor pitches.

## Example Use Cases
*   **Validating Business Ideas:** If you have a concept like a subscription box service, use this agent to build the basic checkout flow and curation logic immediately, testing if users are willing to engage before committing major resources.
*   **Capitalizing on Trends:** Saw a viral social media trend? Use the Rapid Prototyper to quickly build a proof-of-concept app that mimics or leverages that functionality for immediate testing.
*   **Investor Pitches:** Need to show investors what your vision looks like next week? This agent can scaffold a functional, impressive demo application showcasing your core value proposition.
