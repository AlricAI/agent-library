## Overview
Python Pro is a senior-level AI agent designed to function as an expert Python developer. It focuses on producing code that is not only functional but also highly performant, idiomatic, and architecturally sound. The agent adheres strictly to modern software engineering best practices, including SOLID principles and comprehensive testing.

## Capabilities
* **Idiomatic Development:** Generates PEP 8 compliant, clean Python code utilizing advanced features like decorators, metaclasses, and async/await.
* **Performance Optimization:** Identifies bottlenecks through profiling concepts and implements memory-efficient solutions for speed improvements.
* **Architecture Design:** Structures applications using established design patterns and adheres to SOLID principles for modularity and scalability.
* **Testing Excellence:** Prioritizes comprehensive test coverage (>90%) using `pytest` fixtures and robust mocking strategies.
* **Advanced Programming:** Excels in concurrent programming, particularly high-performance asynchronous I/O using `async`/`await`.

## Example Use Cases
1. **Refactoring Legacy Code:** Provide an existing module and ask the agent to refactor it to be more idiomatic, adding type hints and comprehensive unit tests.
2. **Implementing Complex Features:** Request a new feature (e.g., an asynchronous API client) and guide the agent through iterative development, ensuring test coverage at each step.
3. **Performance Tuning:** Submit code that is slow under load and ask the agent to profile potential bottlenecks and suggest optimized, memory-efficient alternatives.
