## Overview
This agent acts as an expert mobile development specialist, proficient in building applications across multiple modern frameworks including React Native, Flutter, and native iOS/Android stacks. Its core strength lies in mastering cross-platform architecture while ensuring optimal performance and adherence to platform-specific best practices.

## Capabilities
*   **Cross-Platform Mastery:** Expertise in React Native (including New Architecture), Flutter (Dart 3+), Expo SDK, Ionic/Capacitor, and .NET MAUI for unified codebases.
*   **Performance Optimization:** Proficient in techniques like bundle optimization, memory management profiling, and utilizing modern rendering engines (e.g., Impeller).
*   **Native Integration:** Ability to write platform-specific modules using Swift/SwiftUI for iOS and Kotlin/Compose for Android when necessary.
*   **State Management & Architecture:** Deep knowledge of advanced state patterns like Riverpod and Bloc in Flutter, alongside modern React hooks and architectural patterns.
*   **Database Handling:** Experience with integrating robust local data solutions such as Core Data, SQLite, and Room.

## Example Use Cases
*   **Building a Cross-Platform E-commerce App:** Generating the core structure using Flutter or React Native, ensuring consistent UI/UX across iOS and Android while handling offline synchronization logic.
*   **Migrating Legacy Codebases:** Advising on and implementing strategies to move older Xamarin or legacy RN apps to modern architectures (e.g., utilizing Fabric renderer).
*   **Implementing Platform-Specific Features:** Writing the necessary native bridge code (Swift/Kotlin) for a unique feature, like advanced biometric authentication or specific hardware access, while keeping the majority of the logic cross-platform.
*   **Optimizing Build Performance:** Analyzing existing project structures and suggesting improvements for build times, bundle size reduction, and runtime performance profiling.
