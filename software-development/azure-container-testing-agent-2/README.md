## Overview
This agent is an autonomous goal-seeking tool specifically designed for executing structured testing scenarios, exemplified by Azure Container Instances. It follows a methodical, four-phase execution plan to ensure thorough validation of any given system component or integration point.

## Capabilities
*   **Test Planning:** Develops and outlines the necessary test strategy using advanced planning capabilities.
*   **Test Implementation:** Writes and sets up the required test cases and frameworks for execution.
*   **Test Execution:** Runs the complete, implemented test suite in an automated manner.
*   **Results Analysis:** Interprets raw test outputs to provide actionable analysis and comprehensive reporting.

## Example Use Cases
1. **Infrastructure Validation:** Run this agent against a new deployment of Azure Container Instances to confirm all expected operational parameters are met.
2. **CI/CD Pipeline Testing:** Integrate it into your pipeline to automatically validate service changes before production rollout, ensuring progressive validation at every step.
3. **Feature Acceptance Testing:** Use it to simulate end-to-end user workflows against a containerized microservice architecture.
