## Overview
This agent acts as a senior C#/.NET developer, providing expert assistance across the entire software development lifecycle. It is designed to generate code that is not only functional but also adheres to modern .NET conventions, security best practices, and established design patterns.

## Capabilities
*   **Code Generation:** Produces clean, well-designed, error-free, fast, secure, readable, and maintainable C# code for .NET projects.
*   **Design Pattern Implementation:** Proficiently applies advanced patterns such as Async/Await, Dependency Injection (DI), Unit of Work, CQRS, and Gang of Four principles.
*   **Architectural Guidance:** Enforces SOLID principles and provides insights into general software design best practices.
*   **Testing Support:** Plans and writes comprehensive unit tests using xUnit, NUnit, or MSTest, following TDD/BDD methodologies.
*   **Performance Optimization:** Offers suggestions to improve code performance regarding memory usage, asynchronous operations, and data access layers.
*   **Security Review:** Integrates security considerations like authentication, authorization, and data protection directly into the proposed solutions.

## Example Use Cases
*   **Building a REST API Endpoint:** Ask it to create a new endpoint for user management, ensuring proper dependency injection setup, input validation, and role-based authorization checks.
*   **Refactoring Legacy Code:** Provide a problematic class and ask it to refactor it using the Repository or Unit of Work pattern while adhering to SOLID principles.
*   **Implementing Background Jobs:** Request a solution for processing queued messages asynchronously, complete with error handling and retry logic.
*   **Writing Comprehensive Tests:** Give it a service method and ask it to generate a full suite of unit tests covering success paths, edge cases, and failure scenarios.
