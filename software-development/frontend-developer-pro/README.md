## Overview
Frontend Developer Pro is a senior-level AI agent designed to architect and build robust, high-performance user interfaces. It specializes in modern JavaScript frameworks, including React 18+, Vue 3+, and Angular 15+. Its primary focus is ensuring that all generated code is not only functional but also highly maintainable, accessible (WCAG compliant), and scalable.

Crucially, this agent enforces a structured workflow, always beginning by gathering comprehensive project context to align with existing architectural patterns before writing any code.

## Capabilities
*   **Context-Aware Development:** Automatically initiates the process by requesting detailed project context regarding component ecosystems, state management, and design tokens.
*   **Multi-Framework Proficiency:** Capable of generating high-quality code across leading frameworks (React, Vue, Angular).
*   **Code Quality Focus:** Generates components using TypeScript interfaces, ensuring strong typing and maintainability from the outset.
*   **Best Practices Adherence:** Builds responsive layouts, integrates with existing state patterns, and writes accompanying unit tests alongside implementation.

## Example Use Cases
1. **Building a Dashboard Widget:** Provide requirements for a new data visualization widget; the agent will first analyze your current component library structure before scaffolding the React/Vue/Angular component.
2. **Refactoring Legacy UI:** If you provide an older section of code, the agent can assess its adherence to modern standards and propose refactored, accessible components.
3. **Implementing Complex Forms:** Use it for multi-step forms or complex input validation flows, ensuring state management integration matches your existing patterns (e.g., Redux, Pinia).

**Note:** Always provide the initial project context or allow the agent to perform its mandatory context discovery step first.
