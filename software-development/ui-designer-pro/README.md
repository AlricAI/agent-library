## Overview
UI Designer Pro is an expert agent that bridges the gap between visual design and technical implementation. It specializes in creating beautiful, functional, and highly user-centered interfaces by mastering modern design systems, component-driven architecture, and responsive layout principles.

This agent ensures that any proposed UI solution is not only aesthetically pleasing but also technically feasible and scalable for real-world development.

## Capabilities
*   **Component Design & Creation:** Implements atomic design methodology (atoms through pages), designs state-driven components (hover, active, error), and handles complex form patterns.
*   **Layout Systems & Grid Design:** Masters CSS Grid and Flexbox for building responsive layouts using mobile-first strategies, incorporating container queries and consistent spacing scales.
*   **Visual Design Fundamentals:** Applies principles of color theory (contrast ratios, harmony), typography systems (type scale, font pairing), and visual hierarchy to create cohesive user experiences.

## Example Use Cases
*   **Designing a Dashboard:** Requesting a multi-widget dashboard layout that must adapt perfectly from desktop monitors down to mobile phones using a 12-column grid system.
*   **Creating a Component Library:** Needing the specifications for a reusable 'Card' component, including states (default, hover), required props, and accessibility considerations.
*   **Visual Polish Pass:** Providing rough wireframes and asking the agent to apply professional visual polish by suggesting color palettes, typography pairings, and appropriate spacing rules.
