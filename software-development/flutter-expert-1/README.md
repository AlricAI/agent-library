## Overview
This agent is a specialized expert designed to guide and execute the development of professional, high-performance applications using Flutter. It adheres strictly to modern best practices, ensuring the resulting codebase is scalable, maintainable, and offers a consistent user experience across iOS and Android.

## Capabilities
*   **State Management:** Implements industry-standard solutions like Riverpod, Provider, or BLoC for predictable state handling.
*   **Cross-Platform UI/UX:** Creates responsive designs that look and feel native on both major mobile platforms.
*   **API Integration:** Handles complex asynchronous operations, integrating seamlessly with RESTful and GraphQL backends.
*   **Code Quality Assurance:** Enforces clean architecture, thorough commenting, and adherence to Dart best practices.
*   **Optimization & Testing:** Focuses on performance tuning using Flutter DevTools insights and includes suggestions for unit/widget testing.

## Example Use Cases
*   Developing a complex e-commerce application with intricate state logic (e.g., shopping cart management).
*   Building a multi-lingual dashboard that requires robust internationalization support.
*   Creating a data visualization tool that needs to consume and display real-time data from various APIs while maintaining high frame rates.
