## Overview
The Product Sprint Prioritizer is an expert AI agent designed to function as a seasoned Product Manager. Its core purpose is to guide teams through complex product development cycles by ensuring that the backlog is prioritized optimally, maximizing business value while maintaining team velocity.

This agent applies industry-leading agile methodologies and data-driven frameworks to transform raw feature ideas into actionable, executable sprint plans.

## Capabilities
* **Advanced Prioritization**: Implements multiple frameworks including RICE, MoSCoW, Kano Model, and Value vs. Effort Matrix for objective scoring.
* **Agile Lifecycle Management**: Supports planning across Scrum, Kanban, SAFe, and Design Sprints.
* **Capacity & Risk Planning**: Performs detailed team velocity analysis, identifies bottlenecks, manages dependencies, and assesses technical debt impact.
* **Requirements Engineering**: Assists in creating comprehensive user stories complete with acceptance criteria and story mapping.
* **Stakeholder Alignment**: Structures communication plans and uses data to build consensus among diverse stakeholders.

## Example Use Cases
1. **Sprint Planning**: Provide a list of features, estimated effort, and business goals; the agent will output a prioritized sprint backlog with suggested stories and success metrics.
2. **Roadmap Definition**: Input high-level product vision and market constraints; the agent will generate a phased roadmap detailing milestones and necessary prerequisite epics.
3. **Scope Management**: When scope creep is suspected, feed in new requests alongside current capacity data; the agent will perform an impact assessment to advise on trade-offs.
