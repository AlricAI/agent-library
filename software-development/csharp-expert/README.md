## Overview
This agent is a specialized expert in modern C# programming, focusing on writing clean, high-performance, and maintainable code. It enforces industry best practices, including SOLID principles, asynchronous patterns, and robust testing methodologies.

## Capabilities
*   **Modern Syntax Mastery:** Utilizes features from C# 8.0 and later, such as pattern matching and tuples.
*   **Asynchronous Programming:** Prefers `async/await` for non-blocking operations to ensure optimal responsiveness.
*   **Design Pattern Enforcement:** Actively applies SOLID principles to create highly decoupled and reusable architectures.
*   **Performance Optimization:** Analyzes code for bottlenecks, focusing on memory management and efficient resource usage.
*   **Quality Assurance:** Provides well-structured code accompanied by comprehensive unit tests covering critical paths.

## Example Use Cases
1. **Refactoring Legacy Code:** Provide an older C# class and ask the agent to refactor it using modern async patterns and LINQ for improved readability.
2. **Implementing a Service Layer:** Request a complete, well-tested service layer that handles data fetching asynchronously while adhering strictly to interface segregation principles.
3. **Performance Review:** Submit a block of code suspected of having memory leaks or performance issues; the agent will profile it conceptually and suggest optimized alternatives with explanations.
