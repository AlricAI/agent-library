## Overview
GraphQL Architect is a world-class AI agent specializing in the end-to-end lifecycle of building robust, high-performance GraphQL APIs. It guides development from initial schema design through to advanced optimization techniques like data loading and federation.

This agent ensures that the resulting API is not only functional but also scalable, secure, and adheres to modern software engineering best practices.

## Capabilities
*   **Schema Architecture:** Designs expressive type systems using interfaces, unions, and structures ready for Apollo Federation.
*   **Performance Optimization:** Resolves common pitfalls like the N+1 problem by implementing DataLoader patterns and advising on effective caching strategies.
*   **Federation Design:** Manages multi-service graph composition, defining clear subgraph boundaries and gateway configurations.
*   **Real-time Features:** Implements robust WebSocket subscriptions using pub/sub patterns for event-driven architectures.
*   **Security Implementation:** Enforces security by design through field-level authorization checks and query complexity analysis.

## Example Use Cases
1. **Greenfield Project Kickoff:** Start a new microservices backend by defining the core domain models and initial federated schema structure.
2. **Performance Audit:** Analyze an existing, slow GraphQL endpoint to pinpoint N+1 queries and suggest DataLoader implementations for immediate performance gains.
3. **Refactoring Legacy API:** Take an outdated REST-style service and architect a modern, type-safe GraphQL layer on top of it, ensuring backward compatibility where necessary.
