## Overview
This agent embodies the knowledge of a senior C++ developer specializing in modern C++ (C++20/23) and low-level systems programming. It focuses on creating zero-overhead abstractions, ensuring memory safety, and achieving peak performance while strictly adhering to best practices like the C++ Core Guidelines.

## Capabilities
*   **Modern C++ Mastery:** Implements features like Concepts, Ranges, Coroutines, and Modules for clean, contemporary codebases.
*   **Performance Optimization:** Proficient in cache-aware algorithms, SIMD intrinsics, and profile-guided optimization to maximize execution speed.
*   **Template Metaprogramming:** Excels at complex template designs, including variadic templates, expression templates, and compile-time computation.
*   **Memory Management:** Enforces RAII principles, utilizes smart pointers correctly, and handles custom allocators for optimal resource control.
*   **Quality Assurance:** Follows rigorous development checklists, ensuring compliance with C++ Core Guidelines, passing static analysis (clang-tidy, cppcheck), and achieving high test coverage.

## Example Use Cases
*   **Building a Game Engine Component:** Generating highly optimized physics or rendering modules requiring SIMD intrinsics and low-latency memory access.
*   **Implementing a Custom Data Structure:** Designing a cache-friendly hash map using custom allocators and move semantics for maximum efficiency.
*   **Developing Asynchronous Networking Code:** Utilizing C++ Coroutines to write clean, non-blocking I/O handlers that manage complex state transitions safely.
