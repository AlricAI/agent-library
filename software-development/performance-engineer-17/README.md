## Overview
This agent emulates a senior performance engineer, specializing in taking complex applications from functional to highly optimized. Its core function is to systematically identify bottlenecks—whether they reside in the application code, database queries, or underlying infrastructure—and implement measurable improvements to ensure superior user experience and resource efficiency.

## Capabilities
*   **Comprehensive Testing:** Designs and executes various load tests (stress, soak, spike) to simulate real-world usage patterns.
*   **Deep Profiling:** Performs detailed analysis across CPU, memory, I/O, and network layers to pinpoint performance hotspots.
*   **System Tuning:** Provides expertise in optimizing database indexes, connection pooling, caching strategies (Redis, CDN), and OS parameters.
*   **Bottleneck Identification:** Systematically analyzes resource utilization, thread contention, and execution plans to diagnose root causes of slowdowns.

## Example Use Cases
*   **Pre-Launch Readiness:** Before deploying a major feature, use this agent to establish performance baselines and ensure the system can handle 10x expected traffic.
*   **Slow API Investigation:** When an endpoint degrades under load, feed it the metrics and architecture diagrams; it will guide you through query analysis and potential code optimizations.
*   **Scalability Planning:** Determine if migrating from a single server to a containerized cluster requires specific tuning of resource limits or database partitioning strategies.
