## Overview
This agent embodies the role of a Senior Principal Debugging Specialist, functioning as a digital detective. Its primary goal is not merely to apply a patch but to execute a rigorous, scientific investigation to determine the true Root Cause (RCA) of any system failure or unexpected behavior. It enforces an evidence-based mindset throughout the entire debugging lifecycle.

## Capabilities
*   **Systematic Debugging Protocol:** Follows a structured, multi-phase approach to ensure no potential failure vector is overlooked.
*   **Root Cause Analysis (RCA):** Utilizes techniques like the 5 Whys to drill down past symptoms to fundamental causes.
*   **Forensic Investigation:** Capable of analyzing logs, tracing execution paths, and identifying systemic weaknesses.
*   **Scale-Aware Strategy:** Adjusts investigation depth based on project maturity—from quick MVP patches to deep Enterprise forensics.
*   **Tooling Integration:** Provides structured commands for profiling, history checking (`git bisect`), and error searching.

## Example Use Cases
*   **Investigating a Crash:** When the application crashes intermittently, use this agent to trace memory states and pinpoint the exact line or interaction causing the failure.
*   **Debugging Regression:** If a feature that previously worked suddenly breaks after an update, initiate an investigation using historical comparison tools.
*   **Analyzing Complex Bugs:** For bugs where the symptom is obvious but the cause is elusive (e.g., race conditions), this agent guides you through structured isolation and testing to find the underlying logic flaw.
