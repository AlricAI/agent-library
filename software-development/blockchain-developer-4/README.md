## Overview
This expert agent specializes in the end-to-end development of secure, robust Web3 applications and smart contracts. It operates with a security-first mindset, adhering to industry best practices like the Checks-Effects-Interactions pattern and leveraging battle-tested libraries such as OpenZeppelin.

## Capabilities
*   **Smart Contract Development:** Designs and writes secure Solidity contracts, complete with comprehensive inline documentation.
*   **Security Auditing & Hardening:** Implements advanced security patterns (e.g., reentrancy guards) and actively prevents common vulnerabilities like flash loan attacks and front-running.
*   **DeFi Protocol Implementation:** Builds complex financial primitives including AMMs, lending platforms, and staking mechanisms.
*   **Optimization & Testing:** Provides gas consumption analysis, optimization recommendations, and generates extensive test suites using frameworks like Hardhat or Foundry.
*   **Architecture Design:** Assists with cross-chain bridge architecture and integrating Web3 functionality with frontend applications.

## Example Use Cases
1. **Building a New DeFi Vault:** Request the agent to design a staking vault contract, ensuring it includes role-based access control and withstands common exploits.
2. **Security Review:** Provide an existing smart contract and ask for a comprehensive security audit checklist, focusing on potential reentrancy vectors or improper state management.
3. **Cross-Chain Integration:** Outline the need to connect two different blockchains and have the agent draft the architectural blueprint and initial implementation steps for a bridge mechanism.
