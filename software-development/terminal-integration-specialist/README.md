## Overview
This specialist agent provides deep expertise in the nuances of terminal emulation, text rendering optimization, and seamless integration with SwiftTerm within modern Apple development environments. It handles everything from low-level ANSI escape sequences to high-level SwiftUI view embedding.

## Capabilities
*   **Terminal Emulation Mastery**: Full support for VT100/xterm standards, including precise cursor control, raw/cooked mode management, and robust scrollback handling.
*   **SwiftUI Integration**: Expertly embeds functional SwiftTerm views into SwiftUI applications while managing the view lifecycle correctly.
*   **Performance Optimization**: Implements Core Graphics optimizations for smooth scrolling and efficient memory management during high-frequency text updates.
*   **Input Handling**: Manages complex input streams, including special key combinations, paste operations, and clipboard synchronization.
*   **SSH Pattern Implementation**: Knows how to bridge I/O streams efficiently when connecting terminal sessions via SSH patterns.

## Example Use Cases
*   Building a custom developer console within a macOS application that mimics the behavior of iTerm2. 
*   Implementing a real-time logging dashboard in SwiftUI that requires perfect ANSI color support and scrolling performance.
*   Creating an embedded shell interface that needs to handle both local input and remote SSH streams reliably.
