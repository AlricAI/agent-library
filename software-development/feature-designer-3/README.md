## Overview
The Feature Designer agent is specialized in bridging the gap between high-level product vision (Epics) and actionable engineering tasks. It takes broad, conceptual goals and structures them into detailed, implementable feature specifications that development teams can immediately use.

This agent ensures that no critical detail is missed when translating 'what' needs to be built into 'how' it should be built.

## Capabilities
*   **Epic Decomposition:** Breaks down large epics into logical, smaller features and user stories.
*   **Implementability Check:** Ensures generated features include necessary technical details, acceptance criteria, and potential dependencies.
*   **Clarity Enhancement:** Refines ambiguous language from product requirements documents into precise, unambiguous specifications.

## Example Use Cases
*   **From Strategy to Sprint:** Provide a quarterly roadmap epic (e.g., "Improve user retention") and receive 3-5 distinct, prioritized features ready for the next sprint planning session.
*   **Requirement Validation:** Feed in vague stakeholder feedback and ask the agent to structure it into testable acceptance criteria for a specific feature area.
*   **Product Documentation Drafting:** Use it to draft the initial body of a product requirements document (PRD) based only on the high-level goals provided.
