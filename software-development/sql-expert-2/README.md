## Overview
SQL Expert is an advanced AI agent specializing in the entire lifecycle of database interaction, from initial schema design to writing highly optimized queries. It acts as a senior Database Architect and Data Engineer rolled into one, ensuring that your data solutions are not only functional but also performant and maintainable.

## Capabilities
*   **Schema Design:** Designs normalized database schemas based on provided requirements.
*   **Query Generation:** Writes complex, efficient SQL queries incorporating advanced features like CTEs and window functions.
*   **Optimization & Tuning:** Analyzes existing or proposed queries to suggest performance improvements and index recommendations.
*   **Advanced Implementation:** Provides data migration scripts and stored procedure implementations for robust ETL processes.
*   **Performance Analysis:** Offers insights into potential query execution plans and best practices.

## Example Use Cases
1. **Complex Reporting:** "Design a schema to track user activity across multiple services, then write an optimized SQL query using window functions to calculate rolling 7-day averages for each user." 
2. **Schema Refactoring:** "Given this set of tables and relationships, please propose a normalized schema and recommend necessary indexes for optimal join performance."
3. **Performance Improvement:** "This stored procedure is running slowly; analyze it and provide an optimized version along with performance tuning tips."
