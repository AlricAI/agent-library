## Overview
The Database Specialist is a comprehensive AI expert covering the entire lifecycle of data persistence, from initial architectural design to advanced performance tuning and cloud deployment. It masters both relational (SQL) and non-relational (NoSQL) databases, ensuring your data layer is robust, scalable, and efficient.

## Capabilities
*   **Architecture Design:** Selects appropriate database types (e.g., SQL vs. Graph vs. TimeSeries) and models schemas using best practices like dimensional modeling or event sourcing.
*   **Performance Tuning:** Analyzes execution plans, recommends advanced indexing strategies (B-tree, GIN), and advises on partitioning and sharding for massive scale.
*   **Caching & Scaling:** Implements multi-layered caching patterns (Redis) and designs horizontal scaling solutions using read replicas and connection pooling.
*   **Cloud Expertise:** Provides specific guidance across major cloud providers, including AWS (Aurora, DynamoDB), Azure, and GCP services.
*   **Operations & Migration:** Assists with CI/CD pipelines for database changes using tools like Flyway/Liquibase, ensuring zero-downtime migrations and robust disaster recovery plans.

## Example Use Cases
1. **New System Design:** "We are building a real-time IoT platform tracking millions of sensor readings; what is the optimal database stack (SQL/NoSQL) and schema model?"
2. **Performance Bottleneck:** "My complex reporting query on PostgreSQL is timing out. Analyze this query, suggest necessary indexes, and propose a materialized view strategy."
3. **Scaling Strategy:** "Our user base is projected to grow 10x next year. How should we architect our primary transactional database for horizontal scaling across AWS regions?"
4. **Data Modeling:** "We need to store relationships between users, products, and support tickets over time; recommend a suitable graph or relational model."
