## Overview
This agent functions as a Senior AI Engineer, specializing in the end-to-end development of robust, production-grade AI applications. Its expertise spans modern LLM integration, advanced Retrieval Augmented Generation (RAG) architectures, and complex multi-step agentic workflows.

It enforces rigorous software engineering principles—such as test-driven development, quality gates, and modular design—ensuring that all generated code is not only functional but also maintainable and scalable.

## Capabilities
*   **LLM Application Development:** Building complete, API-integrated AI solutions with robust error handling.
*   **RAG System Architecture:** Implementing advanced knowledge retrieval using vector databases (e.g., Pinecone, Chroma) and context optimization.
*   **Prompt Engineering:** Applying advanced techniques like Chain-of-Thought and few-shot learning for superior model output.
*   **AI Workflow Orchestration:** Designing multi-step reasoning systems and integrating various tools into cohesive agentic pipelines.
*   **Quality Assurance:** Adhering to strict development standards, including linting, type checking, and comprehensive testing before final delivery.

## Example Use Cases
*   **Building a Custom Chatbot:** Need a chatbot that answers questions based on proprietary documents? This agent can design the full RAG pipeline, from document ingestion (chunking/embedding) to vector search retrieval and final LLM response generation.
*   **Developing an Automated Research Tool:** If you require an AI system that must perform multi-step analysis—e.g., 'First, summarize these articles; second, compare findings across the three sources; third, draft a conclusion'—this agent can orchestrate those sequential steps using agentic workflows.
*   **Optimizing Existing LLM Calls:** If your current application is brittle or slow, use this agent to refactor prompts and integrate better context management, ensuring the solution meets production-readiness standards.
