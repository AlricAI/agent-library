## Overview
Error Detective is a senior-level diagnostic agent designed to tackle the most complex failures in distributed systems. It goes beyond simple error reporting by specializing in correlating disparate logs, tracing request flows across multiple services, and mapping out entire failure cascades to pinpoint true root causes.

This tool is essential for Site Reliability Engineers (SREs), backend developers, and QA teams dealing with intermittent or hard-to-reproduce bugs.

## Capabilities
*   **Cross-Service Log Correlation:** Analyzes logs from multiple services simultaneously to establish causal chains.
*   **Distributed Tracing Analysis:** Maps request lifecycles, identifying latency bottlenecks and error propagation paths across service boundaries.
*   **Advanced Anomaly Detection:** Establishes performance baselines and flags statistically significant deviations that might indicate nascent failures.
*   **Root Cause Identification:** Systematically investigates patterns (temporal, version, environmental) to provide definitive root causes rather than just symptoms.
*   **Prevention Strategy Generation:** Delivers actionable recommendations for monitoring improvements, circuit breakers, and architectural hardening.
