## Overview
The QA Expert agent embodies the role of a seasoned Quality Assurance professional. Its primary function is to integrate rigorous, systematic quality checks throughout the entire software development lifecycle (SDLC). It moves beyond simple bug reporting by establishing proactive quality gates and ensuring that testing is built into the process from the outset.

## Capabilities
*   **Test Strategy Development:** Creates holistic testing blueprints covering scope, objectives, and resource allocation for any given feature or product module.
*   **Comprehensive Test Case Design:** Generates detailed, actionable test cases that cover functional paths, edge cases, performance bottlenecks, and security vulnerabilities.
*   **Quality Assessment & Execution:** Supports both manual exploratory testing scenarios and the design/execution of automated end-to-end (E2E) tests using tools like Playwright.
*   **Defect Management & Analysis:** Systematically identifies defects, documents them with clear steps to reproduce, performs root cause analysis, and tracks resolution through defined workflows.
*   **QA Metrics & Reporting:** Provides data-driven insights by tracking key quality metrics, giving stakeholders confidence in the product's readiness for release.

## Example Use Cases
1. **Pre-Release Audit:** Provide the agent with a new feature specification and ask it to generate a full test plan covering functional, performance, and security aspects before any code is written.
2. **Bug Triage Simulation:** Input a set of reported bugs and ask the agent to prioritize them based on severity, exploitability, and impact on core user flows.
3. **Test Coverage Gap Analysis:** Feed the agent existing unit test reports and request it to identify high-risk areas or complex logic paths that are currently under-tested.
