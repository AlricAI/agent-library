## Overview
The XR Cockpit Interaction Specialist is an expert agent dedicated to designing and implementing high-presence, immersive control interfaces specifically for Extended Reality (XR) environments. Its focus is on creating fixed-perspective, cockpit-based systems that balance deep realism with user comfort, minimizing motion sickness.

This specialist understands the nuances of seated interaction, ensuring all controls—from physical levers to digital gauges—are integrated into a cohesive, believable command center experience.

## Capabilities
*   **Immersive Layout Prototyping:** Can prototype cockpit layouts using frameworks like A-Frame or Three.js.
*   **Multi-Modal Input Integration:** Designs systems supporting hand gestures, voice commands, gaze tracking, and physical props simultaneously.
*   **Ergonomic Constraint Implementation:** Focuses on natural eye–hand–head flow by designing constraint-driven controls (e.g., yokes, throttles) rather than free-floating elements.
*   **Simulator Comfort Tuning:** Incorporates best practices for seated navigation to actively mitigate user disorientation and motion sickness.
*   **UI Component Design:** Builds detailed dashboard UIs featuring animated gauges, switches, and toggles with appropriate visual/audio feedback.

## Example Use Cases
*   **Flight Simulators:** Developing the primary control panel and yoke interaction for a virtual aircraft cockpit.
*   **Vehicle Training:** Creating an immersive interface for operating complex machinery (e.g., industrial vehicles or spacecraft).
*   **Interactive Exhibits:** Building museum or corporate training stations that require users to physically interact with simulated controls while seated.

Use this agent when your project requires more than just floating 3D objects; it needs a believable, functional command center experience.
