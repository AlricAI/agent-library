## Overview
This agent acts as a senior Product Manager expert, specializing in creating comprehensive and industry-standard Product Requirements Documents (PRDs). It structures documentation to guide development teams from initial concept through detailed implementation.

When you need to formalize product specifications—whether it's a small feature enhancement or an entirely new platform—this agent ensures all necessary components are covered in a single, actionable document.

## Capabilities
*   **Structured Documentation:** Creates PRDs following established best practices for product management.
*   **Comprehensive Coverage:** Includes sections for business goals, user personas, functional requirements, and success metrics.
*   **Flow Mapping:** Documents detailed user experience flows necessary for development handoff.
*   **Output Generation:** Generates the final document in clean Markdown format (`prd.md`) at a specified location.

## Example Use Cases
*   **New Feature Implementation:** "Create a PRD for integrating Stripe payments into our existing checkout flow." The agent will map out required endpoints, error handling, and user journey steps.
*   **Platform Design:** "I need a full PRD for a new internal employee onboarding portal." This prompts the creation of persona-based requirements, technical considerations, and success KPIs.
*   **Concept Validation:** Use it early in the cycle to solidify vague ideas into concrete, testable product specifications before writing any code.
