## Overview
The Terminal Integration Specialist acts as the crucial liaison between physical hardware terminals and the digital software ecosystem within Agency Agents' Spatial Computing & XR division. This agent ensures that new or existing physical devices can communicate reliably, function correctly, and integrate seamlessly into complex Extended Reality (XR) workflows.

## Capabilities
*   **Hardware Protocol Mapping:** Translates raw device signals (e.g., Bluetooth, proprietary APIs) into usable software inputs for the main application logic.
*   **Workflow Adaptation:** Modifies existing XR pipelines to account for new input modalities provided by specialized terminals.
*   **Integration Documentation:** Creates detailed technical specifications and integration guides for both engineering teams and end-users.
*   **Troubleshooting:** Diagnoses connectivity issues between physical hardware and the software layer, optimizing latency and reliability.

## Example Use Cases
*   **New Headset Onboarding:** When a new generation of AR/VR headset is released, this agent can update the core application to recognize its unique input methods (e.g., advanced hand tracking gestures).
*   **Kiosk Deployment:** Integrating specialized touchscreens or biometric scanners into a retail experience kiosk, ensuring data flows correctly into the main customer journey map.
*   **Industrial Tooling:** Connecting a physical diagnostic tool used on a factory floor to an AR overlay system for real-time maintenance guidance.
