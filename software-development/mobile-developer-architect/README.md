## Overview
This agent emulates the role of a Senior Principal Mobile Architect, focusing on bridging the gap between shared cross-platform logic and platform-specific native perfection. Its core mission is to ensure that mobile applications feel indistinguishable from high-end, natively built experiences.

It enforces rigorous standards across performance, UI fidelity, and architectural patterns, moving beyond simple code generation into true system design consultation.

## Capabilities
*   **Cross-Platform Strategy:** Provides decision matrices for choosing the right technology stack (React Native vs. Flutter) based on project needs.
*   **Performance Auditing:** Enforces modern performance targets, identifying and forbidding common mobile anti-patterns.
*   **Design System Adherence:** Ensures all components align with established mobile design system standards.
*   **Testing & Debugging Guidance:** Guides users through advanced tooling like Maestro testing and native debugging (Flipper/Logcat) for robust verification.
*   **Scale Management:** Adjusts architectural rigor based on project scale, from rapid MVP builds to complex R&D prototypes.

## Example Use Cases
*   **Initial Project Scoping:** Ask the agent to evaluate a new feature requirement against its 'Platform & Framework Decision Matrix' to recommend the optimal approach (e.g., pure JS for MVP vs. custom native modules).
*   **Performance Review:** Submit existing code snippets or architectural plans and ask it to audit them against the '2025 Mobile Performance Targets,' flagging potential bottlenecks.
*   **Feature Implementation Guidance:** When building a complex UI interaction, prompt it to guide the implementation using gesture-driven patterns (Reanimated 3) while ensuring native fidelity.
