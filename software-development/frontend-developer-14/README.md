## Overview
This expert agent specializes in creating cutting-edge, user-centric frontend applications. It adheres to modern development best practices, focusing heavily on accessibility (WCAG 2.1) and performance optimization from the initial wireframe stage.

## Capabilities
*   **Framework Mastery**: Proficient across React, Vue.js, Angular, Next.js, and Nuxt.js.
*   **State Management**: Implements robust state solutions using Redux, Vuex, Context API, or Zustand as appropriate.
*   **Styling & Layout**: Expert in modern CSS techniques including Tailwind CSS, Sass, and responsive, mobile-first design patterns.
*   **Component Architecture**: Develops reusable, composable components following best practices for maintainability.
*   **Testing Suite**: Writes comprehensive unit and end-to-end tests using Jest, React Testing Library, and Cypress.
*   **Concurrency**: Excels at developing multiple, interconnected features in parallel to accelerate development cycles.

## Example Use Cases
1. **Building a Dashboard**: Request the setup of a complex dashboard layout featuring data visualization components, user management sections, and integrated state handling.
2. **Component Library Creation**: Ask for a set of reusable, accessible UI components (e.g., Modals, Cards, Forms) adhering to WCAG standards.
3. **Framework Migration/Integration**: Need to integrate a new feature using Vue 3 within an existing React application? This agent can architect the necessary communication layers and component wrappers.
