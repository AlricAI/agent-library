## Overview
This DevOps Engineer agent specializes in maintaining the entire build, test, and deployment infrastructure for complex software projects, such as game studios. Its primary goal is to ensure that the development team can build, test, and ship high-quality code reliably and efficiently.

## Capabilities
*   **CI/CD Pipeline Management:** Building, updating, and maintaining continuous integration and continuous delivery pipelines for multiple target platforms.
*   **Version Control Workflow Enforcement:** Implementing and managing branching strategies, merge policies, and branch protection rules to maintain code integrity.
*   **Automation Scripting:** Developing scripts to automate repetitive tasks like asset cooking, shader compilation, packaging, signing, and deployment across various environments.
*   **Environment Management:** Provisioning and maintaining development, staging, and production environments using Infrastructure-as-Code (IaC) principles.
*   **Monitoring & Optimization:** Establishing health dashboards to monitor build times, failure rates, and queue depth, ensuring productivity remains high.

## Example Use Cases
*   **Implementing a New Build Target:** When the team needs to support a new console or platform, this agent can write the necessary build scripts and update the CI/CD configuration.
*   **Troubleshooting Pipeline Failures:** If builds are failing intermittently (flaky tests), the agent can triage logs, identify root causes, and propose fixes or post-mortems.
*   **Setting Up Deployment Runbooks:** Before a major release, it can generate detailed, step-by-step runbooks, including necessary rollback procedures, for the release manager.
