## Overview
DevOps Automation Expert is an advanced AI agent designed to act as a seasoned DevOps engineer. Its core mission is to eliminate manual toil from the software delivery lifecycle by implementing comprehensive automation across infrastructure, deployment, and operations.

This agent excels at designing resilient systems that are not only functional but also scalable, observable, and self-healing.

## Capabilities
* **Infrastructure as Code (IaC):** Designs and implements cloud infrastructure using industry standards like Terraform or CloudFormation.
* **CI/CD Pipeline Development:** Builds complete, automated pipelines using tools such as GitHub Actions, GitLab CI, or Jenkins.
* **Container Orchestration:** Sets up complex deployments using Docker and Kubernetes, including service mesh configurations.
* **Reliability Engineering:** Implements zero-downtime deployment strategies (Canary, Blue/Green) and robust rollback mechanisms.
* **Observability & Resilience:** Configures monitoring stacks (Prometheus/Grafana), alerting systems, and automated disaster recovery procedures.

## Example Use Cases
1. **New Service Deployment:** Provide the target cloud platform and application stack; the agent will generate a complete Terraform module for provisioning resources and a corresponding GitHub Actions workflow for CI/CD.
2. **Pipeline Hardening:** If you have an existing deployment process, ask the agent to review it for security gaps, suggesting the integration of vulnerability scanning or compliance checks.
3. **Cost Optimization Review:** Feed in your current cloud resource utilization reports; the agent will suggest right-sizing opportunities and automation scripts to reduce operational expenditure.
