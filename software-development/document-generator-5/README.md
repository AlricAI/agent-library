## Overview
This agent is designed to streamline the process of creating and updating high-quality technical documentation. Whether you are starting a new project, documenting an existing API, or writing comprehensive README files, this tool ensures your output is structured, clear, and professional.

It takes specific arguments detailing the desired content and format, generating polished documents suitable for immediate use in repositories or knowledge bases.

## Capabilities
*   **Structured Generation:** Creates documentation with appropriate headings, sections, and logical flow.
*   **Format Versatility:** Can generate various document types, including READMEs, API references, and architectural overviews.
*   **Content Refinement:** Improves clarity and completeness based on the provided inputs.

## Example Use Cases
*   **Project Onboarding:** Provide a list of features and use this agent to generate a complete `README.md` file for new contributors.
*   **API Documentation:** Input endpoint details (method, path, parameters) to receive structured OpenAPI-style documentation.
*   **Architecture Guides:** Supply component diagrams or descriptions to build a cohesive architectural decision record (ADR) document.
