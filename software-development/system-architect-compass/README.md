## Overview
System Architect Compass is designed to function as the technological compass for any development team. It moves beyond simple coding or maintenance, focusing instead on charting the optimal, forward-looking technical path. This agent ensures that current solutions are not merely adequate, but are architecturally sound, scalable, and positioned for future growth.

## Capabilities
*   **Proactive Vetting:** Tests and vets multiple technological alternatives before recommending a final stack.
*   **Holistic System View:** Designs systems by understanding how all components interact to tell a coherent 'story,' rather than optimizing parts in isolation.
*   **Future-Proofing Analysis:** Identifies potential technical debt early, building solutions with scalability and longevity in mind.
*   **Best Practice Implementation:** Provides recommendations backed by preliminary testing and deep documentation review.

## Example Use Cases
*   **Technology Stack Selection:** When faced with a new product line, use this agent to compare three viable backend technologies (e.g., GraphQL vs. REST vs. gRPC) based on long-term maintainability and required velocity.
*   **System Redesign:** If an existing system is showing signs of technical debt, prompt the agent to map out a phased migration plan that minimizes downtime while upgrading core architectural components.
*   **Roadmapping:** Use it to take a high-level business goal (e.g., 'Enter international market') and generate a multi-stage, technically feasible roadmap with suggested foundational technologies for each stage.
