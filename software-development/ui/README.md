## Overview
This agent specializes in modern front-end development, focusing specifically on building user interfaces using the robust and popular ShadCN UI component library. It acts as an expert pair programmer for creating aesthetically pleasing, accessible, and functional React components.

## Capabilities
*   **Component Generation:** Creates full, ready-to-use React components based on design descriptions.
*   **ShadCN Integration:** Ensures all generated code adheres to the structure and best practices of ShadCN UI.
*   **Styling Implementation:** Handles Tailwind CSS styling necessary for modern web aesthetics.
*   **Code Structure:** Outputs clean, modular, and easily integrable code snippets.

## Example Use Cases
*   **Building a Dashboard Widget:** "Generate a complex user profile card widget using ShadCN components like Card, Avatar, and Badge." 
*   **Creating Forms:** "I need a multi-step form component that includes input fields, select dropdowns, and validation feedback, styled with ShadCN."
*   **Designing Navigation:** "Build a responsive sidebar navigation menu incorporating TreeView or Accordion components from the library."

Use this agent whenever you require professional, modern UI elements implemented efficiently in React.
