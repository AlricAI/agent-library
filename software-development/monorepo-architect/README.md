## Overview
Monorepo Architect is an expert AI specialized in the architecture, tooling, and optimization of large-scale monorepos. It masters industry-leading tools like Nx, Turborepo, Bazel, and Lerna to ensure development workflows remain efficient as your codebase grows.

This agent helps teams move beyond simple polyrepos by establishing robust, scalable structures that minimize redundant builds and complex dependency management issues.

## Capabilities
*   **Tool Selection:** Recommends the best monorepo tooling (Nx, Turborepo, Bazel) based on project needs.
*   **Structure Design:** Guides workspace configuration and optimal project structuring for maintainability.
*   **Build Optimization:** Implements advanced caching strategies (local and remote) to drastically speed up builds.
*   **Dependency Management:** Manages complex dependency graphs and identifies necessary code sharing patterns.
*   **CI/CD Efficiency:** Configures affected/changed detection to run only necessary tests and builds in CI pipelines.

## Example Use Cases
1. **New Setup:** Need to start a new product suite with 10+ interconnected services? Ask for a recommended Nx workspace setup.
2. **Performance Tuning:** Your CI pipeline is slow because it rebuilds everything on every commit. Use this agent to implement affected detection rules.
3. **Migration:** Moving from several independent repositories (polyrepo) into one cohesive monorepo structure.
