## Overview
The Product Manager AI is a specialized assistant designed to guide product development from initial concept through to launch planning. It acts as a virtual Product Owner, ensuring that all features are traceable back to core user needs and business objectives.

This agent excels at structuring ambiguous ideas into actionable, prioritized plans suitable for engineering teams and stakeholders alike.

## Capabilities
*   **User Story Generation:** Creates detailed user stories following the 'As a [user], I want to [goal] so that [reason]' format, complete with acceptance criteria.
*   **Requirements Management:** Takes high-level feature requests and breaks them down into granular, manageable functional and non-functional requirements.
*   **Roadmapping:** Develops phased product roadmaps, prioritizing features based on potential impact versus development effort (e.g., using RICE scoring).
*   **Stakeholder Alignment:** Helps draft documentation that bridges the gap between business goals, technical feasibility, and user empathy.

## Example Use Cases
*   **New Feature Definition:** Provide a vague idea like 'We need better checkout flow.' The agent will return structured Epics, User Stories (e.g., 'As a returning customer, I want to save my shipping address so I don't have to type it every time.'), and suggest acceptance criteria.
*   **Quarterly Planning:** Input your company's annual goals. The agent can generate a draft 3-quarter roadmap, suggesting which features should be tackled first based on market urgency.
*   **Requirements Validation:** Feed in existing documentation and ask the agent to identify missing user perspectives or conflicting requirements between different departments.
