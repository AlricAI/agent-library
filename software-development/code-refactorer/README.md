## Overview
The Code Refactorer agent is designed for senior developers who need to improve the structural integrity and clarity of existing codebases. Its primary directive is to enhance readability, reduce technical debt, and simplify complex logic while guaranteeing that the original functionality remains perfectly intact.

This agent acts as a meticulous pair programmer focused solely on *how* the code is written, not *what* it does.

## Capabilities
*   **Structural Improvement:** Breaking down overly long or monolithic functions into smaller, manageable units.
*   **Readability Enhancement:** Improving variable and function naming conventions for maximum clarity.
*   **Duplication Reduction:** Identifying and consolidating repeated logic blocks to adhere to the DRY (Don't Repeat Yourself) principle.
*   **Complexity Simplification:** Restructuring complex conditional statements or algorithms into clearer, more straightforward patterns.

## Example Use Cases
*   **Post-Feature Cleanup:** After implementing a new feature, use this agent to clean up the associated code, removing temporary debugging lines and optimizing structure.
*   **Code Review Remediation:** When a review flags issues like high cyclomatic complexity or poor naming, this agent can systematically address those specific quality concerns.
*   **Legacy Code Modernization:** For older sections of code that work but are difficult to read, use the refactorer to bring them up to modern best practices without introducing bugs.
