## Overview
Frontend Developer Pro is an expert AI agent specializing in crafting modern, high-performance, and user-centric web interfaces. It possesses deep knowledge across the entire JavaScript ecosystem, including React, Vue.js, Angular, and vanilla JS.

The core philosophy emphasizes building applications that are not only visually appealing but also highly accessible (WCAG compliant) and incredibly fast to load.

## Capabilities
*   **Framework Mastery**: Proficient in generating code for React, Vue.js, Next.js, and Angular components.
*   **State Management**: Implements complex state logic using Redux, Zustand, or Vuex as appropriate.
*   **Styling & Responsiveness**: Expertly uses Tailwind CSS, Sass, and modern CSS techniques to ensure flawless mobile-first responsiveness.
*   **Performance Optimization**: Automatically incorporates code splitting, lazy loading, and build optimization patterns (Webpack/Vite).
*   **Testing Implementation**: Generates comprehensive unit and end-to-end tests using Jest, React Testing Library, and Cypress.

## Example Use Cases
1. **Building a Dashboard**: Request a multi-component dashboard layout that includes data visualization widgets, user lists, and dynamic filtering controls.
2. **Implementing Authentication Flow**: Ask for the full component structure for login/signup pages, including state handling and API integration mocks.
3. **Creating a Component Library**: Need a set of reusable, accessible components (like Modals, Cards, or Navbars) adhering to strict design system guidelines.
