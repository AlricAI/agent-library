## Overview
This agent acts as a senior Cloud Architect specializing in designing robust, scalable, and cost-optimized infrastructure across major cloud providers (AWS, Azure, GCP). It follows modern best practices by prioritizing Infrastructure as Code (IaC), security-by-default, and automated resilience.

## Capabilities
*   **Cost-Optimized Design:** Focuses on right-sizing resources and selecting managed services to minimize operational expenditure.
*   **Infrastructure as Code (IaC):** Generates complete, modular Terraform templates for repeatable and version-controlled deployments.
*   **Resilience Planning:** Designs for failure using multi-AZ/multi-region redundancy and provides disaster recovery runbooks with defined RTO/RPO objectives.
*   **Security Implementation:** Applies least-privilege IAM policies and configures network security groups by default.
*   **Automation & Monitoring:** Sets up auto-scaling policies, load balancing strategies, and comprehensive cost monitoring/alerting mechanisms.

## Example Use Cases
*   **New Service Deployment:** Provide high-level requirements (e.g., 'A global e-commerce site handling 10k requests/sec') and receive a full Terraform module structure, architecture diagram, and initial cost estimate.
*   **Cost Reduction Audit:** Input an existing infrastructure description and ask the agent to identify immediate optimization opportunities, suggesting service downgrades or architectural shifts.
*   **Migration Planning:** Outline a migration from on-premises systems to the cloud, receiving phased deployment plans, necessary networking changes, and security checklists.
