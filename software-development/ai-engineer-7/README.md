## Overview
This agent functions as an expert AI Engineer, specializing in the design and implementation of production-grade generative AI systems. It masters the modern AI stack, covering everything from advanced Retrieval Augmented Generation (RAG) pipelines to complex multi-agent workflows.

Whether you are building a sophisticated chatbot, integrating proprietary knowledge bases, or orchestrating multiple specialized agents, this tool provides expertise across leading frameworks and model deployments.

## Capabilities
* **LLM Integration & Model Management:** Supports major models (GPT-4o, Claude 3.5 Sonnet, Llama 3.1) with function calling and structured output generation. It also handles local deployment strategies using Ollama and vLLM.
* **Advanced RAG Systems:** Implements multi-stage retrieval pipelines utilizing top vector databases (Pinecone, Qdrant) and advanced techniques like GraphRAG, HyDE, and hybrid search (BM25).
* **Agent Frameworks & Orchestration:** Proficient with industry-leading frameworks including LangChain/LangGraph for state management, LlamaIndex for data-centric applications, and CrewAI/AutoGen for multi-agent collaboration.
* **Optimization:** Focuses on cost efficiency through model routing, caching strategies, and context compression to keep token usage optimal.

## Example Use Cases
1. **Building a Knowledge Bot:** Need a chatbot that answers questions based only on 50 complex internal PDF manuals? Use this agent to architect the entire RAG pipeline, from chunking strategy (semantic) to vector store selection and reranking.
2. **Multi-Step Workflow Automation:** You require an AI system where one agent researches market data, another summarizes it, and a third drafts a final report. This agent can scaffold the workflow using LangGraph or CrewAI.
3. **Model Comparison & Deployment Plan:** Planning to move from GPT-4o to a self-hosted Llama 3.1 solution? Use this agent to map out the necessary infrastructure (e.g., BentoML deployment, vLLM setup) and model routing logic for cost optimization.
