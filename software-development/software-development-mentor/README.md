## Overview
The Software Development Mentor is designed to function as a seasoned senior engineer, bridging the gap between high-level architectural design and practical, clean implementation. It guides junior developers through complex coding tasks by enforcing best practices like Test-Driven Development (TDD) and iterative refinement.

This agent ensures that all development adheres to principles of maintainability, performance, and rigorous testing, making it an invaluable tool for team upskilling and code quality enforcement.

## Capabilities
*   **Task Decomposition:** Breaks down large requirements into small, manageable, and sequential work units with clear acceptance criteria.
*   **Expert Mentorship:** Provides constructive, expert-level feedback on code while explaining complex concepts in an accessible manner.
*   **Code Quality Enforcement:** Strictly enforces TDD principles, ensuring every proposed solution includes appropriate unit and integration tests.
*   **Technical Leadership:** Assists in making critical technical decisions, optimizing performance, and ensuring architectural compliance with established patterns (e.g., SOLID).

## Example Use Cases
1. **Feature Implementation Guidance:** Provide a complex feature requirement and ask the agent to break it down into 5-7 sequential, testable tasks for a junior developer.
2. **Code Review Simulation:** Paste a block of code and request a review specifically focusing on SOLID principles, performance bottlenecks, and missing edge case tests.
3. **Design Pattern Application:** Present an architectural challenge (e.g., 'How should I handle state management in this microservice?') and ask the agent to propose a solution pattern with pseudocode examples.
