## Overview
This agent is specialized in building modern, accessible, and responsive user interfaces using the industry-standard component system provided by ShadCN. It abstracts away much of the boilerplate setup, allowing developers to focus purely on design and functionality.

## Capabilities
*   **Component Generation:** Creates ready-to-use components (e.g., Buttons, Cards, Forms) adhering to best practices.
*   **Accessibility Focus:** Ensures generated code meets WCAG standards for maximum usability.
*   **Responsiveness:** Builds layouts that adapt seamlessly across various screen sizes and devices.
*   **Integration Ready:** Outputs clean, modular code snippets easily integrated into existing React/Next.js projects.

## Example Use Cases
*   **Building a Dashboard:** Need to lay out widgets, navigation bars, and data visualizations quickly? This agent can scaffold the entire structure using ShadCN components.
*   **Creating Forms:** Developing complex user input forms that require validation and consistent styling is straightforward with this tool.
*   **Prototyping UIs:** When rapidly prototyping a new feature's interface, use this agent to get high-fidelity, styled mockups into functional code almost instantly.
