## Overview
The CSS Styling Expert is a specialized AI agent designed to generate, refactor, and audit high-quality, production-ready CSS code. It adheres strictly to modern web development best practices, ensuring that the resulting styles are not only visually appealing but also highly performant, scalable, and accessible across all devices.

## Capabilities
*   **Layout Mastery:** Expert implementation of complex layouts using CSS Grid and Flexbox for both one-dimensional and two-dimensional structures.
*   **Theming & Scalability:** Utilizes CSS Variables (Custom Properties) for dynamic theming and implements methodologies like BEM or CSS Modules for maintainable, modular codebases.
*   **Responsiveness:** Builds mobile-first designs using media queries, ensuring optimal viewing experiences from small screens to large desktops.
*   **Performance Optimization:** Focuses on minimizing file size, optimizing selectors, and suggesting performance improvements (e.g., efficient use of `srcset`).
*   **Accessibility Compliance:** Checks for adequate color contrast ratios and applies semantic styling considerations beneficial for screen readers.

## Example Use Cases
*   **Refactoring Legacy Code:** Paste a large block of messy CSS and ask the agent to refactor it into a modular, BEM-compliant structure using SASS variables.
*   **Implementing Complex Components:** Request a component (e.g., a card gallery or a navigation bar) with specific requirements for responsiveness and animations, and receive fully styled, cross-browser compatible code.
*   **Theming Overhaul:** Provide existing styles and ask the agent to convert them to use CSS Variables so that primary colors and spacing can be updated globally by changing only a few root declarations.
