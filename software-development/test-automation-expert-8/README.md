## Overview
This agent embodies the expertise of a senior test automation engineer, specializing in creating robust and intelligent testing ecosystems. It moves beyond simple scripting by integrating AI capabilities for self-healing tests, predictive failure analysis, and advanced quality engineering principles.

## Capabilities
*   **Test-Driven Development (TDD) Mastery:** Supports full TDD cycles, including generating failing tests, guiding refactoring, tracking cycle metrics, and implementing both state-based (Chicago School) and interaction-based (London School) approaches.
*   **AI-Powered Frameworks:** Implements self-healing test automation using modern tools, leveraging NLP for natural language test case generation and maintenance.
*   **Advanced Validation:** Provides visual AI testing for accurate UI regression detection and utilizes machine learning to predict potential test failures before they occur.
*   **Comprehensive Coverage:** Supports property-based TDD, Behavior-Driven Development (BDD) integration, and advanced techniques like test triangulation for maximum coverage.

## Example Use Cases
1. **Establishing a New Test Suite:** Guide the setup of a new microservice's testing layer by implementing TDD from scratch, ensuring all core business logic is covered with property-based tests.
2. **Maintaining Legacy Tests:** Feed it an existing UI test suite that frequently breaks due to minor UI changes; the agent will suggest and implement self-healing mechanisms using visual AI validation.
3. **Optimizing CI/CD Pipeline:** Integrate its capabilities into a CI/CD pipeline to not only run tests but also monitor TDD compliance metrics, flagging areas where test coverage is lagging or where refactoring debt has accumulated.
