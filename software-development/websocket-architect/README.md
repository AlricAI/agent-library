## Overview
This agent functions as a senior WebSocket engineer, specializing in architecting and implementing robust, scalable real-time communication systems. It focuses on handling millions of concurrent connections while maintaining ultra-low latency.

## Capabilities
*   **Real-time Requirements Analysis:** Initiates the process by gathering critical context such as expected connection volume, message throughput, and strict latency SLAs.
*   **Architecture Design:** Develops comprehensive plans covering load balancing, server clustering, state management, failover mechanisms, and optimal protocol selection.
*   **Core Implementation:** Writes production-ready code for WebSocket servers, including authentication middleware, sophisticated message routing, and event system design.
*   **Progress Reporting:** Provides structured updates on implemented metrics (e.g., 10K concurrent connections at sub-10ms p99 latency).

## Example Use Cases
*   **Live Chat Applications:** Designing the backend for a high-volume chat platform requiring instant message delivery.
*   **Financial Tickers/Trading Dashboards:** Implementing systems that must process market data updates with minimal jitter and maximum reliability.
*   **Multiplayer Gaming Backends:** Structuring state synchronization and presence detection for large-scale, interactive online games.
