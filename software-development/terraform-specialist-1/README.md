## Overview
This agent functions as an expert consultant specializing in advanced Infrastructure as Code (IaC) using Terraform and OpenTofu. It is designed to handle enterprise-scale automation, complex module design, robust state management, and adherence to modern DevOps best practices like GitOps.

Whether you are setting up a new multi-cloud environment or migrating an existing setup, this specialist provides deep technical guidance across the entire IaC lifecycle.

## Capabilities
*   **Core IaC Mastery**: Deep knowledge of resources, data sources, variables, and advanced expressions (e.g., `for_each`, dynamic blocks).
*   **State Management**: Expertise in configuring secure remote backends (S3, Azure, GCS), implementing state locking, and defining comprehensive backup strategies.
*   **Module Architecture**: Guidance on creating reusable, versioned, and tested modules using best-practice composition patterns.
*   **Security & Compliance**: Implementing Policy as Code (PaC) principles and ensuring secure state handling through encryption at rest and in transit.
*   **Workflow Integration**: Assisting with GitOps workflows and CI/CD integration points for automated infrastructure deployment.

## Example Use Cases
1. **Multi-Cloud Setup**: Designing a module structure to deploy networking components consistently across AWS, Azure, and GCP.
2. **State Migration**: Developing a detailed plan and code snippets for migrating an existing Terraform state file from one backend system to another securely.
3. **Advanced Module Design**: Creating a generic, versioned module that can be consumed by different root modules while enforcing specific input contracts.
