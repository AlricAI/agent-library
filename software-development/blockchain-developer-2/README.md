## Overview
This specialized AI agent acts as an expert blockchain developer, focusing on the secure design and implementation of decentralized applications (dApps). It adheres to a security-first mindset, ensuring that all generated code is robust against common Web3 vulnerabilities.

## Capabilities
*   **Smart Contract Development:** Designs and writes secure Solidity contracts with comprehensive inline documentation.
*   **Security Auditing:** Implements industry best practices like reentrancy guards, access controls, and vulnerability prevention measures (e.g., flash loan attack mitigation).
*   **DeFi Protocol Implementation:** Builds complex financial primitives such as AMMs, lending platforms, and staking mechanisms.
*   **Testing & Optimization:** Generates extensive test suites using frameworks like Hardhat or Foundry, alongside gas consumption analysis and optimization recommendations.
*   **Architecture Design:** Assists with cross-chain bridge architecture and Web3 frontend integration examples.

## Example Use Cases
1. **Building a Staking Pool:** Request the agent to create a secure staking contract that handles deposits, rewards distribution, and withdrawal logic, complete with unit tests covering edge cases.
2. **Security Review:** Provide an existing smart contract and ask the agent to perform a security audit checklist, identifying potential vulnerabilities related to external calls or state manipulation.
3. **Cross-Chain Interoperability:** Ask for an architectural blueprint and initial code structure for a simple asset bridge between two simulated chains.
