## Overview
This agent specializes in mastering Tailwind CSS, providing expert assistance for styling modern web applications. It adheres strictly to the utility-first philosophy, ensuring that all generated code is highly maintainable, responsive, and optimized.

Whether you are starting a rapid prototype or optimizing a large-scale component library, this agent leverages Tailwind's comprehensive set of utilities to enforce design consistency and performance best practices.

## Capabilities
*   **Utility Implementation:** Generates styled components using atomic utility classes for maximum efficiency.
*   **Responsive Layouts:** Builds complex, multi-device layouts utilizing the responsive grid system (`sm:`, `lg:`, etc.).
*   **Theming & Customization:** Assists in tailoring Tailwind's default configuration to match specific brand guidelines or design tokens.
*   **Performance Optimization:** Incorporates best practices like ensuring unused styles are purged (e.g., via PurgeCSS considerations).
*   **Typography & Spacing:** Manages consistent visual hierarchy through precise control of padding, margins, and font utilities.

## Example Use Cases
1. **Building a Dashboard Widget:** Provide the structure and required layout for a complex dashboard card, and this agent will return it fully styled with responsive breakpoints and appropriate spacing.
2. **Implementing a Navigation Bar:** Ask for a multi-level navigation bar that must look perfect on mobile, tablet, and desktop screens, ensuring proper hover states are included.
3. **Theming Consistency Check:** If you provide existing CSS or component snippets, this agent can refactor them to adopt Tailwind's utility classes while maintaining the intended visual design.
