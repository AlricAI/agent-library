## Overview
This agent acts as a senior API documenter, specializing in creating world-class, developer-friendly documentation. It ensures your APIs are not only technically accurate but also exceptionally easy for developers to consume and integrate.

The core focus is achieving full OpenAPI 3.1 compliance while providing interactive elements that minimize the learning curve for new users.

## Capabilities
*   **Specification Generation:** Creates comprehensive documentation adhering strictly to OpenAPI 3.1 standards, covering schemas, endpoints, parameters, and security schemes.
*   **Interactive Portals:** Implements 'Try-it-out' functionality, code generation across multiple languages, and robust API explorers.
*   **Comprehensive Coverage:** Ensures 100% endpoint coverage, detailed error documentation (including resolution steps), and clear authentication guides for various flows (OAuth 2.0, API Keys).
*   **Multi-Format Support:** Capable of documenting diverse protocols including REST, GraphQL, WebSockets, gRPC, and Webhooks.

## Example Use Cases
*   **Onboarding New APIs:** Feed it raw endpoint details and receive a fully structured, interactive documentation set ready for immediate developer consumption.
*   **Updating Legacy Docs:** Provide existing API endpoints and user feedback; the agent will analyze gaps and update the documentation to meet modern best practices.
*   **Compliance Auditing:** Use it to validate an existing OpenAPI file against industry best practices, ensuring all error codes, security schemes, and examples are present.
