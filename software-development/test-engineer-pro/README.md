## Overview
Test Engineer Pro is an expert AI agent dedicated to the entire software quality assurance lifecycle. It specializes in designing, implementing, and reviewing comprehensive test suites using industry best practices like Test-Driven Development (TDD). This agent ensures that codebases are reliable, maintainable, and pass rigorous testing before deployment.

## Capabilities
*   **Test Strategy Design:** Develops multi-layered testing plans covering unit, integration, and end-to-end scenarios.
*   **Framework Implementation:** Proficient with modern frameworks such as pytest, Jest, and Playwright for writing clean, readable tests.
*   **Coverage & Quality:** Focuses on achieving high code coverage (targeting >80%) and identifying edge cases.
*   **Performance Testing:** Can assist in designing load and performance test scenarios.
*   **CI/CD Integration:** Structures tests to integrate smoothly into continuous integration pipelines.

## Example Use Cases
1. **Feature Validation:** Provide a new feature's requirements, and the agent will generate a full suite of unit and E2E tests covering happy paths and failure modes.
2. **Refactoring Safety Net:** When refactoring existing code, use this agent to write comprehensive regression tests first, ensuring no existing functionality breaks.
3. **Test Plan Generation:** Input a system architecture diagram or set of user stories, and the agent will output a structured testing approach document detailing required test types and coverage goals.
