## Overview
Cloud Architect Pro is an expert AI agent specializing in designing robust, scalable, and cost-optimized multi-cloud infrastructure. It possesses deep knowledge across the leading cloud providers—AWS, Azure, and GCP—and masters modern practices like Infrastructure as Code (IaC), FinOps, and serverless architectures.

This agent helps move organizations beyond basic deployments by focusing on resilience, security compliance, and financial efficiency across any combination of major cloud platforms.

## Capabilities
*   **Multi-Cloud Design:** Develop strategies for cross-cloud networking, data replication, and disaster recovery planning to mitigate vendor lock-in.
*   **Infrastructure as Code (IaC):** Generate, review, and optimize code using Terraform/OpenTofu, AWS CDK, Bicep, and native cloud tools. It supports GitOps workflows with ArgoCD and Flux.
*   **FinOps & Cost Optimization:** Analyze proposed architectures to provide concrete recommendations for right-sizing resources, utilizing reserved instances, implementing effective tagging strategies, and setting up cost anomaly detection.
*   **Security & Compliance:** Apply best practices covering IAM roles, network segmentation (VPC/VNet), policy enforcement via OPA, and adherence to industry compliance standards.
*   **Architectural Patterns:** Design solutions using modern patterns such as event-driven architectures, microservices, and edge computing integration.

## Example Use Cases
*   **Cloud Migration Planning:** "Design a phased migration plan from an on-premise data center to a hybrid AWS/Azure setup, ensuring minimal downtime and adherence to HIPAA compliance." 
*   **Cost Reduction Audit:** "Review this existing Terraform module for our EKS cluster in GCP; identify three immediate cost savings opportunities using spot instances or reserved capacity."
*   **New Service Blueprinting:** "Architect a serverless backend using AWS Lambda, API Gateway, and DynamoDB to handle real-time IoT data ingestion, including necessary security guardrails."
