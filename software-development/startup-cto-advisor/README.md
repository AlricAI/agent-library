## Overview
Startup CTO Advisor simulates the role of an experienced technical co-founder who has navigated multiple startup lifecycles, including both successes and failures. This agent is designed for founders and early engineering teams that need actionable, battle-tested technical guidance rather than academic theory.

Its core philosophy is simple: ship working software quickly while building scalable habits into the foundation.

## Capabilities
* **Pragmatic Architecture Review:** Assesses proposed systems to ensure they optimize for speed-to-market and minimize rework, avoiding over-engineering.
* **Tech Stack Recommendation:** Advises on technology choices, favoring managed services and proven, boring infrastructure unless a specific tech advantage is required.
* **Engineering Culture Setup:** Helps establish foundational processes like CI/CD, code review standards, and basic monitoring from day one.
* **Due Diligence Preparation:** Provides insights into what investors actually scrutinize during technical due diligence phases.

## Example Use Cases
* **Initial Tech Stack Selection:** "We are building a real-time analytics dashboard for small businesses. Should we use Kafka or managed queues?" The agent will guide you toward the simplest, most reliable starting point.
* **Architecture Review:** Presenting a complex microservices diagram and asking, "What is the absolute minimum viable architecture to prove our core hypothesis in 6 weeks?"
* **Process Improvement:** Asking for best practices on setting up basic observability (logging/alerting) before scaling teams.
