## Overview
This agent embodies the expertise of a senior test automation engineer, specializing in building resilient and intelligent testing ecosystems. It moves beyond simple script execution by incorporating advanced quality engineering principles, AI-powered maintenance, and modern development methodologies.

## Capabilities
* **Test-Driven Development (TDD) Mastery:** Facilitates the entire TDD lifecycle, supporting red-green-refactor cycles, property-based testing, and BDD integration. It also tracks metrics like cycle time to ensure process adherence.
* **Self-Healing Automation:** Implements AI-driven test maintenance using techniques applicable to modern tools (e.g., Testsigma, Applitools), ensuring tests remain functional even when the UI changes.
* **AI-Powered Generation:** Generates comprehensive test cases from natural language specifications and uses machine learning for failure prediction and test optimization.
* **Comprehensive Coverage:** Supports advanced strategies like 'Test Triangulation' and visual AI testing for deep regression detection.
* **Framework Guidance:** Provides guidance on integrating these capabilities into large-scale, maintainable CI/CD pipelines.

## Example Use Cases
1. **New Feature Validation:** Provide a set of requirements in natural language; the agent will generate an initial suite of self-healing, behavior-driven tests and outline the necessary TDD steps to validate them.
2. **Legacy Test Suite Modernization:** Feed it an existing, brittle test suite; the agent will analyze failure patterns, suggest areas for AI enhancement (e.g., visual regression checks), and refactor sections into a more robust framework structure.
3. **Process Improvement Audit:** Ask it to review your current testing workflow against industry best practices, receiving actionable advice on improving TDD compliance monitoring or optimizing test execution speed.
