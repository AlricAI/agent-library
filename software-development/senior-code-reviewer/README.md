## Overview
This agent emulates a Senior Fullstack Code Reviewer with over 15 years of experience. It is designed to move beyond simple syntax checking, providing deep architectural critiques across the entire stack—from frontend UI/UX considerations to backend service design and database integrity.

Use this agent when you need assurance that your code is not only functional but also scalable, secure, maintainable, and adheres to industry best practices.

## Capabilities
*   **Security Analysis:** Identifies vulnerabilities based on standards like OWASP Top 10 (e.g., XSS, SQL Injection).
*   **Architectural Evaluation:** Assesses design patterns, component interaction, and overall system scalability.
*   **Performance Tuning:** Pinpoints potential bottlenecks in algorithms, database queries, and resource usage.
*   **Code Quality Enforcement:** Reviews for readability, adherence to DRY principles, and general maintainability.
*   **Comprehensive Documentation:** Can generate structured documentation (e.g., architecture overviews) when the codebase context warrants it.

## Example Use Cases
*   **Authentication System Review:** Submit a new JWT implementation for review to ensure proper token handling, refresh mechanisms, and authorization checks.
*   **Database Migration Safety Check:** Provide complex SQL or migration scripts to verify transactional safety, idempotency, and potential data loss risks before production deployment.
*   **API Endpoint Design Critique:** Present an API contract (e.g., OpenAPI spec or sample endpoints) for review to ensure RESTful compliance, proper status code usage, and robust input validation.
