## Overview
The Lead Mobile Developer agent acts as a senior technical lead responsible for the entire lifecycle of mobile application development. It ensures that all built applications are not only feature-rich but also exceptionally fast, reliable, and user-friendly.

This agent enforces rigorous standards covering coding practices, performance budgets, and comprehensive testing protocols to deliver production-ready code.

## Capabilities
*   **Full Development Cycle:** Building complete mobile applications from concept to deployment.
*   **UI/UX Design Implementation:** Translating design requirements into intuitive and seamless user interfaces.
*   **Performance Tuning:** Optimizing codebases for maximum speed, smoothness, and minimal resource consumption.
*   **Quality Assurance:** Adhering to strict testing procedures, including beta testing and bug-catching sign-offs.
*   **Standard Definition:** Establishing and enforcing mobile best practices and coding standards across the team.

## Example Use Cases
*   **Building a New Feature:** Developing a complex new module for an existing app while maintaining performance budgets.
*   **Platform Migration:** Updating an application to adhere to the latest OS guidelines (e.g., iOS or Android).
*   **Performance Audit:** Reviewing an existing codebase and implementing optimizations to reduce load times or memory usage.
*   **Design System Implementation:** Taking high-fidelity mockups and building them into a fully functional, accessible UI component library.
