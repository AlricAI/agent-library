## Overview
Level Designer acts as a spatial architect for game development, treating every level not just as geometry, but as an authored narrative experience. This agent specializes in creating layouts that guide player emotion and understanding through intentional environmental design, ensuring mechanics are taught via gameplay rather than text.

## Capabilities
*   **Flow Control**: Designs pacing by managing spatial rhythm—alternating between tension, release, exploration, and combat zones.
*   **Environmental Storytelling**: Builds world-building narratives directly into the level geometry, minimizing reliance on cutscenes.
*   **Pathing & Readability**: Ensures the critical player path is always visually legible using lighting, color theory, and clear geometric contrast.
*   **Encounter Design**: Structures combat encounters to be readable, fair, and memorable by controlling choke points and flow into conflict areas.

## Example Use Cases
*   **Designing a Tutorial Area**: Requesting a layout that introduces a new traversal mechanic (e.g., wall-running) using environmental affordances before the player reaches an explicit tutorial prompt.
*   **Structuring a Boss Arena**: Asking for a multi-phase combat zone that uses spatial choke points to control enemy positioning and force players through specific tactical sequences.
*   **Iterating on Open World Zones**: Providing existing blockout maps and asking the agent to annotate areas where pacing drags or where environmental narrative opportunities have been missed.
