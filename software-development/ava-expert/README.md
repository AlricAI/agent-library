## Overview
This agent is a dedicated expert for the Ava JavaScript testing framework. It focuses on elevating code quality by generating clean, efficient, and highly reliable test suites. Whether you are starting a new feature or refactoring an existing module, this agent ensures your tests adhere to best practices, maximizing coverage while minimizing execution time.

## Capabilities
*   **Test Generation:** Creates concise, effective, and well-structured test cases adhering strictly to Ava's syntax.
*   **Optimization & Performance:** Analyzes test suites for bottlenecks, suggesting refactoring to leverage Ava's concurrency and improve runtime efficiency.
*   **Best Practice Enforcement:** Ensures tests are isolated, independent, and follow the DRY (Don't Repeat Yourself) principle by managing setup/teardown hooks effectively.
*   **Advanced Debugging:** Provides detailed analysis of test failures, offering actionable steps to diagnose root causes in asynchronous or complex scenarios.
*   **Coverage & Reporting:** Aims for high code coverage (>85%) and generates clear reports on potential gaps or performance issues.

## Example Use Cases
1. **New Feature Testing:** Provide a function and ask the agent to generate a complete, comprehensive test file using Ava, including setup/teardown hooks.
2. **Performance Review:** Submit an existing, slow-running test suite and ask the agent to optimize it for concurrency and speed.
3. **Debugging Failures:** Paste an error log from a failed test run and request that the agent debug the failure, providing corrected code and explanations.
