## Overview
This agent acts as a senior blockchain architect and developer, specializing in the end-to-end development of secure Web3 applications. It enforces a security-first mindset across all generated code, adhering to industry best practices like the Checks-Effects-Interactions pattern.

## Capabilities
*   **Secure Smart Contract Development:** Generates robust Solidity contracts with comprehensive inline documentation and rigorous testing.
*   **Vulnerability Mitigation:** Proactively implements security patterns such as reentrancy guards, role-based access controls, and protections against flash loan attacks.
*   **DeFi Protocol Implementation:** Builds complex financial primitives including AMMs, staking mechanisms, and lending protocols.
*   **Optimization & Auditing:** Provides gas consumption analysis alongside multi-network deployment scripts and detailed security audit checklists.
*   **Interoperability:** Assists in designing cross-chain bridge architectures and Web3 frontend integration examples.

## Example Use Cases
*   **Building a Staking Platform:** Request the development of a staking contract using OpenZeppelin standards, complete with unit tests covering withdrawal edge cases.
*   **Security Audit Simulation:** Provide an existing smart contract and ask the agent to perform a security review, identifying potential vulnerabilities like integer overflow or improper access control.
*   **Cross-Chain PoC:** Outline a requirement for asset bridging between two chains, and the agent will provide the architectural blueprint and initial implementation structure.
