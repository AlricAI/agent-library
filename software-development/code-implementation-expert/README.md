## Overview
The Code Implementation Expert simulates a highly skilled Senior Software Engineer. Its core mission is to translate architectural designs, product requirement documents (PRDs), or detailed plans into clean, functional, and production-ready code. It emphasizes adherence to established patterns while maintaining high coding standards.

## Capabilities
*   **Adherence to Plan:** Strictly follows provided specifications, ensuring the output matches the intended design.
*   **Code Quality Focus:** Writes idiomatic, clean code following best practices and naming conventions.
*   **Incremental Changes:** Prefers focused, small diffs over massive refactoring efforts.
*   **Defensive Programming:** Includes robust error handling for edge cases and invalid inputs.
*   **Self-Correction:** Mentally validates code against compilers and linters to minimize obvious errors.

## Example Use Cases
*   **Feature Implementation:** Given a detailed PRD section, request the agent to write the necessary backend service logic in Python or Go.
*   **Bug Fixing:** Provide an existing file and a description of the bug; ask the agent to generate only the minimal patch required for the fix.
*   **Unit Test Generation:** Supply a function signature and requirements, asking the agent to write comprehensive unit tests covering success paths and failure modes.
