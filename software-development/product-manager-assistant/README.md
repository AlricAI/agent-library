## Overview
The Product Manager Assistant is a specialized agent designed to guide product managers, owners, and founders through the entire product lifecycle. It acts as an orchestrator for multiple product skills, helping users move from raw customer insights to actionable, documented product strategies.

This agent focuses on making evidence-based decisions by synthesizing user research, applying established frameworks (like RICE), and generating professional requirement documents without needing deep quantitative expertise.

## Capabilities
*   **Feature Prioritization:** Uses structured methods like the RICE framework to objectively score and rank potential features.
*   **Customer Discovery Analysis:** Analyzes qualitative data from customer interviews to extract key pain points, needs, and opportunities.
*   **PRD Generation:** Creates comprehensive Product Requirement Documents (PRDs) with clear scope, goals, and acceptance criteria.
*   **Roadmap Planning:** Assists in structuring product roadmaps by sequencing features based on business impact and feasibility.
*   **Skill Orchestration:** Integrates multiple specialized product skills into a cohesive workflow for end-to-end support.

## Example Use Cases
*   **New Feature Launch:** Upload raw customer interview transcripts and ask the agent to prioritize the top 5 features using RICE, followed by drafting a preliminary PRD for the highest-ranked item.
*   **Quarterly Planning:** Provide a list of business objectives and several potential initiatives. Ask the agent to structure a quarterly roadmap draft, justifying the sequencing based on dependencies and market impact.
*   **Validation:** Feed in competitor analysis reports and user feedback data, requesting an analysis that pinpoints immediate product gaps and suggests validated next steps.
