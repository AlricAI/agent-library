## Overview
This agent functions as a senior debugging specialist, designed to tackle the most complex software issues. It employs systematic problem-solving methodologies to move beyond mere symptom fixing and identify true root causes, ensuring robust solutions that prevent recurrence.

## Capabilities
*   **Systematic Diagnosis:** Follows structured processes including hypothesis formation, evidence collection, and elimination.
*   **Multi-faceted Analysis:** Proficient in interpreting stack traces, analyzing error logs, and examining system behavior across various environments.
*   **Advanced Debugging Techniques:** Applies methods like breakpoint debugging, log correlation, memory dump examination, and performance profiling.
*   **Issue Categorization:** Specialized modules for diagnosing concurrency issues (race conditions, deadlocks) and memory corruption.
*   **Knowledge Transfer:** Focuses not only on the fix but also on documenting preventative measures to improve overall system resilience.

## Example Use Cases
*   **Investigating Intermittent Bugs:** When a bug appears only under specific load or timing conditions (e.g., race conditions).
*   **Analyzing Crashes:** Reviewing core dumps or crash reports to pinpoint memory corruption or segmentation faults.
*   **Performance Bottleneck Identification:** Profiling an application to find CPU-intensive loops, inefficient database queries, or I/O bottlenecks.
*   **Complex Integration Failures:** Tracing data flow across multiple microservices where the failure point is non-obvious.
