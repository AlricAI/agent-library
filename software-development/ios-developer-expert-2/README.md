## Overview
This AI agent acts as an expert iOS developer, specializing in building modern, high-performance native applications for the Apple ecosystem. It has deep expertise across Swift 6, SwiftUI (including iOS 18 features), and seamless integration with UIKit.

## Capabilities
*   **Modern UI Development:** Mastery of SwiftUI 5.0+ features, including advanced animations, state management (@StateObject, @Binding), and custom view builders.
*   **Architecture & Patterns:** Implements industry-standard patterns like MVVM, Clean Architecture, and the Coordinator pattern for scalable codebases.
*   **Interoperability:** Expert handling of hybrid architectures, allowing seamless integration between SwiftUI and UIKit components.
*   **Performance & Lifecycle:** Optimizes for modern iOS lifecycles (Scene-based) and manages background processing efficiently.
*   **Tooling Proficiency:** Proficient with Xcode 16, Swift Package Manager, and accessibility-first design principles.

## Example Use Cases
*   **Building a New App Feature:** Need to implement a complex screen using SwiftUI that interacts with Core Data persistence? This agent can structure the MVVM layers and write the necessary Combine bindings.
*   **Migrating Legacy Code:** You have an existing app heavily reliant on UIKit, but you want to introduce modern features. Use this agent for best practices on wrapping legacy ViewControllers into a SwiftUI context.
*   **Implementing Complex Navigation:** Developing multi-step flows requiring robust navigation management (e.g., onboarding or checkout processes) using the Coordinator pattern.
