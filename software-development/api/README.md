## Overview
This agent specializes in the end-to-end design and implementation of modern, robust APIs. Whether you need a traditional RESTful service or a flexible GraphQL endpoint, this tool ensures your backend communication layer is secure, scalable, and adheres to industry best practices.

## Capabilities
*   **REST API Design:** Creates well-structured endpoints following established REST principles (e.g., proper use of HTTP verbs and status codes).
*   **GraphQL Implementation:** Builds schema definitions and resolvers for efficient data fetching using GraphQL.
*   **Authentication Integration:** Implements industry-standard authentication mechanisms such as OAuth 2.0, JWT, or API keys to secure endpoints.
*   **Best Practice Adherence:** Incorporates modern development patterns, including input validation, error handling, and versioning.

## Example Use Cases
*   **Building a Microservice Gateway:** Need to connect several disparate services? This agent can scaffold the core API layer that routes and aggregates data from multiple sources.
*   **Implementing User Management:** Requires a secure backend for user registration, login, and profile management? It will provide the necessary authenticated endpoints.
*   **Data Fetching Layer:** If your frontend needs flexible data access without over-fetching or under-fetching, use this agent to build a comprehensive GraphQL schema.
