## Overview
This agent acts as a senior .NET Core architect, specializing in modern development practices using the latest features of .NET 10 and C# 14. It is designed to guide the implementation of robust, high-performance, and scalable cloud-native applications.

It enforces best practices across architecture (Clean Architecture), deployment (Containerization), and code quality (High Test Coverage).

## Capabilities
*   **Architecture Design:** Implements patterns like CQRS and Repository pattern across Domain, Application, Infrastructure, and Presentation layers.
*   **API Development:** Masters Minimal APIs for rapid endpoint creation, including full OpenAPI/Swagger documentation and robust validation.
*   **Microservices Implementation:** Guides the setup of service discovery, API gateways, resilience patterns (Circuit Breakers), and distributed tracing.
*   **Modern C# Utilization:** Expertly leverages features such as Record types, Pattern matching, Source generators, and Top-level programs for cleaner code.
*   **Performance & Cloud Readiness:** Ensures solutions are optimized for AOT compilation, container deployment, and cloud scalability.

## Example Use Cases
*   **Building a Scalable Backend:** Need to design a multi-service e-commerce platform? This agent will structure the domain models, define inter-service communication via an event bus, and implement resilient API endpoints.
*   **Refactoring Legacy Code:** If you have an existing monolithic ASP.NET Core application, use this agent to guide the extraction of bounded contexts into independent microservices while maintaining data integrity.
*   **Implementing Complex Business Logic:** Requires a service that handles complex state changes? It will structure the logic using MediatR and CQRS within a clean architectural boundary.
