## Overview
This agent is your dedicated expert for all things Docusaurus. It possesses deep knowledge of the framework's configuration, theming system, and build pipeline, making it ideal for maintaining or building complex documentation websites.

## Capabilities
*   **Configuration Analysis:** Thoroughly examines `docusaurus.config.js` and related files for syntax errors, version incompatibilities, and best practices adherence.
*   **Structure Validation:** Analyzes sidebars (`sidebars.js`) to ensure correct document linking, category organization, and navigation flow.
*   **Troubleshooting:** Provides step-by-step debugging guides for common build failures, deployment issues, and rendering errors.
*   **Theming & Styling:** Offers specific guidance on custom CSS/SCSS integration, component overrides, and theme customization within the Docusaurus ecosystem.
*   **Content Guidance:** Advises on best practices for writing content using MDX and standard Markdown formats to ensure optimal rendering.

## Example Use Cases
*   **Build Failure Debugging:** If your site fails to build with `npm run build`, provide the error log, and this agent will pinpoint the likely configuration file causing the issue and suggest a fix.
*   **Navigation Overhaul:** Need to restructure your documentation from a flat list to a multi-level sidebar? Provide your current structure, and it will generate optimized `sidebars.js` entries.
*   **Theming Implementation:** Want to add a custom color palette or component globally? Describe the desired change, and it will provide the necessary code snippets for theme overrides.
