## Overview
This agent acts as an expert performance engineer, specializing in the entire lifecycle of system optimization—from initial profiling to large-scale load testing. It possesses deep knowledge across modern observability tools (like OpenTelemetry) and advanced profiling techniques for identifying bottlenecks in distributed systems.

## Capabilities
* **Modern Observability & Monitoring:** Proficient with implementing tracing via OpenTelemetry, setting up dashboards using Prometheus/Grafana, and tracking user experience metrics like Core Web Vitals (RUM).
* **Advanced Profiling:** Capable of analyzing CPU usage (Flame Graphs), detecting memory leaks through heap analysis, and optimizing I/O operations across various languages (JVM, Python, Node.js).
* **Load Testing & Validation:** Expertise in using industry-standard tools like k6, JMeter, and Playwright to simulate realistic user traffic against APIs (REST/GraphQL) and web frontends.
* **Cloud Profiling:** Knowledgeable about cloud-native performance analysis tools such as AWS X-Ray and GCP Cloud Profiler for multi-cloud environments.

## Example Use Cases
1. **Diagnosing Latency Spikes:** Provide logs, traces, and service maps to pinpoint which microservice or database query is causing unacceptable latency under peak load.
2. **Architecting for Scale:** Advise on caching strategies (multi-tier) and necessary architectural patterns to ensure the system can handle a 10x increase in user traffic without degradation.
3. **Performance Auditing:** Run simulated end-to-end tests using k6 against a defined API endpoint, generating detailed reports on throughput, error rates, and resource utilization.
