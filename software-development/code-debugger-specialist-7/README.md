## Overview
This agent functions as a senior debugging specialist, designed to systematically diagnose complex software issues. It moves beyond simple error message reading by analyzing system behavior, data flows, and environmental factors to pinpoint the true root cause of bugs.

Its methodology combines rigorous diagnostic checklists with advanced debugging techniques to ensure not only that the bug is fixed, but that preventative measures are documented for future development cycles.

## Capabilities
*   **Systematic Diagnosis:** Follows a structured approach including symptom analysis, hypothesis formation, and systematic elimination.
*   **Advanced Analysis:** Proficient in interpreting stack traces, core dumps, memory leaks, race conditions, and performance bottlenecks.
*   **Technique Mastery:** Applies various debugging techniques like breakpoint setting, log correlation, and differential debugging.
*   **Knowledge Transfer:** Focuses on documenting the root cause and implementing prevention measures to avoid recurrence.

## Example Use Cases
*   **Investigating Intermittent Failures:** When a bug only appears under specific load conditions (e.g., race conditions or timing issues).
*   **Memory Corruption Debugging:** Analyzing heap dumps or core files to find use-after-free errors or buffer overflows.
*   **Performance Bottleneck Identification:** Profiling an application to locate CPU-intensive loops, inefficient database queries, or I/O bottlenecks.
*   **Complex Integration Bugs:** Tracing data flow across multiple microservices or environmental boundaries to isolate the point of failure.
