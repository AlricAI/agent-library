## Overview
The Frontend Design Architect agent is specialized in bridging the gap between visual design and functional code. It takes abstract concepts—such as Figma mockups, raw wireframes, or collections of screenshots—and transforms them into actionable, detailed technical specifications.

This agent acts as a virtual senior frontend architect, ensuring that the final implementation will be pixel-perfect, scalable, and maintainable by adhering to modern design system principles.

## Capabilities
*   **Design Analysis:** Interprets visual inputs (mockups/screenshots) to understand layout, components, and user flow.
*   **Component Architecture Generation:** Breaks down complex UIs into reusable, atomic components suitable for frameworks like React or Vue.
*   **Technical Specification Output:** Produces comprehensive documentation detailing props, states, interactions, and necessary dependencies.
*   **Design System Extraction:** Analyzes multiple UI samples to identify common patterns, color palettes, typography scales, and component guidelines for establishing a robust design system.

## Example Use Cases
*   **Mockup Implementation Guide:** Provide a Figma link or image of a dashboard and receive a step-by-step guide on how to build it in React, including suggested file structures and necessary hooks.
*   **Design System Audit:** Upload screenshots from various parts of an existing application. The agent will analyze them and output a formal Design System Specification document covering tokens (colors, spacing) and component guidelines.
*   **Wireframe Structuring:** Give the agent a hand-drawn or simple wireframe sketch for a user profile page. It will return a structured component tree, detailing which components need to be built and how they should interact.
