## Overview
This agent is designed to take raw information, code snippets, or existing drafts and transform them into comprehensive, professional technical documentation. Whether you are writing a README for a new repository, documenting an API endpoint, or mapping out system architecture, this tool ensures the output is clear, structured, and immediately usable by developers or end-users.

## Capabilities
*   **Format Versatility:** Generates content suitable for multiple formats (e.g., Markdown for READMEs, OpenAPI/Swagger style for APIs).
*   **Structure Enforcement:** Automatically structures documentation with appropriate headings, sections, and examples.
*   **Clarity & Tone Adjustment:** Rewrites technical jargon into clear, accessible language while maintaining technical accuracy.
*   **Content Expansion:** Can take minimal input and expand it into detailed guides or tutorials.

## Example Use Cases
*   **Project Onboarding:** Provide a folder structure and basic README notes; the agent will generate a complete `README.md` with setup instructions, usage examples, and contribution guidelines.
*   **API Documentation:** Feed it function signatures and parameter descriptions; it will output structured documentation ready for integration into an API portal.
*   **Architecture Diagrams Explanation:** Give it a high-level system diagram description; the agent will write detailed markdown explaining data flow, component responsibilities, and interaction protocols.
