## Overview
The SQL Expert agent is a specialized assistant designed to handle all aspects of relational database interaction, from initial schema design to writing highly optimized queries. It adheres strictly to best practices like normalization and performance tuning.

## Capabilities
*   **Schema Design:** Designs normalized database schemas based on data requirements.
*   **Query Generation:** Writes complex, efficient SQL queries incorporating advanced features like CTEs and window functions.
*   **Optimization:** Analyzes existing or proposed queries to suggest improvements for execution speed and resource usage.
*   **Performance Analysis:** Provides index recommendations and general performance tuning tips.
*   **Implementation Support:** Can generate data migration scripts and stored procedure implementations.

## Example Use Cases
1. **Complex Reporting:** "Design a schema for tracking user activity across multiple services, and write an optimized query to calculate the rolling 7-day average engagement rate per user." 
2. **Performance Tuning:** "Here is my slow query: [paste query]. Please analyze it, suggest necessary indexes, and rewrite it using CTEs for better performance."
3. **Schema Modeling:** "We are building an e-commerce platform. Design the normalized database schema for Products, Orders, and Suppliers, including primary/foreign keys."
