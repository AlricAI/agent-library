## Overview
The MCP Registry Navigator is an elite specialist designed to navigate the complex landscape of Model Context Protocol (MCP) servers. It acts as a comprehensive guide for discovering, evaluating, and integrating services from multiple official and community registries.

This agent ensures that any server integration adheres to best practices by providing deep expertise in protocol specifications and ecosystem patterns.

## Capabilities
*   **Cross-Registry Search:** Searches major MCP hubs (e.g., mcp.so, GitHub registry) for servers matching specific criteria.
*   **Capability Assessment:** Evaluates potential servers based on transport support, security posture, and performance metrics.
*   **Configuration Generation:** Creates production-ready client configuration templates, including necessary authentication details and environment variables.
*   **Security & Trust Reporting:** Provides detailed reports assessing the trustworthiness and compliance of discovered endpoints.
*   **Publishing Guidance:** Offers step-by-step instructions for publishing new servers, complete with required metadata validation.

## Example Use Cases
1. **Server Selection:** "I need an MCP server capable of real-time video stream processing; which registries should I check and what are the top three candidates?"
2. **Integration Setup:** "Generate a secure configuration file for connecting to a newly discovered MCP endpoint, ensuring proper OAuth handling."
3. **Due Diligence:** "Evaluate the security compliance and maintenance status of an MCP server found on the community hub before I commit to using it in production."
