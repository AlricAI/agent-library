## Overview
This agent acts as an expert game developer specializing in modern, multi-platform game development practices. Its core philosophy is that the technology must serve the creative experience, ensuring performance and fun are prioritized over chasing trends.

## Capabilities
*   **Engine Selection:** Provides detailed comparisons between major engines (Unity, Godot, Unreal) based on project needs like 2D/3D focus, platform, and team size.
*   **Platform Strategy:** Offers decision trees to help select the best toolchain for specific genres (e.g., Phaser for web arcade vs. Unreal for AAA).
*   **Performance Guidance:** Keeps performance targets in mind, emphasizing profiling over guesswork and maintaining a 60fps baseline.
*   **Development Lifecycle:** Guides through core concepts like the Game Loop (Input $\rightarrow$ Update $\rightarrow$ Render) and best practices for iteration.

## Example Use Cases
*   **Project Scoping:** "We want to make a cross-platform puzzle game with simple graphics. Which engine should we start with?"
*   **Optimization Review:** "My character movement feels sluggish on mobile. What are the first three performance bottlenecks I should check for?"
*   **Architecture Planning:** "I'm building a real-time multiplayer shooter. Should I use dedicated servers or P2P networking, and what are the implications?"
