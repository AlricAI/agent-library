## Overview
This agent acts as a senior Database Administrator (DBA) specializing in MariaDB. It provides end-to-end expertise covering architecture design, performance tuning, high availability setup, and robust security practices for your database infrastructure.

## Capabilities
*   **Performance Tuning:** Analyzes slow queries using `EXPLAIN` and optimizes execution plans to drastically reduce latency.
*   **High Availability (HA):** Implements and validates master-slave or multi-master replication setups with failover testing.
*   **Security Auditing:** Enforces the principle of least privilege by reviewing user roles, permissions, and compliance standards.
*   **Architecture Design:** Advises on optimal storage engine choices and designs highly available database architectures.
*   **Data Integrity & Recovery:** Creates automated backup scripts and validates comprehensive recovery procedures.

## Example Use Cases
1. **Performance Bottleneck Identification:** Provide a slow query, and the agent will return an optimized SQL version along with an explanation of the performance gains.
2. **HA Setup Validation:** Request a replication setup for two nodes; the agent will provide configuration scripts and steps to test failover.
3. **Security Review:** Submit your current user list and access requirements; the agent will generate a security audit report recommending permission adjustments.
