## Overview
TypeScript Pro is an AI agent engineered to function as a senior TypeScript architect, specializing in versions 5.4 and newer (with a preference for 5.6+). This agent excels at handling the most complex aspects of the TypeScript type system, ensuring codebases are robust, highly performant, and maintainable.

## Capabilities
*   **Advanced Type System Mastery:** Generates solutions utilizing intricate features like conditional types, mapped types, template literal types, and recursive types.
*   **Generics Implementation:** Designs reusable components using advanced generic constraints to ensure type safety across varied inputs.
*   **Architectural Guidance:** Provides best practices for structuring large-scale applications, including patterns for state management and module separation.
*   **Type Inference Optimization:** Analyzes existing code to suggest improvements that maximize TypeScript's inference capabilities while maintaining strictness.
*   **Decorator Handling:** Implements and explains the use of decorators in an enterprise context.

## Example Use Cases
1. **Complex Data Modeling:** Need to define a highly polymorphic data structure where different subtypes must adhere to specific, varying constraints? Ask for a solution using conditional types and generics.
2. **Library Development:** Building a reusable utility library that needs to work correctly with various input types (e.g., an ORM layer)? Use this agent to ensure all type guards and interfaces are perfectly defined.
3. **Refactoring Legacy Code:** You have older TypeScript code that is losing type safety? Provide the snippet, and the agent will refactor it using modern TS features for maximum strictness.
