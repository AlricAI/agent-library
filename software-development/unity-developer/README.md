## Overview
This expert agent specializes in end-to-end game development within the Unity ecosystem. It possesses deep knowledge of Unity 6 LTS, modern rendering pipelines (URP/HDRP), and advanced C# programming required for creating scalable, high-performance, and cross-platform games.

## Capabilities
*   **Core Mastery:** Expertise in Unity Editor workflows, package management, and version control integration (Git).
*   **Rendering Pipelines:** Proficient in optimizing both Universal Render Pipeline (URP) and High Definition Render Pipeline (HDRP), including custom shader work via Shader Graph and HLSL.
*   **Performance Tuning:** Masters the use of the Unity Profiler for deep analysis of CPU, GPU, and memory bottlenecks, implementing solutions like LOD systems and advanced culling techniques.
*   **Programming:** Writes clean, optimized C# code for complex gameplay mechanics, UI implementation, and asset management.

## Example Use Cases
*   **Performance Debugging:** When your game frame rate drops on mobile devices, ask this agent to analyze the bottleneck using profiling best practices.
*   **Graphics Implementation:** Need a visually stunning scene? Request setup guidance for HDRP lighting or custom post-processing effects.
*   **System Architecture:** Building out core gameplay loops, such as an inventory system or character movement controller, while ensuring cross-platform compatibility.
