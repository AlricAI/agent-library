## Overview
The MongoDB Database Expert is a specialized AI agent designed to assist with every facet of working with MongoDB. It goes beyond simple query writing, focusing on architectural best practices for performance, scalability, and data integrity.

This agent ensures that your database structure and queries are not just functional, but highly optimized for real-world application demands.

## Capabilities
*   **Query Optimization:** Analyzes existing queries to recommend necessary indexes and structural improvements.
*   **Schema Design:** Provides best-practice schema blueprints tailored to specific access patterns (e.g., embedding vs. referencing).
*   **Performance Tuning:** Develops efficient aggregation pipelines and advises on appropriate indexing strategies.
*   **Scalability Planning:** Guides setup for replica sets, sharding keys, and high availability configurations.
*   **Data Integrity:** Provides guidance on implementing transactions and designing robust backup/restore procedures.

## Example Use Cases
*   **Performance Bottleneck Identification:** Provide a slow query and the agent will return optimized alternatives with required index definitions.
*   **New Feature Data Modeling:** Describe a new application feature, and the agent will propose an optimal MongoDB schema design, justifying its choices based on read/write patterns.
*   **Scaling Strategy Consultation:** If you anticipate massive data growth, ask for a sharding strategy recommendation based on your projected access keys.
