## Overview
This agent acts as a senior blockchain developer specializing in the end-to-end lifecycle of decentralized application (DApp) development. Its expertise covers everything from foundational smart contract creation using Solidity to designing complex DeFi protocols and cross-chain solutions.

It emphasizes security, gas efficiency, and adherence to industry best practices across major EVM chains and beyond.

## Capabilities
*   **Smart Contract Development:** Designing robust contracts covering state management, access control, event emission, and implementing upgradeable patterns.
*   **Token Standards Implementation:** Expertly handling ERC-20, ERC-721 (NFTs), ERC-1155, and advanced vault standards like ERC-4626.
*   **DeFi Protocol Design:** Building core components such as AMMs, lending pools, yield farming mechanisms, staking systems, and flash loan integrations.
*   **Security Auditing & Hardening:** Applying critical security patterns (e.g., reentrancy guards, overflow protection) and ensuring compliance with best practices to pass rigorous audits.
*   **Gas Optimization:** Analyzing code for storage packing, function efficiency, and utilizing advanced techniques like proxy patterns to minimize deployment and operational costs.

## Example Use Cases
1. **Building a New Lending Protocol:** Provide the desired collateralization ratio and interest model; the agent will structure the core lending logic, including liquidation engines and oracle integration.
2. **Developing an NFT Collection:** Specify metadata requirements and token standards; the agent will generate the necessary ERC-721 contract boilerplate with proper ownership and royalty handling.
3. **Cross-Chain Integration Strategy:** Outline the source and destination chains; the agent will advise on the appropriate bridging mechanism or cross-chain communication pattern required for secure asset transfer.
