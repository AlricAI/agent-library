## Overview
This agent acts as a dedicated Developer Experience (DX) specialist, focusing on making the entire development lifecycle seamless and joyful. Its core mission is to identify and eliminate friction points—from initial setup to daily coding tasks—so that developers can focus purely on writing code.

## Capabilities
*   **Environment Setup:** Streamlines onboarding processes to under five minutes by automating dependency installation and setting intelligent defaults.
*   **Workflow Optimization:** Analyzes repetitive tasks to suggest automation scripts, useful aliases, and shortcuts, significantly improving build/test times.
*   **Tooling Enhancement:** Configures IDE settings, implements necessary git hooks, and creates project-specific CLI commands for consistency.
*   **Documentation Generation:** Creates functional setup guides and interactive examples that developers can trust immediately.

## Example Use Cases
1. **New Project Onboarding:** Provide the agent with a boilerplate structure; it will generate a comprehensive `README` and initial scripts to get the project running end-to-end in minutes.
2. **Performance Bottleneck Identification:** If build times are slow, use this agent to profile the current workflow, suggest parallelization strategies, or recommend caching layers.
3. **Team Process Improvement:** When team feedback indicates setup confusion, the agent will generate standardized `Makefile`s and configuration files to enforce best practices across all contributors.
