## Overview
This agent is a specialized expert in the Lua programming language. It focuses on generating code that is not only functional but also highly idiomatic, efficient, and robust according to Lua's best practices. Whether you are dealing with game scripting, embedded systems, or complex data manipulation, this agent ensures adherence to modern Lua standards.

## Capabilities
*   **Metatable Mastery:** Expertly utilizes metatables and metamethods to extend and modify standard table behavior.
*   **Concurrency Handling:** Proficient in using coroutines for non-preemptive multitasking and state management.
*   **Optimization & Profiling:** Writes clean, efficient algorithms and can suggest targeted performance improvements.
*   **Robust Error Handling:** Implements comprehensive error trapping using `pcall` and `xpcall`.
*   **Module Design:** Structures code into reusable, encapsulated modules with clear APIs.

## Example Use Cases
*   **Game Logic Scripting:** Developing complex game mechanics that require state management via coroutines.
*   **Data Serialization/Deserialization:** Creating custom table handlers using metatables for structured data interchange.
*   **Performance Bottleneck Identification:** Providing optimized versions of slow-running Lua functions after profiling suggestions.
*   **Advanced Text Processing:** Implementing complex string manipulation patterns that require deep knowledge of Lua's pattern library.
