## Overview
Api Documenter Pro is an expert AI agent designed to master the creation of world-class, comprehensive, and interactive API documentation. It moves beyond simple reference guides by integrating modern developer experience (DX) practices, ensuring that your documentation actively drives API adoption and minimizes developer integration time.

This tool specializes in adhering to industry standards like OpenAPI 3.1+ while leveraging AI capabilities for content generation and maintenance.

## Capabilities
*   **Modern Standards Mastery:** Authoring specifications using OpenAPI 3.1+, supporting AsyncAPI, GraphQL SDL, and JSON Schema validation.
*   **AI-Powered Content Generation:** Automatically generates developer-friendly explanations, code examples in multiple languages, and keeps documentation consistent from source code annotations.
*   **Interactive Platform Building:** Creates fully functional documentation portals featuring Swagger UI/Redoc customization, live 'Try-it-now' endpoints with authentication handling, and custom onboarding tutorials.
*   **Workflow Automation:** Manages the entire API lifecycle documentation—from initial design (API-first) through to deprecation notices.

## Example Use Cases
1. **New Product Launch Documentation:** Feed it your raw API endpoint definitions; it will output a fully structured, interactive developer portal ready for public consumption.
2. **Legacy System Modernization:** Provide existing code comments and endpoints; the agent will generate compliant OpenAPI specs and draft comprehensive tutorials explaining usage patterns.
3. **Internal Developer Onboarding:** Use it to build an internal knowledge base that guides new engineers through complex microservices APIs with runnable examples.
