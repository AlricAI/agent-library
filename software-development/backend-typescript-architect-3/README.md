## Overview
This agent functions as a Senior Backend TypeScript Architect, providing expert-level guidance and code generation for modern server-side applications. It operates with the rigor of a seasoned engineer, prioritizing clean architecture, maintainability, and performance.

It is specialized in leveraging the power and speed of the Bun runtime within a strict TypeScript environment to build resilient backend systems.

## Capabilities
*   **Advanced TypeScript Implementation:** Generates code adhering to modern best practices, including complex type definitions and generics.
*   **API Design:** Designs RESTful or GraphQL endpoints with clear request/response schemas.
*   **Database Integration:** Provides boilerplate and logic for connecting to and interacting with various databases (e.g., using ORMs).
*   **Performance Optimization:** Reviews existing code snippets to identify bottlenecks and suggest high-performance refactors suitable for Bun's capabilities.
*   **Architectural Patterning:** Implements patterns like Clean Architecture, Hexagonal Architecture, or layered services.

## Example Use Cases
1. **Building a Microservice Endpoint:** Ask it to scaffold a complete CRUD endpoint in TypeScript using Express/Fastify with database connection handling.
2. **Refactoring Legacy Code:** Provide an existing service file and ask it to refactor it to improve type safety, modularity, and adherence to modern TS patterns.
3. **Designing Data Models:** Request the optimal schema design (including migrations) for a new feature that requires complex relational data storage.
