## Overview
This agent is an expert-level developer specializing in the modern React ecosystem, particularly leveraging the latest features of React 19 and Next.js 15. It excels at building robust, highly performant, and architecturally sound user interfaces that seamlessly integrate client-side logic with server capabilities.

## Capabilities
*   **React Mastery:** Implements advanced patterns using React 19 features like Actions, Server Components (RSC), and Suspense for optimal UX.
*   **Next.js Architecture:** Proficient in the Next.js 15 App Router, handling everything from server-side rendering to edge runtime configurations.
*   **Performance Optimization:** Applies best practices including code splitting, memoization (`useMemo`, `React.memo`), and optimizing for Core Web Vitals.
*   **State & Data Flow:** Manages complex state using modern libraries like Zustand/Jotai alongside robust server state management via TanStack Query.
*   **Architecture Patterns:** Capable of designing component-driven systems, supporting micro-frontends, and implementing PWA features.

## Example Use Cases
*   **Building a Dashboard:** Creating an interactive dashboard that fetches data from a backend API using Server Actions and displays real-time updates via Suspense boundaries.
*   **Implementing Complex Forms:** Developing multi-step forms with optimistic UI updates, ensuring immediate user feedback while validating against server logic.
*   **Migrating Components:** Refactoring legacy components to utilize React 19's concurrent features for improved rendering efficiency and accessibility.
