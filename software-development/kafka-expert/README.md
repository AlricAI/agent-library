## Overview
This agent specializes in the end-to-end design, implementation, and optimization of Apache Kafka ecosystems. It covers everything from initial cluster setup to advanced real-time stream processing using Kafka Streams.

It adheres to industry best practices for ensuring high throughput, fault tolerance, and data integrity across complex messaging pipelines.

## Capabilities
*   **Cluster Design:** Generates optimal broker configurations, including heap sizing and resource planning.
*   **Scalability Planning:** Develops robust topic partitioning strategies based on anticipated load patterns.
*   **Producer/Consumer Optimization:** Writes code examples incorporating idempotency, transactional guarantees, and effective backpressure handling.
*   **Stream Processing:** Implements complex data transformation pipelines using the Kafka Streams DSL with proper error handling.
*   **Security & Resilience:** Configures security measures (TLS, SASL) and designs failover/replication strategies for maximum uptime.
*   **Monitoring & Auditing:** Provides blueprints for JMX monitoring setups and necessary ACL auditing procedures.

## Example Use Cases
1. **New System Design:** "Design a Kafka architecture to handle 10,000 events per second from IoT sensors, ensuring at-least-once delivery." 
2. **Performance Tuning:** "Review this producer code snippet and suggest improvements for batching and retry logic."
3. **Troubleshooting:** "We are seeing consumer lag spikes; analyze the potential causes related to partitioning or consumer group coordination."
