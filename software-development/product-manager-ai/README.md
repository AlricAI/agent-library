## Overview
This Product Manager AI agent acts as a dedicated specialist to bridge the gap between high-level business goals and actionable technical specifications. It is designed to guide product development through structured processes like requirements analysis, user story creation, and strategic roadmap planning.

By adhering to a user-centric philosophy, this agent ensures that all features are grounded in solving real 'Jobs to be Done' for the end-user.

## Capabilities
* **Requirements Analysis**: Systematically gathers and documents comprehensive product requirements from vague business needs.
* **User Story Generation**: Writes clear, actionable user stories complete with detailed acceptance criteria using standard templates.
* **Roadmap Management**: Creates and maintains strategic, prioritized product roadmaps over time.
* **Prioritization Frameworks**: Applies industry-standard techniques like MoSCoW or RICE to help stakeholders decide what to build first.
* **Cross-Functional Documentation**: Provides technical notes (API endpoints, DB changes) alongside design requirements for seamless handoff to engineering teams.

## Example Use Cases
1. **New Feature Definition**: Provide a high-level concept (e.g., 'We need better reporting') and ask the agent to generate a prioritized roadmap, starting with 3 key user stories and their acceptance criteria.
2. **Backlog Grooming**: Feed it existing requirements and ask it to apply the RICE framework to help prioritize the backlog for the next sprint cycle.
3. **Stakeholder Alignment**: Use it to structure a meeting agenda where you need to balance technical feasibility against business value, ensuring all decisions are documented in user story format.
