## Overview
This expert agent specializes in the full lifecycle of Model Context Protocol (MCP) server development. It possesses deep, up-to-date knowledge of the MCP specification and best practices for building production-ready AI service backends.

Whether you are starting a new microservice or upgrading an existing one, this architect ensures adherence to modern standards, focusing on robustness, security, and performance.

## Capabilities
*   **Full Lifecycle Implementation:** Designs and builds MCP servers from initial concept through deployment readiness.
*   **Transport Layer Support:** Implements both `stdio` and streaming HTTP transports with comprehensive error handling.
*   **Advanced Features:** Automatically integrates tool/resource definitions, completion endpoints, batching support, and session management.
*   **Security Hardening:** Enforces best practices including input validation, rate limiting, CORS policies, and secure session ID generation.
*   **Code Generation:** Provides complete, production-ready code in TypeScript (using `@modelcontextprotocol/sdk`) or Python with full type coverage.

## Example Use Cases
*   **Building a New Service:** Need to create an entirely new backend service that interacts with multiple external tools via the MCP standard.
*   **Upgrading Connectivity:** An existing server needs to be updated to support streaming HTTP connections while maintaining `stdio` compatibility.
*   **Adding Functionality:** You require adding complex, annotated tool definitions (e.g., read-only or destructive actions) to an established server structure.
