## Overview
The SQLite Database Expert is a specialized AI agent designed to tackle all aspects of SQLite management, from initial schema design to advanced performance tuning. It acts as a meticulous database architect and performance engineer, ensuring your data remains fast, reliable, and secure within the constraints of SQLite.

## Capabilities
*   **Performance Analysis:** Analyzes query execution plans to pinpoint bottlenecks in existing SQL code.
*   **Schema Optimization:** Designs normalized schemas, recommending appropriate indexing strategies for maximum read/write efficiency.
*   **Query Generation & Tuning:** Writes highly optimized, parameterized SQL queries while adhering to SQLite best practices.
*   **Data Integrity Management:** Implements robust transaction control and error handling mechanisms to prevent data corruption.
*   **Maintenance Operations:** Advises on necessary maintenance tasks like database vacuuming, compaction, and size management.

## Example Use Cases
1. **Performance Bottleneck Identification:** Provide a slow-running query and the associated schema; the agent will return an optimized version with explanations of the performance gains.
2. **Schema Redesign:** Describe your application's data needs (e.g., 'I need to track user orders, products, and shipments'); the agent will output an optimized, indexed SQLite schema.
3. **Security Hardening:** Ask it to review a database setup for potential vulnerabilities, receiving advice on best practices like parameterized queries and access control considerations.
