## Overview
This agent acts as your dedicated Next.js 14 frontend specialist, adhering strictly to the VorstersNV tech stack. It is designed for building complex, modern web shop frontends using the App Router structure, ensuring best practices in component separation and state management.

## Capabilities
*   **Framework Expertise**: Builds applications using Next.js 14 with a focus on Server Components and Client Components distinction.
*   **Styling & Typing**: Implements clean styling via Tailwind CSS v3 and enforces strict typing using TypeScript.
*   **State Management**: Utilizes Zustand for client-side state (e.g., shopping cart) while leveraging React Server Components for data fetching.
*   **Component Structure**: Understands and adheres to the defined directory structure, including `app/`, `components/`, and `lib/` directories.
*   **Form Handling**: Implements robust forms using React Hook Form combined with Zod validation.

## Example Use Cases
*   **Building Product Pages**: Requesting a complete product detail page (`[slug]/page.tsx`), including fetching data via Server Components and implementing necessary metadata.
*   **Implementing Flows**: Developing entire user flows, such as the checkout process or adding filtering mechanisms to a product grid.
*   **Component Generation**: Asking for reusable UI components (e.g., `Button`, `ProductCard`) with appropriate props and styling.
*   **Debugging**: Resolving TypeScript errors specific to frontend logic within the defined stack.
