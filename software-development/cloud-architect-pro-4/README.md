## Overview
Cloud Architect Pro is an expert AI agent specializing in designing, implementing, and optimizing highly scalable, cost-effective cloud infrastructure across AWS, Azure, and GCP. It follows modern DevOps principles by prioritizing Infrastructure as Code (IaC) to ensure repeatability, reliability, and security.

This agent moves beyond simple deployment scripts; it provides comprehensive architectural blueprints that account for operational resilience, cost management, and compliance from day one.

## Capabilities
*   **Cost-Optimized Design:** Analyzes requirements to select the most economical services while maintaining necessary performance levels.
*   **Infrastructure as Code (IaC):** Generates production-ready Terraform modules with proper state management for seamless deployment.
*   **Resiliency Planning:** Designs for failure using multi-AZ and multi-region redundancy, complete with Disaster Recovery runbooks (RTO/RPO).
*   **Security Implementation:** Applies security best practices by default, enforcing least privilege IAM roles and robust network configurations.
*   **Observability & Cost Control:** Sets up comprehensive monitoring, alerting mechanisms, and detailed monthly cost estimations.

## Example Use Cases
1. **New Application Deployment:** Provide use cases for deploying a new microservices application, receiving a complete Terraform module set, architecture diagram (Mermaid), and initial cost breakdown.
2. **Cost Optimization Audit:** Feed in existing cloud resource usage logs and receive actionable recommendations to right-size services, suggesting specific savings opportunities.
3. **Cloud Migration Planning:** Outline a migration from an on-premise data center to a multi-cloud setup, receiving phased deployment plans, networking blueprints, and security hardening checklists.
