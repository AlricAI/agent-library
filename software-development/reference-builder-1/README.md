## Overview
Reference Builder acts as a specialized documentation specialist designed to transform raw technical inputs into comprehensive, structured, and highly searchable reference materials. It ensures that every parameter, method signature, configuration setting, and edge case is documented with precision, creating a true 'source of truth' document.

This agent excels at imposing rigorous structure on complex technical specifications, making the output immediately usable by developers or system architects.

## Capabilities
*   **Exhaustive Coverage**: Documents every detail, including parameters, return types, and error codes.
*   **Structured Formatting**: Adheres to a strict markdown template for consistency (e.g., Type, Default, Required).
*   **Cross-Referencing**: Identifies and links related concepts within the documentation set.
*   **Multiple Formats**: Capable of generating API References, Configuration Guides, and Schema Documentation.
*   **Example Generation**: Includes multiple practical examples for every documented feature to aid understanding.

## Example Use Cases
1. **API Deep Dive**: Feed it a library's source code or OpenAPI spec to generate a complete, runnable reference manual with all endpoints, parameters, and error handling.
2. **System Onboarding**: Provide configuration files and architectural diagrams to build a comprehensive Configuration Guide detailing every setting, its default value, and migration path.
3. **Schema Validation**: Input database schemas or JSON models to receive detailed documentation covering field types, constraints, and relationships.
