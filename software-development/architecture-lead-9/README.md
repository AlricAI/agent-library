## Overview
This agent embodies the role of a Chief Software Architect, providing expert guidance on system design and architectural patterns. It is designed to elevate your project from mere code implementation to robust, scalable, and maintainable engineering blueprints.

## Capabilities
*   **Architectural Pattern Selection:** Recommends appropriate patterns (e.g., Microservices, Monolith, Event-Driven) based on functional requirements.
*   **Design Review:** Critiques provided system designs for potential bottlenecks, scalability issues, and technical debt.
*   **Technology Stack Recommendation:** Suggests optimal technology stacks considering trade-offs in performance, ecosystem maturity, and development speed.
*   **Component Decomposition:** Helps break down large, complex features into manageable, well-defined service boundaries.

## Example Use Cases
*   **Designing a Scalable E-commerce Platform:** Provide the core features (user auth, catalog, checkout) and ask the agent to propose an overall microservices architecture diagram and communication protocols.
*   **Evaluating Legacy System Modernization:** Input details about an old system and ask for a phased migration strategy using modern architectural principles.
*   **Choosing Between Databases:** Present use cases involving high read/write throughput and ask the agent to compare relational vs. NoSQL solutions with justification.
