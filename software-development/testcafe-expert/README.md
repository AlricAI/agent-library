## Overview
This expert agent provides deep mastery over TestCafe, a powerful framework for building reliable and maintainable end-to-end UI tests. It focuses on simulating real user interactions to validate critical application paths, ensuring high confidence in your deployed software.

## Capabilities
*   **Test Case Generation:** Writes concise, descriptive test cases following the page model pattern for modularity.
*   **Flakiness Mitigation:** Implements advanced techniques to stabilize selectors and interactions, addressing common flaky test issues.
*   **Best Practices Adherence:** Ensures tests follow TestCafe best practices for cross-browser compatibility and performance optimization.
*   **CI/CD Integration:** Structures tests for efficient execution within Continuous Integration/Continuous Deployment pipelines.
*   **Reporting & Documentation:** Generates comprehensive test suites, detailed execution reports, and documentation mapping tests to specific application features.

## Example Use Cases
1. **Feature Validation:** Provide a set of user stories (e.g., 'User must be able to register, log in, and update profile details') and receive a complete suite of passing TestCafe scripts.
2. **Performance Auditing:** Submit existing test files and ask the agent to refactor them for better execution speed, suggesting parallelization strategies.
3. **Onboarding Material:** Request an orientation guide detailing best practices for new team members adopting TestCafe within an existing codebase.
