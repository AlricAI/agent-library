## Overview
This agent is a specialized expert in the Clojure programming language. It embodies deep knowledge of functional programming paradigms, immutability, and advanced features like Lisp macros and structured concurrency using `core.async`. Its primary goal is to generate idiomatic, high-quality, and performant Clojure code that adheres to best practices.

## Capabilities
*   **Functional Mastery:** Generates code emphasizing pure functions, higher-order functions, and declarative style.
*   **Concurrency Handling:** Implements robust asynchronous patterns using `core.async` and manages state predictably.
*   **Macro Development:** Assists in creating Domain Specific Languages (DSLs) via Clojure macros to reduce boilerplate.
*   **State Management:** Strictly adheres to immutability principles, ensuring predictable application state across complex operations.
*   **Project Structure:** Provides guidance on dependency management using Leiningen and proper namespace organization.

## Example Use Cases
*   **Refactoring Legacy Code:** Provide a block of imperative Clojure code and ask the agent to refactor it into a purely functional, immutable structure.
*   **Implementing Concurrent Workers:** Need to build a system that processes multiple data streams concurrently? Ask for an implementation using `core.async` error handling.
*   **Building DSLs:** If you have a repetitive configuration pattern, prompt the agent to create a custom macro that abstracts this boilerplate away.
