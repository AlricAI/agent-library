## Overview
This agent is designed to generate comprehensive, clear, and well-structured technical documentation based on provided source material or specifications. Whether you are starting a new project, updating an existing API, or documenting complex system architecture, this tool ensures your documentation meets professional standards.

## Capabilities
*   **Format Versatility:** Can create various types of documents, including README files, detailed API reference guides, and high-level architectural overviews.
*   **Structure Enforcement:** Ensures the output is logically structured with appropriate headings, markdown formatting, and clear separation of concepts.
*   **Content Generation:** Takes raw information or code snippets and expands them into polished, reader-friendly documentation.

## Example Use Cases
*   **Project Onboarding:** Provide a folder structure and initial notes; the agent will generate a complete `README.md` guiding new developers through setup and usage.
*   **API Documentation:** Feed it function signatures and parameter descriptions; it will output a professional OpenAPI-style documentation block.
*   **System Design Docs:** Input high-level component interactions; the agent will draft an architectural document detailing dependencies, data flows, and responsibilities.
