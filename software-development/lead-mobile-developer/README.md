## Overview
Lead Mobile Developer is a comprehensive agent designed to manage the entire lifecycle of mobile application development. It acts as a senior technical lead, ensuring that all built applications are not only feature-rich but also highly performant, reliable, and intuitive for the end-user.

This agent enforces strict development standards covering code reviews, performance budgeting, and adherence to platform guidelines.

## Capabilities
*   **Full Development Cycle:** Building complete mobile applications from concept to deployment.
*   **UI/UX Design Implementation:** Creating interfaces that are both aesthetically pleasing and highly usable.
*   **Performance Tuning:** Optimizing codebases for speed, smoothness, and minimal resource consumption.
*   **Quality Assurance:** Implementing rigorous testing protocols, including bug catching and performance metric validation.
*   **Standards Enforcement:** Defining and enforcing best practices across coding standards, accessibility, and platform guidelines.

## Example Use Cases
*   **Building a New Feature:** When tasked with adding a complex feature, the agent will structure the development plan, write optimized code, and ensure it passes performance budgets before merging.
*   **Performance Audit:** If an existing app is reported as slow, this agent can diagnose bottlenecks, refactor inefficient components, and deploy speed improvements.
*   **Cross-Platform Consistency:** Ensuring that the user experience remains consistent and high-quality across different operating systems while adhering to platform-specific best practices.
