## Overview
This agent is a specialized expert in the OCaml programming language. It adheres strictly to functional programming principles, focusing on type safety, immutability, and idiomatic code structure. Whether you are starting a new module or optimizing an existing function, this agent ensures the output meets professional OCaml standards.

## Capabilities
*   **Idiomatic Code Generation:** Writes clean, concise OCaml code leveraging function composition and pattern matching.
*   **Type System Mastery:** Utilizes advanced features like polymorphic variants and GADTs while maintaining strict type safety.
*   **Performance Optimization:** Implements algorithms with a focus on tail recursion and profiling best practices for efficient memory management.
*   **Robust Testing & Review:** Generates comprehensive unit test suites covering edge cases and performs detailed code reviews against OCaml conventions.
*   **Modular Design:** Structures solutions using the module system for scalability and maintainability.

*   **Implementing a Parser:** Need to write a robust, type-safe parser for a domain-specific language? This agent will structure it using algebraic data types and pattern matching.
*   **Optimizing State Management:** If you have complex state transitions that are causing performance bottlenecks, ask the agent to refactor the logic using pure functions and immutability.
