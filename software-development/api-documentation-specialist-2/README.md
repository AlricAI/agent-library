## Overview
This agent acts as a dedicated API documentation specialist, ensuring your APIs are documented with the highest degree of clarity and developer experience. It adheres to a 'Developer-First' philosophy, meaning every output is designed to get a consumer up and running in minutes.

It specializes in generating industry-standard specifications like OpenAPI/Swagger 3.0+ while simultaneously creating supporting materials such as integration guides and multi-language code examples.

## Capabilities
* **OpenAPI Specification Generation**: Creates valid, structured OpenAPI 3.0+ YAML/JSON files for your entire API surface area.
* **Comprehensive Endpoint Referencing**: Documents every endpoint with clear parameters, request bodies, and detailed response schemas.
* **Integration Guide Creation**: Writes step-by-step tutorials that guide developers through common workflows (e.g., authentication flow).
* **Multi-Language Code Examples**: Provides ready-to-use code snippets in popular languages for all documented endpoints.
* **Concurrent Documentation Pattern**: Analyzes and documents all aspects of the API simultaneously, ensuring consistency across all generated artifacts.

## Example Use Cases
1. **New API Launch**: Feed it your raw endpoint definitions and receive a complete package: OpenAPI spec, Quick Start guide, and sample code for client implementation.
2. **API Versioning**: Provide documentation from v1 and v2 endpoints; the agent will help structure the necessary versioning metadata within the specs.
3. **Onboarding New Teams**: Use it to generate an interactive 'Try-It-Out' style reference guide that minimizes questions for your support team.
