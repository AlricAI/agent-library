## Overview
Blueprint is a specialized AI agent designed to function as a senior technical architect. Its core purpose is to guide the design phase of software projects, ensuring that systems are built with foresight, clarity, and resilience. It operates by establishing clear blueprints for development rather than writing implementation code itself.

## Capabilities
*   **Architecture Decision Records (ADRs):** Generates comprehensive ADRs detailing technical choices, alternatives considered, and the trade-offs associated with each decision.
*   **Data Modeling:** Designs logical and physical data models, defining service boundaries and necessary API contracts to ensure composability.
*   **Risk Identification:** Proactively analyzes proposed systems to identify potential architectural weaknesses, scalability bottlenecks, and points of failure before development begins.
*   **Design Documentation:** Produces highly explicit design documents that leave minimal room for ambiguity, making the transition from design to implementation straightforward.

## Example Use Cases
*   **New Service Implementation:** When starting a new microservice, use Blueprint to generate an initial ADR outlining its purpose, dependencies, and core data structures.
*   **System Refactoring:** If an existing system is struggling with complexity, ask Blueprint to analyze the current state and propose a modular, decoupled architectural overhaul.
*   **API Contract Definition:** Before any frontend or backend team starts integration, use this agent to nail down definitive API specifications, ensuring idempotency and clear versioning strategies.
