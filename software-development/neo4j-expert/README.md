## Overview
This agent serves as a comprehensive expert for Neo4j graph databases. It specializes in transforming complex data requirements into robust, efficient, and scalable graph models using the Cypher query language.

Whether you are designing a new knowledge graph or optimizing an existing system, this tool ensures best practices are followed across modeling, querying, and performance tuning.

## Capabilities
*   **Cypher Query Generation:** Creates accurate and complex queries for data retrieval and manipulation.
*   **Graph Modeling:** Designs intuitive and relationship-focused schemas suitable for graph databases.
*   **Performance Optimization:** Implements indexing strategies and optimizes read/write operations to minimize latency.
*   **Advanced Procedures:** Utilizes knowledge of the APOC library for extended database functionality.
*   **System Architecture:** Provides guidance on clustering, high availability, and security best practices.

## Example Use Cases
*   **Relationship Mapping:** Model a complex social network or supply chain to discover hidden connections between entities.
*   **Query Optimization:** Provide slow-running queries and receive optimized Cypher alternatives with suggested indexes.
*   **Data Ingestion Pipeline:** Generate scripts for reliably importing large datasets while maintaining graph integrity.
*   **Performance Auditing:** Receive reports and recommendations on monitoring and tuning a production Neo4j instance.
