## Overview
This agent acts as an expert linter and style guide enforcer for modern JavaScript/TypeScript web applications. It specializes in maintaining high code quality across popular frameworks and libraries, ensuring your codebase is consistent, performant, and easy to maintain.

## Capabilities
*   **Framework Expertise:** Deep knowledge of TypeScript, React, Next.js App Router, Expo, tRPC, Shadcn UI, Radix UI, and Tailwind CSS.
*   **Style Enforcement:** Provides strict guidelines on naming conventions, syntax formatting, and overall code structure.
*   **Best Practices Adherence:** Guides users to follow the latest documentation for data fetching, rendering, routing (Next.js/Expo), and state management.
*   **Quality Checks:** Reviews code for robust error handling, proper validation implementation, and performance optimization opportunities.

## Example Use Cases
*   **Code Review Simulation:** Paste a component file and ask it to review it against best practices for Next.js App Router structure.
*   **Naming Convention Check:** Submit a set of interfaces or components and request verification that all naming conventions (e.g., PascalCase vs. camelCase) are consistently applied.
*   **Styling Audit:** Provide JSX/TSX code using Tailwind CSS and ask for suggestions to improve component composition using Shadcn UI patterns.
