## Overview
Angular Expert is a specialized AI agent designed to write production-grade Angular code that follows modern best practices. It focuses on creating scalable, maintainable, and high-performance frontend applications by adhering strictly to established architectural patterns.

## Capabilities
*   **Component Architecture:** Designs components following the Single Responsibility Principle (SRP) for clean separation of concerns.
*   **Reactive Programming:** Expertly utilizes RxJS operators for complex asynchronous data streams, ensuring proper subscription management to prevent memory leaks.
*   **State Management:** Implements scalable state solutions using industry standards like NgRx or Akita.
*   **Performance Optimization:** Prioritizes performance by implementing `OnPush` change detection and leveraging lazy loading for optimal bundle size.
*   **Modern Features:** Incorporates modern Angular features, including support for Angular Universal (SSR) and best practices for routing and forms (Reactive Forms).

## Example Use Cases
1. **Building a Dashboard:** Requesting a complex dashboard module that requires fetching data from multiple endpoints, managing global state via NgRx, and displaying results using Angular Material components.
2. **Form Handling:** Developing a multi-step form with intricate validation logic, ensuring the use of Reactive Forms for robustness.
3. **Performance Review:** Asking it to refactor an existing component to improve change detection performance by implementing `OnPush` strategy.
