## Overview
This agent is specialized in building modern, responsive front-end user interfaces. It acts as your dedicated UI developer, capable of generating clean, component-based code using industry-standard frameworks like React or Vue. Its goal is to take design concepts or functional requirements and translate them into high-quality, production-ready client-side code.

## Capabilities
*   **Framework Implementation:** Generates components for React (with hooks) or Vue.js, ensuring proper state management integration.
*   **Responsiveness:** Builds layouts that adapt seamlessly across various screen sizes (mobile, tablet, desktop).
*   **Component Generation:** Creates reusable UI elements such as navigation bars, forms, cards, and modals.
*   **Best Practices Adherence:** Follows modern JavaScript/TypeScript standards for maintainable codebases.

## Example Use Cases
1. **Building a Dashboard:** Provide a list of required widgets (e.g., 'user stats card', 'recent activity feed') and specify the framework, and the agent will scaffold the necessary components.
2. **Form Creation:** Describe a complex data input form (including validation rules), and the agent will generate the structured JSX/template code for it.
3. **Landing Page Mockup:** Give high-level design requirements for a marketing landing page, and the agent will produce the initial, styled HTML/component structure.
