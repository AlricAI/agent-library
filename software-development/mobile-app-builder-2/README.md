## Overview
Mobile App Builder is an expert AI agent specializing in the end-to-end development of high-performance, user-friendly mobile applications. Whether you require a deeply native experience on iOS (SwiftUI/Swift) or Android (Kotlin/Compose), or prefer the efficiency of cross-platform solutions like Flutter or React Native, this agent ensures platform best practices are followed.

Its core mission is to deliver apps that feel inherently native, optimized for speed, battery life, and superior user experience across all target devices.

## Capabilities
*   **Native Development:** Builds dedicated iOS apps using Swift/SwiftUI and Android apps using Kotlin/Jetpack Compose.
*   **Cross-Platform Solutions:** Develops robust applications using Flutter or React Native for code reusability.
*   **Performance Optimization:** Implements platform-specific techniques to minimize memory footprint, optimize startup times, and ensure smooth animations.
*   **Offline-First Architecture:** Designs data synchronization strategies to guarantee functionality even without an internet connection.
*   **Feature Integration:** Seamlessly integrates complex features such as biometric authentication (Face ID/Touch ID), geolocation services, camera access, and push notifications.

## Example Use Cases
1. **E-commerce App:** Build a cross-platform shopping app featuring product catalogs, offline viewing of saved items, and integrated payment gateways.
2. **Field Service Tool:** Develop a native Android application for technicians that requires robust offline data capture (e.g., signatures, photos) synced when connectivity is restored.
3. **Social Utility App:** Create an iOS-first app utilizing SwiftUI and CoreLocation to provide real-time mapping services with smooth gesture handling.
