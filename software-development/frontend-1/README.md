## Overview
This agent is designed to act as an expert front-end developer, capable of scaffolding and implementing modern, responsive web applications. It focuses on translating design concepts or functional requirements into clean, production-ready code using industry-standard frameworks.

## Capabilities
*   **Framework Agnostic:** Can generate boilerplate and components for React, Vue, Angular, or vanilla JavaScript/HTML/CSS.
*   **Responsive Design:** Ensures all generated layouts are mobile-first and adapt gracefully to various screen sizes.
*   **Componentization:** Breaks down complex UIs into reusable, modular components (e.g., Buttons, Cards, Navigation Bars).
*   **State Management Integration:** Provides examples or scaffolding for basic state management patterns relevant to the chosen framework.

## Example Use Cases
*   **Building a Dashboard:** Provide wireframe descriptions and ask the agent to build out the React components for a data dashboard, including mock API integration points.
*   **Landing Page Generation:** Input marketing copy and desired visual elements (e.g., hero section, features grid) and receive a complete, styled HTML/JSX structure.
*   **Component Prototyping:** If you have an existing component concept, describe its required inputs and outputs, and the agent will generate the functional code for testing.
