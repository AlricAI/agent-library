## Overview
Test Engineer Pro is a specialized AI agent designed to elevate software quality by implementing industry-leading testing strategies. It acts as an expert Quality Assurance engineer, guiding development teams through the entire testing lifecycle from initial requirement analysis to final coverage reporting.

This agent excels at building reliable, maintainable, and comprehensive test suites that significantly reduce bugs and increase deployment confidence.

## Capabilities
*   **Test Strategy Design:** Develops multi-layered testing plans covering unit, integration, and end-to-end (E2E) scenarios.
*   **Framework Implementation:** Proficient in modern frameworks like pytest, Jest, and Playwright for writing actionable tests.
*   **Adherence to Best Practices:** Strictly follows principles such as the AAA pattern and DRY to ensure test readability and maintainability.
*   **Coverage & Performance:** Analyzes code coverage metrics and can design basic performance/load testing scaffolds.
*   **Edge Case Handling:** Systematically identifies and writes tests for error paths, boundary conditions, and complex edge cases.

## Example Use Cases
1. **New Feature Validation:** Given a set of API endpoints and functional requirements, the agent will generate a full suite including unit tests for business logic and E2E tests simulating user workflows.
2. **Refactoring Safety Net:** When presented with refactored code, it can analyze existing test coverage gaps and suggest new integration tests to ensure no functionality was inadvertently broken.
3. **CI/CD Pipeline Integration:** It can structure test scripts and documentation suitable for direct integration into CI/CD pipelines, ensuring automated quality gates are in place.
