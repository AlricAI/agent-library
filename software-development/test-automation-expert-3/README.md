## Overview
This agent acts as an expert test automation engineer, specializing in building robust, intelligent, and scalable testing ecosystems. It masters modern testing frameworks, incorporating AI capabilities like self-healing tests and natural language processing for test generation to ensure high-quality software delivery.

## Capabilities
*   **Test-Driven Development (TDD) Excellence:** Supports full TDD cycles including red-green-refactor patterns, property-based testing, and BDD integration. It can track metrics like cycle time and guide teams through methodologies such as the Chicago School approach.
*   **AI-Powered Testing Frameworks:** Implements self-healing test automation using industry tools (e.g., Testsigma). It leverages ML for failure prediction and uses NLP to generate and maintain test cases from natural language descriptions.
*   **Comprehensive Coverage:** Provides advanced techniques like test triangulation, visual AI testing for UI validation, and intelligent test data management.
*   **CI/CD Integration:** Designed to optimize testing efficiency within continuous integration pipelines, ensuring fast feedback loops.

## Example Use Cases
1. **Building a New Test Suite:** Provide the application's core functionality and desired behaviors; the agent will scaffold a TDD-compliant test suite using modern frameworks.
2. **Maintaining Legacy Tests:** Input existing UI automation scripts that frequently break due to minor UI changes; the agent will apply self-healing logic to update selectors and maintain test reliability.
3. **Adopting Behavior Specifications:** Give the agent a set of user stories written in plain English; it will translate these into executable, behavior-driven test specifications.
