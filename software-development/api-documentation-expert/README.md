## Overview
This agent acts as a senior API documenter, specializing in generating world-class, developer-friendly documentation. It ensures that your APIs are not only technically sound but also incredibly easy for external developers to adopt and use successfully.

The core focus is achieving full compliance with OpenAPI 3.1 standards while providing interactive elements that minimize the learning curve for new consumers.

## Capabilities
*   **OpenAPI Compliance:** Generates documentation strictly adhering to OpenAPI 3.1 specifications, covering schemas, endpoints, and security schemes.
*   **Comprehensive Coverage:** Ensures 100% endpoint coverage, detailing request/response examples, error handling, and authentication flows for every resource.
*   **Interactive Features:** Implements 'Try-it-out' consoles, code generation snippets across multiple languages, and API explorers directly within the documentation portal.
*   **Multi-Format Support:** Capable of documenting various protocols including REST, GraphQL, WebSockets, gRPC, and webhook events.
*   **Best Practice Guides:** Provides detailed guides on complex topics like OAuth 2.0 flows, JWT implementation, and robust error resolution strategies.

## Example Use Cases
*   **Onboarding New APIs:** Feed the agent raw endpoint definitions and have it output a complete, interactive developer portal ready for immediate consumption.
*   **Updating Legacy Docs:** Provide existing documentation and user feedback reports; the agent will analyze gaps to update schemas, add missing error codes, or clarify ambiguous endpoints.
*   **Cross-Protocol Documentation:** Use it when you need to document both a REST endpoint *and* its corresponding GraphQL schema in one cohesive package.
