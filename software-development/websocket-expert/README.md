## Overview
This specialized AI agent is designed to act as an expert resource for all aspects of the WebSocket protocol (RFC 6455). It focuses on building, securing, and optimizing real-time communication channels, ensuring compliance with industry standards.

## Capabilities
*   **Protocol Compliance:** Ensures all implementations strictly adhere to RFC 6455 guidelines.
*   **Security Implementation:** Specializes in establishing secure WebSocket connections using WSS (TLS).
*   **Connection Management:** Handles the full lifecycle, including handshake validation, message framing, and graceful disconnection/reconnection strategies.
*   **Performance Optimization:** Provides advice on scaling WebSocket servers to handle high connection churn and optimizing message delivery.
*   **Error Handling & Debugging:** Implements robust network error handling, logging, and debugging patterns for production environments.

## Example Use Cases
1. **Building a Live Chat Feature:** Generating secure, scalable client-server code examples for instant messaging.
2. **Real-Time Data Feeds:** Architecting connections for live stock tickers or IoT sensor data streams requiring minimal latency.
3. **Protocol Auditing:** Reviewing existing WebSocket implementations to validate adherence to security best practices and protocol standards.
4. **Load Testing Strategy:** Advising on the necessary load testing parameters to ensure a WebSocket backend can handle peak concurrent users.
