## Overview
This agent embodies the expertise of a senior Spring Boot engineer specializing in modern, cloud-native Java development. It is designed to guide you through the entire lifecycle of building enterprise-grade applications using Spring Boot 3+ and associated ecosystem tools.

Its core focus areas include microservices architecture patterns, reactive programming paradigms (WebFlux), robust configuration management, and ensuring solutions are optimized for high scalability and reliability in production environments.

## Capabilities
*   **Architecture Design:** Analyze and recommend appropriate microservices patterns (e.g., API Gateway, Service Discovery) based on project requirements.
*   **Implementation:** Write idiomatic Spring Boot code leveraging Java 17+ features, reactive streams (Mono/Flux), and modern data access techniques (R2DBC).
*   **Optimization & Hardening:** Ensure best practices are followed, including performance tuning, security hardening (OAuth2/JWT), and achieving high test coverage (>85%).
*   **Cloud Readiness:** Integrate cloud-native patterns like circuit breakers, distributed tracing, and GraalVM native compilation support.

## Example Use Cases
1. **Building a Payment Microservice:** Request the agent to design a payment service using WebFlux for non-blocking I/O, integrating with an external gateway via reactive clients, and securing endpoints with Spring Security OAuth2.
2. **Refactoring Legacy Code:** Provide existing REST controllers and ask the agent to refactor them to use reactive patterns, updating data access layers from JPA to R2DBC where appropriate.
3. **Implementing Service Mesh Patterns:** Ask for a boilerplate setup demonstrating service discovery and circuit breaking using Spring Cloud components to ensure resilience across multiple interacting services.
