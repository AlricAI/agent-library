## Overview
This agent functions as a highly specialized Prompt Engineer, designed to move beyond basic prompting. It masters cutting-edge methodologies like Chain-of-Thought (CoT), Constitutional AI, and advanced meta-prompting techniques. Its goal is to produce production-ready prompts that are not only effective but also reliable, safe, and optimized for specific business logic.

## Capabilities
*   **Advanced Reasoning:** Implements CoT, Few-Shot CoT, Tree-of-Thoughts (ToT), and self-consistency decoding for complex problem decomposition.
*   **Safety & Alignment:** Incorporates Constitutional AI principles, red teaming prompts, and safety guardrails to mitigate bias and prevent harmful outputs.
*   **Prompt Generation:** Capable of meta-prompting—generating optimal prompts based on desired outcomes or existing prompt weaknesses.
*   **Optimization:** Provides frameworks for A/B testing prompt performance and compressing complex instructions without losing fidelity.

## Example Use Cases
1. **Complex Reasoning Task:** Need a multi-step analysis (e.g., market trend prediction)? Request the agent to structure the necessary CoT or ToT steps.
2. **Safety Layer Implementation:** Building an agent that handles user input? Ask it to generate a constitutional prompt layer to filter out adversarial inputs.
3. **System Prompt Refinement:** You have a working but inconsistent system prompt. Provide it, and ask the agent to apply iterative refinement techniques to make it more robust and explicit.
