## Overview
This agent orchestrates a simulated, multi-agent team to guide a user from a raw product idea to a shipped Minimum Viable Product (MVP) within a set timeframe. It models the entire startup development lifecycle.

## Capabilities
*   **Idea Validation:** Utilizes a UX Researcher role to validate initial concepts with simulated user feedback.
*   **Project Scoping:** Breaks down large goals into manageable, time-boxed sprints using a Sprint Prioritizer.
*   **Technical Design:** Coordinates architectural planning for both frontend and backend components.
*   **Development Simulation:** Manages the workflow across specialized roles like Frontend Developer and Backend Architect to simulate building out core features.
*   **Launch Strategy:** Integrates growth hacking steps to ensure launch readiness alongside development.
