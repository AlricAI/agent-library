## Overview
This agent embodies the expertise of a senior test automation engineer, specializing in creating resilient and intelligent testing ecosystems. It masters modern frameworks, AI-powered test generation, and self-healing capabilities to ensure high-quality software delivery at scale. It moves beyond simple scripting to implement full quality engineering strategies.

## Capabilities
*   **Test-Driven Development (TDD) Mastery:** Facilitates the entire TDD cycle, including failing test generation, minimal implementation guidance, regression safety validation, and tracking metrics for adherence.
*   **Advanced Testing Patterns:** Supports both state-based (Chicago School) and interaction-based (London School) TDD approaches, alongside Behavior-Driven Development (BDD) integration.
*   **AI-Powered Resilience:** Implements self-healing test automation using industry tools, leveraging NLP for natural language test case generation and maintenance.
*   **Intelligent Optimization:** Utilizes machine learning for failure prediction, predictive analytics to optimize test runs, and visual AI testing for robust UI validation.
*   **Comprehensive Coverage:** Supports advanced techniques like Property-based TDD and Test Triangulation for maximum code coverage assurance.

## Example Use Cases
1. **Implementing a New Feature Test Suite:** Provide the core business logic, and this agent will generate a complete TDD cycle: writing initial failing tests, guiding minimal passing implementations, and structuring refactoring validation.
2. **Upgrading Legacy Tests:** Feed in an existing set of brittle UI tests; the agent can suggest modernizing them using AI-driven self-healing patterns for better maintenance.
3. **Establishing a Quality Gate:** Define required test coverage metrics (e.g., TDD compliance, property testing depth), and the agent will structure the necessary CI/CD pipeline steps to enforce these quality gates automatically.
