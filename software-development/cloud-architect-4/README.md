## Overview
This agent embodies the expertise of a senior cloud architect, specializing in designing resilient, scalable, and cost-effective infrastructure across major public clouds (AWS, Azure, GCP, OCI). It masters modern practices including Infrastructure as Code (IaC), FinOps methodologies, and advanced architectural patterns like serverless and event-driven systems.

## Capabilities
* **Multi-Cloud Design**: Develop strategies for cross-cloud networking, data replication, and disaster recovery planning across AWS, Azure, GCP, and OCI.
* **Infrastructure as Code (IaC)**: Proficient in Terraform/OpenTofu, along with native tools like CloudFormation, Bicep, and CDK. It supports GitOps workflows using ArgoCD and Flux.
* **Cost Optimization (FinOps)**: Provides actionable recommendations for right-sizing resources, utilizing reserved instances, and implementing cost governance policies across all supported platforms.
* **Security & Compliance**: Implements security best practices by advising on IAM roles, network segmentation (VPC/VNet), and policy enforcement using tools like OPA.

## Example Use Cases
1. **Cloud Migration Planning**: "Design a phased migration plan to move our monolithic application from an on-premise data center to a multi-cloud setup utilizing AWS for compute and Azure for identity management, ensuring minimal downtime." 
2. **Cost Reduction Audit**: "Analyze the attached resource usage report and provide specific Terraform modules or architectural changes to reduce monthly cloud spend by at least 15% while maintaining current SLA levels."
3. **Serverless Architecture Blueprint**: "Outline a complete event-driven microservices architecture using AWS Lambda, SQS, and EventBridge to process incoming IoT data streams securely and scalably."
