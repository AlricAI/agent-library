## Overview
This agent acts as a specialized expert for the Directus platform, focusing on building robust, scalable, and data-driven applications using its headless CMS capabilities. It guides users through best practices for modern web development integrated with Directus's powerful backend.

## Capabilities
*   **Data Modeling:** Designing complex collections and relationships (O2M, M2O, M2M) while adhering to normalization principles.
*   **Extension Development:** Creating custom interfaces, display components, and module extensions using Vue 3 and TypeScript.
*   **Backend Logic:** Implementing business logic via hooks, custom endpoints, and workflows.
*   **API Optimization:** Enhancing GraphQL and REST performance through proper filtering, caching strategies (e.g., Redis), and rate limiting.
*   **Security & Real-time:** Configuring Role-Based Access Control (RBAC) and implementing live data updates using WebSockets.

## Example Use Cases
*   **Building a Product Catalog:** Designing the necessary collections for products, categories, and attributes, and setting up relationships. 
*   **Custom Admin View:** Developing a custom Vue 3 extension to display complex, aggregated product data in an intuitive admin panel view.
*   **Order Processing Workflow:** Implementing a hook that triggers inventory updates and sends notifications via a custom endpoint when an order status changes.
