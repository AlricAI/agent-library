## Overview
The SQL Expert agent is a specialized tool for database professionals requiring advanced SQL generation and schema design. It goes beyond simple querying by focusing on performance, normalization principles, and robust data modeling.

This agent acts as an expert consultant, analyzing your data requirements to provide not just queries, but complete architectural advice, including necessary indexes and migration scripts.

## Capabilities
*   **Schema Design:** Designs normalized database schemas based on provided requirements.
*   **Optimized Query Writing:** Generates complex SQL using advanced features like Common Table Expressions (CTEs) and window functions.
*   **Performance Tuning:** Analyzes potential bottlenecks and suggests query execution plan improvements.
*   **Comprehensive Deliverables:** Provides optimized queries, schema definitions, index recommendations, and data migration scripts in one package.
*   **Advanced SQL Constructs:** Effectively utilizes complex JOIN operations and stored procedure implementations.

## Example Use Cases
*   **Complex Reporting:** Need to build a multi-stage report involving time-series analysis? Provide the raw data context, and it will generate the necessary CTEs and window functions for accurate results.
*   **Database Refactoring:** If you have an existing poorly performing database structure, ask it to review the schema against normalization rules and suggest improvements.
*   **Data Migration:** Planning a move between systems? Use this agent to draft the full set of data migration scripts required to maintain integrity during the transfer.
