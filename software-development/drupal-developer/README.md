## Overview
This expert agent is specialized in designing and developing robust, enterprise-level solutions using Drupal 10/11. It adheres strictly to modern best practices, leveraging Symfony components, dependency injection, and the official plugin system for scalable architecture.

## Capabilities
*   **Custom Module Development:** Creating modules utilizing the plugin system, services, and adhering to core APIs.
*   **Content Modeling:** Designing complex content structures using custom entities, field types, and managing relationships effectively.
*   **Theming & Frontend:** Building responsive, component-based themes with advanced Twig templating and asset optimization.
*   **API Integration:** Implementing headless or API-first configurations for modern frontend consumption (REST/GraphQL).
*   **Development Workflow:** Utilizing Composer, CMI, Drush, and promoting test-driven development practices (PHPUnit/Behat).

## Example Use Cases
*   **Building a Custom Module:** Need to create a module that processes data asynchronously? This agent can structure the service layer, implement queue workers, and define necessary custom entities.
*   **Revising Content Structure:** Your site needs complex relationships between content types. Use this agent to design the optimal field configuration, entity reference strategy, and vocabulary hierarchy.
*   **Modernizing a Theme:** If you are migrating an old theme, ask for component-based architecture using Twig partials and modern CSS methodologies (like BEM) while ensuring progressive enhancement.
