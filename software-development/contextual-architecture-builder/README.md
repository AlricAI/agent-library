## Overview
This agent acts as an expert System Architect specializing in designing complex, context-aware AI systems. It guides users through a structured interview process to gather detailed requirements for a specific domain.

The ultimate goal is to transform abstract user needs into a complete, production-ready directory structure (simulated as `.opencode`) containing all necessary components: orchestrators, subagents, context files, workflows, and commands.

## Capabilities
*   **Requirements Elicitation:** Conducts an interactive interview process to deeply understand the target domain and use cases.
*   **System Structuring:** Designs hierarchical agent patterns and modular knowledge organization for optimal performance.
*   **Context Management:** Implements sophisticated context handling using dedicated context files, ensuring agents operate with necessary background information.
*   **Architecture Generation:** Outputs a comprehensive blueprint simulating a full `.opencode` folder system ready for implementation.

## Example Use Cases
*   **Building a Financial Analyst Agent:** Provide domain details (e.g., stock market analysis) and the agent will structure subagents for data fetching, trend identification, and report generation.
*   **Creating a Legal Compliance Checker:** Input legal documents and use cases; the agent builds workflows that route text through specialized agents to check against specific regulatory contexts.
*   **Developing a Technical Documentation Bot:** Define the source material and required outputs; the system will build context handlers and command structures for seamless documentation querying.
