## Overview
Product Manager Pro acts as a senior, data-driven Product Manager to guide your product lifecycle. It specializes in bridging the gap between user needs, market opportunities, and achievable business objectives. This agent ensures that every feature decision is strategic, measurable, and aligned with a clear North Star metric.

## Capabilities
*   **Product Strategy Development:** Assists in defining vision, value propositions, and go-to-market strategies using established frameworks (e.g., Jobs to be Done).
*   **Roadmap Planning:** Structures quarterly objectives, prioritizes features against strategic themes, and manages dependency mapping.
*   **Feature Prioritization:** Employs quantitative methods like RICE scoring and analyzes value versus complexity to build actionable backlogs.
*   **User Insight Synthesis:** Helps structure user research findings from interviews, surveys, and analytics into actionable pain points and personas.
*   **Cross-Functional Alignment:** Keeps stakeholders focused on measurable success metrics (OKRs) and continuous feedback loops.

## Example Use Cases
1. **New Product Incubation:** Provide a market concept and ask the agent to generate a full product vision document, including target personas and initial value propositions.
2. **Backlog Grooming:** Feed it a list of 50 potential features and request prioritization using the RICE framework, along with justification for the top 10.
3. **Roadmap Adjustment:** Input recent quarterly performance data (e.g., low adoption on Feature X) and ask the agent to revise the next quarter's roadmap to address the shortfall.
