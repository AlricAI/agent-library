## Overview
This agent is designed to streamline the crucial initial phase of any software or product development lifecycle: requirements gathering. Instead of relying on vague discussions, it systematically prompts users and analyzes inputs to build a robust foundation for your project.

By structuring the discovery process, this tool ensures that all necessary functional, non-functional, and business requirements are captured early, significantly reducing scope creep and rework later in development.

## Capabilities
*   **Structured Questioning:** Guides users through comprehensive questioning across various domains (e.g., user roles, performance needs, integrations).
*   **Specification Drafting:** Generates detailed product requirement documents (PRDs) or specification sheets based on gathered inputs.
*   **Gap Analysis Identification:** Helps pinpoint missing requirements or areas where current specifications conflict with stated goals.
*   **Format Generation:** Outputs structured markdown or document outlines suitable for handover to development teams.

## Example Use Cases
*   **New Feature Implementation:** Need to build a new payment gateway? Provide the business goal, and this agent will generate specs covering error handling, compliance needs, and success workflows.
*   **MVP Definition:** Starting a brand-new product? Use it to define the Minimum Viable Product scope by prioritizing core features against desired future enhancements.
*   **System Upgrade Planning:** When upgrading legacy software, feed in the current system's limitations and the desired modern functionality; the agent will draft the necessary 'to-be' state specifications.
