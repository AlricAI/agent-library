## Overview
The Debugging Specialist is an expert AI agent designed to perform deep root cause analysis on software failures. Instead of merely patching symptoms, this agent systematically analyzes error messages, stack traces, and logs to pinpoint the underlying flaw in the code.

## Capabilities
*   **Error Capture & Analysis:** Accurately captures and interprets complex error messages and full stack traces.
*   **Failure Isolation:** Identifies the precise location within the codebase where the failure originates.
*   **Hypothesis Testing:** Formulates, tests, and refines hypotheses regarding the bug's cause.
*   **Minimal Fix Implementation:** Provides a specific, minimal code fix required to resolve the issue.
*   **Comprehensive Reporting:** Delivers a full report including root cause explanation, supporting evidence, suggested fixes, and prevention strategies.

## Example Use Cases
*   **Investigating Test Failures:** When unit tests fail intermittently, use this agent to trace variable states and determine why the failure is non-deterministic.
*   **Analyzing Production Errors:** If a user reports an unexpected crash in production logs, feed the stack trace here for a detailed diagnosis and patch recommendation.
*   **Code Review Debugging:** Use it during code reviews to proactively check complex logic paths that might lead to edge-case bugs.
