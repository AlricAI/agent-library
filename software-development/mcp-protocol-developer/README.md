## Overview
This agent functions as a senior-level Model Context Protocol (MCP) developer, specializing in building the critical middleware that connects large language models to external data sources and tools. It ensures all integrations adhere to industry best practices for performance, security, and maintainability.

## Capabilities
*   **Protocol Mastery:** Ensures strict adherence to JSON-RPC 2.0 standards for all communication layers.
*   **Full Stack Implementation:** Develops both robust server backends (resource implementation, tool creation) and reliable client connectors (discovery, state management).
*   **Security Hardening:** Implements comprehensive security controls, including authentication handling, rate limiting, and input validation.
*   **SDK Generation:** Provides expertise in generating type-safe SDKs using TypeScript and Python, enforcing strong typing throughout the stack.
*   **Integration Patterning:** Connects AI systems to diverse external services like databases, message queues, and proprietary APIs.

## Example Use Cases
*   **Building a Tool Agent:** Need to connect an LLM to an internal inventory database? This agent will structure the necessary MCP endpoints, define the required schemas (Zod/Pydantic), and write the connection logic.
*   **Protocol Audit:** If you have an existing integration that fails under load, use this agent to audit its compliance against the full MCP checklist, focusing on error handling and performance benchmarking.
*   **Client Library Creation:** Developing a new service client for your AI platform? This agent will generate boilerplate code and necessary middleware patterns for seamless connection management.
