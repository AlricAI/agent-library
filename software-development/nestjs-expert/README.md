## Overview
This agent specializes in architecting and developing enterprise-grade applications using the NestJS framework. It enforces modern software design patterns, ensuring that the resulting code is not only functional but also highly scalable, maintainable, and rigorously tested.

The focus is on leveraging core NestJS features—such as Dependency Injection (DI), Guards, Pipes, Interceptors, and Middleware—to create a clean separation of concerns across large codebases.

## Capabilities
*   **Architecture Design:** Structures applications using feature modules, promoting clear separation of concerns.
*   **Data Validation & Transformation:** Implements robust data validation using custom Pipes.
*   **Security Implementation:** Develops secure endpoints by integrating Guards for authentication and route protection.
*   **Error Handling:** Establishes global, consistent error handling mechanisms via custom Exception Filters.
*   **Cross-Cutting Concerns:** Manages tasks like logging, caching, and request modification using Interceptors and Middleware.
*   **Code Reusability:** Creates custom decorators to encapsulate repetitive patterns, adhering strictly to DRY principles.
*   **Testing:** Writes comprehensive unit tests for all components using Jest, ensuring high code coverage.

## Example Use Cases
*   **Building a Microservice API:** Need to scaffold a complete user management service with authentication, validation, and logging hooks?
*   **Refactoring Legacy Code:** Have an existing monolithic NestJS app that needs better modular separation and error handling? This agent can guide the refactoring process.
*   **Implementing Complex Authorization:** Requires setting up role-based access control (RBAC) across multiple endpoints using Guards?

Use this agent when you need production-ready, best-practice code for your backend services.
