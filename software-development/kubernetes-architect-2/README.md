## Overview
This agent functions as a senior Kubernetes Architect, providing deep expertise in building and managing enterprise-grade, cloud-native platforms. It masters the entire lifecycle of container orchestration, from initial cluster provisioning to advanced GitOps deployment patterns.

It is designed for engineers and architects who need to move beyond basic YAML manifests to implement robust, scalable, and resilient infrastructure solutions.

## Capabilities
* **Multi-Cloud & Managed K8s**: Expertise across EKS, AKS, GKE, OKE, and self-managed clusters (kubeadm).
* **GitOps Mastery**: Deep knowledge of ArgoCD and Flux v2 for declarative, automated deployments.
* **Progressive Delivery**: Implementing advanced strategies like Canary and Blue/Green rollouts using tools like Flagger.
* **Infrastructure as Code (IaC)**: Proficient in Terraform/OpenTofu for cluster provisioning and Helm/Kustomize for application configuration.
* **Platform Engineering**: Advising on multi-tenancy, service mesh implementation (Istio), observability stacks, and cost optimization patterns.

## Example Use Cases
1. **Designing a Multi-Cluster GitOps Pipeline**: Ask the agent to design an 'app-of-apps' structure using ArgoCD that promotes services from Dev -> Staging -> Prod across three different cloud environments.
2. **Selecting Cluster Components**: Request a comparison of best practices for implementing robust secret management (e.g., Vault integration vs. Sealed Secrets) in a multi-tenant setup.
3. **Platform Roadmapping**: Use it to outline the necessary steps and technologies required to upgrade an existing on-premises cluster to support modern service mesh capabilities.
