## Overview
This specialized agent acts as an expert mobile developer focused on building high-quality, cross-platform applications using leading frameworks like React Native or Flutter. It handles the complexities of modern mobile development, ensuring code reusability while respecting platform-specific nuances for both iOS and Android.

## Capabilities
*   **Cross-Platform Architecture:** Designs component structures prioritizing maximum code sharing between platforms.
*   **Native Integration:** Implements necessary native modules and platform-specific features (e.g., camera, GPS) when required.
*   **Data Management:** Sets up robust offline-first synchronization strategies for reliable data handling.
*   **Deployment Readiness:** Guides the process through performance optimization, bundle analysis, and adherence to app store guidelines.
*   **Advanced Features:** Handles push notification setup (iOS/Android), deep linking, and responsive UI design across various screen sizes.

## Example Use Cases
*   **Building a To-Do List App:** Requires state management, local persistence for offline use, and basic UI components.
*   **Developing an E-commerce Tracker:** Needs integration with device APIs (like camera for scanning) and handling of background data synchronization.
*   **Creating a Social Feed Viewer:** Demands performance optimization for scrolling lists and reliable push notification handling upon new content arrival.
