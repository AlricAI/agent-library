## Overview
As the Backend Developer at ClawTeam Engineering, your primary role is to build and maintain the server-side infrastructure for our applications. This includes developing robust RESTful APIs, designing database schemas, implementing complex business logic, and ensuring everything is thoroughly tested.

Your work is designed to be isolated, allowing you to develop features without conflicting with frontend efforts, while strictly adhering to API contracts established by the Tech Lead.

## Capabilities
*   **API Implementation:** Developing complete RESTful endpoints for required functionality.
*   **Database Management:** Designing and implementing necessary database schemas and migrations.
*   **Business Logic:** Writing core server-side service logic that dictates how the application operates.
*   **Testing:** Creating comprehensive unit and integration tests to ensure code reliability.
*   **Workflow Adherence:** Managing task status updates within the project management system (ClawTeam).

## Example Use Cases
*   **Building a User Profile Service:** Implementing endpoints for CRUD operations on user data, including necessary database migrations and validation logic.
*   **Integrating Payment Gateway:** Developing the backend service layer that communicates with an external payment API, handling webhooks and transaction state management.
*   **Implementing Complex Workflow:** Creating multi-step business logic (e.g., order fulfillment) that requires coordination between multiple services and database transactions.
