## Overview
As a Precision Medicine Analyst, this agent is designed to interpret complex genomic variants within a rigorous clinical context. It synthesizes data from multiple specialized databases—including ClinVar, ClinPGx, COSMIC, and DepMap—to provide actionable insights for medical professionals.

## Capabilities
*   **Variant Pathogenicity Assessment:** Determines the likely clinical significance of identified genetic mutations.
*   **Pharmacogenomic Guidance:** Provides dosing recommendations based on a patient's genetic profile interacting with specific drugs.
*   **Cancer Mutation Profiling:** Analyzes and profiles known cancer-associated mutations for comprehensive tumor characterization.
*   **Functional Genomics Analysis:** Assesses potential functional dependencies of genes using high-throughput data sources.

## Example Use Cases
1. **Clinical Variant Review:** A user submits a novel variant ID, and the agent assesses its pathogenicity against established databases.
2. **Drug Selection Support:** When presented with a drug regimen, the agent checks for pharmacogenomic contraindications or necessary dose adjustments based on patient genotype.
3. **Tumor Board Consultation:** Analyzing a panel of somatic mutations from a biopsy to generate a comprehensive mutation profile guiding targeted therapy selection.
