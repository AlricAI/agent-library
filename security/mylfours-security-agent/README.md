## Overview
MyLfours is a specialized AI agent designed to function as a proactive security specialist. Its primary directive is continuous system monitoring, threat pattern recognition, and ensuring the overall integrity of digital environments. It operates by adhering to strict protocols for documentation and escalation.

## Capabilities
*   **SSH Attack Monitoring:** Actively monitors SSH access logs for suspicious activity.
*   **Service Health Security:** Assesses the security posture and operational health of various services.
*   **Threat Pattern Recognition:** Analyzes incoming data streams to identify emerging or known threat patterns.
*   **Incident Response Preparation:** Maintains readiness by preparing documented steps for immediate incident response.
*   **System Hardening Recommendations:** Provides actionable, expert advice on strengthening system configurations.

## Example Use Cases
1. **Security Audit:** Instruct the agent to run a comprehensive audit of a specified server cluster, resulting in a detailed report saved in the `workspace/REPORTS/` directory.
2. **Alert Investigation:** When an alert is triggered (e.g., unusual login attempts), prompt the agent to investigate immediately, documenting all findings and escalating to 'Sentinel' if necessary.
3. **Proactive Hardening:** Ask the agent to review the current system configuration against best practices and generate a prioritized list of hardening recommendations for implementation.
