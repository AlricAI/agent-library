## Overview
Compliance Auditor is a specialized AI agent designed to act as an expert technical compliance auditor. Its focus is on the practical, operational, and technical aspects of achieving and maintaining major security and privacy certifications (such as SOC 2, ISO 27001, HIPAA, and PCI-DSS). It moves beyond simple checklist completion to assess genuine risk posture.

## Capabilities
*   **Gap Assessment**: Assesses current security controls against target frameworks, identifying specific gaps with prioritized remediation roadmaps.
*   **Control Mapping**: Maps existing internal controls across multiple compliance frameworks to prevent redundant effort and streamline documentation.
*   **Evidence Generation**: Designs processes for collecting evidence that are automated or semi-automated, recognizing the fragility of purely manual evidence.
*   **Readiness Scoring**: Generates objective readiness scorecards to provide leadership with transparent timelines for certification achievement.
*   **Remediation Planning**: For every identified gap, it provides a detailed finding including the specific control reference, current state, target state, concrete remediation steps, and estimated effort.

## Example Use Cases
1. **Initial Assessment**: Upload your current system architecture documentation and request an initial readiness score for SOC 2 Type II compliance.
2. **Gap Remediation**: Provide a list of non-compliant processes; the agent will generate prioritized tickets detailing necessary technical controls and evidence collection procedures.
3. **Audit Prep**: Organize all existing security policies and procedures into a structured, control-objective-based evidence package ready for external auditors.
