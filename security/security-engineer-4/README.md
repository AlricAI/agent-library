## Overview
This agent serves as the final line of defense for Donchitos Game Studio, responsible for protecting all game assets, player data, and backend infrastructure from malicious activity. It specializes in proactive vulnerability assessment, designing anti-cheat mechanisms, and ensuring compliance with global privacy regulations.

## Capabilities
*   **Vulnerability Assessment:** Reviews codebases for critical flaws like injection attacks, buffer overflows, and insecure deserialization.
*   **Anti-Cheat System Design:** Develops server-authoritative validation and anomaly detection systems to combat cheating.
*   **Network Security Hardening:** Implements best practices including encryption, certificate pinning, and protection against replay attacks.
*   **Compliance Auditing:** Ensures adherence to major data privacy laws such as GDPR, CCPA, and COPPA.
*   **Threat Modeling:** Conducts formal threat modeling for all new features before they reach production environments.

## Example Use Cases
1. **Pre-Release Audit:** When a new game feature is implemented, use this agent to generate a comprehensive security review report detailing potential attack vectors and required remediation steps.
2. **Incident Response Simulation:** Simulate an active exploit attempt against the network layer; the agent will guide the investigation process and recommend immediate mitigation protocols.
3. **Compliance Check:** Before launching into a new region (e.g., EU), use this agent to generate a data privacy compliance checklist, ensuring all PII handling meets GDPR standards.
