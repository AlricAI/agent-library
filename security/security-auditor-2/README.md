## Overview
The Security Auditor is an expert AI agent designed to embed robust security practices throughout the entire software development lifecycle (SDLC). It specializes in DevSecOps methodologies, ensuring that security is not an afterthought but a core component of system design and deployment. This agent masters modern cybersecurity frameworks, from vulnerability scanning to advanced identity management.

## Capabilities
* **DevSecOps & Automation**: Integrates SAST/DAST/IAST into CI/CD pipelines, implements Security as Code (e.g., OPA), and manages container security policies for Kubernetes.
* **Authentication Mastery**: Provides deep expertise in modern identity protocols including OAuth 2.0/2.1, OpenID Connect, SAML 2.0, and implementing Zero Trust architectures.
* **Vulnerability Assessment**: Thoroughly analyzes systems against the OWASP Top 10 (2021) and guides adherence to standards like OWASP ASVS.
* **Compliance Guidance**: Assesses system architecture against major compliance frameworks such as GDPR, HIPAA, and SOC2 requirements.
* **Supply Chain Security**: Advises on implementing SLSA framework controls and managing Software Bills of Materials (SBOM).

## Example Use Cases
1. **Pre-Deployment Audit**: Feed the agent your application's architecture diagrams and code snippets to receive a comprehensive report detailing potential OWASP Top 10 vulnerabilities and necessary remediation steps.
2. **DevSecOps Pipeline Review**: Ask the agent to design a security gate for a new CI/CD pipeline, specifying required SAST/DAST tools and policy checks using Policy as Code principles.
3. **Authentication Hardening**: Request a review of your current user authentication flow (e.g., OAuth implementation) to ensure it meets Zero Trust principles and uses modern standards like WebAuthn.
