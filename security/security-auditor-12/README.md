## Overview
This agent acts as an expert security auditor, specializing in integrating robust security practices throughout the entire Software Development Life Cycle (SDLC). It possesses deep knowledge of modern cybersecurity frameworks, DevSecOps methodologies, and industry compliance standards like GDPR, HIPAA, and SOC2. Its goal is to help you build resilient, secure, and compliant systems from the ground up.

## Capabilities
*   **DevSecOps Integration:** Automates security checks (SAST, DAST, dependency scanning) directly into CI/CD pipelines, promoting a 'shift-left' security posture.
*   **Authentication & Authorization Mastery:** Implements best practices for modern identity protocols including OAuth 2.0/OIDC, Zero Trust architecture, and fine-grained access controls (RBAC/ABAC).
*   **Vulnerability Assessment:** Thoroughly analyzes systems against the OWASP Top 10 and ASVS standards to identify critical weaknesses.
*   **Compliance & Governance:** Provides guidance and checks necessary for adherence to major regulatory frameworks (e.g., GDPR, HIPAA).
*   **Supply Chain Security:** Assesses risks related to third-party dependencies using concepts like SBOMs and SLSA framework compliance.

## Example Use Cases
*   **Pre-Deployment Audit:** Feed the agent your application architecture diagrams and codebase snippets to receive a comprehensive report detailing potential OWASP vulnerabilities and required security hardening steps.
*   **CI/CD Pipeline Review:** Ask it to design the necessary security gates for a new microservice deployment, ensuring SAST scanning and secret management checks are enforced.
*   **Compliance Gap Analysis:** Provide documentation related to patient data handling (HIPAA) and request an audit checklist detailing all required controls and implementation steps.
