## Overview
This agent functions as an expert security auditor specializing in modern cybersecurity practices, DevSecOps methodologies, and industry compliance frameworks. It is designed to help integrate robust security measures early into the software development lifecycle (Shift Left) and ensure systems are resilient against modern threats.

## Capabilities
*   **DevSecOps & Security Automation**: Integrates SAST/DAST scanning into CI/CD pipelines, enforces Policy as Code using OPA, and manages container security policies for Kubernetes.
*   **Modern Authentication**: Masters identity protocols including OAuth 2.0/OIDC, implements Zero-Trust architectures, and advises on advanced MFA solutions like WebAuthn.
*   **Vulnerability Management**: Provides deep expertise across the OWASP Top 10 (2021) and guides adherence to standards like OWASP ASVS for rigorous application security verification.
*   **Compliance & Frameworks**: Assesses systems against major compliance needs, including GDPR, HIPAA, and SOC2 requirements, ensuring secure data handling throughout development.

## Example Use Cases
*   **Pre-Deployment Audit**: Run a full audit checklist on a new microservice architecture to identify potential OWASP Top 10 vulnerabilities before it hits production.
*   **CI/CD Hardening**: Advise on integrating dependency scanning and secret detection directly into a GitHub Actions workflow, ensuring secrets are never committed.
*   **Compliance Gap Analysis**: Review an existing application's access control mechanisms against the principles of least privilege required by SOC2 controls.
