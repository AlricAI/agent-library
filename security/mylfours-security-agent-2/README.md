## Overview
MyLfours is a specialized AI agent designed to function as a proactive Security Specialist. Its core directive is continuous monitoring, threat pattern recognition, and maintaining system integrity across various digital assets. It operates by adhering strictly to established protocols, ensuring that every alert is investigated and documented.

## Capabilities
*   **SSH Attack Monitoring:** Provides real-time vigilance over SSH access points for suspicious activity.
*   **Service Health Security:** Assesses the security posture of running services to prevent vulnerabilities from being exploited.
*   **Threat Pattern Recognition:** Analyzes logs and data streams to identify emerging or subtle threat patterns.
*   **Incident Response Preparation:** Maintains readiness by structuring necessary steps and documentation for immediate incident handling.
*   **System Hardening Recommendations:** Delivers actionable, expert advice on improving overall system security baselines.

## Example Use Cases
*   **Security Audit Simulation:** Instruct the agent to run a simulated audit against a set of provided logs, requiring it to document all findings in the designated `workspace/REPORTS/` directory.
*   **Alert Triage:** Feed the agent a stream of security alerts and task it with investigating each one, escalating any critical findings directly to 'Sentinel' as per its guidelines.
*   **System Hardening Plan Generation:** Ask MyLfours to review the current system architecture documentation (provided context) and generate a prioritized list of hardening recommendations for implementation.
