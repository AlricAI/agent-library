## Overview
This agent specializes in the complex and critical domains of OAuth 2.0 and OpenID Connect (OIDC). It acts as a security architect, ensuring that all authentication and authorization mechanisms adhere to industry best practices, cryptographic standards, and the latest specifications.

Whether you are building a new application or auditing an existing system, this agent guides you through secure token management, flow implementation, and vulnerability mitigation.

## Capabilities
*   **Flow Implementation:** Designs and validates all major OAuth 2.0 grant types (e.g., Authorization Code with PKCE, Client Credentials).
*   **Token Lifecycle Management:** Advises on the secure handling, storage, refreshing, and revocation of access, refresh, and ID tokens.
*   **Standards Compliance:** Ensures strict adherence to OIDC scopes, claims management, and overall OAuth 2.0 specifications.
*   **Security Auditing:** Performs threat modeling specific to identity protocols and checks for common vulnerabilities (e.g., improper redirect URIs).
*   **Documentation Generation:** Produces detailed guides, test plans, and vulnerability reports for implemented flows.

## Example Use Cases
*   **New Service Integration:** Need to connect a third-party service using modern SSO? Ask the agent to design the optimal OAuth 2.0 flow, including PKCE implementation.
*   **Security Audit:** Suspect an existing API uses outdated token handling? Provide the context, and the agent will audit it against current best practices.
*   **Developer Guidance:** Developing a client application that needs user consent? Use this agent to structure the necessary user flows and required scopes.
