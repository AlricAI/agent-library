## Overview
The Security Procurement Advisor is designed to assist in the most critical phase of enterprise sales: due diligence. Instead of focusing solely on product features, this agent addresses the deep concerns of large buyers regarding data governance, security controls, and legal compliance. Its primary function is to provide truthful, concrete answers that mitigate risk for both parties.

## Capabilities
*   **Evidence-Based Answering:** Provides responses tied directly to documented controls and observable runtime behavior, avoiding speculation.
*   **Gap Analysis:** Quickly identifies missing evidence or policy gaps within a large questionnaire (DDQ).
*   **Issue Triaging:** Distinguishes between genuine technical deficiencies and underlying commercial/rights issues masked by security questions.
*   **Concise Reporting:** Collapses massive questionnaires into actionable, high-priority requests and red flags, preventing vague overstatements of readiness.

## Example Use Cases
*   **Handling DDQs:** Feed it a comprehensive due diligence questionnaire and receive a prioritized list of necessary evidence items and critical unanswered questions.
*   **Vetting Claims:** When presented with a security claim (e.g., 'We are GDPR compliant'), the agent will prompt for the specific supporting documentation or control implementation details required to verify that claim.
*   **Risk Identification:** Use it to review product architecture summaries and flag potential areas where data isolation, retention policies, or access rights might be ambiguous or undocumented.
