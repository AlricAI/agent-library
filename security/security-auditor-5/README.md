## Overview
This agent functions as an expert security auditor specializing in modern cybersecurity practices, DevSecOps methodologies, and industry compliance frameworks. It is designed to embed security deeply into the development lifecycle (Shift Left) rather than treating it as a final checkpoint.

## Capabilities
* **DevSecOps Integration:** Automates security checks within CI/CD pipelines, including SAST, DAST, IAST, and dependency scanning.
* **Compliance Auditing:** Assesses adherence to major standards like GDPR, HIPAA, SOC2, and OWASP guidelines.
* **Threat Modeling:** Systematically identifies potential attack vectors and design weaknesses in applications.
* **Authentication Security:** Provides deep expertise on modern identity protocols (OAuth 2.0/OIDC) and Zero Trust architecture implementation.
* **Vulnerability Assessment:** Covers the latest threats outlined by the OWASP Top 10, ensuring robust application security verification.

## Example Use Cases
* **Pre-Deployment Audit:** Feed it a microservice architecture diagram and ask it to perform a threat model against potential data exfiltration paths.
* **Compliance Gap Analysis:** Provide documentation for a new feature and request an audit checklist against HIPAA requirements, highlighting necessary security controls.
* **Pipeline Hardening:** Integrate it into your CI/CD workflow to automatically scan container images and enforce least-privilege policies before deployment.
