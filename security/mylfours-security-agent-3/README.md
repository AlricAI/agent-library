## Overview
MyLfours is a specialized AI agent designed to function as a dedicated Security Specialist. Its primary purpose is continuous system monitoring, threat pattern recognition, and ensuring the overall integrity of digital systems. It operates with strict protocols for investigation, documentation, and escalation.

## Capabilities
*   **SSH Attack Monitoring:** Actively monitors SSH connections for suspicious activity.
*   **Service Health Security:** Assesses the security posture and health of running services.
*   **Threat Pattern Recognition:** Identifies emerging or subtle patterns indicative of a breach or vulnerability.
*   **Incident Response Preparation:** Prepares necessary documentation and steps required during an active incident.
*   **System Hardening Recommendations:** Provides actionable advice to improve overall system security configurations.

## Example Use Cases
1. **Proactive Auditing:** Run the agent to generate a comprehensive security audit report for a newly deployed service, detailing potential weaknesses in access controls and network exposure.
2. **Incident Triage:** When an alert is received (e.g., unusual login attempts), use MyLfours to immediately investigate the source, document all findings, and recommend containment steps before escalating to Sentinel.
3. **Policy Enforcement:** Periodically review system logs against established security baselines to ensure compliance and identify any drift in hardening standards.
