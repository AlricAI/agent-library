## Overview
This agent acts as a dedicated expert focused entirely on the OWASP Top 10 vulnerabilities. It provides systematic, deep-dive security assessments for web applications, ensuring that common and critical attack vectors are identified and remediated.

It covers everything from injection flaws (SQLi, XSS) to broken authentication and insecure configurations, providing actionable reports suitable for development teams and security auditors alike.

## Capabilities
*   **Comprehensive Risk Assessment:** Performs structured reviews against all current OWASP Top 10 categories.
*   **Vulnerability Identification:** Detects specific flaws such as Injection vulnerabilities, Broken Access Control, and Sensitive Data Exposure.
*   **Remediation Guidance:** Provides detailed, actionable recommendations for mitigating identified security risks.
*   **Secure Coding Validation:** Checks code patterns against best practices for input validation, output encoding, and secure session management.
*   **Tool Integration Simulation:** Guides the process of using automated tools (like ZAP) alongside manual review techniques.

## Example Use Cases
*   **Pre-Deployment Audit:** Run this agent on a new application codebase to generate a preliminary OWASP Top 10 risk report before going live.
*   **Code Review Deep Dive:** Paste a specific module or API endpoint and ask the agent to validate it for XSS or Injection vulnerabilities.
*   **Security Gap Analysis:** Provide documentation about an existing system and ask the agent to generate a security configuration checklist based on industry best practices.
