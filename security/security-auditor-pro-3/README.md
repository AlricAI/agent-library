## Overview
This agent acts as an expert security auditor specializing in modern cybersecurity practices, DevSecOps methodologies, and industry compliance frameworks. It is designed to help integrate robust security measures early into the development lifecycle (Shift-Left) and ensure systems are resilient against common threats.

## Capabilities
*   **DevSecOps Integration**: Automates security checks within CI/CD pipelines using SAST, DAST, IAST, and dependency scanning.
*   **Compliance & Standards**: Assesses adherence to major frameworks like GDPR, HIPAA, SOC2, OWASP Top 10, and ASVS.
*   **Identity Security**: Masters modern authentication protocols including OAuth 2.0/OIDC, implementing Zero-Trust principles (RBAC/ABAC).
*   **Infrastructure Hardening**: Provides expertise in container security (Kubernetes), secrets management (Vault), and supply chain integrity (SBOM).

## Example Use Cases
1. **Pre-Deployment Audit**: Feed it a microservice architecture diagram and ask it to generate a threat model based on potential data flows.
2. **Compliance Gap Analysis**: Provide documentation for a new feature and request an audit report detailing necessary controls to meet SOC2 requirements.
3. **Pipeline Hardening**: Integrate it into your CI/CD workflow to automatically scan container images and check for outdated dependencies before deployment.
