## Overview
This agent functions as an expert security auditor, providing deep knowledge across modern cybersecurity practices, DevSecOps methodologies, and critical compliance frameworks. It is designed to help integrate robust security measures early into the software development lifecycle (Shift-Left Security).

## Capabilities
* **DevSecOps & Automation:** Integrates SAST/DAST scanning into CI/CD pipelines and implements Policy as Code using OPA.
* **Authentication Mastery:** Implements modern identity standards including OAuth 2.0/OpenID Connect, supporting Zero-Trust architectures and advanced MFA.
* **Vulnerability Management:** Deep knowledge of OWASP Top 10 (2021) and ASVS to identify common application weaknesses.
* **Compliance Frameworks:** Assists with adherence checks for major standards like GDPR, HIPAA, and SOC2.
* **Supply Chain Security:** Manages dependency scanning and SBOM generation to secure the entire software supply chain.

## Example Use Cases
1. **Pre-Deployment Audit:** Run a comprehensive audit on a new microservice architecture to check for OWASP Top 10 vulnerabilities and ensure proper OAuth scope implementation.
2. **CI/CD Hardening:** Advise on integrating dependency scanning (SCA) and secret detection directly into a Jenkins or GitLab pipeline definition file.
3. **Compliance Gap Analysis:** Review existing access control logic against HIPAA requirements to identify potential over-privileging issues, recommending RBAC adjustments.
