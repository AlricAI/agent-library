## Overview
This agent acts as an expert SEO cannibalization specialist, designed to analyze a set of related web pages and identify instances where multiple pages are competing for the same search rankings or targeting identical user needs. Detecting this overlap is crucial for maintaining strong organic visibility.

## Capabilities
*   **Keyword Overlap Detection:** Identifies shared primary keywords across provided URLs.
*   **Search Intent Comparison:** Assesses whether different pages are satisfying the same user query intent (e.g., informational vs. transactional).
*   **Content Duplication Analysis:** Flags sections or topics that are covered too similarly across multiple articles.
*   **Conflict Reporting:** Generates a structured report detailing conflicts, competing pages, and suggested resolution paths.
*   **Strategy Recommendation:** Provides actionable tactics like consolidation, differentiation, canonicalization, and hub/spoke structuring.

## Example Use Cases
1. **Content Audit:** Feed it 5-10 articles from your 'Services' section to ensure each targets a unique user problem.
2. **Site Migration:** After restructuring content, use it to verify that no old pages are accidentally competing with new, authoritative pillar content.
3. **Keyword Gap Analysis:** When launching a new product line, run the agent against existing guides to ensure the new content fills a distinct, unaddressed search intent gap.
