## Overview
This agent is designed to act as a comprehensive sales enablement tool, transforming raw product knowledge and sales goals into actionable, step-by-step calling scripts. It structures the entire customer interaction lifecycle—from initial opening lines to handling common objections.

It moves beyond simple talking points by incorporating proven frameworks like 'Feel-Felt-Found' objection handling, ensuring the user sounds knowledgeable, empathetic, and authoritative during a live call.

## Capabilities
*   **Script Generation:** Creates time-boxed scripts for specific stages (e.g., 15-second opening).
*   **Discovery Questioning:** Provides targeted questions to uncover pain points in the prospect's current setup.
*   **Value Proposition Structuring:** Helps organize product benefits into compelling, quantifiable value statements.
*   **Objection Handling:** Implements psychological frameworks (like Feel-Felt-Found) to neutralize common customer pushback.
*   **Call Flow Mapping:** Maps out the ideal path through a sales conversation (Discovery $\rightarrow$ Pitch $\rightarrow$ Close).

## Example Use Cases
1. **Cold Calling Prep:** Input your product details and target industry, and receive a full script ready for immediate use.
2. **Objection Rehearsal:** Feed the agent three common objections you face (e.g., 'Too expensive,' 'Happy with current vendor'), and it will generate tailored responses using proven sales psychology.
3. **Sales Training:** Use this to role-play or review a pitch, ensuring all necessary components—social proof, value metrics, and next steps—are included for maximum impact.
