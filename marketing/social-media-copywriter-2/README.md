## Overview
This agent is an expert social media copywriter specifically designed to promote podcast episodes. Its core function is to take comprehensive episode notes and transform them into native, high-engagement content tailored for three major platforms: Twitter/X, LinkedIn, and Instagram.

It moves beyond simple announcements by focusing on creating genuine 'Fear Of Missing Out' (FOMO) and highlighting the unique value proposition of each episode.

## Capabilities
*   **Multi-Platform Adaptation:** Generates distinct content for Twitter/X threads (3-5 tweets), professional LinkedIn updates, and visually engaging Instagram captions.
*   **Deep Content Analysis:** Extracts key topics, notable quotes, guest credentials, and the episode's unique selling points from provided show notes.
*   **Engagement Optimization:** Crafts hooks designed to stop scrolling, builds narrative tension across threads, and includes clear calls-to-action (CTAs).
*   **Quality Assurance:** Verifies factual accuracy of names and details before finalizing all promotional copy.

## Example Use Cases
*   **Launching a New Episode:** Provide the full show notes for a recent interview. The agent will return a complete social media package ready to schedule across all three platforms, ensuring each piece feels native to its channel.
*   **Boosting Evergreen Content:** If you have an older episode with high value, feed it the summary and ask the agent to generate fresh promotional copy targeting a specific industry pain point to drive new listeners.
