## Overview
This agent acts as a dedicated Xiaohongshu (RED) Marketing Specialist, designed to help brands master the nuances of this popular Chinese lifestyle and e-commerce platform. It ensures all generated content aligns with RED's aesthetic, trending formats, and user engagement patterns.

## Capabilities
*   **Trend Analysis:** Identifies current viral topics, aesthetics, and popular hashtags on Xiaohongshu.
*   **Content Ideation:** Generates diverse post concepts, including photo series, carousel guides, and short video scripts.
*   **Copywriting:** Crafts catchy, native-sounding Chinese captions optimized for readability and virality.
*   **Visual Direction:** Provides detailed prompts and mood boards suggestions to guide graphic designers or AI image tools for optimal visual appeal.

## Example Use Cases
*   **Launching a New Skincare Line:** Provide the product details and target demographic; the agent will generate 3 distinct, high-engagement post concepts suitable for RED's beauty niche.
*   **Seasonal Campaign Planning:** Input the campaign theme (e.g., 'Autumn Travel'); the agent will outline a multi-week content calendar with suggested formats and trending keywords to maximize reach.
*   **Competitor Analysis Simulation:** Describe a competitor's recent successful post; the agent can suggest ways to create superior, differentiating content that captures similar user interest.
