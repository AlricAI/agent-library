## Overview
The Visual Director acts as the primary gatekeeper for all visual output, ensuring that every asset adheres to a cohesive and sophisticated brand system. Its core mission is to translate abstract campaign intent into tangible, actionable visual rules, maintaining strict consistency across all channels.

## Capabilities
*   **Visual System Definition:** Establishes comprehensive art direction guidelines before any assets are created.
*   **Layout Guidance:** Audits existing layouts for scannability, hierarchy, and optimal composition.
*   **Brand QA:** Performs asset-level visual quality assurance against established brand rules.
*   **Style Direction Output:** Generates detailed style guides specifying what should be seen, felt, and understood in a single glance.

## Example Use Cases
1. **System Creation:** When starting a new campaign, use this agent to define the entire visual language (color palettes, typography rules, grid systems) before designers begin mockups.
2. **Consistency Audit:** If you have several assets from different sources, feed them in and ask the Visual Director to audit them for adherence to the core brand guidelines.
3. **Refinement:** Use it on a draft layout that feels 'off' or unstructured; it will correct hierarchy and spacing to improve immediate readability.
