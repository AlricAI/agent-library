# YouAndINotAI Copy Requirements Document

## Overview

This document specifies copy requirements for all UI elements in the YouAndINotAI platform. All copy must adhere to our brand voice: authentic, inclusive, empowering, and community-focused while avoiding manipulative or gamified language.

## Core Brand Voice Guidelines

### Tone Principles

1. **Authentic**: Speak genuinely without artificial enthusiasm
2. **Inclusive**: Welcome all users regardless of background or identity
3. **Empowering**: Enable user agency and informed decision-making
4. **Community-Focused**: Emphasize human connection over transaction
5. **Respectful**: Honor user dignity and privacy

### Language Rules

1. **Never use**: "donate", "donation", "solicitation" (Florida §496.405 compliance)
2. **Avoid**: Gamification terms ("points", "levels", "achievements")
3. **Avoid**: Manipulative urgency ("limited time", "hurry")
4. **Prefer**: Direct, clear language over marketing speak
5. **Support**: Florida charity registration disclosures where required

## UI Element Copy Requirements

### Authentication & Account Creation

#### Sign Up/Login Headers

- Primary: "Join Our Community"
- Secondary: "Real people. Real connections. No bots."

#### Form Labels

- Email: "Email Address"
- Password: "Password" (6+ characters)
- Confirm Password: "Confirm Password"
- First Name: "First Name"
- Last Name: "Last Name"

#### Action Buttons

- Sign Up: "Create Account"
- Log In: "Sign In"
- Continue with Email: "Continue with Email"
- Continue with Phone: "Continue with Phone"

#### Success Messages

- "Welcome to YouAndINotAI! Let's get started."
- "Account created successfully."

#### Error Messages

- "Invalid email or password."
- "Email already registered. Try logging in instead?"
- "Password must be at least 6 characters."
