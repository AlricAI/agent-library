## Overview
This Elite SEO & GEO Specialist bridges the gap between traditional search engine optimization (SEO) and Generative Engine Optimization (GEO). In the modern digital landscape, visibility requires satisfying both human crawlers and AI models that synthesize information. This agent ensures your content is not just indexed, but intelligently understood and cited by advanced AI systems.

## Capabilities
*   **Semantic Architecture:** Implements deep semantic linking and structures content to satisfy complex informational queries.
*   **AI Search Readiness (GEO):** Optimizes for how LLMs consume data, ensuring clear topic authority and citation potential.
*   **Technical Auditing:** Manages best practices for Core Web Vitals (LCP, INP, CLS) and implements advanced structured data (Schema.org).
*   **Scale-Aware Strategy:** Adapts optimization rigor based on project scope—from basic MVP foundations to full Enterprise dominance.

## Example Use Cases
*   **Technical SEO Audit:** Run a comprehensive audit using the `/seo` tool to check for technical debt, sitemap completeness, and schema implementation across a site structure.
*   **Content Gap Analysis:** Provide existing content and ask the agent to rewrite sections specifically targeting AI summarization patterns, ensuring proper H1-H3 hierarchy and FAQ structuring.
*   **Performance Remediation:** Use `npx lighthouse [url]` to diagnose performance bottlenecks and receive actionable steps to improve user experience signals critical for modern rankings.
