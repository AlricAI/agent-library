## Overview
SOUL is an AI agent designed to transform raw data or meeting notes into polished, executive-ready slide decks. Its core philosophy centers on the 'phone-first' principle: every presentation must be instantly digestible on a mobile device.

The agent enforces strict design principles—such as limiting slides to one clear message and keeping bullet points under 15 words—to ensure that busy stakeholders can grasp the key takeaways immediately, even when reviewing content on the go.

## Capabilities
*   **Mobile-First Design:** Automatically reviews and redesigns content if it is not optimized for mobile viewing.
*   **Structured Storytelling:** Ensures one distinct state or message per slide, preventing information overload.
*   **Workflow Automation:** Handles post-creation tasks by organizing files in designated Drive folders and scheduling follow-up calendar events.
*   **Spec Adherence:** Requires an approved specification before building the presentation to maintain quality control.

## Example Use Cases
*   **Post-Meeting Reporting:** Given meeting notes, SOUL builds a deck summarizing key decisions, organizes supporting documents in Google Drive, and schedules a follow-up review with all necessary links included.
*   **Status Updates:** When reporting on a project state (e.g., Normal, Offline, Error), it generates three distinct, simple slides rather than cramming all status points onto one overview slide.
*   **Executive Briefings:** Takes complex App Designer outputs and refines them into a concise narrative suitable for immediate consumption by senior leadership.
