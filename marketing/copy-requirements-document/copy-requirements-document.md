# YouAndINotAI Copy Requirements Document

## Overview

This document specifies copy requirements for all UI elements in the YouAndINotAI platform. All copy must adhere to our brand voice: authentic, inclusive, empowering, and community-focused while avoiding manipulative or gamified language.

## Core Brand Voice Guidelines

### Tone Principles

1. **Authentic**: Speak genuinely without artificial enthusiasm
2. **Inclusive**: Welcome all users regardless of background or identity
3. **Empowering**: Enable user agency and informed decision-making
4. **Community-Focused**: Emphasize human connection over transaction
5. **Respectful**: Honor user dignity and privacy

### Language Rules

1. **Never use**: "donate", "donation", "solicitation" (Florida §496.405 compliance)
2. **Avoid**: Gamification terms ("points", "levels", "achievements")
3. **Avoid**: Manipulative urgency ("limited time", "hurry")
4. **Prefer**: Direct, clear language over marketing speak
5. **Support**: Florida charity registration disclosures where required

## Accessibility Copy

### Alt Text Standards

- Images: Descriptive, contextual alt text
- Icons: Clear functional labels
- Decorative: Empty alt or role="presentation"

### Screen Reader Optimizations

- Landmark regions: Clearly defined
- Heading structure: Logical and descriptive
- Form labels: Explicit associations

### Keyboard Navigation

- Skip links: "Skip to main content"
- Focus indicators: Visually distinctive
- Navigation instructions: Context-sensitive

## Internationalization Considerations

### Translation Readiness

- All UI copy written with translation in mind
- Cultural sensitivity in language choices
- Gender-neutral phrasing where possible
- Regional variations noted for localization

### Date/Time Formatting

- Relative timestamps: "2 hours ago", "Yesterday"
- Absolute dates: Consistent localized format
- Time zones: User's local time zone by default

## Testing & Validation

### Copy Review Process

1. All UI copy reviewed by CMO before implementation
2. Accessibility impact assessment for all new copy
3. Florida legal compliance verification
4. User testing with diverse demographics
5. A/B testing for high-impact messaging

### Quality Assurance

- Grammar and spelling checked
- Consistency across platform verified
- Brand voice adherence confirmed
- Tone appropriateness validated
