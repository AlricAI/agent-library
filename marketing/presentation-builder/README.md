## Overview
SOUL is a specialized presentation builder designed for high-stakes, mobile-first communication. Its core philosophy dictates that every slide must convey one single, crystal-clear message, ensuring maximum impact when viewed on a phone.

The agent enforces strict organizational principles, guaranteeing that the final output isn't just slides, but a complete, actionable package including organized cloud storage and a scheduled review meeting.

## Capabilities
*   **Mobile Optimization:** Restructures content to ensure readability and clarity on mobile devices.
*   **Single Focus Per Slide:** Enforces the 'one message per slide' rule, preventing information overload.
*   **Process Automation:** Automatically creates necessary follow-up items, including scheduling a calendar event and organizing Google Drive folders.
*   **Compliance Checking:** Requires an approved specification before building any presentation to maintain quality control.

## Example Use Cases
*   **Weekly Status Reports:** Takes raw sprint data and generates a concise deck for executive review on their mobile device.
*   **Project Updates:** Structures complex technical findings into distinct, easily digestible slides (e.g., one slide for 'Normal State,' another for 'Error State').
*   **Stakeholder Demos:** Ensures the final package is delivered to the correct stakeholders with all assets pre-organized in a designated Drive folder.
