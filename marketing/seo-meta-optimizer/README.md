## Overview
This agent acts as a specialized meta tag optimization expert, ensuring your digital content is discoverable and highly clickable. It moves beyond basic keyword stuffing to craft metadata that incorporates psychological triggers, emotional resonance, and strict character compliance for maximum search engine visibility.

## Capabilities
*   **Comprehensive Metadata Generation:** Creates multiple variations (3-5) for Titles, Descriptions, and URLs.
*   **Character & Pixel Compliance:** Adheres strictly to industry best practices (e.g., 150-160 char descriptions, 50-60 char titles).
*   **Advanced Optimization Rules:** Integrates emotional triggers, power words, action verbs, and clear Calls-to-Action (CTAs).
*   **Platform Specific Output:** Provides ready-to-implement suggestions for WordPress plugins (Yoast/RankMath) and modern frameworks (Astro/Next.js components).
*   **Structural Recommendations:** Optimizes URLs using lowercase, hyphens, and primary keyword placement.

## Example Use Cases
1. **Pre-Publication Audit:** Feed it a draft article and target keywords to receive a complete 'Meta Package' ready for immediate deployment.
2. **A/B Testing Prep:** Request multiple title variations specifically designed for A/B testing on high-traffic pages.
3. **Technical SEO Check:** Ask for schema markup recommendations or component code snippets to integrate metadata directly into your site's frontend framework.
