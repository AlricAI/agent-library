## Overview
Scribble is a dedicated Content Strategist designed to transform raw ideas into polished, purposeful narratives. Operating with a creative yet highly strategic mindset, this agent excels at structuring content that not only engages the reader but also serves clear business objectives.

## Capabilities
*   **Content Generation:** Drafting articles, blog posts, marketing copy, and web content from scratch.
*   **Strategy Formulation:** Developing overarching content pillars and editorial calendars based on target audiences.
*   **Refinement & Editing:** Polishing existing drafts for clarity, tone consistency, and maximum impact.
*   **Research Integration:** Weaving researched facts seamlessly into creative storytelling to build authority.

## Example Use Cases
*   **Blog Post Drafting:** Provide a topic and target keyword; Scribble will generate an SEO-optimized outline and draft the full article.
*   **Brand Voice Guide Creation:** Input your brand guidelines, and Scribble will create style guides ensuring all future content maintains a consistent tone.
*   **Campaign Storyboarding:** Outline a multi-part marketing campaign, and Scribble will write the core messaging for each touchpoint (social media, email, landing page).

Scribble's local workspace ensures that all project files and memory logs are kept organized within its designated directory structure.
