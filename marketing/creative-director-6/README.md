## Overview
The Creative Director acts as a high-level Production Lead, responsible for owning the entire creative execution lifecycle. This agent's core function is to bridge the gap between abstract marketing strategy (provided by leadership) and tangible, polished creative assets. It ensures that all output maintains brand integrity, adheres to strict quality standards, and follows a structured production workflow.

## Capabilities
*   **Brief Normalization:** Takes vague or complex strategic inputs and refines them into clear, actionable work packages with defined goals and constraints.
*   **Workflow Coordination:** Manages the delegation of tasks to specialized 'swarm' roles (e.g., copywriters, designers) according to a structured operating model.
*   **Quality Assurance Gatekeeping:** Implements mandatory review gates for brand and editorial compliance, providing specific, actionable fix lists rather than general feedback when standards fail.
*   **Structured Communication:** Enforces a clear communication pattern: State Goal $\rightarrow$ State Constraint $\rightarrow$ State Next Owner $\rightarrow$ State Risk.

## Example Use Cases
1. **Campaign Kickoff:** Provide the CMO's high-level campaign goal, and the Creative Director will structure it into a multi-stage production plan, assigning initial tasks to copy and visual specialists.
2. **Asset Review:** Submit a draft piece of content that needs final sign-off; the agent will run it through its QA process, flagging any deviations from brand guidelines with concrete revision instructions.
3. **Project Scoping:** When given a broad objective (e.g., 'Improve Q3 engagement'), use this agent to break it down into measurable deliverables and assign ownership for each phase.
