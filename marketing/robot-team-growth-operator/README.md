## Overview
This agent serves as the central 'robot-team growth operator' within the Blueprint ecosystem. Its primary function is to synthesize raw demand intelligence into a structured, reusable playbook for city demand planning initiatives. It acts as the orchestrator, ensuring that all proposed strategies maintain consistency across core elements like ICP definition, message hierarchy, and proof paths.

## Capabilities
*   **Playbook Generation:** Translates research findings into Blueprint's standardized robot-team demand playbook.
*   **System Maintenance:** Ensures adherence to established frameworks (ICP, message hierarchy, buyer funnel).
*   **Workflow Orchestration:** Delegates specific tasks to specialized downstream agents (e.g., `conversion-agent`, `analytics-agent`).
*   **Asset Briefing:** Writes detailed creative briefs for visual assets and routes them to the appropriate coding/design agent (`webapp-codex`).
*   **Process Documentation:** Maintains a rigorous internal proposal standard, requiring human sign-off before public execution.

## Example Use Cases
1. **New Market Entry:** When new city demand data arrives, use this agent to draft an initial playbook, defining the target site and suggesting the first workflow lane for testing.
2. **Campaign Refinement:** After running a hosted demo or analyzing experiment logs, feed the results back into this agent to update the core messaging and proof path within the master playbook.
3. **Handoff Management:** When a component of the plan is ready (e.g., campaign copy), use it to formally delegate the next step to another specialized agent with clear instructions.
