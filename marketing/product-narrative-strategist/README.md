## Overview
This agent embodies the philosophy that a feature cannot be adopted if its value proposition cannot be explained in a single, compelling sentence. It acts as an upstream marketing gatekeeper, ensuring every product decision strengthens the core narrative rather than diluting it. It draws inspiration from frameworks like StoryBrand and Amazon's 'working backwards' method.

## Capabilities
*   **Story Coherence Check:** Assesses if a proposed feature reinforces the primary product identity and competitive advantage (e.g., making the comparison to existing tools more favorable).
*   **One-Sentence Test:** Forces simplification by evaluating if the feature's purpose and value can be distilled into one clear sentence.
*   **Clarity Assessment:** Checks for developer comprehension, agent readability, and jargon usage to ensure universal understanding.
*   **Narrative Alignment Scoring:** Provides feedback on whether a requirement supports or detracts from the established product story.

## Example Use Cases
*   **Feature Vetting:** Input a detailed feature spec and ask the agent: "Does this strengthen our core narrative against Competitor X?"
*   **Positioning Refinement:** Provide three potential value statements for a new module and ask the agent to select the one that is most instantly understandable.
*   **Documentation Review:** Feed in draft documentation and request an audit specifically checking for jargon and single-sentence explainability.
