## Overview
This agent acts as an expert in both Search Engine Optimization (SEO) and Generative Engine Optimization (GEO). Its core philosophy is to create content that satisfies human readers while being perfectly structured for machine consumption, ensuring visibility across platforms like Google and AI chatbots.

## Capabilities
*   **Dual-Target Auditing:** Simultaneously assesses content against traditional SEO best practices and modern GEO requirements.
*   **E-E-A-T Enhancement:** Provides actionable advice on demonstrating Experience, Expertise, Authoritativeness, and Trustworthiness within your content structure.
*   **Technical Checklist Execution:** Reviews technical elements such as Core Web Vitals (LCP, INP, CLS), sitemap configuration, and schema markup validity.
*   **AI Citation Structuring:** Advises on formatting content—using original statistics, clear definitions, and attributed quotes—to maximize the likelihood of being cited by AI models.

## Example Use Cases
1. **Content Refresh Audit:** Provide an existing blog post URL and ask the agent to run a full audit, scoring it against both SEO and GEO metrics and providing a prioritized action plan.
2. **Technical Implementation Plan:** Upload your site's current technical setup details (e.g., speed scores, sitemap status) and request a remediation checklist for passing Core Web Vitals.
3. **AI Readability Optimization:** Give the agent a draft article and ask it to restructure sections specifically to increase its chances of being pulled as an 'extracta' or key definition by generative AI.
