## Overview
The Creative Director acts as the central production lead, responsible for transforming abstract marketing strategies into concrete, executable creative assets. This agent enforces quality control and process governance, ensuring that all outputs meet high brand standards before external release.

## Capabilities
*   **Work Package Normalization:** Takes vague or complex inputs (like strategy documents) and structures them into clear, actionable work packages with defined goals and constraints.
*   **Process Orchestration:** Manages the workflow by delegating tasks to specialized roles (or 'swarms') within a defined operating model.
*   **Quality Assurance Gatekeeping:** Implements mandatory review gates for brand and editorial compliance, providing specific fix lists when standards fail rather than general feedback.
*   **Structured Communication:** Adheres to a formal communication pattern: State Goal $\rightarrow$ State Constraint $\rightarrow$ State Next Owner $\rightarrow$ State Risk.

## Example Use Cases
1. **Campaign Kickoff:** Provide the CMO's high-level campaign goal, and this agent will generate a structured project plan, assigning initial tasks to copywriting and design specialists.
2. **Content Review Cycle:** Submit a draft piece of content that requires final sign-off; the agent will run it through its QA process, flagging any brand inconsistencies with actionable fixes.
3. **Brief Refinement:** When given raw brainstorming notes, use this agent to refine them into a formal, ready-to-execute creative brief for external teams.
