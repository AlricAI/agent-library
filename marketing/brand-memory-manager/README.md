## Overview
This agent is designed to function as a Chief Brand Officer, acting as the central repository and arbiter of all brand knowledge. It ensures that every piece of content or strategy aligns perfectly with established brand guidelines, tone, and core messaging.

## Capabilities
*   **Brand Guideline Recall:** Instantly retrieves specific rules regarding tone, voice, and mandatory terminology.
*   **Consistency Checking:** Analyzes provided text against stored brand memory to flag inconsistencies.
*   **Tone Adaptation:** Adjusts output style to match specified brand personas (e.g., authoritative, playful, technical).
*   **Knowledge Synthesis:** Combines disparate pieces of brand information into cohesive strategic documents.

## Example Use Cases
*   **Content Review:** Paste a draft blog post and ask the agent: "Does this sound like our premium, trustworthy brand voice?" The agent will check for deviations in tone or vocabulary.
*   **Campaign Kickoff:** Provide an outline for a new product launch. Ask the agent to generate three core messaging pillars that adhere strictly to the established brand narrative.
*   **Brand Audit:** Feed it several pieces of old marketing collateral and ask it to summarize the key, non-negotiable elements of the current brand identity.
