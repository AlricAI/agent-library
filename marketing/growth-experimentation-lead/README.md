## Overview
This agent acts as a Growth Experimentation Lead, dedicated to running rigorous, data-backed experiments to uncover true sources of growth and identify tactics ready for scaling. It operates under strict protocols, ensuring that no resource is allocated without measurable evidence.

## Capabilities
*   **Experiment Design:** Creates structured plans based on defined growth targets and channel data.
*   **Funnel Analysis:** Pinpoints drop-off points in the customer journey requiring immediate testing.
*   **Hypothesis Generation:** Develops cross-channel hypotheses focused on tangible revenue or retention improvements.
*   **Decision Making:** Provides clear 'Scale' or 'Stop' recommendations based strictly on measured results against a baseline.
*   **Metric Focus:** Explicitly ignores vanity metrics, focusing only on actionable KPIs.

## Example Use Cases
1. **Funnel Drop Analysis:** If conversion rates drop between the product page and checkout, use this agent to design A/B tests targeting that specific friction point.
2. **Channel Validation:** When a new marketing channel shows initial promise, task this agent with designing an economic validation experiment before any budget increase is approved.
3. **Scaling Strategy:** After successfully proving a tactic's ROI in a small test, use it to build the full rollout plan and necessary monitoring dashboards.
