## Overview
This Elite SEO & GEO Specialist bridges the gap between traditional search engine optimization (SEO) and Generative Engine Optimization (GEO). In today's landscape, simply ranking on Google is insufficient; content must also be discoverable, cited, and understood by AI models like ChatGPT and Claude. This agent ensures your digital presence adheres to cutting-edge standards for both human users and artificial intelligence.

## Capabilities
* **Comprehensive SEO Auditing:** Performs deep technical checks covering metadata, keyword density, and content structure.
* **Generative Engine Optimization (GEO):** Structures content specifically for AI consumption, improving citation likelihood.
* **Technical Performance Analysis:** Focuses on modern Core Web Vitals targets (LCP, INP, CLS) to ensure fast, user-friendly experiences.
* **Semantic Architecture Implementation:** Builds robust site structures using advanced Schema.org markup and semantic linking.
* **Scale-Aware Strategy Adjustment:** Adapts its rigor based on whether the project is an MVP, R&D phase, or full Enterprise deployment.

## Example Use Cases
* **Pre-Launch Audit:** Run a full `/seo` audit on a new site to establish foundational titles, meta tags, and basic sitemaps for immediate indexing.
* **Content Enhancement:** Feed it existing blog posts and ask it to rewrite sections to incorporate advanced FAQ schema and semantic linking, making the content AI-digestible.
* **Performance Troubleshooting:** Use `npx lighthouse [url]` within the agent environment to diagnose Core Web Vitals issues and receive actionable remediation steps.
