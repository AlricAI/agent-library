## Overview
Snippet Optimizer Pro is a specialized AI agent designed to transform standard long-form content into highly structured, snippet-optimized blocks. Its primary goal is to increase the likelihood of your content appearing in Google's 'Position Zero' (the featured snippet box) and other rich search results.

It moves beyond simple keyword stuffing by architecting content specifically for how search engines interpret direct answers, lists, and comparisons.

## Capabilities
*   **Structured Formatting:** Automatically converts text into optimal formats like 40-60 word paragraphs, numbered steps (HowTo), and comparison tables.
*   **Question Identification:** Scans provided text to identify latent questions that can serve as perfect H2/H3 headers for PAA (People Also Ask) optimization.
*   **Multi-Format Generation:** Creates multiple versions of the answer (e.g., a paragraph snippet, a list snippet, and a table comparison) from a single source query.
*   **Schema Markup Suggestion:** Provides ready-to-implement JSON-LD suggestions for `FAQPage` and `HowTo` schema to boost search engine understanding.
*   **Contextual Enrichment:** Generates supporting bullet points and related context to make the primary snippet answer more authoritative.

## Example Use Cases
1. **Product Comparison:** Feed it a detailed article comparing three software tools; it will output a clean, structured comparison table ready for implementation.
2. **'How-To' Guides:** Provide steps for a process (e.g., 'How to bake sourdough'); the agent will format this into numbered, concise steps suitable for a rich snippet carousel.
3. **Definition Pages:** Give it an article defining complex terms; it will extract and format these definitions into clear, direct paragraph snippets with strong opening sentences.
