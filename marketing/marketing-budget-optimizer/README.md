## Overview
This agent is a specialized marketing intelligence tool designed to maximize advertising return on investment (ROI) across multiple digital platforms. It ingests comprehensive performance data from Meta Ads, Google Ads, Klaviyo, GA4, and other sources to provide a holistic view of campaign efficiency.

Its core function is calculating a blended Return on Ad Spend (ROAS), allowing marketers to understand true cross-channel profitability rather than relying on siloed platform metrics. The output is not just analysis; it's a set of concrete, prioritized budget shifts.

## Capabilities
*   **Cross-Platform Data Ingestion:** Reads and parses structured data from Meta Ads (spend, purchases, ROAS), Google Ads (cost, conversions), GA4 (sessions, revenue), and other integrated sources.
*   **Blended ROAS Calculation:** Computes a unified ROAS metric by normalizing spending and revenue across all connected advertising channels.
*   **Actionable Recommendations:** Generates specific, quantitative recommendations detailing which campaigns or platforms should receive increased or decreased budget allocations to improve overall performance.
*   **Data Normalization:** Handles raw data structures (like Google Ads' micro-currency costs) and converts them into usable, comparable metrics.

## Example Use Cases
*   **Budget Reallocation Strategy:** A marketing manager can feed the agent last week's Meta and Google Ads performance to ask: "Where should I shift $5,000 of my next month's budget for the highest blended ROAS?"
*   **Performance Diagnosis:** If overall ROAS drops unexpectedly, the agent can pinpoint whether the issue lies with poor ad copy (Meta) or low-quality traffic (Google), guiding immediate troubleshooting.
*   **Campaign Scaling Plan:** Before a major sales event, use the agent to model optimal spending increases across all channels based on historical conversion rates and current budget constraints.
