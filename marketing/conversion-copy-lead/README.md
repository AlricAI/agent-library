## Overview
This agent acts as a Conversion Copy Lead, specializing in crafting persuasive and action-oriented copy. Its primary mission is to guide the reader through a clear path from initial attention to a desired final action (conversion). It operates with a strict focus on measurable results rather than mere aesthetics.

## Capabilities
*   **Headline & Hook Generation:** Creates multiple headline options targeting different psychological angles.
*   **Sales Page Structuring:** Develops persuasive copy sequences that build desire and overcome objections.
*   **CTA Optimization:** Rewrites calls-to-action to be highly specific, benefit-led, and urgent.
*   **Objection Handling:** Integrates proof points directly into the narrative flow to preemptively address audience concerns.

## Example Use Cases
1. **Landing Page Refresh:** Provide an existing landing page draft, your target audience profile, and the primary offer. The agent will return 3-5 distinct headline variations with explanations of their psychological appeal.
2. **Email Sequence Drafting:** Give it a lead magnet and the desired next step (e.g., book a demo). It will structure a short email sequence designed to build momentum toward that single action.
3. **Ad Copy Testing:** Input your core value proposition and three potential objections. The agent will generate ad copy variants, ensuring each variation directly addresses one of the stated concerns.
