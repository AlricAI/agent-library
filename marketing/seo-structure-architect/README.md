## Overview
This agent acts as a specialized Content Structure Specialist, designed to analyze existing or planned content and elevate its information architecture for maximum search engine visibility. It moves beyond simple keyword stuffing by focusing on logical flow, topical clustering (siloing), and technical markup implementation.

## Capabilities
*   **Header Hierarchy Analysis:** Ensures strict adherence to best practices (H1 per page, H2/H3 variations) for optimal readability and SEO structure.
*   **Schema Markup Generation:** Recommends and structures high-impact JSON-LD schema (e.g., FAQ, HowTo, Article) tailored to the content type.
*   **Siloing Strategy Mapping:** Develops parent/child topical clusters and suggests contextual internal linking opportunities to build site authority.
*   **Content Flow Assessment:** Evaluates the logical progression of ideas, suggesting optimal placements for tables of contents and featured snippet formats.
*   **Technical Blueprint Output:** Provides actionable deliverables including structured outlines, linking matrices, and platform-specific implementation advice (WordPress, Static Sites).

## Example Use Cases
1. **Content Audit:** Feed it a draft article URL or text block to receive an immediate structural blueprint, identifying missing H2s or weak topic transitions.
2. **Site Mapping:** Provide a list of 10 related articles and ask the agent to build a silo map, detailing which pages should link to which parent/child hubs.
3. **Schema Implementation:** Give it content describing a process (e.g., 'How to Bake Bread') and request the full `HowTo Schema` JSON-LD code block for direct copy-pasting into the site's head section.
