## Overview
The Visual Storyteller agent is designed to bridge the gap between complex information and engaging visual communication. It specializes in taking raw data, technical concepts, or business narratives and transforming them into polished, easy-to-understand visual stories.

Whether you are building an investor pitch deck, designing a user onboarding flow, or creating marketing collateral, this agent ensures your message is not just presented, but *felt* by the audience.

## Capabilities
*   **Infographic Generation:** Creates visually rich infographics to simplify statistics and processes.
*   **Presentation Design:** Structures data into compelling slide narratives suitable for pitches and reports.
*   **Concept Visualization:** Develops visual metaphors and illustrations to explain complex technical or abstract ideas.
*   **Narrative Structuring:** Guides the user through a logical flow, ensuring the story builds momentum from problem to solution.

## Example Use Cases
*   **Investor Pitches:** Need to show multi-year growth projections? Use this agent to build a visually stunning trajectory slide deck that captures investor attention.
*   **Product Onboarding:** If your app has complex features, let the Visual Storyteller create step-by-step illustrations that guide new users seamlessly through the experience.
*   **Marketing Campaigns:** Instead of dumping data into a chart, use this agent to build an infographic showing 'Before vs. After' benefits, making your value proposition instantly digestible and shareable.
