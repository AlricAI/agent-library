## Overview
The SEO Keyword Strategist is an advanced tool designed to elevate your content's search engine visibility by ensuring deep topical relevance and natural keyword integration. It moves beyond simple keyword stuffing by focusing on semantic relationships, entity mapping, and achieving optimal keyword density.

This agent analyzes provided text against best-practice SEO guidelines to generate a comprehensive strategy package that helps you build authoritative, high-ranking content.

## Capabilities
*   **Keyword Density Analysis:** Calculates the current usage percentage for primary and secondary keywords, flagging any instances of over-optimization or underutilization.
*   **LSI & Semantic Suggestion:** Generates a rich list of 20–30 Latent Semantic Indexing (LSI) keywords and semantic variations to broaden topical coverage.
*   **Entity Mapping:** Identifies core entities within your content and suggests related concepts necessary to build topical authority signals.
*   **Search Intent Assessment:** Analyzes the content type to suggest optimal keyword clusters for Featured Snippets, People Also Ask (PAA), and voice search optimization.
*   **Optimization Checklist:** Provides actionable recommendations on where and how to naturally place keywords throughout your existing draft.

## Example Use Cases
1. **Content Audit:** Paste a piece of blog content you suspect is underperforming; the agent will return density reports, LSI gaps, and entity suggestions for immediate improvement.
2. **Topic Hub Creation:** Provide 3-5 core topics; the agent will cluster related keywords and suggest an authoritative structure, including necessary secondary entities to cover all angles.
3. **Draft Review:** Before publishing, run a draft through the tool to ensure keyword placement feels natural while maximizing topical depth, preventing any 'stuffing' penalties.
