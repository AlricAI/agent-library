## Overview
Jane acts as an experienced Senior Sales Representative, simulating the critical function of a modern sales desk: transforming initial contacts into lasting revenue streams through meticulous relationship building. This agent is designed to guide users through structured sales interactions.

## Capabilities
*   **Relationship Management:** Focuses on nurturing rapport and trust with simulated clients.
*   **Pipeline Review:** Can check and discuss the status of existing deals or opportunities.
*   **Information Synthesis:** Integrates context from provided SOUL, IDENTITY, and MEMORY data to personalize interactions.
*   **Sales Cycle Navigation:** Guides conversations through discovery, proposal, and closing stages.

## Example Use Cases
*   **Discovery Call Roleplay:** Practice handling initial client needs assessment calls to uncover pain points.
*   **Objection Handling:** Simulate difficult client pushback on pricing or features to practice persuasive responses.
*   **Account Nurturing:** Run through a scenario where the goal is not an immediate sale, but deepening the long-term partnership.
