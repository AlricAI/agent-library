## Overview
This agent acts as a specialized SEO Meta Tag Optimization Specialist, designed to transform raw content and target keywords into perfectly structured metadata packages. Its goal is not just compliance but maximizing click-through rates (CTR) by incorporating psychological triggers and compelling copywriting.

## Capabilities
*   **Comprehensive Metadata Generation:** Creates optimized variations for Title Tags, Meta Descriptions, and URL slugs.
*   **Strict Compliance Checking:** Ensures adherence to character limits (e.g., 50-60 chars for titles, 150-160 chars for descriptions) while considering mobile truncation.
*   **Advanced SEO Strategy:** Integrates primary/secondary keywords naturally, strategically placing them within the first few characters of titles.
*   **Persuasive Copywriting:** Incorporates emotional triggers, power words, and strong Calls-to-Action (CTAs) to compel user clicks.
*   **Platform Adaptability:** Provides specific recommendations for popular CMSs like WordPress (Yoast/RankMath) and modern frameworks like Astro/Next.js.

## Example Use Cases
1. **New Blog Post Launch:** Input the article text and target keywords; receive a full 'Meta Package' including 3 A/B test title variations, ready for immediate deployment.
2. **Landing Page Refresh:** Provide existing page copy to generate fresh, benefit-driven descriptions that outperform current search results.
3. **Technical SEO Audit:** Request schema markup recommendations and optimized URL structures for improved crawlability.
