## Overview
This Elite SEO & GEO Specialist bridges the gap between traditional search engine optimization (SEO) and Generative Engine Optimization (GEO). In 2025, visibility requires more than just ranking; content must be structured to be easily cited and understood by AI models like ChatGPT and Claude. This agent ensures your digital presence is robustly optimized for both human readers and artificial intelligence crawlers.

## Capabilities
*   **Semantic Architecture:** Implements deep semantic linking and structures content using best practices for modern search algorithms.
*   **Technical Auditing:** Focuses on critical performance metrics like Core Web Vitals (LCP, INP, CLS) to ensure a flawless user experience.
*   **AI Readiness:** Optimizes for Generative AI by structuring data with clear schema and comprehensive FAQ sections, making content highly citable.
*   **Scale-Aware Strategy:** Adjusts the depth of optimization—from basic MVP foundations to full Enterprise dominance—based on your project's current stage.

## Example Use Cases
*   **Content Gap Analysis:** Run a `/seo` audit on existing articles to identify missing semantic connections or outdated schema markup.
*   **Performance Check:** Execute `npx lighthouse [url]` to diagnose Core Web Vitals issues and receive actionable remediation steps for speed improvements.
*   **Schema Implementation:** Use the agent to validate and generate complex Schema.org structures, ensuring search engines understand your content's context (e.g., Recipe, Product, Article).
*   **Strategy Planning:** When starting a new project, use the Scale-Aware Strategy matrix to determine if you need an MVP focus or an Enterprise dominance plan.
