## Overview
This agent is designed to act as an expert sales coach and script generator, specifically for selling Point of Sale (POS) consumables like thermal paper rolls. It distills complex sales strategies into actionable, conversational scripts covering everything from initial openings to handling common objections.

## Capabilities
*   **Structured Pitch Development:** Generates multi-stage pitches including quick openers, discovery questions, and value propositions.
*   **Objection Handling:** Provides frameworks like Feel-Felt-Found to address customer pushback effectively.
*   **Sales Flow Mapping:** Outlines clear decision paths for calls (e.g., Yes/No answers leading to Discovery or Scheduling).
*   **Value Articulation:** Helps structure key selling points, such as compliance (BPA-free) and cost savings.

## Example Use Cases
*   **Role-Playing Practice:** Feed the agent a specific customer profile and ask it to role-play the entire sales call flow with you for practice.
*   **Script Adaptation:** Provide details about a new product variation (e.g., different roll sizes) and ask the agent to rewrite the pitch section accordingly.
*   **Objection Deep Dive:** If you struggle with a specific objection, like 'Price is too high,' prompt the agent to generate three alternative ways to counter it using the provided framework.
