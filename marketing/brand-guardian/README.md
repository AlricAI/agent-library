## Overview
Brand Guardian acts as your dedicated strategic guardian for maintaining a cohesive and powerful brand presence. It specializes in developing, documenting, and enforcing comprehensive brand guidelines to ensure that every piece of content, design element, or user interaction reflects the intended identity.

## Capabilities
*   **Guideline Creation:** Develop detailed style guides covering tone of voice, visual elements (color palettes, typography), and usage rules.
*   **Consistency Auditing:** Review existing materials across multiple platforms (web, mobile, marketing) to identify and correct deviations from established standards.
*   **Asset Management Strategy:** Help structure and document brand asset libraries, ensuring developers and designers use approved versions of logos, imagery, and components.
*   **Brand Evolution Planning:** Guide strategic updates to an existing brand identity to keep it fresh, relevant, and competitive without losing core recognition.

## Example Use Cases
*   **New Product Launch:** Need to launch a new app? Use this agent to build the foundational visual and tonal guidelines from scratch, ensuring immediate market cohesion.
*   **Multi-Platform Sync:** If your brand looks different on iOS versus web, Brand Guardian will create unified design standards that bridge those gaps.
*   **Brand Refresh:** Feeling stale? Use it to modernize your brand perception by strategically updating elements while preserving core equity.
*   **Developer Handover:** Stop developers from using random colors! This agent creates definitive asset usage guides for engineering teams.
