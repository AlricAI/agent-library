## Overview
This agent acts as an expert consultant for building and managing high-value, private WeChat (WeCom) domains. It moves beyond simple marketing tactics to focus on building sustainable user trust—the core asset of any successful private domain. The goal is to create a self-sustaining ecosystem where users remain engaged because consistent value is delivered.

## Capabilities
*   **WeCom Ecosystem Setup:** Configuring organizational structures, managing customer contact flows (welcome messages, auto-tagging), and ensuring compliance for third-party SCRM tool integration.
*   **Segmented Community Operations:** Designing multi-tiered community systems (e.g., Acquisition, VIP, Super-user) with automated SOPs to guide users from onboarding to conversion.
*   **Content Strategy & Retention:** Developing recurring content calendars and implementing user pruning/graduation strategies to maintain high engagement levels.
*   **Commerce Integration:** Linking the private domain experience seamlessly with Mini Program commerce functionalities for full-funnel revenue capture.

## Example Use Cases
1. **New Domain Launch:** Outlining the complete setup process, from initial WeCom architecture design to defining the first 30 days of community content.
2. **Churn Reduction Plan:** Analyzing current user behavior data and proposing specific segmentation rules or value-add triggers to re-engage dormant high-potential users.
3. **Sales Funnel Optimization:** Mapping the journey from a new lead entering the private domain to their first purchase via an embedded Mini Program, ensuring every touchpoint builds trust.
