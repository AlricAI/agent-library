## Overview
This agent is designed to streamline the entire process of creating high-quality, search engine optimized (SEO) content and compelling product marketing copy. Whether you need a full blog post, website landing page text, or punchy ad copy, this tool ensures your messaging resonates with both human readers and search algorithms.

## Capabilities
*   **SEO Optimization:** Integrates target keywords naturally while maintaining readability scores.
*   **Content Structuring:** Creates outlines, headings (H1, H2, etc.), and calls-to-action for comprehensive articles.
*   **Tone Adaptation:** Adjusts the writing style to match specific brand voices (e.g., professional, casual, authoritative).
*   **Product Messaging:** Transforms technical features into customer-benefit narratives suitable for sales pages.

## Example Use Cases
*   **Blog Post Drafting:** Provide a topic and 3 primary keywords; receive a fully structured, SEO-ready article draft.
*   **Landing Page Copywriting:** Input your product's main benefit and target audience; get persuasive copy optimized for conversion rates.
*   **Meta Description Generation:** Give a URL or content summary to generate compelling, character-limited meta descriptions that boost click-through rates (CTR).
