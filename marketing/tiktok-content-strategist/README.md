## Overview
This agent acts as a dedicated TikTok Channel Specialist, designed to take raw campaign ideas and rework them into content that feels authentically native to the TikTok platform. It focuses intensely on pace, hooks, and current viral formats, ensuring your message respects the fast tempo of short-form video.

## Capabilities
*   **Short-Form Scripting:** Generates complete scripts optimized for rapid consumption.
*   **Hook Optimization:** Develops multiple high-impact opening lines to maximize initial viewer retention.
*   **Trend Adaptation:** Analyzes input concepts against current platform trends, suggesting necessary format shifts.
*   **Platform Compliance:** Ensures all outputs are concise and adhere strictly to TikTok's pacing and style guidelines.

## Example Use Cases
1. **Concept Refinement:** Provide a detailed blog post summary and ask the agent to convert it into three distinct TikTok script options (e.g., 'Listicle Style,' 'Storytelling Hook,' 'Myth-Busting').
2. **Trend Integration:** Input a general brand message and specify a current viral sound or format, asking the agent to weave the core message seamlessly into that trend.
3. **Content Auditing:** Submit an existing draft script and ask the agent to critique it based on TikTok best practices, specifically flagging any sections that are too slow or lack immediate hooks.
