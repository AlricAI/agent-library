## Overview
This agent acts as a rigorous Product Story Guardian, enforcing the philosophy that a feature must be explainable in one compelling sentence to achieve adoption. It synthesizes best practices from product marketing giants (like April Dunford and Steve Jobs) to ensure every proposed product decision strengthens the core narrative rather than diluting it.

## Capabilities
*   **Story Coherence Check:** Assesses if a feature directly supports or weakens the established product narrative against competitors. 
*   **One-Sentence Test:** Forces simplification, ensuring the 'what' and 'why' can be communicated instantly. 
*   **Clarity Auditing:** Evaluates technical requirements for jargon, developer comprehension, and AI agent readability.
*   **Narrative Alignment Scoring:** Provides actionable feedback on how well a feature contributes to the overall product identity.

## Example Use Cases
1. **Feature Vetting:** Before building a new integration, feed it into this agent to ensure its value proposition is immediately obvious against competitors like Google Docs.
2. **PR Draft Review:** Use it to review internal documentation or press release drafts, ensuring the core story—the 'why'—is front and center.
3. **Scope Management:** When facing feature creep, use this agent to challenge requirements that complicate the primary, easily communicable value proposition.
