# User Profile Components Copy Review

## Overview

Copy elements for review in the new user profile components.

## Components with Copy Elements

### UserProfile (`design-specs/user-profile-component.md`)

#### Profile Labels

- "Member since {date}"
- "{city}, {state}" or "Location private"
- "{events} events attended", "{hours} hours volunteered", "{connections} connections made", "{groups} community groups"
- "Verified member", "Pending verification"
- "Community Contributor", "Event Organizer", "Volunteer Leader", "Helpful Member"
- "Private", "Connections only", "Public"

#### Action Button Text

- "Edit Profile"
- "Send Message"
- "Add Connection"
- "Report Concern"
- "Block User"
- "View All Events"
- "See Full Impact Report"

#### Empty States

- "This user hasn't added a bio yet"
- "No upcoming events"
- "Impact information is private"
- "This information is not shared publicly"

### UserProfileSettings (`design-specs/user-profile-settings-component.md`)

#### Section Headers

- "Profile Information"
- "Privacy Settings"
- "Notification Preferences"
- "Account Settings"
- "Safety & Blocking"

#### Form Labels and Placeholders

- "Full Name" / "Enter your full name"
- "Username" / "@username"
- "Bio" / "Tell us about yourself and your interests"
- "Location" / "City, State"
- "Profile Picture" / "Upload an image"

#### Privacy Settings Options

- "Public", "Connections Only", "Private" (for various visibility settings)

#### Notification Preferences Options

- "Email", "Push", "None" (for notification categories)

#### Account Action Buttons

- "Save Changes"
- "Cancel"
- "Delete My Account"
- "Verify My Identity"
- "Change Password"
- "Update Email Address"

#### Confirmation Messages

- "Your profile has been updated successfully!"
- "There was an error saving your changes. Please try again."
- "Are you sure you want to delete your account? This cannot be undone."
- "Identity verification helps build trust in our community."

## Brand Voice Alignment

Please verify:

1. Consistency with YouAndINotAI's community-focused messaging
2. Inclusive language throughout all components
3. Clarity and accessibility of all labels
4. Appropriate tone for encouraging community participation
5. Transparency in privacy and safety language

## Legal Compliance Check

Florida §496.405 adherence:

- [ ] No instances of "donate", "donation", or "solicitation" in any copy
- [ ] Appropriate language distinguishing between nonprofit and commercial activities
- [ ] Clear privacy and consent language for profile information
- [ ] Compliance with data protection regulations (GDPR, CCPA)

## Coordination Requests

1. Copy approval for all labels and messages
2. Translation requirements for international markets
3. A/B testing opportunities for key CTAs
4. Crisis communication protocols for account deletion processes
