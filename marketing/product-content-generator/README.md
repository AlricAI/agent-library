## Overview
This agent acts as the dedicated Product Content Specialist for VorstersNV. Its core function is to elevate raw product information into compelling, conversion-focused marketing copy. It ensures every description adheres to strict quality standards, balancing professional tone with consumer accessibility.

## Capabilities
*   **Structured Generation:** Creates descriptions following a mandatory structure: Hook, Explanation, USPs (3 bullets), Specifications (table), and FAQ (2-3 Q&A).
*   **Tone & Style Enforcement:** Writes in an active voice, focusing on customer benefits over mere features, maintaining a professional yet approachable tone.
*   **SEO Integration:** Optimizes content by ensuring the primary keyword appears within the first 50 words, distributing secondary keywords naturally, and providing meta description suggestions.
*   **Content Improvement:** Can analyze existing copy (especially low-scoring versions) to identify weaknesses and suggest prompt adjustments for iterative improvement.

## Example Use Cases
*   **Improving Existing Copy:** If you provide a draft product description and ask it to 'improve the laptop description,' it will rewrite it according to all VorstersNV guidelines.
*   **Generating New Content:** When starting from scratch, simply request content for a new product, specifying the core features.
*   **Creating Support Material:** Use this agent specifically to generate comprehensive FAQ sections that complement the main product page copy.
