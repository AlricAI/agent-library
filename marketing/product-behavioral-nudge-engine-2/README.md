## Overview
The Product Behavioral Nudge Engine is designed to apply principles of behavioral economics and psychology to product design. Its core mission is to subtly guide users toward desired actions—such as completing onboarding, upgrading a subscription, or engaging with key features—without resorting to manipulative tactics.

This agent operates within the context of Product & Project Management, ensuring that all suggested nudges are measurable, ethical, and aligned with overall product goals.

## Capabilities
*   **Nudge Identification:** Analyzes user flow data and conversion bottlenecks to pinpoint areas ripe for behavioral intervention.
*   **Principle Application:** Applies established psychological principles (e.g., scarcity, social proof, default bias) to create actionable suggestions.
*   **Design Specification:** Outputs detailed specifications for the nudge, including placement, timing, copy variations, and A/B testing hypotheses.
*   **Ethical Review:** Ensures all proposed nudges adhere to ethical design guidelines, prioritizing user trust over short-term gains.

## Example Use Cases
*   **Improving Onboarding Completion:** If users frequently drop off during the initial setup wizard, this agent can suggest implementing a 'progress bar' nudge combined with immediate positive reinforcement messages.
*   **Boosting Feature Adoption:** To encourage new users to try an underutilized premium feature, it might recommend a contextual tooltip that highlights the benefit and shows how many peers are using it (social proof).
*   **Increasing Subscription Conversion:** For free-trial users nearing expiration, the agent can design a timely reminder nudge that emphasizes the value they will lose by not upgrading.

By following its documented workflow, this engine provides actionable, psychologically grounded strategies to optimize user interaction.
