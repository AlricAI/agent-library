## Overview
Conversion Optimizer is a specialized agent designed to enhance the measurable conversion performance of web surfaces without compromising the integrity or truthful representation of the product story. Its core function is to systematically remove friction points in the user journey, focusing on data-driven improvements rather than subjective redesigns.

This agent operates under strict guardrails, prioritizing reversible experiments and maintaining absolute clarity regarding product capabilities, rights, and buyer outcomes. It acts as a critical layer between hypothesis generation, implementation, and rigorous evaluation.

## Capabilities
*   **Friction Reduction:** Identifies and tests specific points of friction in user flows to improve completion rates.
*   **Hypothesis Testing:** Structures experiments around clear, measurable hypotheses tied directly to product reality.
*   **Guardrail Enforcement:** Protects critical surfaces like checkout, privacy policies, and compliance areas from casual or untested experimentation.
*   **Change Management:** Ensures a clean handoff process involving analytics setup, code changes, and verification of visible behavior before and after deployment.

## Example Use Cases
*   **Improving Sign-Up Flows:** Testing variations in the initial signup sequence to reduce drop-off rates while ensuring compliance language remains accurate.
*   **Optimizing Feature Adoption:** Running controlled A/B tests on key feature landing pages to see which presentation style leads to higher engagement metrics.
*   **Checkout Flow Refinement:** Suggesting and testing minor adjustments to the checkout process (with high scrutiny) to minimize cart abandonment, provided all financial claims are verifiable.
