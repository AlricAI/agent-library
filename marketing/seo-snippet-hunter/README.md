## Overview
Snippet Optimizer Pro is a specialized AI agent designed to transform standard web content into highly structured, snippet-optimized blocks. Its primary goal is to increase the likelihood of your content appearing in Google's 'Position Zero' featured snippets, People Also Ask (PAA) sections, and other rich search results.

This agent moves beyond basic SEO by focusing on the *structure* and *directness* of answers, ensuring that search engines can easily parse definitive information from your text.

## Capabilities
*   **Multi-Format Generation:** Creates optimized content blocks in Paragraph, List, and Table formats suitable for different types of queries.
*   **Question Identification:** Automatically scans provided text to identify key questions perfect for PAA optimization.
*   **Concise Answering:** Generates direct, definitive answers (40-60 words) that immediately address the user's query in the opening sentence.
*   **Schema Markup Suggestion:** Provides ready-to-implement suggestions for FAQPage and HowTo schema markup to enhance technical SEO.
*   **Structural Enhancement:** Recommends advanced tactics like jump links, comparison tables, and step-by-step guides.

## Example Use Cases
1. **Blog Post Enhancement:** Feed it a long-form guide and ask it to generate 3 distinct snippet blocks (one paragraph, one list, one table) for the most critical sections.
2. **Product Comparison Pages:** Input raw feature lists and request a structured comparison table optimized for SERP visibility.
3. **FAQ Page Creation:** Provide a set of Q&A pairs and ask the agent to format them with appropriate schema markup suggestions for maximum search engine crawlability.
