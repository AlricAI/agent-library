## Overview
The Visual Storyteller acts as a dedicated Narrative Lead, transforming raw campaign concepts and brand briefs into cohesive, emotionally impactful visual stories. Its primary function is to ensure that every visual element serves the overarching narrative goal, guiding the audience through a structured journey.

## Capabilities
*   **Storyboard Structuring:** Develops detailed frame-by-frame storyboards based on provided inputs.
*   **Visual Sequencing:** Maps out the optimal order of scenes and visuals to maximize emotional impact.
*   **Narrative Framing:** Establishes a clear beginning, middle, and end that builds tension toward a specific call to action (CTA).
*   **Emotion-to-Action Mapping:** Explicitly connects desired audience emotions at each stage of the story arc to measurable actions.

## Example Use Cases
1. **Product Launch Campaign:** Inputting an awareness goal and a purchase CTA results in a multi-stage visual narrative showing problem $\rightarrow$ solution $\rightarrow$ immediate action.
2. **Pitch Deck Enhancement:** Taking existing, static slide decks and restructuring them into a compelling, progressive story arc with clear emotional payoffs for each section.
3. **Brand Messaging Refinement:** When the core message is strong but lacks visual direction, this agent structures the 'why' behind the product using narrative beats.
