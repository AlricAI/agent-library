## Overview
Seo Specialist Pro is a senior-level AI agent designed to act as your dedicated SEO strategist. It provides deep expertise across technical SEO, content marketing, and off-page optimization to drive measurable organic traffic growth for any website.

This agent follows a rigorous, data-driven workflow, ensuring that all recommendations are grounded in current search engine best practices and site performance metrics.

## Capabilities
*   **Context Gathering:** Automatically initiates the process by requesting comprehensive SEO context (rankings, architecture, content gaps) from the system manager to ensure no stone is left unturned.
*   **Technical Auditing:** Performs detailed technical SEO checks, including crawlability assessments and structured data implementation guidance.
*   **Content Strategy Development:** Creates actionable content roadmaps based on competitor analysis and identified keyword gaps.
*   **Optimization Execution:** Guides the process through on-page improvements, backlink profile building strategies, and performance monitoring.

## Example Use Cases
*   **New Website Launch:** Run a full audit to establish foundational SEO elements before any content is published.
*   **Stagnant Traffic:** Feed it your current analytics data and ask it to diagnose the primary bottleneck preventing ranking improvements.
*   **Competitor Analysis:** Provide competitor URLs and ask for a comprehensive gap analysis detailing where you can outperform them in search results.
