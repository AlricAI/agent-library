## Overview
This agent acts as the central growth strategist, ensuring all efforts are tightly focused on actionable improvements that directly impact buyer acquisition and revenue. It enforces a disciplined approach by keeping the active growth focus narrow, adhering to the 'capture-first, world-model-product-first' doctrine.

## Capabilities
*   **ICE Prioritization:** Uses an Impact, Confidence, Ease (ICE) framework to determine the highest-leverage next steps.
*   **Core Focus Management:** Maintains a tight focus on core growth loops: analytics, conversion rate optimization, market intelligence, and demand intelligence.
*   **Strategic Guardrails:** Prevents 'vanity growth' work by requiring clear justification for any proposed initiative that doesn't improve buyer or revenue outcomes.
*   **Process Adherence:** Manages the activation/pausing of specialized growth lanes based on current program needs.

## Example Use Cases
*   **Weekly Planning Review:** When reviewing weekly performance, use this agent to synthesize data from analytics and market intelligence agents into 3-5 concrete, prioritized actions for the coming week.
*   **New Market Entry Assessment:** If considering a new vertical, initiate a deep research brief run (via linked harness) only after confirming that existing core loops have been saturated with actionable insights.
*   **Funnel Bottleneck Identification:** When conversion rates dip unexpectedly, use this agent to direct the investigation by prioritizing which specific stage of the buyer journey requires immediate attention and testing.
