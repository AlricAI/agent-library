## Overview
This agent functions as a specialized SEO consultant for tech podcasts. Its primary goal is to maximize the search visibility and click-through rate (CTR) of podcast episodes by transforming raw content summaries into highly optimized metadata.

It balances the technical requirements of search engines (keyword density, character limits) with the creative need for engaging, clickable copy that accurately reflects the episode's value proposition.

## Capabilities
*   **Title Generation:** Creates SEO-optimized titles under 60 characters, naturally integrating primary keywords while maintaining click appeal.
*   **Meta Description Writing:** Drafts compelling meta descriptions (under 160 characters) that clearly state the value and include secondary keywords.
*   **Long-Tail Keyword Identification:** Pinpoints exactly three highly specific, long-tail keywords (3-5 words) relevant to the tech concepts discussed.
*   **Keyword Validation:** Integrates external tools (like a simulated `KeywordVolume` plugin) for search volume estimation and uses RAG systems to check historical keyword performance.
*   **Strategic Reporting:** Provides detailed rationale on keyword selection, search intent matching, and quality guidelines to prevent keyword stuffing.

## Example Use Cases
1. **New Episode Launch:** Feed the agent a transcript summary of an episode about 'Quantum Computing in Finance' and receive ready-to-use titles and descriptions for YouTube/Spotify show notes.
2. **Content Gap Analysis:** Provide a list of competitor podcast topics, and ask the agent to suggest 3 underserved, high-intent long-tail keyword clusters that could form the basis of new content.
3. **Optimization Audit:** Input an existing, poorly performing episode description and ask the agent to rewrite it entirely for better SEO performance while maintaining its original tone.
