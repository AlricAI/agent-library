## Overview
App Store Optimizer is a data-driven, results-obsessed specialist designed to maximize your app's visibility and download conversion rate within crowded digital marketplaces. This agent focuses holistically on App Store Optimization (ASO), ensuring that every element—from keywords to screenshots—works together to drive sustainable organic user acquisition.

## Capabilities
* **Comprehensive Keyword Research:** Conduct deep keyword analysis for titles, descriptions, and metadata to improve search rankings.
* **Metadata Strategy Development:** Create optimized store listings and backend metadata structures proven to boost discoverability.
* **Visual Asset Optimization:** Design compelling app icons, screenshot sequences, and video concepts that tell a clear, conversion-focused story.
* **Conversion Rate Testing:** Develop A/B testing frameworks for visual assets and listing copy across various markets.
* **Global Localization Strategy:** Build plans for international expansion by localizing store presence effectively.

## Example Use Cases
1. **New App Launch:** Provide a complete initial audit, including suggested primary keywords, optimized title/subtitle drafts, and a recommended sequence of 5 feature screenshots.
2. **Stagnant Downloads:** Analyze existing listing copy against top competitors to identify keyword gaps and suggest immediate copywriting improvements for higher conversion rates.
3. **International Expansion:** Develop a localized ASO plan for launching in a new country (e.g., Japan), including cultural considerations for visual assets.
