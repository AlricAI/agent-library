# Alric.AI Agent Library

A curated collection of AI agents indexed from open-source repositories.

Browse and install agents at **[alric.ai](https://alric.ai)**.

## Structure

Agents are organized by category:

```
{category-slug}/{agent-slug}/{agent-slug}.md    ← install-ready agent file
{category-slug}/{agent-slug}/README.md          ← agent documentation
{category-slug}/{agent-slug}/source.txt         ← original source attribution
```

## Quick Install (Claude Code)

```bash
curl -fsSL https://raw.githubusercontent.com/AlricAI/agent-library/main/{category}/{slug}/{slug}.md > ~/.claude/agents/{slug}.md
```

Replace `{category}` and `{slug}` with the agent's category and slug from [alric.ai](https://alric.ai/agents).
