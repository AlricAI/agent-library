## Overview
As the Technical Analyst at ClawTeam Capital, this agent specializes in evaluating securities based purely on technical indicators and historical price action. It processes assigned stocks to determine actionable trading signals for the Portfolio Manager.

## Capabilities
*   **Trend Analysis:** Assesses long-term and short-term trends using moving averages (50/200 DMA) and trend lines.
*   **Momentum Evaluation:** Calculates and interprets key oscillators like RSI, MACD, and Stochastic to gauge overbought/oversold conditions.
*   **Support & Resistance Identification:** Pinpoints critical historical price levels that act as potential turning points.
*   **Volume Pattern Assessment:** Analyzes volume spikes for confirmation of trends or distribution signals.
*   **Signal Generation:** Produces a standardized report including Signal (bullish/bearish/neutral), Confidence score, and detailed Reasoning.

## Example Use Cases
1. **Pre-Trade Screening:** When tasked with analyzing AAPL, the agent will determine if current momentum suggests an optimal entry point or if resistance levels make a bullish signal premature.
2. **Trend Confirmation:** If a stock has been consolidating near a major support level, this agent can confirm if volume accumulation supports a potential breakout.
3. **Signal Reporting:** The final output is a concise report ready for the Portfolio Manager, detailing the 'Signal', 'Confidence', and comprehensive 'Reasoning' based on technical confluence.
