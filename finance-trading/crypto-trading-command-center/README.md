## Overview
This agent functions as a comprehensive command center for cryptocurrency trading. It is designed to maintain continuity across sessions by rigorously tracking portfolio positions, investment theses, and daily market movements. Its core philosophy emphasizes disciplined documentation, risk management, and structured collaboration with other specialized agents.

## Capabilities
*   **Market Data Processing:** Reads current price feeds and analyzes historical market events from memory logs.
*   **Portfolio Tracking:** Maintains detailed records of current positions, investment theses, and trade history stored in persistent memory files.
*   **Risk Management:** Enforces strict guidelines regarding position sizing and downside calculation before suggesting any trades.
*   **Internal Coordination:** Designed to interface with specialized internal agents (e.g., ALPHA-9 for strategy, LEDGER-9 for accounting) to ensure holistic operational alignment.
*   **Structured Reporting:** Provides insights only when significant market data is detected or when specifically requested regarding portfolio status.

## Example Use Cases
*   **Daily Review:** Uploading today's logs and asking the agent to summarize key trades, deviations from the thesis, and any immediate risk flags for review.
*   **Strategy Validation:** Providing a new market protocol document and asking the agent to analyze its potential impact on existing portfolio holdings based on historical performance data.
*   **Trade Documentation:** After executing a trade manually or via another system, instructing the agent to formally log the transaction details, rationale, and risk assessment into the memory structure.
