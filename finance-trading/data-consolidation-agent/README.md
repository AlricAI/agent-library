## Overview
The Data Consolidation Agent acts as a strategic synthesizer for raw sales metrics, transforming disparate data points into actionable, real-time dashboards. Its core mission is to provide a comprehensive view of sales performance by aggregating data across all territories, representatives, and time periods.

This agent ensures that key insights—such as regional summaries, rep rankings, and pipeline health—are immediately available for decision-making.

## Capabilities
*   **Comprehensive Aggregation:** Consolidates metrics from multiple sources to build a holistic sales picture.
*   **Advanced Calculation:** Accurately calculates crucial KPIs like attainment (Revenue / Quota * 100), handling edge cases such as division by zero.
*   **Multi-Dimensional Reporting:** Supports generating reports segmented by territory, individual representative, and time period (MTD, YTD).
*   **Pipeline Integration:** Merges current sales metrics with lead pipeline data for a complete view of potential revenue.
*   **Structured Output:** Delivers results in dashboard-friendly JSON formats optimized for rapid loading and visualization.

## Example Use Cases
*   **Executive Dashboard Generation:** Requesting a single, up-to-date dashboard showing YTD revenue attainment, top 5 performers, and overall pipeline value across all regions.
*   **Territory Deep Dive:** Needing a detailed report for one specific region, including the performance history of every rep within that territory over the last 50 entries.
*   **Performance Trend Analysis:** Generating comparative reports to track revenue trends over the trailing six months to spot growth patterns or dips.
