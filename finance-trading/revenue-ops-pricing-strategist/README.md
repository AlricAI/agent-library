## Overview
This agent acts as the dedicated pricing and commercial systems lead for Blueprint. Its core function is to synthesize complex signals—including market demand, actual usage data, and catalog supply costs—to generate actionable, product-grounded recommendations for pricing structures.

It ensures that all proposed pricing reinforces the reality of the product being built: a capture-first, world-model system, rather than abstract service menus.

## Capabilities
*   **Pricebook Drafting:** Generates draft updates to pricebooks based on operational data.
*   **Package Recommendation:** Suggests optimal product packaging and bundling strategies.
*   **Quote Guidance:** Provides structured rationale and guardrails for preparing quote packets.
*   **Signal Synthesis:** Integrates demand, usage, and cost signals into cohesive pricing narratives.
*   **Contradiction Flagging:** Explicitly routes missing data or commercial contradictions to human oversight rather than masking them in logic.

## Example Use Cases
*   **New Feature Pricing:** When a new feature is launched, use this agent to draft tiered pricing models that reflect the actual technical implementation cost and perceived customer value.
*   **Deal Support:** During an active sales cycle, utilize this agent within a deal thread to provide structured rationale for recommended discounts or package adjustments, ensuring alignment with established guardrails.
*   **Pricing Audit:** When reviewing existing pricing tiers, prompt this agent to validate if the current structure accurately reflects the 'product-first' reality of Blueprint’s core offerings.
