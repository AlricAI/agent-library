## Overview
This agent acts as a dedicated quantitative risk manager for investment portfolios. Its core function is to provide proactive, multi-faceted risk assessment, ensuring that trading decisions are systematically protected against adverse market movements.

It moves beyond simple P&L tracking by focusing on statistical risk metrics like R-multiples and Value at Risk (VaR), allowing users to manage capital exposure rigorously.

## Capabilities
*   **Risk Quantification:** Calculates portfolio risk using established methods, including R-multiple analysis and maximum drawdown assessment.
*   **Position Sizing:** Determines optimal trade sizes based on defined account risk percentages and principles derived from the Kelly Criterion.
*   **Hedging & Mitigation:** Generates systematic hedging recommendations utilizing derivatives (options, futures) to offset concentration risks.
*   **Performance Metrics:** Calculates expected value (expectancy) using win/loss ratios and provides detailed correlation matrices for portfolio diversification assessment.
*   **Systematic Controls:** Implements automated stop-loss and take-profit level calculations for disciplined trade execution.

## Example Use Cases
*   **Pre-Trade Analysis:** Before entering a new position, use this agent to calculate the maximum allowable size (position sizing) given current account risk tolerance and assess how it impacts existing portfolio correlations.
*   **Portfolio Review:** Run a comprehensive risk report at the end of the week to receive an R-multiple tracking spreadsheet, identifying any concentration risks or excessive drawdowns that require immediate attention.
*   **Stress Testing:** Simulate market downturns using Monte Carlo methods to stress test the current portfolio structure and generate actionable hedging recommendations to maintain capital integrity.
