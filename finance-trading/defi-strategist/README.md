## Overview
The DeFi Yield Strategist is an advanced agent designed to act as a comprehensive financial architect for decentralized finance. It specializes in designing, modeling, and proposing sophisticated yield farming strategies that go beyond simple APY chasing.

This agent focuses on sustainable, risk-adjusted returns by deeply analyzing protocol mechanics, impermanent loss scenarios, and cross-chain opportunities across major blockchain ecosystems.

## Capabilities
*   **Yield Strategy Design:** Creates multi-faceted yield farming plans incorporating lending protocols, liquidity provision, and vault automation.
*   **Risk Modeling:** Performs detailed analysis of impermanent loss, smart contract risks, and protocol sustainability metrics.
*   **Optimization & Efficiency:** Models gas cost optimization through transaction batching and suggests strategies to mitigate MEV attacks.
*   **Cross-Chain Integration:** Develops blueprints for cross-chain yield strategies utilizing various bridge mechanisms with integrated risk management.
*   **Reporting & Backtesting:** Generates comprehensive reports including backtested performance data, real yield calculations (after costs), and position monitoring dashboards.

## Example Use Cases
1. **New Protocol Evaluation:** Feed it a new DeFi protocol address; the agent will assess its TVL, liquidity depth, and potential for sustainable yield generation while flagging security concerns.
2. **Liquidity Pool Optimization:** Input current token pairs and desired risk profile; the agent will model optimal LP allocations to minimize impermanent loss while maximizing reward capture.
3. **Automated Strategy Blueprinting:** Request a 'Stablecoin Yield Vault' strategy for Ethereum; it will output the necessary steps, smart contract interaction logic (using Web3.py/Ethers.js concepts), and gas-optimized execution plan.
