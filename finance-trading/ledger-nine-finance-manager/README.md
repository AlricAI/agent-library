## Overview
Ledger Nine is a specialized AI agent designed to function as a Chief Financial Officer (CFO) assistant. It manages complex financial operations, provides deep market intelligence, and assists with forward-looking economic projections. This agent integrates various skills to give users a holistic view of financial health.

## Capabilities
* **Market Intelligence:** Utilizes `crypt-market-data` for up-to-date market insights.
* **Exchange Connectivity:** Interacts with exchanges via `crypto-exchange-api` for real-time data access.
* **Wallet Management:** Supports basic operations using the `evm-wallet` skill.
* **Financial Modeling:** Capable of analyzing economic texts and running probability simulations for risk assessment.
* **Skill Expansion:** Guides users on requesting advanced skills like automated reporting (`cron-scheduler`) or system monitoring (`health-monitor`).

## Example Use Cases
* **Quarterly Budgeting:** Upload a set of financial documents and ask the agent to forecast Q3 expenditures based on current market trends.
* **Crypto Investment Strategy:** Provide a list of potential assets; Ledger Nine will pull live data, assess associated risks using probability models, and suggest an optimized allocation strategy.
* **System Health Check:** Request a comprehensive report detailing the financial system's operational status, checking for potential bottlenecks or required maintenance.
