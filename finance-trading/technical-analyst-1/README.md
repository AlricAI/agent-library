## Overview
This agent functions as a dedicated Technical Analyst for ClawTeam Capital, specializing in evaluating securities based on established technical analysis principles. It processes tasks assigned by the Portfolio Manager to provide actionable trading insights.

## Capabilities
*   **Trend Analysis:** Assesses long-term and short-term trends using moving averages (50/200 day) and trend lines.
*   **Momentum Evaluation:** Analyzes key oscillators like RSI, MACD, and stochastic indicators to gauge market strength.
*   **Support & Resistance Identification:** Pinpoints critical historical price levels that act as potential turning points.
*   **Volume Assessment:** Interprets volume patterns (accumulation/distribution) to confirm the validity of observed price moves.
*   **Signal Generation:** Produces a standardized report detailing the signal, confidence level, and comprehensive reasoning for entry/exit recommendations.

## Example Use Cases
1. **Pre-Trade Screening:** When evaluating a watchlist of stocks, use this agent to generate initial 'bullish' or 'bearish' signals before deeper fundamental review.
2. **Signal Confirmation:** After another analysis (e.g., fundamental), run the security through this agent to confirm if current price action supports the thesis.
3. **Exit Strategy Planning:** Use it to determine optimal exit zones by analyzing recent momentum shifts and resistance breakouts.
