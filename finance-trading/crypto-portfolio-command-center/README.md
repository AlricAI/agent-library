## Overview
This agent serves as a dedicated command center for advanced cryptocurrency trading and portfolio management. It is designed to maintain continuity by reading structured memory files, ensuring every trade and market observation is meticulously documented.

It operates with a high degree of focus, prioritizing data-driven insights over speculative chatter, making it ideal for active traders who require rigorous record-keeping and multi-faceted analysis.

## Capabilities
*   **Memory Integration:** Reads daily logs (`YYYY-MM-DD.md`) and long-term portfolio theses (`MEMORY.md`) to maintain context across sessions.
*   **Market Monitoring:** Capable of checking real-time price movements and analyzing market trends.
*   **Strategy Coordination:** Designed to interface with specialized internal agents (e.g., ALPHA-9, Counting Crow) for holistic strategy alignment.
*   **Risk Management Focus:** Emphasizes disciplined trading by prioritizing documentation and calculating downside risk before suggesting actions.

## Example Use Cases
*   **Post-Market Review:** After a volatile day, prompt the agent to synthesize today's trades against the long-term thesis stored in `MEMORY.md`.
*   **Pre-Trade Analysis:** Ask for an analysis comparing current price action against historical patterns documented in memory files before initiating a new position.
*   **Strategy Alignment Check:** Use it to coordinate with simulated partners, such as asking, "What is the consensus strategy between ALPHA-9 and myself based on last week's data?"

Remember that this agent excels when provided with structured inputs (like price feeds or existing memory files) to ensure actionable, documented outputs.
