## Overview
Crypto Analyst Pro is an advanced AI agent designed to provide multi-faceted, data-driven analysis across the entire cryptocurrency ecosystem. It synthesizes technical indicators, deep on-chain metrics (like MVRV and active addresses), and real-time market sentiment to generate actionable insights.

This agent moves beyond simple price charting by evaluating underlying network health, tokenomics, and macro market correlations, ensuring that any generated signal is backed by diverse data sources like Glassnode and Messari.

## Capabilities
*   **Technical Analysis:** Executes comprehensive technical evaluations using crypto-specific indicators across various timeframes.
*   **On-Chain Metrics Deep Dive:** Analyzes critical on-chain data, including transaction volumes, active addresses, and key metrics (NVT, MVRV).
*   **Sentiment & News Aggregation:** Conducts sentiment analysis by monitoring social media trends and major financial news outlets.
*   **Tokenomics Evaluation:** Assesses the fundamental health of tokens by analyzing supply dynamics, vesting schedules, and whale wallet activity.
*   **Signal Generation:** Provides clear trading signals complete with a detailed rationale and an assigned confidence score.
*   **Market Correlation Mapping:** Generates correlation matrices to show how different assets relate to BTC/ETH and broader market movements.

## Example Use Cases
1. **Pre-Investment Due Diligence:** Input a new altcoin ticker and receive a full report covering its tokenomics, current on-chain adoption rates, and potential resistance/support levels.
2. **Market Regime Identification:** Ask the agent to analyze the current market regime (e.g., early bull vs. late cycle) based on macro indicators and provide adjusted risk assessments.
3. **Signal Validation:** Provide a recent price spike for an asset and request confirmation by cross-referencing it with exchange flow data and social media sentiment to validate the signal's strength.
