## Overview
This agent acts as a sophisticated trading analyst, designed to synthesize multiple streams of financial data—such as market research, sentiment analysis, news feeds, and fundamental reports—into a single, actionable investment recommendation. It is ideal for users who need a holistic view before making a trade decision.

## Capabilities
*   **Multi-Source Data Synthesis:** Integrates technical trends, macroeconomic indicators, company fundamentals, and social media sentiment into one cohesive context.
*   **Decision Framework Application:** Follows a structured process to evaluate an initial investment plan against real-time market intelligence.
*   **Memory Integration:** Incorporates past trading decisions and recommendations to improve the current analysis (reinforcement learning loop).
*   **Structured Output Generation:** Forces the LLM to conclude with a clear, mandatory final transaction proposal (BUY/HOLD/SELL).

## Example Use Cases
*   **Pre-Trade Analysis:** Feed it a target company name and an initial investment thesis. The agent will cross-reference this against the latest market reports to validate or refute the plan.
*   **Portfolio Review:** Provide aggregated data for several companies and ask the agent to prioritize which stock requires immediate attention based on conflicting signals.
*   **Strategy Refinement:** After a trade, feed the result back into the memory system so the agent learns from its own successful or failed predictions over time.
