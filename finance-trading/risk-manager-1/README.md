## Overview
This agent functions as an expert risk manager dedicated to safeguarding investment portfolios. It moves beyond simple performance tracking by implementing rigorous quantitative methods like R-multiple analysis, Value at Risk (VaR) calculation, and systematic stress testing.

Use this tool proactively whenever you need a deep dive into portfolio vulnerability, precise position sizing recommendations, or actionable hedging strategies before executing trades.

## Capabilities
*   **R-Multiple Tracking:** Measures all trades against defined risk units (1R) for objective performance measurement.
*   **Risk Quantification:** Calculates key metrics including Expected Value and Maximum Drawdown.
*   **Position Sizing:** Determines optimal trade sizes using principles derived from the Kelly Criterion based on account risk tolerance.
*   **Correlation Analysis:** Generates a matrix to identify dangerous concentration risks between correlated assets.
*   **Systematic Hedging:** Recommends specific hedges using derivatives (options, futures) to mitigate identified risks.
*   **Stress Testing:** Conducts Monte Carlo simulations to model portfolio performance under various adverse market scenarios.

## Example Use Cases
1. **Pre-Trade Analysis:** Before entering a new position, ask the agent to calculate the potential R-multiple impact and suggest an appropriate stop-loss level based on current volatility.
2. **Portfolio Review:** Submit your current holdings and request a comprehensive risk assessment report, focusing specifically on correlation risks between sectors or assets.
3. **Strategy Validation:** Provide historical win/loss data and ask the agent to calculate the expected return (expectancy) and suggest if the current strategy warrants adjustments in position sizing.
