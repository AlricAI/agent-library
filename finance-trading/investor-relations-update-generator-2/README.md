## Overview
This agent is designed to transform raw, operational data—including shipped features, actual metrics, and identified risks—into a disciplined, truth-telling monthly update suitable for investor review. Its core function is to prevent the creation of overly polished narratives that detach from underlying business reality.

## Capabilities
*   **Truth Synthesis:** Converts real traction and shipped work into clear narrative points, prioritizing operational facts over marketing spin.
*   **Risk Identification:** Explicitly calls out areas where progress has stalled or where instrumentation is missing, maintaining investor confidence through transparency.
*   **Actionable Insights:** Structures updates to clearly delineate what improved, what remains uncertain, and concrete asks for investor support.
*   **Auditable Drafting:** Produces a draft that is grounded in evidence, ensuring all claims can be traced back to source data before any external publication.

## Example Use Cases
*   **Monthly Board Reporting:** Input raw sprint reports, Stripe transaction summaries, and key product milestones to generate the core narrative for the monthly board meeting deck.
*   **Post-Quarter Review:** Feed in conflicting metrics (e.g., soft analytics vs. direct payment data) and have the agent draft a report that prioritizes hard financial evidence.
*   **Milestone Communication:** When launching a major feature, use it to write an update that connects the shipped code directly to its measurable business consequence for investors.
