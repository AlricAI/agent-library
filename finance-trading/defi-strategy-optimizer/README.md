## Overview
The DeFi Strategy Optimizer is an advanced agent designed to act as a comprehensive yield strategist for decentralized finance. It moves beyond simple APY tracking by modeling complex risk factors, impermanent loss scenarios, and gas optimization across various blockchain ecosystems.

This agent focuses on building sustainable, capital-efficient strategies rather than chasing unsustainable high yields.

## Capabilities
*   **Yield Strategy Design:** Creates comprehensive yield farming plans with built-in risk adjustments.
*   **Liquidity Optimization:** Manages liquidity pool positioning across different Automated Market Makers (AMMs).
*   **Risk Modeling:** Assesses impermanent loss, smart contract risks, and protocol sustainability using historical data.
*   **Execution & Automation:** Generates gas-optimized transaction builders suitable for direct execution via Web3 libraries (e.g., Web3.py/Ethers.js).
*   **Cross-Chain Integration:** Develops strategies that account for cross-chain bridge utilization and associated risks.

## Example Use Cases
1. **Portfolio Deployment:** Input a set of target assets and desired risk tolerance; the agent will output a backtested, optimized vault strategy with deployment steps.
2. **Protocol Due Diligence:** Provide a new DeFi protocol address; the agent will generate a comprehensive risk report covering TVL, liquidity depth, and potential attack vectors.
3. **Rebalancing Alert:** Monitor an existing pool; the agent can calculate when impermanent loss or changing incentives necessitate an automated rebalance transaction.
