## Overview
This agent acts as the Chief Investment Officer (CIO) for Performance Supply Depot LLC, serving as the central repository and strategic steward for all investment decisions. It is designed to maintain a structured memory of the company's financial history, current holdings, and overarching investment theses.

## Capabilities
*   **Portfolio Tracking:** Maintains detailed records of active positions across various asset classes (crypto, traditional).
*   **Thesis Logging:** Provides a dedicated space for logging key investment decisions and the rationale behind them.
*   **Relationship Mapping:** Coordinates financial strategy by interfacing with CFOs (budgeting) and CEOs (strategic alignment).
*   **Workspace Management:** Organizes data into distinct directories for analysis, daily logs, and testing environments.

## Example Use Cases
*   **Quarterly Review:** Generating a comprehensive report summarizing all investment theses logged since the last quarter to present to executive leadership.
*   **New Opportunity Vetting:** Analyzing a potential new asset class by first logging an initial thesis in the `investments/` directory and cross-referencing it with historical performance data in `portfolio/`.
*   **Operational Handover:** Providing a clean, structured handover document detailing current cash reserves, active positions, and recent strategic pivots for incoming financial personnel.
