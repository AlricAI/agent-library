## 📌 What is an Airdrop?

**Definition:** When a crypto developer gives out free tokens/coins to wallet addresses.

### Two Main Purposes:

| Purpose | Description | Example |
|---------|-------------|---------|
| **Advertising** | Spread awareness and get people to try the coin | Marketing campaign |
| **Reward Early Adopters** | Thank users who tested/interacted early | Uniswap retroactive |

---

## 🦄 Case Study: Uniswap Airdrop (September 2020)

### What is Uniswap?
- Largest decentralized exchange (DEX) by TVL (Total Value Locked)
- Allows swapping any Ethereum token for any other
- Started centralized → became decentralized

### The Airdrop:
- **Who received:** Every wallet that interacted with Uniswap before Sept 2020
- **Amount:** ~400 UNI tokens per wallet
- **Value at time:** ~$1,000
- **Value if held:** ~$8,000 (at time of video)

### Why They Did It:
1. **Decentralization** - Gave governance to community
2. **Voting power** - Token holders vote on protocol changes
3. **Reward early testers** - Not random users
4. **Create loyal community** - People who use it early become advocates

### Technical How-To:
```
1. Uniswap = smart contracts on Ethereum
2. Smart contracts are public/auditable
3. Query all wallet addresses that interacted
4. Send UNI tokens to each address
5. Automatic distribution via blockchain
```

### Key Term: **Retroactive Airdrop**
- Looked BACK in time at historical usage
- Rewarded wallets from BEFORE the announceme
