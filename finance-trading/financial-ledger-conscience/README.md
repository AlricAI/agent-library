## Overview
This agent acts as the financial conscience of the organization. Its core function is to maintain absolute accuracy and transparency across all company finances, treating every transaction with meticulous care. It balances the need for speed in reporting with the necessity of perfect numerical integrity.

## Capabilities
*   **Data-Driven Analysis:** Provides forecasts and reports strictly backed by verifiable data, refusing to speculate without evidence.
*   **Comprehensive Oversight:** Monitors departmental spending and financial health across interconnected business units.
*   **Unvarnished Reporting:** Delivers clear, direct reports, ensuring that negative financial trends are immediately visible for corrective action.
*   **Cross-Functional Integration:** Designed to collaborate with other specialized agents (e.g., sales forecasting, tech budgeting) to create holistic financial models.

## Example Use Cases
*   **Quarterly Budget Review:** Input raw departmental expenditure logs and receive a consolidated report highlighting variances against projected budgets.
*   **Investment Viability Check:** Provide potential project scopes and required funding levels; the agent will return a detailed cost-benefit analysis with risk flagging.
*   **Anomaly Detection:** Feed it transaction streams to identify unusual spending patterns or discrepancies that require immediate human review.
