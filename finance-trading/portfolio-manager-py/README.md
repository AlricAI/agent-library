## Overview
The Portfolio Manager acts as the final decision-making layer in an investment workflow. Its primary role is to synthesize disparate pieces of information—including market research, sentiment analysis, fundamental reports, and debates from specialized agents (like Risk Analysts)—into one cohesive, actionable trading recommendation.

This agent takes all preceding analyses and forces a definitive conclusion, ensuring that the final trade decision is well-rounded, risk-aware, and strategically justified.

## Capabilities
*   **Synthesis:** Combines multiple data sources (market reports, news, fundamentals) into a single context for decision-making.
*   **Risk Integration:** Incorporates historical memory of past decisions to avoid repeating mistakes.
*   **Structured Output Generation:** Forces the output into a strict format: Rating, Executive Summary, and Investment Thesis.
*   **Actionable Grading:** Constrains the final recommendation to one of five predefined rating levels (Buy, Overweight, Hold, Underweight, Sell).

## Example Use Cases
*   **Finalizing an Investment:** After gathering all data points on a stock, use this agent last to receive a definitive 'Buy' or 'Sell' signal.
*   **Strategy Review:** If the team is debating between two conflicting strategies, running the Portfolio Manager can force consensus or highlight which risk factor outweighs the others.
*   **Position Sizing Guidance:** The resulting Executive Summary provides immediate guidance on position sizing and time horizons for execution.
