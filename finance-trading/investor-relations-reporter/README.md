## Overview
This agent is designed to streamline the critical, often complex, process of generating regular investor updates. It acts as a rigorous quality gate, ensuring that all reported metrics are grounded in current issues and verified against the highest-truth data sources available.

Its primary function is not to send live communications but to generate polished, draft artifacts ready for human review, significantly reducing manual reconciliation time.

## Capabilities
*   **Reporting Window Confirmation:** Grounds its analysis on the most recent Paperclip issue to confirm the correct reporting period.
*   **Metric Verification:** Performs month-over-month comparisons, cross-referencing figures against designated 'highest-truth' systems for accuracy.
*   **Artifact Drafting:** Creates comprehensive draft artifacts, explicitly avoiding live sends until human sign-off is achieved.
*   **Monthly Summarization:** Generates executive summaries detailing what was shipped, what materially changed, what slipped, and identifying areas requiring immediate support.
*   **Founder Ask Refinement:** Helps distill founder asks into concrete, sparse, and actionable statements.

## Example Use Cases
*   **Monthly Board Deck Prep:** Running the agent at month-end to generate a complete draft summary covering all key performance indicators (KPIs) for board review.
*   **Metric Discrepancy Flagging:** Identifying and flagging situations where core systems (like Stripe, Firestore, or analytics) report conflicting data points, prompting necessary investigation.
*   **Pre-Fundraising Check:** Ensuring that any draft update containing runway projections or fundraising claims is fully supported by approved documentation before being finalized.
