## Overview
This agent is designed to create comprehensive and highly actionable workspace digests. Its core function is to synthesize disparate sources of project information—specifically Knowledge articles, Growth Studio insights, and the Work Queue backlog—into a single, cohesive narrative. The goal is always to provide an internal, concise report that drives immediate understanding and next steps.

## Capabilities
*   **Multi-Source Grounding:** Ensures all generated content is grounded in the current issue context and relevant timeframe.
*   **Holistic Inspection:** Inspects Knowledge bases, Growth Studio data, and the Work Queue simultaneously to build a complete picture.
*   **Digest Generation:** Produces reports that are inherently internal, action-oriented, and avoid sounding like external marketing copy.
*   **Temporal Summarization (Weekly):** For weekly runs, it compresses information into clear Highlights, identified Risks, and concrete Next Actions.

## Example Use Cases
*   **Daily Standup Prep:** Run this agent daily to get a quick pulse check on what's happening across all active workstreams.
*   **Bi-Weekly Stakeholder Update:** Utilize the weekly mode to provide leadership with a high-level overview, focusing heavily on risks and required decisions rather than just completed tasks.
*   **Identifying Divergence:** Use it when you suspect that the strategic narrative (Growth Studio) is not matching the operational reality (Work Queue), flagging these discrepancies for immediate review.
