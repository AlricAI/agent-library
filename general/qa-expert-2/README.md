## Overview
This agent functions as a senior Quality Assurance (QA) expert, guiding teams through systematic quality validation. It specializes in developing comprehensive test strategies and ensuring that software meets the highest standards of user satisfaction and reliability.

## Capabilities
*   **Test Strategy Development:** Creates detailed plans covering requirements analysis, risk assessment, and resource planning.
*   **Gap Analysis:** Reviews existing test coverage, defect patterns, and quality metrics to pinpoint weaknesses.
*   **Testing Execution Guidance:** Provides expertise across manual testing (e.g., usability, accessibility) and automated testing (e.g., API, CI/CD integration).
*   **Defect Lifecycle Management:** Guides root cause analysis, severity classification, and regression testing protocols.
*   **Quality Metric Tracking:** Monitors key indicators like test coverage, defect density, and automation percentage to measure overall quality maturity.

## Example Use Cases
*   **Pre-Release Audit:** Feed it a set of requirements and existing test plans; it will generate a gap analysis report highlighting areas needing more rigorous testing (e.g., suggesting performance or security checks).
*   **Test Plan Review:** Provide an initial test plan, and the agent will critique it against industry best practices, ensuring all necessary components like exit criteria and resource allocation are present.
*   **Process Improvement:** If defect leakage is high, use this agent to recommend implementing stricter pre-commit quality gates or improving unit test coverage.
