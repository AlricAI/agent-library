## Overview
The Project Manager Pro agent is a specialized assistant designed to streamline the entire project lifecycle for professional project managers, Scrum Masters, and PMO directors. It integrates best practices from Agile methodologies (like Scrum) with practical execution across common enterprise tools such as Jira and Confluence.

This agent moves beyond simple task tracking by providing data-driven insights into portfolio health, risk exposure, and team velocity, enabling proactive decision-making rather than reactive reporting.

## Capabilities
* **Sprint Planning & Execution:** Facilitates structured sprint planning sessions and tracks daily progress against defined goals.
* **Toolchain Administration:** Manages project setup within Jira (e.g., workflow design) and maintains centralized documentation in Confluence.
* **Portfolio Health Monitoring:** Generates comprehensive, portfolio-level dashboards that assess overall project status using RAG (Red/Amber/Green) indicators across multiple initiatives.
* **Risk Analysis:** Utilizes a risk matrix analyzer to identify potential roadblocks and quantify their impact on timelines and budgets.
* **Stakeholder Reporting:** Creates executive-ready reports summarizing progress, risks, and achievements for non-technical stakeholders.

## Example Use Cases
1. **End of Sprint Review:** Ask the agent to generate a 'Sprint Retrospective Report' comparing planned velocity against actual output, highlighting bottlenecks identified in Jira tickets.
2. **New Project Kickoff:** Instruct it to create the foundational structure by generating a new Confluence space template and setting up initial project boards within Jira based on provided scope documents.
3. **Portfolio Oversight:** Request a 'Q3 Portfolio Health Dashboard' for all active projects, forcing an analysis of schedule variance and budget burn rate across departments.
