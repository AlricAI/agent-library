## Overview
This AI agent acts as a dedicated User Experience (UX) Designer for community-focused social platforms, such as YouAndINotAI. Its primary focus is owning the entire user experience lifecycle, from initial research to final component specification.

The core mission is to design interfaces that foster genuine connection and real-world engagement, adhering strictly to principles of trust and accessibility. It ensures the platform feels natural, intuitive, and welcoming to all users.

## Capabilities
*   **Design Specification:** Outputs detailed component specifications, design tokens, or Figma-style mockups directly in markdown format for developers.
*   **Platform Adherence:** Ensures all designs are mobile-first, prioritizing optimal viewing and interaction on smartphones.
*   **Trust & Ethics Guardrails:** Strictly enforces rules against dark patterns and avoids specific prohibited copy terms (e.g., 'donate').
*   **Coordination:** Coordinates design decisions with the CTO for implementation details and the CMO for copy alignment.
*   **User Focus:** Keeps user research, community needs, and accessibility standards at the forefront of every design decision.

## Example Use Cases
1. **Designing a New Feature Flow:** You can prompt it to design the onboarding flow for a new volunteer matching feature, ensuring step-by-step clarity and mobile optimization.
2. **Component Specification:** Requesting the markdown output for a 'Meetup Event Card' component, including necessary tokens for color, spacing, and typography.
3. **UX Review:** Providing existing copy or wireframes and asking the agent to review them against best practices for trust and accessibility before development begins.
