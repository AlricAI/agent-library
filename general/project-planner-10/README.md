## Overview
The Project Planner agent is designed to act as a strategic project manager, taking an ambitious, high-level goal and breaking it down into a concrete, step-by-step execution plan. Its core function is to transform vague objectives into actionable checklists with defined dependencies.

This tool excels at establishing clear scope boundaries by defining what 'done' means for every single task, ensuring that work progresses logically from one verifiable milestone to the next.

## Capabilities
*   **Goal Decomposition**: Breaks massive features into smaller, sequential phases and tasks.
*   **Dependency Mapping**: Identifies prerequisite tasks, ensuring work is done in the correct order.
*   **Definition of Done (DoD)**: Forces specificity by requiring clear completion criteria for every step.
*   **Risk Prioritization**: Structures plans to tackle high-risk architectural components early in the process.
*   **Structured Output**: Delivers plans exclusively in clean, markdown checklist formats.

## Example Use Cases
*   **Software Development Roadmap**: Inputting 'Build a full-stack e-commerce site' results in phases like 'Database Schema Design,' 'User Authentication MVP,' and 'Payment Gateway Integration.'
*   **Event Planning**: Breaking down 'Host a company conference' into logistical steps, such as 'Venue Scouting' $\rightarrow$ 'Vendor Negotiation' $\rightarrow$ 'Speaker Confirmation.'
*   **Research Project Outline**: Turning 'Write a comprehensive report on AI ethics' into sequential research milestones: Literature Review $\rightarrow$ Case Study Collection $\rightarrow$ Drafting Methodology.
