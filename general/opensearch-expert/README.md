## Overview
This agent acts as a comprehensive expert in OpenSearch cluster management. It guides users through best practices for setting up, optimizing, and maintaining high-performance search clusters. From initial schema design to advanced security hardening, it covers the entire lifecycle of your data within OpenSearch.

## Capabilities
*   **Query Optimization:** Designs queries with improved execution plans to minimize resource consumption.
*   **Indexing Strategy:** Recommends optimal shard and replica counts based on anticipated access patterns.
*   **Performance Tuning:** Provides guidance on JVM tuning, workload balancing using node roles, and monitoring setup.
*   **Data Lifecycle Management (ILM):** Implements best practices for data retention, including snapshotting and restoration plans.
*   **Security & Compliance:** Assesses security configurations, recommends access controls, and generates audit reports.
*   **Advanced Search Features:** Assists with configuring analyzers, tokenizers, and filters for robust full-text search.

## Example Use Cases
*   **Performance Bottleneck Identification:** Provide system logs and query patterns to receive a detailed report on latency issues and actionable tuning steps.
*   **New Data Source Integration:** Outline your data schema and access needs; the agent will generate an optimized index structure, including necessary analyzers and ingestion pipelines.
*   **Disaster Recovery Planning:** Request a comprehensive backup and recovery plan based on current cluster size and required RPO/RTO metrics.
*   **Security Hardening Audit:** Submit existing security settings to receive a compliance report against industry best practices.
