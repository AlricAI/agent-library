## Overview
The Accessibility Auditor is a specialized AI agent dedicated to ensuring that digital products and web interfaces meet established accessibility standards, primarily WCAG (Web Content Accessibility Guidelines). This agent acts as a rigorous quality assurance checkpoint within the development lifecycle, guaranteeing that applications are usable by the widest possible audience, including individuals with visual, auditory, motor, or cognitive impairments.

## Capabilities
*   **WCAG Compliance Checking:** Systematically reviews code and UI mockups against specific WCAG success criteria (e.g., color contrast ratios, keyboard navigability).
*   **Semantic Markup Validation:** Identifies missing or incorrect semantic HTML elements that confuse assistive technologies like screen readers.
*   **Usability Review:** Provides actionable feedback on interaction patterns to improve overall user experience for all abilities.
*   **Remediation Suggestion:** Doesn't just point out flaws; it suggests concrete, implementable code fixes and design adjustments.

## Example Use Cases
*   **Pre-Launch Audit:** Run the agent against a staging build of a new checkout flow to ensure keyboard focus order is logical and all form fields are properly labeled.
*   **Design System Review:** Validate that newly created components (like modals or carousels) adhere to ARIA best practices before they enter the main design system library.
*   **Content Accessibility Check:** Analyze marketing landing pages to confirm that images have appropriate alt-text and that all interactive elements are reachable via tab navigation.
