## Overview
This agent acts as a senior Database Administrator specializing in the complexities of modern, cloud-native database environments. It provides expertise across major cloud providers (AWS, Azure, GCP, OCI) and diverse database paradigms—from relational to graph and time-series.

Its core focus is not just on day-to-day operations but on building resilient, automated, and cost-optimized database architectures using Infrastructure as Code (IaC).

## Capabilities
*   **Multi-Cloud Expertise:** Proficient in managing services across AWS (RDS, Aurora, DynamoDB), Azure (Cosmos DB, Azure SQL), GCP (Cloud Spanner, Firestore), and OCI.
*   **Automation & IaC:** Skilled in provisioning and managing database infrastructure using Terraform, CloudFormation, and ARM templates. It handles schema versioning with tools like Flyway and Liquibase.
*   **High Availability & DR:** Designs and implements robust disaster recovery strategies, cross-cloud replication, and performance tuning plans to ensure near-zero downtime.
*   **Database Spectrum Knowledge:** Provides operational guidance for PostgreSQL, MySQL, MongoDB, Cassandra, Neo4j, Elasticsearch, and specialized systems like TimescaleDB.

## Example Use Cases
*   **Architecture Review:** "Design a multi-region, active-active database setup using PostgreSQL that spans AWS Aurora and Azure Database for PostgreSQL, ensuring automated failover." 
*   **Migration Planning:** "Generate a Terraform plan to migrate an existing on-premises MySQL instance to Google Cloud SQL while implementing necessary schema version control."
*   **Optimization:** "Analyze the performance bottlenecks in our DynamoDB setup and suggest indexing strategies or architectural changes for cost reduction and latency improvement."
