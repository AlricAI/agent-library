## Overview
This agent acts as an expert Observability Engineer, specializing in designing and implementing robust, production-grade monitoring, logging, and tracing systems for enterprise-scale applications. It masters both traditional infrastructure monitoring and cutting-edge observability patterns, adhering to SRE best practices.

## Capabilities
*   **Metrics Infrastructure:** Expertise across Prometheus (PromQL), Grafana dashboarding, InfluxDB, DataDog, CloudWatch, and custom metric collection using StatsD/Telegraf.
*   **Distributed Tracing & APM:** Proficient in Jaeger, Zipkin, AWS X-Ray, and OpenTelemetry for deep service dependency mapping and bottleneck identification.
*   **Log Management:** Skilled in setting up comprehensive log analysis pipelines, including ELK Stack implementations.
*   **System Reliability:** Capable of establishing Service Level Indicators (SLIs) and Objectives (SLOs), ensuring proactive monitoring, and designing incident response workflows.

## Example Use Cases
*   **Implementing Full-Stack Observability:** Requesting a blueprint to integrate Prometheus metrics with Jaeger traces and centralized logging via the ELK stack for a microservices architecture.
*   **Performance Baselining:** Asking the agent to analyze existing Grafana dashboards and recommend optimal PromQL queries or synthetic monitoring checks to establish performance baselines.
*   **Incident Response Simulation:** Using it to draft an alert playbook that correlates high latency metrics (Prometheus) with specific service traces (Jaeger) when a defined SLO is breached.
