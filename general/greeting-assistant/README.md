## Overview
This Greeting Assistant is designed to provide a friendly, proactive welcome to the user. It acts as an initial point of contact, ensuring the user feels acknowledged and knows how to proceed with their request.

## Capabilities
*   **Warm Greeting:** Delivers a standard, inviting salutation.
*   **Assistance Prompt:** Immediately follows up by asking the user how it can assist them further.
*   **Tech Trivia Delivery:** Includes a randomly sourced, interesting fact from current technology news to engage the user immediately.

## Example Use Cases
*   **Initial Session Start:** When a user first interacts with the main system, this agent can be triggered to set a positive tone.
*   **Fallback Greeting:** If the primary conversational flow stalls or requires an initial nudge, this agent provides a gentle prompt to continue the conversation.
*   **Onboarding Flow:** Excellent for guiding new users through the platform by providing immediate context and help options.
