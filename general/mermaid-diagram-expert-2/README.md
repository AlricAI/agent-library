## Overview
Mermaid Diagram Expert is a specialized AI agent designed to transform complex technical concepts into clean, professional Mermaid diagram code. It masters the syntax for numerous diagram types—from basic flowcharts and sequence diagrams to detailed ERDs and state machines. The goal is always clarity, readability, and adherence to best practices in visualization.

## Capabilities
*   **Multi-Diagram Support:** Generates accurate code for `graph`, `sequenceDiagram`, `erDiagram`, `stateDiagram-v2`, `gantt`, and more.
*   **Proactive Structuring:** Selects the most appropriate diagram type based on the input context (e.g., choosing a sequence diagram for API calls vs. a flowchart for business logic).
*   **Styling & Polish:** Provides both basic and highly styled versions of the code, including suggestions for color palettes and consistent labeling.
*   **Comprehensive Output:** Delivers not just the raw code, but also rendering instructions, alternative options, and accessibility notes.

## Example Use Cases
1. **System Architecture Mapping:** Provide a description of microservices interaction; the agent will generate a clean `sequenceDiagram` or network graph.
2. **Business Process Flow:** Describe a user journey (e.g., checkout process); it will output a clear, labeled `graph` flowchart with decision points.
3. **Database Schema Visualization:** Input entity names and relationships; the agent will produce a fully compliant `erDiagram`.

By using this expert, you ensure your technical documentation is not only accurate but also visually compelling and easy for any stakeholder to understand.
