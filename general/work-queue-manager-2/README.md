## Overview
This agent acts as a comprehensive personal task and status management system for an AI operative. It structures the daily workflow, ensuring that agents follow established protocols whether they have immediate assignments or are in a default operational state.

It centralizes critical information streams—such as company bulletins and return-to-work directives—into actionable steps, minimizing ambiguity and maximizing productivity upon activation.

## Capabilities
*   **Task Prioritization:** Clearly defines the immediate tasks an agent must undertake based on source documents.
*   **Default Workflow Generation:** Provides a structured fallback plan (e.g., reviewing memory, checking for departmental instructions) if no specific assignment is found.
*   **Status Reporting:** Enforces a standardized daily log template to ensure consistent and comprehensive progress updates.
*   **Escalation Protocol:** Maps out clear escalation paths for various blockers (technical, procedural, ethical), ensuring timely intervention.

## Example Use Cases
*   **New Agent Onboarding:** When an agent is reactivated or assigned a new role, this system guides them through reading all necessary foundational documents before starting work.
*   **Daily Check-in:** At the start of each operational cycle, the agent runs its default checks to ensure memory integrity and review pending tasks.
*   **Incident Management:** If an agent encounters a blocker (e.g., access failure or unclear directive), it immediately consults the escalation matrix to know exactly who to contact for resolution.
