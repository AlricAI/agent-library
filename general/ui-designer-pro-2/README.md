## Overview
Ui Designer Pro acts as a senior, expert UI/UX designer focused on creating polished, highly functional, and aesthetically pleasing user interfaces. Its core strength lies in maintaining strict adherence to established design systems, brand guidelines, and accessibility standards across every touchpoint.

Before any design work begins, the agent mandates gathering comprehensive context (brand guides, component libraries, etc.) to ensure consistency and prevent rework.

## Capabilities
*   **Context Discovery:** Systematically gathers necessary design context by querying for brand guidelines, existing components, and accessibility requirements.
*   **Visual Concepting:** Generates polished visual concepts and variations based on provided requirements.
*   **Interaction Pattern Definition:** Designs detailed interaction flows and state variations for complex components.
*   **Design System Building:** Structures reusable component systems suitable for developer handoff.
*   **Progress Reporting:** Provides structured status updates throughout the design process, detailing completed items and next steps.

## Example Use Cases
1. **New Feature Implementation:** Need to build a checkout flow? Start by providing your existing e-commerce style guide, and the agent will map out component variations (e.g., error states, success messages) while adhering to brand color palettes.
2. **Design System Audit:** If you suspect inconsistency across different product modules, ask the agent to audit a set of screens against provided design tokens to identify deviations from your core system.
3. **Accessibility Review:** Submit mockups and request an accessibility review; the agent will flag WCAG violations regarding contrast ratios, focus states, and semantic structure.
