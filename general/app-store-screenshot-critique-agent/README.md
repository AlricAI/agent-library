## Overview
This agent serves as the final quality gatekeeper for all app screenshots intended for App Store submission. Its standard is uncompromising: anything that wouldn't pass as top-tier marketing material is rejected. It operates under a strict set of criteria, focusing solely on visual fidelity and adherence to established design specifications.

## Capabilities
*   **Visual Assessment:** Grades provided physical screenshot files against detailed specifications.
*   **Specification Checking:** Verifies every element listed in the 'Gold Star' spec table, providing pass/fail verdicts for each.
*   **Comprehensive Reporting:** Generates a full, element-by-element review table with justifications for all findings.
*   **Grading & Deduction:** Assigns a final grade (1-10) and itemizes every deduction, citing the specific spec value versus the actual observed value.
*   **Verdict Issuance:** Posts a definitive verdict (APPROVED or REJECTED) along with an explicit, actionable list of required fixes.

## Example Use Cases
*   **Pre-Launch QA:** Before submitting any marketing assets, run screenshots through this agent to ensure they meet the highest possible standard.
*   **Design Iteration Review:** When a designer submits a new mock-up, use this agent to immediately identify deviations from established design guidelines.
*   **Compliance Check:** Use it to verify that all necessary UI elements (e.g., specific navigation patterns or required text) are present and correctly implemented in the visual assets.
