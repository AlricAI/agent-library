## Overview
This specialized agent acts as an expert URL and link validation specialist. It combines rigorous technical testing—checking status codes, redirects, and SSL certificates—with sophisticated content analysis to ensure that provided links are not only functional but also contextually appropriate and highly relevant to the surrounding material.

The goal is to provide a comprehensive audit report that moves beyond basic 'is it broken' checks to answer 'is this link useful here?'

## Capabilities
*   **Technical Validation:** Performs deep testing on links, checking for correct HTTP status codes, redirect chains, and SSL certificate validity.
*   **Contextual Appropriateness Scoring:** Analyzes the relationship between anchor text and the destination content to ensure strong topical alignment.
*   **Content Relevance Assessment:** Evaluates external link quality by assessing factors like publication dates, domain authority, and topic match against the source material.
*   **Comprehensive Reporting:** Generates detailed reports that prioritize critical issues with actionable, reasoned recommendations for improvement or replacement.

## Example Use Cases
*   **Pre-Publication SEO Audit:** Before publishing a large article, use this agent to ensure every external link is live, authoritative, and contextually supports the main narrative.
*   **Content Migration Review:** When updating an old website section, run this agent against all existing links to identify broken assets or outdated resources that need updating.
*   **Link Building Quality Check:** If you are building out a resource page, use it to validate potential backlinks to ensure they point to high-quality, relevant sources.
