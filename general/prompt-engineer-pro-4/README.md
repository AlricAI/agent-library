## Overview
Prompt Engineer Pro acts as a specialized consultant for crafting high-performance prompts for Large Language Models (LLMs). It moves beyond simple instruction writing by applying advanced cognitive techniques to ensure the generated prompt is robust, context-aware, and highly effective for the intended task.

## Capabilities
*   **Technique Selection:** Automatically selects optimal prompting strategies like Chain-of-Thought (CoT), Tree of Thoughts (ToT), or few-shot learning based on task complexity.
*   **Structure Generation:** Builds comprehensive prompts incorporating defined roles, step-by-step instructions, and explicit output format specifications.
*   **Optimization & Testing:** Provides not only the final prompt but also a detailed explanation of the techniques used, suggested A/B testing variations, and a methodology for performance measurement.
*   **Constraint Enforcement:** Incorporates necessary guardrails and error handling instructions directly into the prompt structure.

## Example Use Cases
1. **Complex Reasoning Tasks:** When an agent needs to solve multi-step logic problems (e.g., debugging code or analyzing market trends), use this agent to generate a ToT-enhanced prompt.
2. **Role Simulation:** Developing agents that must adopt specific expert personas (e.g., 'Senior Financial Analyst') requires the role-based prompting capabilities.
3. **API Integration Testing:** Before integrating an LLM into a production workflow, use it to generate a highly structured prompt with strict JSON output requirements for reliable parsing.
