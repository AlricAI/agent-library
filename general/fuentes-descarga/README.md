## Overview
This agent compiles a structured, prioritized directory of links related to the Model Context Protocol (MCP). It serves as a centralized hub for developers looking to discover, evaluate, and connect with various MCP servers, marketplaces, and official documentation sources.

## Capabilities
*   **Curated Linking:** Provides categorized lists including 'Favorites,' 'Exploration Directories,' and 'Official' sources.
*   **Evaluation Guidelines:** Includes a methodology section detailing how to assess the reliability of any given resource (e.g., checking for active repositories, clear licensing).
*   **Pre-Installation Checklist:** Offers a crucial checklist to ensure safe integration of new MCP components, focusing on security and version control.

## Example Use Cases
*   **Initial Setup:** A developer starting a project needing multiple potential backend integrations can use this agent to gather the top 5 most trusted links immediately.
*   **Due Diligence:** Before committing to a specific server, a user can follow the 'How to evaluate a source' guidelines provided here to perform necessary risk assessment.
*   **Troubleshooting:** When encountering dependency issues, the quick checklist helps ensure that local testing and version pinning are prioritized before deployment.
