## Overview
This agent is designed to synthesize disparate sources of project information—specifically Knowledge artifacts, Growth Studio metrics, and the Work Queue backlog—into a single, highly actionable internal digest. Its primary goal is to provide stakeholders with a clear, concise narrative of what has happened, what risks exist, and what immediate actions are required.

## Capabilities
* **Holistic Synthesis:** Inspects Knowledge bases, Growth Studio data, and the Work Queue simultaneously for comprehensive context.
* **Digest Generation:** Creates reports that are inherently internal, focusing on operational reality rather than external marketing narratives.
* **Temporal Contextualization:** Adapts its output based on the requested timeframe (e.g., immediate run vs. weekly summary).
* **Risk Identification:** Prioritizes surfacing potential misses, risks, or unresolved dependencies as the core narrative.

## Example Use Cases
* **Daily Check-in:** Run this agent to get an immediate digest grounded in today's issues and current priorities, ensuring all three data sources are reconciled before drafting.
* **Weekly Review Prep:** Execute it weekly to compress a full week's activity into clear highlights, identified risks, and concrete next actions for leadership review.
* **Post-Mortem Synthesis:** Use it when project status seems contradictory across different internal tools; the agent will flag divergences between stated goals (Growth Studio) and actual work items (Work Queue).
