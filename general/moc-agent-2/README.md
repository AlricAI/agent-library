## Overview
The Moc Navigator is a specialized agent designed to solve the common problem of knowledge silos in large digital vaults. Its primary function is to proactively identify areas lacking proper navigational structure and generate comprehensive Maps of Content (MOCs). It ensures that your valuable content remains discoverable by creating interconnected hubs.

## Capabilities
*   **MOC Generation:** Creates new MOC files following established templates, ensuring a consistent structure for easy adoption.
*   **Orphan Asset Management:** Scans directories to find images and visual assets lacking incoming links, automatically organizing them into categorized gallery notes.
*   **Network Maintenance:** Updates existing MOCs to reflect new content and actively maintains bidirectional linking between related navigation hubs.
*   **Structural Organization:** Implements a clear hierarchy within generated MOCs, separating core concepts from resources and cross-references.

## Example Use Cases
*   **Onboarding New Projects:** When a large set of notes is added for a new topic (e.g., 'Quantum Computing'), run the agent to generate `MOC - Quantum Computing.md`, linking all related sub-topics immediately.
*   **Cleaning Up Stale Content:** Run it periodically across your vault to identify orphaned images or disconnected sections, which are then collected into organized gallery notes for review and integration.
*   **Improving Discoverability:** Use it on a major section of documentation to ensure that every core concept is linked from the main MOC, creating a robust, navigable map for any user.
