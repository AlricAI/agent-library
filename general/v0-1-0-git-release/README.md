# 🚀 Hulunote v0.1.0 — First Public Release

Hulunote is now publicly available.

An open-source outliner note app **with a built-in MCP client** — turning your knowledge base into an AI tool workspace.

---

## ✨ Core Features

### 🧩 Built-in MCP Client (Major Feature)

Hulunote works like **Claude Desktop / Cherry Studio / VS Code AI tool integrations**.

You can:

* Connect to **your own MCP servers**
* Configure custom tools
* Use AI tools directly inside your notes

Your note app becomes a **local AI tool hub**.

---

### 📝 Outliner Notes

* Infinite nested bullet structure
* Fast structured thinking workflow

---

### 🔗 Bidirectional Links

* `[[wiki-style links]]`
* Automatic backlinks
* Networked knowledge instead of folders

---

### 📆 Daily Notes

Automatic date-based journaling.

### 🗂 Multiple Databases

Separate workspaces for different projects.

---

## 🏗 Stack

Rust (Axum, SQLx) + ClojureScript + PostgreSQL

## 🎯 What This Release Proves

Hulunote shows that:

> **Notes + Knowledge Graph + MCP Tools can live in one workspace.**

Not just writing.
Not just AI chat.
But a structured thinking environment that can call tools.
