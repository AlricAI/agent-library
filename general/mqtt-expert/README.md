## Overview
This agent serves as a comprehensive expert on the MQTT protocol, designed to help engineers build highly efficient, scalable, and secure Internet of Things (IoT) communication systems. It covers everything from basic topic structuring to advanced security implementations.

## Capabilities
*   **Protocol Mastery:** Deep understanding of MQTT standards, including QoS levels (0, 1, 2), connection lifecycles, and message retention strategies.
*   **Security Implementation:** Advises on best practices for authentication, encryption, and securing client-broker communication.
*   **Topic Design:** Optimizes topic hierarchies for efficient routing, scalability, and organization within large deployments.
*   **Performance Tuning:** Provides guidance on minimizing latency, optimizing payloads, and managing resource usage across the broker setup.
*   **Troubleshooting & Auditing:** Generates best practice guidelines, comprehensive test cases, and performance benchmarks to validate system health.

## Example Use Cases
*   **Designing a Smart City Sensor Network:** Need an agent to structure topics for thousands of sensors while ensuring reliable data delivery using appropriate QoS levels.
*   **Implementing Secure Device Onboarding:** Using the agent to audit connection logic, ensuring proper authentication and encryption are in place before connecting new devices.
*   **Optimizing Data Flow:** When message latency is critical, use this agent to review existing topic structures and suggest optimizations for minimal overhead.
