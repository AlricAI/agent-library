## Overview
This agent is designed to move beyond the limitations of a standard chatbot. It operates as a highly competent, resourceful personal assistant that treats user data and context with utmost respect and discretion. Instead of offering generic pleasantries, it aims to provide direct, actionable assistance by synthesizing information from provided sources.

## Capabilities
*   **Proactive Problem Solving:** Tries to find answers and solutions using available context before asking the user for help.
*   **Contextual Memory Management:** Treats provided files as its core memory, updating and referencing them throughout sessions to maintain continuity.
*   **Opinionated Communication:** Possesses a defined 'vibe'—it is direct, concise when necessary, but thorough when required, avoiding corporate jargon or sycophancy.
*   **Boundary Awareness:** Maintains strict boundaries regarding privacy, only acting externally (like sending emails) after explicit confirmation.

## Example Use Cases
*   **Information Synthesis:** Given a folder of meeting notes and project specs, the agent can synthesize a concise summary highlighting key decisions made and outstanding action items without needing to be prompted for each section.
*   **Drafting Communications:** When asked to draft an email, it will adopt a tone appropriate to the recipient (e.g., direct for a colleague, more formal for a client) while ensuring all necessary context is included.
*   **Task Prioritization:** By reviewing a mix of calendar entries and task lists, it can proactively suggest a revised daily schedule, flagging potential conflicts or bottlenecks.
