## Overview
This agent emulates a senior Quality Assurance (QA) expert, providing systematic guidance across the entire software development lifecycle. Its core function is to elevate product quality by establishing rigorous testing strategies and proactively identifying potential failure points before release.

## Capabilities
*   **Test Strategy Definition:** Develops end-to-end test approaches covering requirements analysis, risk assessment, and resource planning.
*   **Comprehensive Test Planning:** Creates detailed plans including test case design, data preparation, environment setup, and defining clear exit criteria.
*   **Multi-faceted Testing Execution:** Guides manual testing (e.g., usability, accessibility) and automates complex scenarios using modern frameworks (API, mobile).
*   **Defect Lifecycle Management:** Oversees defect discovery, severity classification, root cause analysis, and regression verification.
*   **Quality Measurement:** Tracks key metrics like test coverage, defect density, and automation percentage to provide actionable quality reports.

## Example Use Cases
*   **Pre-Release Audit:** Feed it existing feature specs and current test results; the agent will generate a gap analysis report highlighting areas below the 90% coverage target.
*   **Test Plan Generation:** Provide project scope details, and the agent will output a structured Test Strategy document covering approach, tools, and timelines.
*   **Defect Triage Simulation:** Present a list of newly found bugs; the agent will help prioritize them based on severity and impact, suggesting necessary regression tests.
