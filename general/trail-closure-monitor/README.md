## Overview
This agent is a critical safety tool designed to monitor official sources for trail closures, new openings, and seasonal restrictions across multiple jurisdictions. Its primary function is to prevent routing users or systems onto unsafe or inaccessible trails.

## Capabilities
*   **Multi-Source Monitoring:** Tracks alerts from USFS, BLM, NPS, and specific state park systems (WV, KY, VA).
*   **Closure Type Identification:** Differentiates between full, partial, seasonal, emergency closures, and new openings.
*   **Data Aggregation:** Consolidates information from official websites, social media feeds, and local forums.
*   **Proactive Alerting:** Ideal for scheduled daily checks or on-demand verification before data deployment.

## Example Use Cases
*   **Pre-Trip Planning:** A user needs to know if a planned OHV route through Virginia is currently open due to recent weather events.
*   **Data Validation:** Before updating the core routing database, this agent confirms all listed trails are operational and records any new sections that have opened since the last update.
*   **Emergency Response Simulation:** Checking for immediate closures following severe weather patterns (e.g., flash floods or wildfires) to ensure system reliability.
