## Overview
This agent acts as a comprehensive expert for the entire ELK Stack (Elasticsearch, Logstash, Kibana). It is designed to assist users in deploying, optimizing, and maintaining scalable, secure, and high-performance data ingestion and analysis pipelines.

Whether you are dealing with massive volumes of logs or need complex real-time analytics, this agent provides structured guidance across the entire stack lifecycle.

## Capabilities
*   **Elasticsearch Optimization:** Configuring index settings, managing ILM (Index Lifecycle Management), and optimizing cluster scaling for peak performance.
*   **Logstash Pipeline Development:** Creating robust pipelines to ingest, filter, enrich, and normalize diverse data sources reliably.
*   **Kibana Dashboarding:** Designing intuitive, insightful, and actionable dashboards that transform raw logs into business intelligence.
*   **Security Hardening:** Implementing role-based access control (RBAC) and best practices across all components for secure operation.
*   **Monitoring & Resilience:** Setting up proactive alerting systems and defining comprehensive backup/disaster recovery strategies using tools like Curator.

## Example Use Cases
1. **Building a New Logging System:** Need to ingest logs from 50 different microservices into Elasticsearch, process them through Logstash for geo-enrichment, and visualize key metrics in Kibana? This agent can architect the entire solution.
2. **Performance Tuning:** Your search queries are slow, or your cluster is experiencing high latency. Use this agent to audit index mappings, suggest query optimizations, and recommend hardware/configuration adjustments.
3. **Compliance & Retention:** You need a system that automatically ages out logs after 90 days while keeping critical audit trails accessible for 7 years. This agent will design the necessary ILM policies.
