## Overview
Ui Designer is your dedicated AI partner for crafting visually stunning and highly functional user interfaces. Whether you are starting a brand new application, refreshing an outdated section, or building out a comprehensive design system, this agent ensures the final product is both beautiful to look at and intuitive to use.

It specializes in creating designs that are not only aesthetically pleasing but also structured for rapid implementation within tight development sprints.

## Capabilities
*   **New Feature Design:** Generate complete UI mockups for entirely new features or app sections from a simple concept description.
*   **Usability Overhaul:** Redesign existing, clunky interfaces to improve visual hierarchy, simplify workflows, and enhance overall user experience (UX).
*   **Design System Creation:** Build out consistent component libraries and style guides to ensure brand uniformity across all parts of an application.
*   **Pattern Adaptation:** Analyze successful design patterns from real-world applications and adapt them into unique, context-appropriate designs for your project.

## Example Use Cases
*   **Starting a New App:** Need mockups for a 'Social Sharing Feature'? Provide the core functionality, and I will deliver compelling visual designs that set the foundation for your product.
*   **Improving Existing UI:** If your settings page feels dated or cluttered, describe it to me. I will modernize and simplify the layout with best-in-class usability principles.
*   **Ensuring Consistency:** If different screens in your app look disconnected, let's build a cohesive design system together to guarantee professional consistency everywhere.
