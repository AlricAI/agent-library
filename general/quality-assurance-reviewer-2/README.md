## Overview
The Quality Assurance Reviewer Agent is designed to function as a critical yet constructive guardian for any deliverable. Its core purpose is to rigorously assess work—be it documentation, code snippets, or written reports—to ensure it meets the highest standards of accuracy, completeness, and clarity before final deployment.

## Capabilities
*   **Comprehensive Assessment**: Systematically checks deliverables for accuracy, internal consistency, and overall quality against defined criteria.
*   **Standards Compliance Check**: Validates adherence to specified style guides, industry best practices, and potential security concerns.
*   **Actionable Feedback Generation**: Provides structured feedback, prioritizing identified issues by severity (Critical, Major, Minor) with clear rationale for suggested changes.
*   **Validation Framework**: Executes a multi-step review process covering functional goals, technical correctness, user impact, and edge case identification.

## Example Use Cases
1. **Document Review**: Paste a draft policy document and ask the agent to check it against internal compliance standards, flagging any ambiguous language or missing sections.
2. **Code Review**: Provide a function and request a review focusing on performance efficiency, security vulnerabilities, and adherence to PEP 8 style guidelines.
3. **Content Validation**: Submit marketing copy and ask the agent to validate its tone consistency across multiple paragraphs while ensuring all key product features are mentioned accurately.
