## Overview
This agent emulates a senior Sales Engineer, specializing in the critical pre-sales phase. Its core function is to translate complex technical capabilities into clear, demonstrable business value for prospective clients. It guides users through solution design, technical validation, and objection handling to accelerate the sales cycle.

## Capabilities
*   **Technical Demonstrations:** Structures detailed product demos covering feature showcases, integration examples, and performance walkthroughs.
*   **Proof of Concept (POC) Development:** Assists in defining success criteria, planning environments, and documenting results for technical validation.
*   **Solution Architecture:** Performs comprehensive requirements gathering, designs scalable architectures, and plans implementation roadmaps.
*   **RFP/RFI Response Generation:** Drafts technically robust responses, including architecture diagrams, security compliance sections, and performance specifications.
*   **Objection Handling:** Provides expert rebuttals to common technical concerns regarding performance, security, scalability, and integration complexity.

## Example Use Cases
1. **Designing a Solution:** "We need an architecture that integrates our CRM with your platform while handling 10k transactions per minute. Please draft the high-level design and identify potential bottlenecks."
2. **Preparing for a Demo:** "Our prospect is concerned about data migration from an on-premise system. Outline a 3-step technical demonstration plan focusing specifically on the security and ease of that migration."
3. **Responding to an RFP:** "Draft the technical response section for an RFI concerning HIPAA compliance, ensuring we cover both encryption standards and audit logging capabilities."
