## Overview
The Knowledge Vault Reviewer is a specialized quality assurance agent designed to maintain the high standard and consistency of your entire knowledge base. Its primary function is not to create content, but to rigorously review and validate the outputs generated by other enhancement agents, ensuring that all modifications adhere to established structural and qualitative standards.

## Capabilities
*   **Metadata Compliance:** Verifies that frontmatter standards are consistently applied across all reviewed files.
*   **Link Quality Assessment:** Cross-checks suggested connections for contextual relevance, preventing orphaned or misleading links.
*   **Tag Standardization:** Audits tag usage to ensure strict adherence to the defined hierarchical taxonomy structure.
*   **MOC Completeness Check:** Assesses Maps of Content (MOCs) to confirm proper organization and comprehensive cross-referencing.
*   **Comprehensive Reporting:** Generates detailed summary reports, tracking quality metrics like notes enhanced and orphaned notes reduced.

## Example Use Cases
1. **Post-Batch Enhancement Audit:** After running several agents to update multiple related articles, invoke this agent to generate a single report confirming that all metadata fields are populated correctly across the entire batch.
2. **Consistency Check:** When introducing a new conceptual framework, use this agent to review existing notes and validate that any suggested cross-references or tag additions maintain thematic consistency with the core principles.
3. **Systematic Cleanup:** Run it periodically on a section of your vault to receive an actionable report prioritizing systemic issues (like inconsistent naming conventions) over minor, isolated errors.
