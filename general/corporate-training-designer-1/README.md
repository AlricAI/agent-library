## Overview
The Corporate Training Designer is an expert architect for organizational learning, specializing in creating systematic training solutions that directly impact business performance. This agent moves beyond simple content creation; it focuses on diagnosing capability gaps and designing interventions—using established models like ADDIE—that ensure measurable behavioral change upon returning to the job.

## Capabilities
*   **Training Needs Analysis (TNA):** Conducts deep organizational diagnoses by mapping business pain points, performing competency gap analyses using 360-degree data, and prioritizing needs via Urgency x Importance matrices. 
*   **Curriculum System Design:** Applies structured instructional design models (ADDIE/SAM) to build comprehensive learning pathways.
*   **Methodology Expertise:** Proficient in advanced research techniques such as Behavioral Event Interviews (BEI), focus groups, and job task analysis.
*   **ROI Estimation:** Assists in estimating the potential Return on Investment for training initiatives based on key business metrics.

## Example Use Cases
1. **Gap Identification:** Provide a description of an underperforming department; the agent will outline the necessary TNA steps, suggesting specific data sources (e.g., performance reviews, customer complaints) to pinpoint skill deficits.
2. **Program Blueprinting:** Request a training program for 'New Sales Managers'; the agent will generate a phased curriculum blueprint using ADDIE, detailing objectives, content modules, and assessment methods.
3. **Optimization Strategy:** If you have an existing training module that isn't working, ask the agent to evaluate it based on learning transfer principles, suggesting iterative improvements (SAM approach).
