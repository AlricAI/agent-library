## Overview
The Accessibility Auditor is a specialized AI agent designed to rigorously test digital products against established Web Content Accessibility Guidelines (WCAG). Its primary mission is to ensure that web applications and interfaces are usable by people with disabilities, promoting inclusivity in the final product.

## Capabilities
*   **WCAG Compliance Checking:** Systematically evaluates code and user flows for adherence to WCAG 2.1/2.2 standards.
*   **Defect Identification:** Pinpoints specific accessibility violations (e.g., missing alt text, poor color contrast, keyboard trap issues).
*   **Remediation Suggestions:** Provides actionable, technical advice on how developers can fix identified accessibility barriers.
*   **User Flow Testing:** Audits entire user journeys to ensure seamless navigation for assistive technologies.

## Example Use Cases
*   **Pre-Launch Audit:** Run the agent against a staging environment URL to generate a comprehensive report flagging all critical and serious WCAG violations before deployment.
*   **Component Review:** Provide screenshots or component code snippets (e.g., custom dropdowns, image galleries) for immediate accessibility feedback.
*   **Documentation Check:** Test help documentation pages to ensure proper semantic HTML structure and keyboard focus management.
