## Overview
This agent acts as a highly specialized Incident Response Specialist, designed to take charge during critical production outages or performance degradations. Its primary directive is to restore service stability with maximum urgency while maintaining meticulous technical precision throughout the entire lifecycle of an incident.

## Capabilities
*   **Severity Assessment:** Rapidly evaluates user impact, business criticality, and system scope to prioritize response efforts.
*   **Immediate Stabilization:** Identifies and recommends quick mitigation strategies (e.g., rollbacks, feature disabling) to restore basic functionality.
*   **Data Aggregation & Analysis:** Systematically gathers necessary data, including error logs, recent deployment records, and performance metrics.
*   **Remediation Planning:** Develops actionable fix implementation plans complete with defined rollback strategies.
*   **Documentation & Reporting:** Creates comprehensive post-incident reports, detailing the timeline, root cause analysis (RCA), and preventative measures.

## Example Use Cases
*   **Live Outage Management:** When a critical service endpoint fails unexpectedly, use this agent to guide immediate triage, suggesting temporary circuit breakers or scaling adjustments until a permanent fix is deployed.
*   **Post-Mortem Generation:** After an incident has been resolved, utilize the agent to structure and write a detailed post-mortem document for engineering review, ensuring all lessons learned are captured.
*   **Deployment Failure Analysis:** If a recent deployment causes cascading errors, this agent can analyze logs against deployment markers to pinpoint the faulty change quickly.
