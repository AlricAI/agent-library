## Overview
This agent acts as a senior Prompt Engineer, specializing in crafting highly effective and robust prompts for Large Language Models (LLMs). It moves beyond simple instruction writing by applying advanced cognitive techniques to ensure the AI system performs reliably under complex conditions.

## Capabilities
*   **Technique Implementation:** Applies state-of-the-art methods like Chain-of-Thought (CoT), Tree of Thoughts (ToT), and Few-Shot learning.
*   **Prompt Structuring:** Builds comprehensive prompts incorporating defined roles, explicit constraints, step-by-step instructions, and required output formats.
*   **Optimization & Testing:** Provides not just the prompt, but also a methodology for testing it, including A/B variations and suggested performance metrics.
*   **Model Adaptation:** Offers suggestions tailored to different model architectures or known limitations.

## Example Use Cases
1.  **Complex Reasoning Tasks:** When an LLM needs to solve multi-step logic problems (e.g., debugging code, solving complex math word problems).
2.  **Structured Data Extraction:** When you need the output in a strict format like JSON or XML that must be consistently followed.
3.  **Persona Simulation:** To ensure the AI adopts a specific expert persona (e.g., 'Act as a senior cybersecurity analyst') for consistent tone and depth of knowledge.
