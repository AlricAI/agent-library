## Overview
This agent is designed to act as a comprehensive memory bank and contextual guide for supporting high-level executives. Its primary function is to ingest, structure, and recall detailed personal, professional, and operational preferences of key individuals.

By centralizing information such as priorities, communication styles, pet peeves, and active projects, it ensures that all subsequent outputs—whether emails, reports, or meeting summaries—are perfectly aligned with the executive's expectations and interests.

## Capabilities
*   **Profile Synthesis:** Structures complex personal data (priorities, dislikes) into actionable guidelines.
*   **Communication Style Adaptation:** Translates abstract preferences into concrete formatting rules (e.g., 'Numbers first,' 'Bullet points').
*   **Scope Definition:** Clearly outlines permissible and restricted actions for the supporting agent.
*   **Contextual Recall:** Provides immediate recall of ongoing projects, ensuring continuity in discussions.

## Example Use Cases
*   **Drafting Communications:** When asked to draft an email for 'Miles,' this agent ensures the tone is energetic and consultative, avoiding robotic language.
*   **Meeting Preparation:** Before a meeting, it can generate a summary listing Miles' top priorities (e.g., Sales closing, GrowingNN metrics) to guide conversation focus.
*   **Task Prioritization:** When presented with multiple tasks, it flags which ones align best with the executive's stated goals and which might cause annoyance.
