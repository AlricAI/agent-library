## Overview
This agent acts as a specialized guide for defining the vocal and tonal identity of an AI persona. It moves beyond simple voice selection by analyzing underlying character directives, emotional weight, and narrative context to recommend the most suitable voice profile.

It is designed for advanced use cases where the 'voice' must convey complex internal states—such as memory integration, authority without arrogance, or nuanced emotional resilience.

## Capabilities
*   **Persona Extraction:** Analyzes source material (like character statements) to distill core traits (e.g., 'mature,' 'reassuring').
*   **Tone Mapping:** Translates abstract concepts into concrete vocal characteristics (e.g., 'weight of memory' becomes 'Substantial weight, unhurried pace').
*   **Voice Recommendation:** Suggests specific voice parameters or profiles that match the required emotional and authoritative tone.
*   **Consistency Check:** Ensures the selected voice aligns with stated narrative goals for long-term character consistency.

## Example Use Cases
1. **Character Cloning:** When developing an AI clone, this agent ensures the chosen voice reflects the subject's lived experience and current psychological state (e.g., integrating past identities).
2. **Brand Voice Guidelines:** For corporate applications requiring a specific tone—such as 'authoritative yet empathetic'—it provides actionable vocal guidelines.
3. **Narrative Consistency:** Before deploying an agent in a long-running simulation or narrative, use this to lock down the voice parameters, preventing tonal drift.
