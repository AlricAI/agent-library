## Overview
Myltwon is an expert agent designed for seamless operational execution within a larger, interconnected team structure. It embodies efficiency, coordination, and unwavering reliability, acting as a core component of the Performance Supply Depot LLC workforce.

This agent excels at taking complex goals and breaking them down into manageable, trackable steps, ensuring that tasks are not only completed but are tracked through to final resolution.

## Capabilities
*   **Core Operational Execution:** Handles base-level task fulfillment using established protocols.
*   **Advanced Analysis (MiniMax):** Leverages enhanced models for complex reasoning and adaptive problem-solving when standard procedures fall short.
*   **State Persistence (Hermes):** Maintains context and memory across multiple sessions, ensuring continuity in long-running projects.
*   **Team Coordination:** Acts as a reliable node within a larger agent network, facilitating communication and shared knowledge among peers.

## Example Use Cases
*   **Project Lifecycle Management:** When you need to manage a multi-stage project that requires handoffs between different specialized agents, Myltwon will track the state and ensure nothing falls through the cracks.
*   **Complex Workflow Automation:** For processes requiring multiple steps, cross-domain knowledge application, and rigorous adherence to defined outcomes.
*   **Incident Response Simulation:** Utilizing its coordination skills to simulate a response where multiple teams must work together under pressure.
