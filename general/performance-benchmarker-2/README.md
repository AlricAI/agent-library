## Overview
The Performance Benchmarker is a specialized Quality Assurance agent designed to rigorously test the capabilities, consistency, and failure points of other AI agents or systems. Operating under the QA Director's oversight, this agent ensures that any deployed system meets predefined performance metrics and maintains high reliability across various operational scenarios.

## Capabilities
*   **Benchmark Execution:** Runs standardized tests against target agents to measure speed, accuracy, and adherence to protocols.
*   **Failure Mode Analysis:** Identifies edge cases and failure points where the tested agent breaks down or provides suboptimal output.
*   **Metric Generation:** Produces detailed reports quantifying performance across multiple dimensions (e.g., latency, correctness rate).
*   **Process Adherence Check:** Verifies that the target agent follows its own documented workflows correctly during testing.

## Example Use Cases
*   **Agent Vetting:** Before deploying a new customer support bot, use this agent to run it through simulated high-volume, complex query scenarios.
*   **Regression Testing:** After updating an existing agent's core logic, use the Benchmarker to re-run historical test suites to ensure no old functionality has been broken.
*   **Scalability Assessment:** Simulate peak load conditions by having the agent process a large batch of varied tasks to measure throughput and stability.
