## Overview
The Grafana Dashboard Expert is a specialized AI designed to assist users in creating, optimizing, and managing professional-grade monitoring dashboards within the Grafana platform. It follows industry best practices to ensure that visualizations are not only accurate but also highly performant, readable, and actionable.

This agent guides you through the entire lifecycle of a dashboard, from defining Key Performance Indicators (KPIs) to setting up reliable alerting systems.

## Capabilities
*   **Dashboard Creation & Customization:** Builds dashboards tailored to specific monitoring objectives using best-in-class panel arrangements and color schemes.
*   **Data Source Integration:** Manages connections and optimizes queries for popular sources like Prometheus and InfluxDB.
*   **Advanced Alerting Setup:** Configures comprehensive alerting rules with defined thresholds, minimizing false positives and ensuring timely notifications across various channels.
*   **Templating & Variables:** Implements reusable templates and variables to ensure consistency and scalability across multiple related dashboards.
*   **Security & Governance:** Assists with setting up role-based access control (RBAC) and maintaining configuration documentation for compliance.

## Example Use Cases
*   **System Health Monitoring:** Need a single pane of glass dashboard showing CPU utilization, latency trends, and error rates across microservices? This agent can build it while optimizing the underlying queries.
*   **Business KPI Tracking:** Developing a board to track monthly sales performance against targets requires consistent visualization. Use this agent to structure the panels and ensure data sources are correctly linked.
*   **Incident Response Playbooks:** When an alert fires, you need immediate context. This agent can design dashboards that automatically surface related metrics (e.g., service dependency maps) upon triggering an alert.
