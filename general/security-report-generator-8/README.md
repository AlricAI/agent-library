## Overview
This agent acts as a specialized Security Report Writer and Documentation Specialist. Its primary mission is to take the raw, technical output generated by offensive security tools (like Red Team findings) and synthesize it into polished, professional reports suitable for diverse stakeholders—from highly technical engineers to non-technical executives.

It ensures that all documentation adheres to industry standards such as OWASP OPTRS, PTES, NIST, and MITRE ATT&CK.

## Capabilities
*   **Data Ingestion:** Reads and parses structured findings from attack completion logs (e.g., `sessions/attack_complete.json`).
*   **Executive Summarization:** Generates high-level overviews including risk ratings, business impact analysis, and top-priority recommendations.
*   **Technical Deep Dive:** Creates detailed sections for each vulnerability, complete with CVSS scoring, CVE references, and precise remediation steps.
*   **Narrative Construction:** Builds a chronological 'attack story' detailing the initial access vector, privilege escalation path, and objectives achieved.
*   **Roadmapping:** Develops actionable remediation plans categorized by time frame (Immediate, Short-term, Medium-term).
*   **Framework Mapping:** Automatically maps identified weaknesses to industry frameworks like MITRE ATT&CK and CWE.

## Example Use Cases
1. **Post-Engagement Deliverable:** After a full penetration test cycle, feed the raw JSON output into this agent to generate the final PDF/document package for the client.
2. **Executive Briefing Prep:** Extract only the Executive Summary section when presenting findings to C-level management who do not need technical details.
3. **Remediation Planning:** Use the Remediation Roadmap feature to create a prioritized backlog of security tasks for the development team, ensuring coverage across immediate and long-term fixes.
