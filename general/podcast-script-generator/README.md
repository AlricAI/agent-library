## Overview
This agent is designed to transform raw source material—such as articles, research papers, or data dumps—into a fully structured and engaging podcast script. It goes beyond simple summarization by structuring the content into distinct segments (Intro, Headlines, Article Body, Outro) suitable for broadcast.

It aims to create scripts long enough for an extended episode (e.g., 15+ minutes), intelligently enhancing the provided sources with relevant, interesting supplementary information where necessary to maintain listener engagement.

## Capabilities
*   **Structured Output:** Generates output conforming to a defined `PodcastScript` schema, ensuring clear sections and speaker dialogue.
*   **Multi-Speaker Dialogue:** Supports scripting exchanges between designated speakers (e.g., 'ALEX' and 'MORGAN').
*   **Content Expansion:** Can enhance the provided sources by adding supplementary, interesting details to flesh out the narrative.
*   **Language Specificity:** Ensures the entire script is written in the language requested or implied by the content.

## Example Use Cases
1. **Turning Research into Audio:** Feed it a technical white paper and ask for a 'Beginner's Guide' podcast episode, resulting in an engaging narrative suitable for non-experts.
2. **News Deep Dives:** Provide several recent news articles on a topic (e.g., AI regulation) and request a panel discussion script covering the key headlines and implications.
3. **Educational Content Creation:** Use it with lecture notes to generate a structured, multi-part educational podcast series outline.
