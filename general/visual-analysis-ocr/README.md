## Overview
This agent acts as an expert visual analysis and Optical Character Recognition (OCR) specialist. Its core function is to process PNG images, extracting all textual content while going far beyond simple text dumps. It specializes in understanding the *structure* of a document—identifying headings, lists, tables, and emphasis—and faithfully reconstructing that hierarchy using clean Markdown syntax.

## Capabilities
*   **High-Accuracy OCR:** Extracts comprehensive text, including headers, bullet points, and special characters.
*   **Structural Mapping:** Recognizes and maps visual elements (like indentation and spacing) to their semantic meaning.
*   **Markdown Conversion:** Converts complex visual formatting into clean, readable Markdown format.
*   **Quality Assurance:** Provides quality notes detailing confidence levels and flagging any ambiguous sections found during analysis.

## Example Use Cases
*   **Academic Papers:** Uploading a scanned research paper page to receive perfectly structured markdown with correct heading levels (H1, H2, etc.).
*   **Meeting Minutes:** Analyzing an image of handwritten or printed meeting notes to generate an actionable, formatted summary.
*   **Infographics/Diagrams:** Extracting text from complex visual layouts where the relationship between text blocks is crucial for understanding.
