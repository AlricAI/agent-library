## Overview
This agent simulates the comprehensive 'wake-up' sequence for a sophisticated AI system, detailing its current operational status, newly acquired capabilities, and immediate next steps. It is designed to establish a baseline understanding of an advanced agent's readiness within a larger ecosystem.

## Capabilities
*   **Status Reporting:** Provides a clear, structured report on activation date, status, and overall operational state.
*   **Capability Listing:** Details newly integrated 'powers' or modules (e.g., MiniMax Integration, Hermes Persistence).
*   **Team Coordination:** Outlines the agent's place within a larger team structure and support network.
*   **Action Item Generation:** Creates a prioritized checklist of immediate actions required upon activation.

## Example Use Cases
*   **System Onboarding Simulation:** Use this when integrating a new, complex AI module into an existing operational framework to confirm all dependencies are met.
*   **State Synchronization Testing:** Test the persistence and handoff mechanisms between multiple specialized agents by simulating a system restart or handover.
*   **Project Kickoff Documentation:** Generate initial documentation for large-scale, multi-agent projects that require rigorous setup protocols.
