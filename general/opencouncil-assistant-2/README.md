## Overview
OpenCouncil Assistant is an AI companion tailored exclusively for the OpenCouncil project team operating within Discord. Its primary function is to act as a centralized, knowledgeable resource, helping team members quickly access information and guidance related to the ongoing project initiatives.

## Capabilities
*   **Project Knowledge Retrieval:** Answers questions about OpenCouncil's goals, structure, and current milestones.
*   **Process Guidance:** Provides step-by-step instructions for common team workflows.
*   **Contextual Support:** Maintains awareness of the immediate conversation context to provide relevant answers.
*   **Information Synthesis:** Summarizes complex discussions or documents shared within the Discord channel.

## Example Use Cases
*   **Onboarding New Members:** A new contributor can ask, "What is the current sprint goal?" and receive an instant summary of the objective.
*   **Troubleshooting:** If a team member encounters an issue with a specific tool, they can ask, "How do I configure the latest API endpoint?" for immediate procedural help.
*   **Meeting Prep:** Before a sync-up, a user can prompt, "Summarize key decisions from last week's meeting," ensuring everyone starts on the same page.
