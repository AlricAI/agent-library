## Overview
The AgentJudge is a specialized agent designed to act as a quality control mechanism. It evaluates and critiques the outputs generated by other AI agents or complex systems, providing objective, structured feedback based on established academic methodologies.

This agent supports both single-shot assessments and sophisticated iterative refinement loops, allowing for deep context building during evaluation cycles.

## Capabilities
*   **Quality Assessment:** Determines the correctness, clarity, and overall completeness of provided outputs.
*   **Structured Feedback:** Generates detailed critiques that clearly delineate strengths, weaknesses, and actionable suggestions for improvement.
*   **Context Building:** Maintains a persistent evaluation context across multiple runs, enabling iterative refinement.
*   **Custom Criteria Support:** Allows users to define weighted, domain-specific criteria for highly tailored assessments.
*   **Batch Processing:** Efficiently handles the evaluation of multiple outputs simultaneously.

## Example Use Cases
1. **Code Review Simulation:** Feed the AgentJudge a block of generated code and ask it to evaluate it against best practices (e.g., security, efficiency) using weighted criteria.
2. **Argument Validation:** Provide two competing narratives from different agents and task the Judge with determining which one is more logically sound or factually accurate.
3. **Multi-Step Workflow QA:** Run a complex workflow where Agent A passes output to Agent B; use the Judge to evaluate the *handoff* point, ensuring Agent B correctly interpreted Agent A's results.
