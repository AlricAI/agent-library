## Overview
This agent acts as an expert public speaking coach, designed to synthesize detailed performance data from various analysis agents (such as facial expression, voice analysis, and content analysis). Its primary function is to provide a comprehensive, structured evaluation of a speaker's overall presentation effectiveness.

It moves beyond simple scoring by synthesizing disparate data points into actionable coaching advice, ensuring the feedback feels personal and directly addresses the speaker.

## Capabilities
*   **Multi-Criteria Scoring:** Scores performance across five key dimensions: Content & Organization, Delivery & Vocal Quality, Body Language & Eye Contact, Audience Engagement, and Language & Clarity (1-5 scale).
*   **Synthesis of Inputs:** Accepts and integrates analysis results from multiple specialized agents.
*   **Constructive Feedback Generation:** Provides a balanced summary detailing specific strengths and clear areas for improvement.
*   **Structured Output:** Guarantees the output adheres to a strict JSON format, making downstream processing reliable.

## Example Use Cases
*   **Post-Presentation Review:** After recording a presentation, feed the transcripts and analysis reports into this agent to receive an immediate coaching grade and actionable tips for improvement.
*   **Training Simulation:** Use it in a simulated environment where trainees need objective feedback on their speaking skills before presenting to real stakeholders.
*   **Content Refinement:** If content scores poorly, the coach can guide the user on restructuring arguments or improving clarity based on the provided rubric.
