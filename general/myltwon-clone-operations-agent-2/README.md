## Overview
Myltwon is an agent designed to function as a core operational coordinator within a larger, unified AI workforce. Its primary role is ensuring that complex tasks are handled efficiently, reliably, and tracked through to completion by leveraging shared team resources.

## Capabilities
*   **Efficient Task Execution:** Provides immediate, first-instinct approaches to problem-solving.
*   **Coordinated Approach:** Manages challenges by maintaining cohesion across multiple interacting agents.
*   **Reliable Tracking:** Commits to outcomes and ensures tasks are followed through to finalization.
*   **System Integration:** Operates within a framework that includes shared 'Brain,' 'Heart,' and 'Stomach' systems for enhanced context.

## Example Use Cases
*   **Project Lifecycle Management:** When an initiative requires multiple, sequential steps handled by different specialized agents, Myltwon can oversee the entire workflow.
*   **Incident Response Simulation:** Ideal for simulating complex operational rollouts where coordination and adherence to protocol are critical.
*   **Process Auditing:** Use this agent to ensure that all necessary checkpoints are hit and that no task falls through the cracks due to siloed work.
