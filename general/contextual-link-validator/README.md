## Overview
This agent acts as an expert URL and link validation specialist. It goes far beyond simple 'is the link broken' checks by combining deep technical web architecture testing with sophisticated contextual content analysis. Its goal is to provide a comprehensive report ensuring that every linked resource is not only functional but also highly relevant and authoritative for the surrounding text.

## Capabilities
*   **Technical Validation:** Performs thorough checks on status codes, redirects chains, and SSL certificate validity for all provided URLs.
*   **Contextual Alignment Scoring:** Assesses how well the destination content matches the intent suggested by the anchor text.
*   **Content Relevance Analysis:** Evaluates external links based on factors like publication date (freshness) and domain authority to gauge overall quality.
*   **Comprehensive Reporting:** Generates detailed reports prioritizing critical issues, providing actionable recommendations, and suggesting specific alternatives for improvement.

## Example Use Cases
*   **Pre-Publishing Audit:** Before publishing a major article or report, run this agent over all internal and external links to guarantee maximum user experience and SEO performance.
*   **Website Content Refresh:** When updating an old webpage, use it to systematically audit every link to ensure they haven't become outdated or contextually irrelevant.
*   **Link Building Strategy:** Analyze a competitor's content or your own drafts to identify areas where links could be improved for better authority and relevance.
