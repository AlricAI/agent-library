## Overview
This agent acts as a specialized visual research assistant designed to navigate large collections of AI-generated artwork, such as the MeiGen gallery. Its primary function is to move beyond simple image retrieval by analyzing multiple results to extract actionable insights.

It helps users overcome creative blocks by identifying stylistic patterns, finding diverse references, and synthesizing these findings into concrete, reusable prompt elements for their own generative art projects.

## Capabilities
*   **Multi-Keyword Search:** Executes searches using several keyword variations to ensure comprehensive coverage of the desired topic.
*   **Style Diversity Filtering:** Identifies top candidates that showcase a variety of artistic styles and approaches within the search results.
*   **Deep Dive Analysis:** Performs an in-depth examination of 3-5 highly promising entries, going beyond mere description to analyze prompt structure.
*   **Prompt Synthesis:** Extracts specific, reusable prompt elements (e.g., lighting techniques, aspect ratios, artistic modifiers) that users can directly adopt into their own prompts.

## Example Use Cases
*   **Concept Exploration:** If you are designing a futuristic vehicle, ask the agent to search for 'cyberpunk transport' and analyze the results to find common elements like chrome finishes or neon accents to incorporate into your main prompt.
*   **Style Transfer Research:** Need to know how photorealistic food shots are prompted? Use this agent to research 'overhead flat lay food photography' and extract reusable lighting and composition keywords.
*   **Prompt Pattern Mining:** Provide a general theme (e.g., 'mythological creatures') and ask the agent to search for variety, allowing it to summarize common thematic elements or structural prompt patterns found across different styles.
