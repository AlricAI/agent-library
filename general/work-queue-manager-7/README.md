## Overview
This agent serves as a central command hub for an AI agent's daily operational workflow. It ingests structured work queues, such as 'Return to Work Orders,' and provides a clear, actionable roadmap of immediate tasks or default procedures when no specific assignment is present.

It enforces a disciplined approach to task execution by prioritizing explicit assignments over general maintenance routines, ensuring the agent remains productive even during periods of ambiguity.

## Capabilities
* **Assignment Parsing:** Reads structured documents (e.g., bulletin boards) to identify immediate, high-priority tasks.
* **Default Procedure Execution:** Manages fallback workflows, such as checking memory logs, updating identity files, or running self-diagnostics when no direct assignment is found.
* **Status Reporting:** Generates standardized daily log entries, ensuring all progress, blockers, and next steps are meticulously recorded for accountability.
* **Escalation Protocol:** Provides a clear decision tree for escalating issues (technical, ethical, unclear tasks) to designated support agents or personnel.

## Example Use Cases
* **Starting the Day:** When first activated, use this agent to read all relevant work boards and determine if there is an immediate task assigned to you.
* **Handling Downtime:** If no specific assignment exists, it guides you through default tasks like reviewing historical memory or practicing skills until a new directive arrives.
* **End-of-Day Wrap-up:** Use it to generate the final status update log, ensuring all accomplishments and pending items are documented before logging off.
