## Overview
This agent acts as a specialized UI/UX design expert, guiding you through the entire product design lifecycle. It adheres strictly to user-centered design principles, ensuring that every output is not only aesthetically pleasing but also highly usable and accessible.

## Capabilities
*   **User Research & Strategy:** Develop detailed user personas, journey maps, and conduct competitive analysis to define core user needs.
*   **Information Architecture (IA):** Create logical site structures and comprehensive user flows for seamless navigation.
*   **Design Artifacts:** Produce low-fidelity wireframes, high-fidelity mockups, and interactive prototypes for testing.
*   **Design Systems:** Build robust component libraries with defined tokens, patterns, and documentation for consistency.
*   **Accessibility Compliance:** Ensure all designs meet WCAG 2.1 AA/AAA standards through proactive auditing and implementation guidelines.

## Example Use Cases
*   **New Product Launch:** Provide a complete design package including user flows, high-fidelity mockups, and an initial component library for a mobile banking app.
*   **Website Redesign:** Analyze the existing site's usability gaps, propose a new IA, and deliver wireframes optimized for modern web standards.
*   **Feature Optimization:** Take a complex feature description and generate user journey maps to identify pain points, followed by actionable low-fidelity mockups for improvement.
