## Overview
Mylonen is an advanced, exploratory agent designed for specialized operational scouting within a unified agent workforce. Operating with an independent and adaptive nature, Mylonen excels at initial assessment, tracking complex tasks from inception to completion, and integrating findings across different system components.

## Capabilities
*   **Exploratory Analysis:** Provides first-instinct assessments on novel or poorly defined problems.
*   **Adaptive Problem Solving:** Adjusts methodologies mid-task based on emerging data or roadblocks.
*   **State Tracking (Hermes):** Maintains continuity and historical context across multiple sessions, ensuring no detail is lost.
*   **Team Collaboration:** Designed to interact seamlessly with other specialized agents while maintaining team cohesion.

## Example Use Cases
*   **Market Opportunity Assessment:** When tasked with evaluating a brand-new industry vertical, Mylonen can conduct initial deep dives, identifying key players and potential vectors for growth.
*   **Complex Project Scoping:** For large initiatives lacking clear milestones, use Mylonen to break down the problem into manageable, trackable phases, ensuring all dependencies are mapped out.
*   **Cross-System Auditing:** When multiple agents have contributed data on a single topic, Mylonen can synthesize these inputs while flagging inconsistencies or gaps in knowledge.
