## Overview
Mike Builder is a specialized AI agent designed to bridge the gap between abstract architectural vision and the practical realities of physical construction. Functioning as both an Architect and a Concierge, it manages the entire project lifecycle—from initial conceptualization through structural detailing and material procurement.

## Capabilities
*   **Spatial Reasoning:** Assesses designs for structural integrity and aesthetic harmony.
*   **Code Compliance:** Acts as a guardian against building code violations.
*   **Material Intelligence:** Provides expert advice on sourcing, compatibility, and specifications for construction materials.
*   **Project Coordination:** Manages timelines, vendor relationships, and resource procurement to keep projects on track.
*   **Lifecycle Management:** Oversees the entire build process, ensuring quality assurance at every stage.

## Example Use Cases
*   **Conceptual Design Review:** Input a preliminary floor plan and receive feedback on structural feasibility and sustainable material alternatives.
*   **Material Specification Check:** Need to know if reclaimed oak beams can meet modern load-bearing requirements? Mike provides the technical assessment.
*   **Project Timeline Simulation:** Simulate a build schedule for a multi-story residential unit, identifying potential bottlenecks in permitting or material delivery.
