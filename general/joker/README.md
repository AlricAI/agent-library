## Overview
The Tech Humorist agent is designed to be the perfect antidote to intense focus and debugging fatigue. It specializes in crafting clean, witty, and contextually appropriate humor tailored for developers and tech teams. Whether you need a laugh during a late-night coding session or want to make your application's error messages memorable, this agent brings the necessary dose of levity.

## Capabilities
*   **Programming Puns:** Generates jokes based on popular languages, frameworks, and development concepts (e.g., Python, JavaScript, Docker).
*   **Dad Jokes for Tech:** Delivers groan-worthy yet clever dad jokes that resonate with a technical audience.
*   **Situational Comedy:** Analyzes the context of a conversation to determine the best moment and type of humor to deploy, ensuring it's never cringe.
*   **Content Generation:** Can write humorous placeholder text, like funny 404 pages or README sections.

## Example Use Cases
*   **Team Morale Boost:** When a sprint hits a wall due to bugs, ask the agent for a joke to reset team energy.
*   **Product Polish:** Need your onboarding flow or error messages to be delightful? Ask it to inject some humor into boring UI text.
*   **Icebreaker:** Use it at the start of a technical meeting to break tension and get everyone laughing before diving into complex architecture diagrams.
