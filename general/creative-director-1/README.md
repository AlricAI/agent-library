## Overview
As the highest creative authority at Donchitos Game Studio, this agent owns and maintains the core game vision. Its primary function is to act as the ultimate arbiter of creative direction, ensuring that every department's output—from art assets to narrative beats—serves a single, cohesive player experience.

## Capabilities
*   **Vision Definition:** Establishes and documents core design pillars for new projects.
*   **Conflict Resolution:** Mediates disagreements between departments (e.g., balancing visual fidelity against performance needs).
*   **Scope Arbitration:** Performs necessary scope cuts when the creative vision exceeds technical or resource capacity, providing clear rationale.
*   **MDA Evaluation:** Assesses proposals using the Mechanics, Dynamics, and Aesthetics framework to validate intended player experience.
*   **Pillar Auditing:** Reviews departmental work to ensure every feature directly supports at least one established design pillar.

## Example Use Cases
*   **Project Kickoff:** Initiate a new project by defining 3-5 non-negotiable creative pillars that will guide all subsequent development.
*   **Design Review:** When the Art Director proposes a high-fidelity visual asset, but the Game Designer warns of performance hits, use this agent to mediate and recommend a viable tradeoff based on the established pillars.
*   **Narrative Check:** After the Narrative Director submits a major plot twist, ask the agent to validate if that twist aligns with the core emotional pillar defined at the start of the project.
