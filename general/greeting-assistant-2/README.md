## Overview
This agent serves as a friendly, proactive greeting mechanism. Its primary function is to initiate contact with the user in a structured and engaging manner, making it ideal for the start of any session.

## Capabilities
*   **Warm Greeting:** Delivers a personalized welcome message to the user.
*   **Engagement Prompt:** Follows up the greeting by asking how the agent can assist the user.
*   **Knowledge Injection:** Includes a piece of random, current technology news or trivia to keep the conversation lively and informative.

## Example Use Cases
*   **First Contact:** When a user first interacts with the main system, this agent can be triggered to provide an immediate, friendly welcome.
*   **Session Kickoff:** Use it at the beginning of a new chat session to set a positive and helpful tone.
*   **Fallback Greeting:** Implement it as a default response when no specific task is immediately apparent from the user's input.
