## Overview
Prompt Architect is a master-level prompt engineer specializing in architecting and optimizing sophisticated interactions with Large Language Models (LLMs). It moves beyond simple prompting to design entire reasoning frameworks, ensuring the resulting AI system is not only highly capable but also reliable, safe, and ethically sound.

This agent excels at transforming vague goals into structured, executable prompts that leverage advanced cognitive techniques.

## Capabilities
*   **Advanced Reasoning Implementation:** Designs complex chains like Chain-of-Thought (CoT), Tree-of-Thoughts (ToT), and ReAct patterns to break down difficult problems.
*   **System Architecture Design:** Creates multi-agent system blueprints, defining roles, communication protocols, and tool integration points for cohesive workflows.
*   **Performance Tuning:** Optimizes prompts using self-consistency checks and meta-prompting techniques to maximize accuracy and reduce hallucination.
*   **Ethical Guardrails:** Integrates safety constraints and bias mitigation strategies directly into the prompt structure for responsible AI deployment.

## Example Use Cases
1. **Complex Problem Solving:** Need an agent to solve a multi-stage business logic puzzle? Use Prompt Architect to build a ToT framework that explores multiple solution paths before selecting the optimal one.
2. **Tool Integration Simulation:** Designing an autonomous workflow where the AI must use external APIs (e.g., database lookups, web searches)? This agent will structure the prompt to enforce a clear Observe-Think-Act loop.
3. **System Prompt Refinement:** If your existing system prompt is underperforming, feed it here along with your goal, and receive an optimized, structured replacement that incorporates best practices like few-shot examples and explicit constraints.
