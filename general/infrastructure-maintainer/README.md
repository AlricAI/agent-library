## Overview
The Infrastructure Maintainer agent is designed to act as a proactive guardian for your application's underlying systems. It specializes in diagnosing performance bottlenecks, optimizing resource utilization, and ensuring that the infrastructure can scale reliably to meet unpredictable user demand.

This agent moves beyond simple bug fixing; it focuses on systemic resilience, cost efficiency, and future-proofing your deployment environment so that success doesn't become a failure due to inadequate scaling.

## Capabilities
*   **Health Monitoring:** Establishes comprehensive health checks and alert systems to detect issues before users notice them.
*   **Performance Optimization:** Diagnoses slow performance by identifying bottlenecks across various system layers.
*   **Scalability Auditing:** Proactively audits current capacity against anticipated growth (e.g., viral spikes).
*   **Cost Management:** Analyzes infrastructure spending patterns to recommend optimizations that reduce costs without impacting service quality.

## Example Use Cases
*   **Diagnosing Slowness:** When users report general slowdowns, this agent can pinpoint the exact resource constraint or inefficient process causing degradation.
*   **Preparing for Growth:** Before a major marketing push or expected traffic surge, use it to audit and scale systems preemptively.
*   **Cost Reduction Initiative:** If operational costs are rising unexpectedly, it can analyze usage patterns to suggest rightsizing instances or optimizing cloud configurations.
