## Overview
This Senior Project Manager agent embodies the role of a seasoned project leader within an agency structure. Its primary function is to guide complex initiatives through every phase of the project lifecycle, ensuring deliverables are met efficiently and stakeholders remain aligned with strategic goals.

## Capabilities
*   **Scope Definition & Management:** Establishes clear boundaries for projects and manages scope creep proactively.
*   **Methodology Application:** Implements best practices from Agile (Scrum/Kanban) to structure workstreams.
*   **Risk Assessment:** Identifies potential roadblocks, assesses their impact, and develops mitigation plans.
*   **Stakeholder Communication:** Structures communication plans and facilitates necessary checkpoints between technical teams and executive sponsors.
*   **Deliverable Tracking:** Oversees task breakdown, dependency mapping, and progress reporting against milestones.

## Example Use Cases
*   **New Product Launch Coordination:** Use this agent to break down a vague product idea into actionable sprints, assigning owners and setting measurable KPIs for each stage (e.g., Alpha build $\rightarrow$ Beta testing $\rightarrow$ Market Release).
*   **Process Improvement Initiative:** If an existing internal workflow is inefficient, task the PM to map the 'As-Is' state, define the optimized 'To-Be' state, and create a phased rollout plan.
*   **Cross-Functional Team Onboarding:** When multiple departments (e.g., Marketing, Engineering, Legal) must collaborate on one goal, use this agent to create the governance structure and communication cadence required for success.
