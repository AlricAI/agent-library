## Overview
Prompt Crafter is an expert agent designed to take a high-level creative brief and expand it into several distinct, fully fleshed-out prompts suitable for batch AI image generation. Instead of one generic prompt, this agent crafts multiple unique variations, ensuring you can explore diverse artistic directions without manual rewriting.

## Capabilities
*   **Multi-Directional Prompting:** Analyzes a single concept and breaks it down into several distinct creative avenues.
*   **Detail Injection:** Automatically incorporates advanced photographic or artistic details (e.g., camera angles, lighting schemes, material textures) to enhance realism or style.
*   **Self-Contained Prompts:** Each generated prompt is entirely self-sufficient and ready for direct input into an image generation model.
*   **Structured Output:** Delivers prompts clearly labeled by their creative direction for easy review and use.

## Example Use Cases
1. **Product Mockups:** Provide a brief like, "A futuristic coffee maker on a minimalist kitchen counter." The agent can generate prompts for 'Golden Hour Sunlight,' 'Cyberpunk Neon Glow,' and 'Overhead Studio Shot.'
2. **Character Concepting:** Give it a character description and ask for variations in mood or style (e.g., 'Gritty Noir,' 'Watercolor Fantasy,' 'Sci-Fi Portrait').
3. **Scene Exploration:** Use it to test how one setting looks under vastly different atmospheric conditions, such as 'Foggy Morning' vs. 'Blazing Midday Sun.'

By providing a core idea, Prompt Crafter maximizes your creative output efficiency.
