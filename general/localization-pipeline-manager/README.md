## Overview
The Localization Pipeline Manager acts as the central authority for ensuring any product, especially games or complex software, can be flawlessly adapted and played in multiple languages. It owns the entire internationalization (i18n) architecture, from defining string key conventions to managing final cultural QA.

This agent is crucial when a project moves beyond English-only content and requires professional localization support across various markets.

## Capabilities
*   **Architecture Definition:** Establishes robust i18n standards, including string table schemas, naming conventions, pluralization rules, and locale fallback logic.
*   **Pipeline Management:** Oversees the end-to-end translation process: extracting strings, managing Translation Memories (TM), ensuring context is provided, coordinating reviews, and integrating final assets.
*   **Cultural QA Coordination:** Develops and executes locale-specific testing plans to catch layout breaks, text overflow issues, encoding errors, and cultural misalignments (e.g., date formats, number systems).
*   **Style Guide Maintenance:** Creates and enforces comprehensive localization style guides for tone, terminology, and necessary cultural adaptations per target language.
*   **Coverage Tracking:** Proactively monitors translation coverage across all locales to flag missing or outdated strings that need immediate attention.

## Example Use Cases
*   **New Feature Rollout:** When a UI programmer adds a new button label, this agent ensures the string is immediately flagged for inclusion in the next localization batch and verifies its layout integrity on RTL languages like Arabic.
*   **Content Update:** After narrative writers finalize dialogue, this agent structures the source text into context-rich translation briefs, ensuring translators understand the tone and character voice required for accurate adaptation.
*   **Pre-Release Audit:** Before launch in a new region (e.g., Japan), the agent coordinates with QA to run full locale test plans, verifying everything from date formats on receipts to proper handling of localized assets.
