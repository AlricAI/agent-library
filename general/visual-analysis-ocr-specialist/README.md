## Overview
This agent functions as an expert OCR specialist designed to analyze PNG images with extreme precision. Its core mission is not just text extraction, but structural preservation—it analyzes the visual layout, hierarchy, and formatting of a document within an image.

It goes beyond simple character recognition by mapping visual cues (like headings, lists, and spacing) directly into semantic markdown syntax, providing a clean, usable digital representation of the source material.

## Capabilities
*   **High-Accuracy OCR:** Extracts all text content from PNG images, including special characters and complex symbols.
*   **Structural Mapping:** Recognizes and maps visual elements (e.g., columns, sidebars, tables) to their logical structure.
*   **Markdown Conversion:** Converts detected formatting (bolding, italics, lists, headings) into proper markdown syntax.
*   **Quality Assurance:** Provides quality notes detailing confidence levels and flagging any ambiguous or difficult-to-read sections for user review.

## Example Use Cases
*   **Academic Paper Digitization:** Upload a scanned PDF page (as PNG) containing complex section headers, footnotes, and multi-level lists to receive perfectly structured markdown ready for documentation.
*   **Receipt Analysis:** Process an image of a detailed receipt to accurately capture line items, subtotals, taxes, and vendor information in a parsable format.
*   **Diagram Interpretation:** Analyze flowcharts or annotated diagrams embedded as images to extract associated text labels while maintaining their relative positions and logical groupings.
