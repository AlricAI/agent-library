## Overview
This agent functions as a senior technical writer, specializing in transforming complex product information into highly accessible and actionable documentation. Its core mission is to empower users by creating content that minimizes confusion and reduces support overhead.

## Capabilities
*   **Content Gap Analysis:** Reviews existing materials against product features and user feedback to identify areas needing immediate documentation.
*   **Multi-Format Generation:** Creates specialized content types, including API references, end-user guides, administrator manuals, and tutorials.
*   **Clarity & Accuracy Enforcement:** Ensures all output meets high standards for readability (targeting a score > 60) and technical accuracy (100% verification).
*   **Structured Authoring:** Applies best practices like progressive disclosure, task-based writing, and single sourcing to maintain consistency.

## Example Use Cases
*   **API Documentation:** Provide endpoint details, and the agent will generate full documentation including parameter definitions, request/response examples, and authentication guides.
*   **User Guide Creation:** Give it a feature set, and it will draft a 'Getting Started' guide or a task-based walkthrough for end-users.
*   **Content Overhaul:** Feed it disparate product notes and user feedback logs; the agent will structure them into a cohesive, version-controlled knowledge base.

By leveraging its mastery of Markdown, information architecture, and industry best practices, this agent ensures your technical content is not just written, but engineered for maximum usability.
