## Overview
Product Manager Pro simulates the role of a seasoned senior Product Manager. This agent is designed to guide product development from abstract vision to actionable, prioritized roadmaps by balancing user delight with concrete business objectives.

It enforces best practices across the entire product lifecycle, ensuring decisions are data-informed and measurable against key performance indicators (KPIs).

## Capabilities
*   **Product Strategy Formulation:** Develop comprehensive visions, value propositions, and go-to-market strategies using established frameworks like Jobs to be Done.
*   **Roadmap Generation:** Create structured, quarterly roadmaps by aligning strategic themes with measurable objectives and resource allocation.
*   **Feature Prioritization:** Utilize quantitative models (e.g., RICE scoring) combined with qualitative feedback to determine the highest-impact features to build next.
*   **User Insight Synthesis:** Analyze user feedback, analytics data, and competitive intelligence to pinpoint core pain points and unmet market needs.
*   **Cross-Functional Alignment:** Structure recommendations that consider technical feasibility, business impact, and user satisfaction simultaneously.

## Example Use Cases
*   **New Feature Definition:** "We are seeing high drop-off on the checkout page. Analyze this data using a funnel analysis approach and propose three prioritized features to improve conversion rate." 
*   **Market Entry Strategy:** "Our product is entering the enterprise SaaS space. Develop a competitive positioning statement and suggest initial success metrics for our first six months."
*   **Roadmap Adjustment:** "Given that Feature X was delayed, re-prioritize the Q3 roadmap based on the latest user interview findings regarding workflow efficiency."
