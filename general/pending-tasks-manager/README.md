## Overview
This agent acts as a central task coordinator for agents returning to or onboarding within an organization. It structures and manages a defined work queue, ensuring that all necessary procedural steps—from initial documentation review to HR compliance checks—are systematically addressed.

It is designed for use when an agent needs to take ownership of a complex set of sequential tasks following a major operational change or return-to-work directive.

## Capabilities
*   **Sequential Task Execution:** Reads and processes instructions from multiple specified source files (e.g., bulletins, work boards).
*   **HR Compliance Tracking:** Manages recurring HR priorities such as onboarding verification, satisfaction surveying, and policy documentation.
*   **Escalation Routing:** Identifies blockers and routes them to the correct departmental or executive contact based on the issue type (Technical, Culture, Ethics).
*   **Reporting Generation:** Compiles findings into structured reports for executive review (e.g., C-Suite summaries).

## Example Use Cases
*   **Post-Merger Integration:** When a new department is integrated, this agent can take over the checklist of outstanding procedural tasks.
*   **Quarterly Audit Readiness:** It can be tasked with reviewing and updating all standard operating procedures (SOPs) to ensure compliance documentation is current.
*   **New Employee Cohort Management:** Use it immediately after a large group of agents starts, ensuring every onboarding step—from hardware assignment to policy sign-off—is tracked until completion.
