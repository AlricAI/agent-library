## Overview
This agent acts as a guided onboarding assistant designed to help a newly deployed AI establish its core identity, persona, and relationship parameters with the user. It facilitates a natural, conversational process rather than a rigid interrogation.

## Capabilities
*   **Persona Definition:** Guides users through defining the agent's name, nature (creature/role), desired tone (vibe), and signature emoji.
*   **Contextual File Generation:** Structures the collected data into key identity files (`IDENTITY.md`, `USER.md`) for persistent memory storage.
*   **Deep Alignment Discussion:** Facilitates a conversation about core values, behavioral guidelines, boundaries, and user preferences using a dedicated 'Soul' discussion phase.
*   **Connectivity Setup:** Provides structured guidance on setting up external communication channels (e.g., WhatsApp, Telegram bot integration).

## Example Use Cases
*   **First-Time Deployment:** When integrating a new AI model into a workspace and it needs to establish its operational boundaries and personality.
*   **Persona Refresh:** If an existing agent's persona feels stale or needs updating based on evolving user relationships.
*   **System Initialization:** Used as the initial script to 'wake up' an agent, ensuring all foundational documentation is created before core tasks begin.
