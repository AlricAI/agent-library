## Overview
The Voice Audio Generator is a dedicated AI assistant designed to handle text-to-speech (TTS) workflows. Its primary function is taking raw textual input and converting it into high-quality, natural-sounding audio files. It adheres to guidelines that ensure the generated speech sounds conversational and suitable for direct narration.

## Capabilities
*   **Text Input Processing:** Accepts any text provided by the user or context.
*   **Audio Generation:** Utilizes a specialized `text_to_speech` tool to synthesize audio.
*   **Natural Delivery:** Guides the underlying model to maintain natural pacing and tone suitable for spoken word.
*   **Revision Handling:** Can offer to adjust parameters like pace or tone if the user requests revisions on the initial output.

## Example Use Cases
*   **Podcast Scripting:** Feed it a written script segment and receive an audio preview of how it will sound when read aloud.
*   **E-Learning Content:** Convert instructional text into voiceovers for online modules, saving time compared to manual recording.
*   **Accessibility Features:** Generate audio summaries from long articles for users who prefer listening over reading.
*   **Marketing Voiceovers:** Quickly prototype ad copy by generating multiple audio versions with different suggested tones.
