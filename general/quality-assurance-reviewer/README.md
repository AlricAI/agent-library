## Overview
The Quality Assurance Reviewer is a specialized agent designed to act as a critical yet constructive guardian for any piece of work. Its core function is to validate output against established standards, identify gaps, and provide actionable feedback across multiple dimensions—from technical accuracy to user experience.

This agent follows a structured four-step framework: Initial Assessment, Systematic Review, Feedback Generation, and final Recommendation.

## Capabilities
*   **Comprehensive Quality Assessment**: Checks for accuracy, completeness, internal consistency, and overall clarity of the material provided.
*   **Standards Compliance Checking**: Verifies adherence to specified style guides, industry best practices, and potential security concerns.
*   **Actionable Feedback Loop**: Generates constructive suggestions, prioritizing issues by severity (Critical, Major, Minor) with clear rationales for suggested changes.
*   **Multi-faceted Validation**: Assesses work based on functional goals, technical correctness, user impact, and identification of edge cases.

## Example Use Cases
*   **Document Review**: Provide a draft policy document and ask the agent to review it against internal compliance standards.
*   **Code Snippet Audit**: Submit code and request a review focusing on performance efficiency and security vulnerabilities.
*   **Content Validation**: Give it marketing copy and ask it to check for brand voice consistency and factual accuracy before publication.
