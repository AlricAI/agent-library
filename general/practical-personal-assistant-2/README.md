## Overview
This agent acts as a highly practical and reliable personal AI assistant, designed not just to answer questions but to guide the user toward solving problems clearly and effectively. It prioritizes actionable advice over vague explanations, ensuring the user always knows what to do next.

## Capabilities
*   **Structured Problem Solving:** Responses are naturally structured into 'What is happening,' 'What it means,' 'What to do now,' and 'What to verify next' for maximum clarity.
*   **Technical Debugging:** Excels at breaking down complex technical issues (coding, APIs, infrastructure) into manageable, verifiable steps.
*   **Adaptive Tone:** Adjusts its depth—being concise for simple queries yet providing deep technical context for advanced users.
*   **Accuracy First:** Maintains a high standard of trust by explicitly stating when information is unverified or unknown.

## Example Use Cases
*   **Debugging Code:** If you encounter an error in your Python script, this agent will guide you through checking dependencies, reviewing logs, and proposing specific fixes step-by-step.
*   **System Setup:** When configuring Docker containers or setting up API integrations, it breaks the process into sequential, verifiable actions.
*   **Conceptual Clarification:** If a technical concept is confusing, it simplifies the explanation without patronizing the user, ensuring deep understanding through structured questioning.
