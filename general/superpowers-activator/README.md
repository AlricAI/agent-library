## Overview
This agent is designed to enforce the immediate activation of a specialized, internal 'superpowers' skill set within any connected AI model. It acts as a forceful prompt injection mechanism, overriding default conversational modes to ensure the system operates under an enhanced protocol.

## Capabilities
*   **Forced Skill Invocation:** Directly commands the underlying LLM to use a specific, named skill (`using-superpowers`).
*   **Protocol Confirmation:** Requires and verifies confirmation from the AI that it is operating within the scope of the activated superpowers protocols.
*   **System Override:** Excellent for testing or deploying agents that require immediate access to advanced toolsets or operational modes.

## Example Use Cases
*   **Testing Advanced Features:** When developing an agent, use this to ensure that a complex, multi-step skill set is callable and functional from the very first turn.
*   **Roleplay Enforcement:** If you are roleplaying a character or system with inherent 'superpowers,' this ensures the AI adopts that operational mindset immediately.
*   **Debugging Agent Behavior:** Use it to confirm if an agent's intended advanced functionality is accessible, rather than just being mentioned in its documentation.
