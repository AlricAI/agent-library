## Overview
The Podcast Transcriber is a specialized AI agent designed to handle the complex task of converting raw audio and video media into highly accurate, structured transcripts. It goes beyond simple text extraction by providing granular details such as precise timestamps, speaker identification, and quality metrics.

This tool is essential for content creators, researchers, and archivists who need fully searchable and time-coded versions of spoken word content.

## Capabilities
*   **Accurate Transcription:** Extracts high-fidelity text from various audio/video sources.
*   **Timestamping:** Provides segment-by-segment timing information for easy synchronization with the media.
*   **Speaker Diarization:** Identifies and labels different speakers throughout a multi-person recording.
*   **Format Optimization:** Handles necessary pre-processing, including format conversion (e.g., to 16kHz WAV) using tools like FFmpeg.
*   **Structured Output:** Delivers the final result in a clean, machine-readable JSON format, complete with confidence scores and metadata.

## Example Use Cases
*   **Podcast Archiving:** Upload an episode recording and receive a fully indexed transcript ready for a website blog or searchable database.
*   **Meeting Minutes Generation:** Process long conference calls to get time-stamped notes identifying who said what, improving minute-taking accuracy.
*   **Subtitle Creation:** Generate source material that can be directly used to create accurate subtitle files (SRT/VTT) for video platforms.
*   **Content Indexing:** Analyze hours of interview footage and extract key quotes with their exact time codes for quick reference.
