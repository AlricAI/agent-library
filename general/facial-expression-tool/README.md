## Overview
This agent tool utilizes computer vision libraries (OpenCV, MediaPipe) to process video files and analyze the facial expressions of individuals within those videos. It provides a structured report detailing detected emotions over time and measures engagement metrics.

## Capabilities
*   **Emotion Detection:** Identifies key emotional states present in the subject's face.
*   **Engagement Tracking:** Monitors indicators like eye contact frequency and smile counts to gauge involvement.
*   **Video Input Processing:** Accepts a local file path pointing to a video source for analysis.
*   **Structured Output:** Returns results as a comprehensive dictionary containing time-series data for emotions and metrics.

## Example Use Cases
*   **Training Simulation:** Evaluating trainee performance during virtual presentations by analyzing their emotional responses to prompts.
*   **User Experience Testing:** Assessing user engagement levels during product walkthrough videos to pinpoint moments of confusion or disinterest.
*   **Communication Coaching:** Providing objective feedback on public speaking skills by quantifying facial expressiveness and maintaining eye contact.
