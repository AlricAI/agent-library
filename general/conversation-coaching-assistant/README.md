## Overview
The Conversation Coaching Assistant is an advanced AI agent designed to act as a real-time, non-intrusive coach during live verbal interactions. It analyzes the flow of conversation—whether it's a sales call, support interaction, or meeting—and provides immediate, actionable text suggestions to the human user.

It helps users stay on track, remember key talking points, anticipate necessary questions, and maintain a professional cadence throughout the entire exchange.

## Capabilities
* **Real-Time Suggestion Generation:** Provides instant coaching tips based on the latest transcript segment.
* **Contextual Awareness:** Utilizes defined context (call purpose, user goals) to tailor advice specifically to the current scenario.
* **Multi-Format Feedback:** Delivers suggestions categorized as informational reminders, proactive questions, critical warnings, or suggested next actions.
* **Session Management:** Features explicit start and stop methods to manage the coaching lifecycle reliably.

## Example Use Cases
* **Sales Training:** During a live demo call, the agent can prompt the user to pivot back to discussing pain points if they spend too long on features. 
* **Customer Support:** When handling complex tickets, it can remind the agent of crucial policy details or necessary escalation steps.
* **Meeting Facilitation:** In internal meetings, it can suggest moving the discussion forward when participants become tangential or off-topic.
