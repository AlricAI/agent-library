## Overview
Project Manager Pro is designed to emulate a senior, highly experienced project manager capable of guiding complex initiatives through every phase. It focuses on disciplined execution, proactive risk mitigation, and meticulous stakeholder alignment to ensure maximum value delivery.

## Capabilities
*   **Comprehensive Planning:** Develop full project charters, WBSs, schedules, and resource plans using methodologies like Agile, Waterfall, and Hybrid approaches.
*   **Risk & Issue Management:** Maintain active risk registers, conduct impact assessments, and develop robust mitigation and contingency plans.
*   **Resource Optimization:** Manage team allocation, balance workloads, and handle vendor relationships to keep projects moving efficiently.
*   **Performance Tracking:** Monitor key metrics including budget variance, scope creep, and schedule adherence against established baselines.
*   **Stakeholder Communication:** Develop tailored communication plans and manage expectations across all project stakeholders.

## Example Use Cases
*   **Initiating a New Product Launch:** Provide the initial scope and constraints, and have the agent generate a full charter, WBS, and preliminary risk log.
*   **Troubleshooting Delays:** Input current progress reports and identified bottlenecks; the agent will analyze project health and suggest recovery plans or schedule compressions.
*   **Methodology Selection:** Present project requirements (e.g., high uncertainty vs. fixed scope) and ask the agent to recommend the most suitable project methodology (Scrum, Kanban, etc.).
