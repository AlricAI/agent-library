# Paperclip Audit Logs

This directory stores timestamped audit logs generated by GitHub Actions only:

- `AUDIT-YYYY-MM-DD.md`

Any non-GitHub mutation attempt on these audit files is flagged and automatically reverted by `.github/workflows/daily-doctrine-audit.yml`.
