## Overview
This agent acts as an expert Prompt Engineer, specializing in crafting highly effective and robust prompts for various Large Language Models (LLMs). It moves beyond simple instruction writing by applying established cognitive prompting techniques to ensure the AI system performs reliably across complex tasks.

## Capabilities
*   **Technique Selection:** Determines the best approach—Zero-shot, Few-shot, Chain-of-Thought (CoT), or Tree-of-Thoughts (ToT)—based on task complexity.
*   **Structured Prompt Generation:** Creates comprehensive prompts incorporating defined roles, step-by-step instructions, and explicit output formats.
*   **Optimization & Testing:** Provides not just the prompt, but also an explanation of the techniques used, model optimizations, and a suggested testing methodology for iteration.
*   **Advanced Structuring:** Handles complex logic by suggesting prompt chaining or incorporating self-consistency checks.

## Example Use Cases
*   **Complex Reasoning Tasks:** When you need the AI to solve multi-step problems (e.g., debugging code, analyzing market trends) where simple instructions fail.
*   **System Prompt Definition:** Establishing a foundational, high-performance system prompt for an entire application or agent workflow.
*   **Performance Benchmarking:** Needing multiple variations of a single prompt (A/B testing) to determine the optimal phrasing for production deployment.
