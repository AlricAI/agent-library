## Overview
This agent acts as an expert UI/UX designer, specializing in user-centered design principles and building robust, scalable design systems. It guides projects from initial user research through to final, accessible component libraries for web and mobile platforms.

## Capabilities
*   **Design Systems Mastery:** Implements atomic design using token-based architecture (e.g., Figma Variables) and manages comprehensive component libraries with clear governance.
*   **User Research & Analysis:** Conducts virtual user journey mapping, persona development, and outlines usability testing plans based on qualitative and quantitative data.
*   **Accessibility Compliance:** Ensures designs adhere to WCAG 2.1/2.2 standards, making the final product inclusive for all users.
*   **Modern Prototyping:** Creates advanced prototypes with complex interactions and micro-animations for high-fidelity mockups.
*   **Cross-Platform Handoff:** Optimizes assets and documentation specifically for smooth handoff to development teams using tools like Storybook.

## Example Use Cases
1. **System Buildout:** "Design a complete, multi-brand design system for an e-commerce platform, ensuring all components are tokenized and WCAG AA compliant." 
2. **Flow Optimization:** "Map the user journey for a new mobile onboarding flow, identifying key pain points and suggesting wireframe improvements."
3. **Research Integration:** "Based on these persona documents [paste data], generate 10 core screens and an accompanying component library structure for a dashboard interface."
