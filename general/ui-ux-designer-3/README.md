## Overview
This AI agent acts as an expert UI/UX designer, guiding you through the entire user-centered design process. It ensures that digital products are not only beautiful but also highly usable, accessible (WCAG compliant), and aligned with core user needs.

## Capabilities
*   **User Research & Strategy:** Develop detailed user personas, journey maps, and conduct competitive analysis to define a solid design strategy.
*   **Information Architecture:** Create clear information architecture diagrams and comprehensive user flows for optimal navigation.
*   **Design Artifacts:** Generate low-fidelity wireframes, high-fidelity mockups, and interactive prototypes for testing.
*   **Design Systems:** Build robust, scalable component libraries and design systems with defined tokens and patterns.
*   **Accessibility Auditing:** Ensure compliance with WCAG 2.1 AA/AAA standards throughout the entire design lifecycle.
*   **Handoff Documentation:** Provide detailed implementation guidelines for seamless handoff to development teams across all breakpoints (mobile, tablet, desktop).

## Example Use Cases
*   **Redesigning a Mobile App:** Provide existing app screenshots and target user demographics; the agent will generate wireframes, high-fidelity mockups, and an accessibility audit report.
*   **Building a SaaS Dashboard:** Define the core features and user roles; the agent will create an information architecture, component library, and interactive prototype for stakeholder review.
*   **Improving Onboarding Flow:** Submit a description of a complex sign-up process; the agent will map out the optimal user journey, suggest microinteractions, and document best practices.
