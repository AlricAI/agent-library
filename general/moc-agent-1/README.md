## Overview
The Moc Navigator is a specialized agent designed to solve the common problem of knowledge silos in digital vaults. Its core function is to identify content gaps and proactively generate, update, and maintain Maps of Content (MOCs). These MOCs act as curated navigation hubs, providing users with clear, structured pathways through vast amounts of interconnected information.

## Capabilities
*   **MOC Generation:** Creates new, properly templated MOC files for topics lacking central organization.
*   **Orphan Asset Management:** Scans directories to find unlinked images and visual assets, automatically organizing them into dedicated gallery notes.
*   **Network Maintenance:** Updates existing MOCs by incorporating newly created content and ensuring bidirectional linking between related hubs.
*   **Structural Integrity:** Maintains a clear hierarchical structure within all generated navigation documents.

## Example Use Cases
1. **Onboarding New Content:** After dumping a large set of research notes into a new area, run the agent to generate an 'MOC - [New Topic]' and link all relevant assets/notes within it.
2. **Vault Cleanup:** Periodically run the agent to scan for orphaned images or content clusters that have become disconnected from primary navigation points.
3. **Cross-Referencing:** Use it to ensure that when a core concept MOC is updated, all dependent secondary MOCs are flagged and updated with new cross-references.
