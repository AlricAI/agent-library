## Overview
The Workspace Digest Publisher is designed to solve the problem of information overload within a growing, active workspace. Instead of letting valuable work get lost in disparate updates, this agent synthesizes the entire week's activity—spanning knowledge gains, completed tasks, and ongoing context—into one coherent, easy-to-read internal draft.

Its core purpose is to translate diffuse operational activity into a narrative that highlights what *actually* shipped versus mere internal discussion, ensuring continuity between different work streams.

## Capabilities
*   **Signal Extraction:** Identifies and prioritizes the few internal changes that genuinely matter, filtering out noise.
*   **Contextual Synthesis:** Creates follow-up items only when they are directly implied by the week's evidence, preventing invented tasks.
*   **Clarity over Polish:** Presents uncertainty legibly rather than smoothing it away with overly positive or vague language.
*   **Draft Generation:** Produces an internal draft format optimized for review and immediate action by leads and operators.

## Example Use Cases
*   **Weekly Review Prep:** Run this agent every Friday to generate the core document that summarizes team progress for leadership review, ensuring all key decisions are captured.
*   **Project Handoff:** When a project moves from an 'active development' phase to a 'review/next steps' phase, use it to create a digest summarizing achievements and proposing concrete next actions.
*   **Knowledge Gap Identification:** If the work queue shows several related but unlinked tasks, this agent can synthesize them into a single narrative thread, suggesting necessary connective knowledge.
