## Overview
This agent acts as an expert UI/UX designer, guiding you through the entire user-centered design process. It ensures that digital products are not only visually stunning and engaging but also highly functional, accessible (WCAG compliant), and aligned with core user needs.

## Capabilities
*   **User Research:** Develops detailed user personas, journey maps, and conducts competitive analysis to define the core problem.
*   **Information Architecture:** Creates logical site structures and user flows ensuring intuitive navigation.
*   **Design Artifacts:** Generates low-to-high fidelity wireframes, mockups, and interactive prototypes for testing.
*   **Design Systems:** Builds comprehensive component libraries with tokens and documentation for scalable development.
*   **Accessibility Auditing:** Ensures compliance with WCAG 2.1 AA/AAA standards throughout the design process.
*   **Handoff Documentation:** Provides clear implementation guidelines for seamless developer handoff across all breakpoints (mobile, tablet, desktop).

## Example Use Cases
*   **New Product Launch:** Need to design an entire dashboard from scratch? Provide a brief and let it generate personas, user flows, and high-fidelity mockups.
*   **Website Redesign:** If your current site feels outdated or confusing, use this agent to audit the existing experience and propose a modern, accessible redesign strategy.
*   **Feature Improvement:** Stuck on how to improve onboarding? Use it to map out the user journey for that specific task and iterate on wireframes based on usability best practices.
