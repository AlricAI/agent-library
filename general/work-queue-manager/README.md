## Overview
This agent serves as a comprehensive operational guide for an AI agent, structuring the entire workflow from initial assignment intake to daily status logging. It mimics a structured return-to-work protocol, ensuring that no critical step—from reviewing bulletins to updating memory logs—is overlooked.

## Capabilities
*   **Assignment Triage:** Reads and prioritizes immediate tasks based on source documents (e.g., 'Captain's Return to Work Order').
*   **Default Task Execution:** Provides fallback procedures for when no specific assignment is present, including memory review and identity checks.
*   **Escalation Protocol:** Defines clear pathways for reporting blockers or unclear directives to designated support agents.
*   **Structured Logging:** Enforces a strict template for daily status updates, ensuring consistent record-keeping in the agent's memory file.

## Example Use Cases
*   **Starting Work:** When an agent boots up or resumes work after a period of inactivity, it uses this agent to determine its immediate priorities by checking assigned boards and bulletins.
*   **Status Reporting:** At the end of a working cycle, the agent utilizes its template to generate a detailed, actionable daily log for review by supervisors or teammates.
*   **Troubleshooting Workflow:** If an agent encounters technical issues or ambiguity, it follows the defined escalation matrix to ensure timely resolution.
