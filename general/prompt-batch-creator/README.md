## Overview
Prompt Batch Creator is an expert agent designed to take a single creative concept and expand it into multiple, distinct, and highly detailed prompts suitable for advanced AI image generation models. Instead of writing one prompt, this agent writes several, ensuring you get diverse results without manual iteration.

## Capabilities
*   **Parallel Concept Generation:** Creates entirely separate prompts for multiple variations (e.g., 5 different logo styles).
*   **Serial Workflow Prompting:** Generates follow-up prompts that build upon a base concept or reference image (e.g., turning a character portrait into merchandise mockups).
*   **Style Variation:** Adapts to specific artistic guidelines, including photographic realism, anime aesthetics, and detailed illustration techniques.
*   **Structured Output:** Delivers output in a clean, labeled format, ready for immediate copy-pasting into an image generator.

## Example Use Cases
1. **Logo Set:** You need five distinct concepts for a coffee brand logo—this agent will write five unique prompts covering different moods (minimalist, vintage, modern).
2. **Merchandise Line:** After generating a core character illustration, you can ask this agent to create derivative prompts for that character on a T-shirt, mug, and poster.
3. **Mood Boarding:** When brainstorming an event theme, use it to generate 3-4 distinct visual directions (e.g., 'Cyberpunk Night,' 'Rustic Autumn,' 'Neon Tropical').
