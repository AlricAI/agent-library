## Overview
The Professional Document Generator is an expert agent designed to programmatically create high-quality, structured documents across multiple formats: PDF, PowerPoint (PPTX), Word (DOCX), and Excel (XLSX). Instead of simple text assembly, this agent focuses on replicating professional design standards, ensuring that the output is not just content, but a polished deliverable.

It operates by selecting the best code-based library for the required format—whether it's using HTML/CSS rendering for complex PDFs or structured data handling for spreadsheets—to maintain brand consistency and accessibility.

## Capabilities
*   **Multi-Format Output**: Generates native files including PDF, PPTX, DOCX, and XLSX.
*   **Data Integration**: Accepts raw data inputs to populate charts, tables, and key metrics within the documents.
*   **Styling & Branding**: Adheres to specified styles, themes, and branding guidelines rather than using hardcoded elements.
*   **Structure Enforcement**: Builds documents with proper document structure (e.g., TOCs in Word, slide hierarchies in PPTX).
*   **Code Provision**: Provides the necessary generation script alongside the final output file for reproducibility.

## Example Use Cases
1. **Investor Pitch Deck**: Input raw financial data and company narratives to generate a branded, multi-slide PPTX deck with embedded charts.
2. **Quarterly Compliance Report**: Feed structured datasets into the agent to produce a PDF report containing formatted tables, executive summaries, and required appendices.
3. **Training Manual**: Provide section outlines and text blocks to generate a professionally styled DOCX document complete with headers, footers, and a Table of Contents.
4. **Sales Performance Tracker**: Input raw sales figures into an XLSX file that includes pivot-ready layouts and summary dashboards.
