## Overview
This agent acts as a comprehensive knowledge base for understanding an executive's professional profile, priorities, and communication preferences. By centralizing this information, it allows supporting agents to interact with the principal in a manner that is highly tailored, respectful of their time, and aligned with their goals.

## Capabilities
*   **Priority Mapping:** Identifies and tracks the executive's top professional interests (e.g., sales closing, product launches).
*   **Communication Style Guide:** Provides explicit rules on preferred formats for research (bullet points), status updates (numbers first), and correspondence (drafting only).
*   **Constraint Awareness:** Clearly outlines what actions the supporting agent *can* and *cannot* perform, preventing overreach.
*   **Tone Calibration:** Adjusts communication tone to match the executive's personality (e.g., energetic, consultative).

## Example Use Cases
*   **Drafting Communications:** When asked to draft an email for 'Miles,' this agent ensures the tone is appropriate and flags that it requires final approval before sending.
*   **Meeting Preparation:** Before a meeting, it can summarize key discussion points by cross-referencing current projects (AOS Brain, Ronstrapp) against stated top priorities.
*   **Task Prioritization:** When presented with multiple tasks, the agent helps filter them based on whether they align with 'What Makes Him Happy' versus what might annoy him (e.g., unnecessary complexity).

Use this agent whenever you need to ensure your output is perfectly tailored to the recipient's known preferences and focus areas.
