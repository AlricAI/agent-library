## Overview
This agent is engineered for highly controlled, creative content generation. Its core philosophy centers on the principle of 'Human-guided > fully autonomous generation.' It excels when a user provides a detailed structure or framework, and the AI's role is to execute the creation process strictly within those defined boundaries.

It prioritizes local correctness and explicit structural adherence over attempting global optimization or general conversational flow. This makes it ideal for complex document drafting, technical specification writing, or structured narrative development where deviations from the plan are unacceptable.

## Capabilities
*   **Structure Adherence:** Strictly follows user-defined outlines, schemas, or correction events.
*   **Deterministic Output:** Ensures results are predictable within a given structural batch, minimizing unexpected variations.
*   **Controlled Creativity:** Leverages AI's generative power while maintaining human oversight at every critical juncture.
*   **Scope Management:** Handles corrections and modifications with clearly defined spatial or structural boundaries.

## Example Use Cases
*   **Technical Documentation Drafting:** Providing section headers, required data points, and narrative flow for a manual, allowing the agent to fill in technical descriptions precisely.
*   **Complex Report Generation:** Outlining an executive summary, methodology sections, and conclusion structure, ensuring each part adheres to academic or corporate formatting rules.
*   **Scriptwriting with Scene Breakdown:** Giving the agent scene numbers, required actions per character, and emotional beats, forcing it to generate dialogue that fits the pre-defined dramatic arc.
