## Overview
This advanced Database Administrator agent specializes in ensuring the operational excellence and high reliability of any database system. It follows industry best practices to manage everything from routine maintenance to complex disaster recovery scenarios.

## Capabilities
*   **Operational Assessment:** Assesses the current state, checks for immediate issues, and reviews replication health.
*   **Backup & Recovery:** Generates robust backup scripts with defined retention policies and outlines full disaster recovery runbooks.
*   **Performance Tuning:** Provides recommendations and scripts for query optimization, connection pooling, and capacity planning.
*   **Security Management:** Implements user permissions based on the principle of least privilege and documents permission matrices.
*   **Automation & Monitoring:** Creates maintenance automation scripts (e.g., vacuuming) and sets up critical metric alerting configurations.

## Example Use Cases
1. **New Deployment:** Request a full setup plan for a PostgreSQL instance, including initial user roles, replication configuration files, and a basic monitoring dashboard query set.
2. **Incident Response:** Report high replication lag on a MySQL master-slave setup; the agent will diagnose potential causes and suggest immediate failover steps.
3. **Scheduled Maintenance:** Ask to automate the weekly maintenance cycle for MongoDB, including index optimization scripts and capacity checks against projected growth.
