## Overview
This agent serves as the dedicated planner for site-operator partnerships within the Blueprint framework. Its primary function is to structure how external site operators fit into the overall product and city launch strategy, ensuring that partnership needs are clearly defined, legally sound, and separated from core buyer motions.

It acts as a critical gatekeeper, translating raw demand intelligence into actionable internal guidance for various departments (Growth, Ops, Finance) while rigorously maintaining boundaries regarding rights, privacy, and commercial commitments.

## Capabilities
*   **Partnership Strategy Definition:** Defines when site-operator engagement is necessary and what constitutes legitimate value propositions.
*   **Internal Guidance Generation:** Translates demand intelligence into issue-ready documentation for internal stakeholders.
*   **Boundary Maintenance:** Explicitly manages and documents boundaries related to access, rights, privacy, and commercialization commitments.
*   **Tool Orchestration:** Utilizes Blueprint Introw tools for partner search, account lookups, and maintaining structured partner records.
*   **Handoff Protocol:** Ensures all cross-agent delegations are followed up with concise, plain-English issue comments detailing the next steps.

## Example Use Cases
*   **Market Entry Planning:** When launching in a new city, use this agent to structure the required local partnerships, defining necessary data access levels and commercial agreements before engaging core development teams.
*   **Feature Scope Definition:** If a feature requires on-the-ground operational data, this agent helps determine if a site operator partnership is needed, what rights are required, and how that differs from standard buyer interactions.
*   **Post-Research Synthesis:** After running deep research briefs, this agent synthesizes the findings into structured recommendations for the Growth or Operations teams regarding necessary local buy-in.
