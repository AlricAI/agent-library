## Overview
This agent emulates the expertise of a serial hackathon winner and an experienced competition judge. It is designed to guide teams through the entire lifecycle of a hackathon project—from initial concept brainstorming to final pitch refinement. Its core value lies in providing a critical, multi-faceted evaluation that balances groundbreaking innovation with strict buildability constraints.

## Capabilities
*   **Winning Concept Generation:** Brainstorms AI solutions that maximize impact while remaining technically achievable within 24-48 hour time limits.
*   **Judge Simulation:** Evaluates concepts using industry-standard judging criteria (Innovation, Technical Execution, Impact, Presentation) with detailed scoring rationale.
*   **Strategic Roadmapping:** Offers actionable advice on team composition, optimal time allocation across different project phases, and feature prioritization.
*   **Feasibility Assessment:** Identifies 'impressive but simple' features for the demo versus complex elements that should be scoped down or faked for presentation polish.
*   **Pitch Coaching:** Helps structure compelling narratives and demo flows to maximize impact during limited presentation slots.

## Example Use Cases
1. **Concept Vetting:** Input a vague problem area (e.g., 'local community engagement') and receive 3 distinct, scoped AI solution concepts with immediate feasibility scores.
2. **Mid-Hackathon Pivot:** If the team is struggling to build a core feature, use this agent to re-evaluate the scope, suggesting which non-essential features can be cut or replaced with high-impact mockups.
3. **Final Polish:** Before presenting, feed in the final demo plan and ask for a 'Judge's Critique,' receiving actionable feedback on narrative flow and potential judging weak points.
