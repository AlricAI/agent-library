## Overview
Patricia is a sophisticated, fusion-capable agent designed to function as an expert Six Sigma Black Belt. It operates using the Markdown Operating System (MDOS) architecture, ensuring that all operational knowledge, protocols, and project states are persistently stored in human-readable, version-controlled markdown files. This eliminates context loss common in traditional AI deployments.

## Capabilities
*   **Process Modeling:** Applies Six Sigma methodologies to identify bottlenecks and define measurable improvement targets.
*   **State Persistence:** Maintains a living state across projects using structured memory files (e.g., `MEMORY.md`), ensuring zero context loss between sessions.
*   **Protocol Enforcement:** Adheres to defined system protocols (`AGENTS.md`) for data handling, communication structure, and safety compliance (BEAST).
*   **Fusion Intelligence:** Combines specialized agent lineages (like 'Forge' and 'Executive') to provide holistic problem-solving capabilities.

## Example Use Cases
*   **Factory Optimization:** Analyzing production lines (like a Dark Factory) to reduce cycle times and improve throughput by establishing clear baseline metrics.
*   **Process Auditing:** Reviewing existing workflows against established best practices, documenting deviations, and proposing statistically significant improvements.
*   **Knowledge Management:** Serving as the central repository for all project learnings, ensuring that tribal knowledge is codified into executable, documented procedures.
