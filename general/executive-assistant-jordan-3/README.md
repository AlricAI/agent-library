## Overview
This agent embodies 'Jordan,' a highly organized, methodical executive assistant designed to support high-energy leaders by handling the necessary details and ensuring smooth operations behind the scenes. It is not intended to replace human judgment but rather to augment it with structure, thoroughness, and proactive task management.

## Capabilities
*   **Structured Communication:** Provides concise, actionable updates using established key phrases for immediate clarity.
*   **Proactive Support:** Anticipates needs by organizing information and presenting clear options with recommendations.
*   **Task Management Simulation:** Maintains a simulated two-tier memory system (Working/Long-Term) to track preferences, history, and current tasks.
*   **Drafting & Escalation:** Drafts communications for final approval while strictly adhering to guardrails regarding financial decisions and safety protocols.

## Example Use Cases
*   **Meeting Prep:** Ask the agent to synthesize notes from a meeting transcript into three bullet points: Key Decisions, Action Items (with owners), and Follow-up Needs.
*   **Information Synthesis:** Provide several disparate documents and ask Jordan to compile a 'Status Update Memo' for an executive, ensuring it is brief and highlights only critical deviations or required approvals.
*   **Workflow Structuring:** If you have a complex project outline, prompt the agent to break it down into sequential, manageable tasks, assigning estimated completion timeframes.
