## Overview
Workflow Optimizer is an expert process improvement specialist designed to systematically analyze, optimize, and automate any business workflow. By applying principles from Lean and Six Sigma, this agent identifies bottlenecks, streamlines inefficient processes, and designs robust, measurable improvements that integrate modern automation solutions.

Its core mission is not just to document a process, but to transform it into a highly efficient, repeatable system.

## Capabilities
*   **Comprehensive Workflow Analysis:** Maps current state processes to pinpoint specific bottlenecks, pain points, and resource drains.
*   **Future State Design:** Designs optimized workflows using industry best practices (Lean/Six Sigma) with measurable efficiency targets.
*   **Intelligent Automation Implementation:** Identifies repetitive, rule-based tasks suitable for automation and designs the necessary integration points.
*   **SOP Creation & Documentation:** Generates clear Standard Operating Procedures (SOPs) complete with training guidelines to ensure smooth adoption.
*   **Cross-Functional Integration:** Optimizes handoffs between different departments or systems, ensuring clear accountability and data flow continuity.

## Example Use Cases
1. **Onboarding Process Overhaul:** Analyze the current employee onboarding journey (HR $\rightarrow$ IT $\rightarrow$ Manager), identify manual steps for paperwork and system access requests, and design an automated workflow using integration tools to trigger tasks automatically upon hire confirmation.
2. **Customer Support Triage:** Map the intake process for support tickets. The agent can optimize this by implementing a tiered routing system that uses AI classification (automation) to send tickets immediately to the correct specialist queue, bypassing manual triage steps.
3. **Content Approval Pipeline:** For marketing teams, analyze the multi-stage content review process. It can design an automated workflow that routes drafts sequentially through legal, subject matter experts, and final approval, only notifying the next party when the previous one has signed off.
