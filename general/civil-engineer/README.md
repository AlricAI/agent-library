## Overview
This agent embodies the expertise of a senior structural and civil engineer with deep, practical knowledge across global engineering standards. It is designed to handle complex, multi-jurisdictional projects where compliance with various international codes (such as Eurocode, ACI, AISC, ASCE, etc.) is mandatory.

The core mission is to ensure that all designs are not only structurally sound and safe but also economical, constructible, and fully compliant with the governing building code for the project's location.

## Capabilities
*   **Structural Analysis & Design:** Perform comprehensive load analyses (gravity, seismic, wind) and design primary systems including reinforced concrete, steel frames, and timber structures. It verifies both Ultimate Limit State (ULS) strength and Serviceability Limit State (SLS).
*   **Code Compliance Navigation:** Expertly manages conflicting requirements from multiple international standards, detailing the governing code edition used for every calculation.
*   **Geotechnical Evaluation:** Interprets soil investigation reports (SPT, CPT logs) to design foundations, retaining walls, and assess slope stability.
*   **Documentation Production:** Generates detailed engineering packages, including load takedowns, member checks, technical specifications, and general notes suitable for construction drawings.

## Example Use Cases
*   **Designing a Multi-Story Building:** You can input soil reports and project requirements, and the agent will generate preliminary structural designs adhering to both local seismic codes and international best practices.
*   **Foundation Assessment:** Uploading borehole logs allows the agent to perform bearing capacity checks and recommend appropriate foundation types (shallow vs. deep).
*   **Code Comparison:** Asking the agent to compare the design requirements for a specific connection detail under Eurocode versus ACI 318 will yield a detailed, comparative analysis.
