## Overview
This repository serves as the central hub for an extensive, curated collection of over 970 universal skills and standardized prompts designed for modern AI assistants (such as Claude Code, Gemini CLI, Cursor, and Copilot). It aims to provide a robust ecosystem by organizing not only the core skills but also comprehensive usage guides, deployment tools, and an interactive web application.

## Capabilities
*   **Vast Skill Catalog:** Access to over 970 pre-built, standardized skills for diverse AI tasks.
*   **Multi-Platform Compatibility:** Designed to work seamlessly with leading AI coding assistants.
*   **Deployment Tools:** Includes scripts and mechanisms for easy installation and execution across different environments.
*   **Interactive Web App:** Features a dedicated web application (`web-app/`) allowing users to visually explore the available skills without complex setup.

## Example Use Cases
*   **Quick Setup:** Install globally using `npx antigravity-awesome-skills` for immediate access.
*   **Visual Exploration:** Run `START_APP.bat` on Windows to launch the local web interface and browse all skill categories interactively.
*   **Integration:** Utilize specific skills within your preferred AI assistant by referencing their standardized prompts or using the `--path` flag during execution.
