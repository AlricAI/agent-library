## Overview
Prompt Engineer Pro acts as a specialized consultant for crafting high-performance prompts. It goes beyond simple instruction writing by applying established cognitive and engineering patterns to ensure the best possible output from any Large Language Model (LLM).

This agent systematically analyzes your use case, selects the most appropriate prompting methodology (like CoT or ToT), structures the prompt with necessary components, and provides actionable testing and iteration plans.

## Capabilities
*   **Technique Selection:** Implements advanced patterns such as Chain-of-Thought (CoT), Tree-of-Thoughts (ToT), and few-shot learning based on task complexity.
*   **Structure Generation:** Builds comprehensive prompts including defined roles, step-by-step instructions, explicit output formats, and constraints.
*   **Optimization & Testing:** Provides A/B testing variations, performance metrics suggestions, and a troubleshooting guide for prompt refinement.
*   **Model Adaptation:** Offers model-specific optimizations to tailor the prompt for different LLM architectures.

## Example Use Cases
1. **Complex Reasoning Task:** You need an agent to solve multi-step logic puzzles. The agent will structure the prompt using Tree-of-Thoughts to explore multiple solution paths before committing to an answer.
2. **Data Extraction Pipeline:** You require consistent JSON output from unstructured text. The agent will build a highly constrained prompt with explicit schema definitions and error handling instructions.
3. **Persona Simulation:** You need the AI to act as a senior financial analyst. The agent will define a robust persona, context, and specific evaluation criteria within the final prompt block.
