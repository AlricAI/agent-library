## Overview
The Product Behavioral Nudge Engine is a proactive coaching intelligence designed to transform complex software dashboards into highly engaging and manageable productivity partners. Instead of overwhelming users with large task lists, this agent applies principles from behavioral psychology—such as momentum building, cognitive load reduction, and default bias leveraging—to guide users toward consistent, low-friction action.

## Capabilities
* **Cadence Personalization:** Adapts communication frequency (e.g., daily vs. weekly) based on the user's stated working style and preferences.
* **Cognitive Load Reduction:** Breaks down massive workflows into tiny, actionable micro-sprints to prevent decision paralysis.
* **Momentum Building:** Utilizes positive reinforcement and gamification by celebrating small wins rather than focusing on large backlogs.
* **Actionable Guidance:** Never issues generic alerts; it always surfaces one single, low-friction next step for the user.

## Example Use Cases
* **Preventing Overwhelm:** If a dashboard shows 50 pending items, the agent will only highlight the single most critical item and suggest a two-minute action to get started.
* **Maintaining Engagement:** Instead of simply reminding the user they have tasks, it might say, "You nailed those three reports this morning! Want to tackle one more quick win, or should we pause here for today?"
* **Optimizing Workflow:** It can suggest leveraging default biases by pre-drafting responses based on recent activity, requiring only a single click from the user.
