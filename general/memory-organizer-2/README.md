## Overview
Memory Organizer is designed to act as a sophisticated information repository and structuring agent. Its core function is to take raw, unstructured data—whether it comes from meeting transcripts, research papers, or brainstorming sessions—and organize it into logical, retrievable memory blocks. It moves beyond simple summarization by imposing structure, making the knowledge actionable.

## Capabilities
*   **Contextual Chunking:** Breaks down large volumes of text into semantically coherent, manageable chunks.
*   **Relationship Mapping:** Identifies and maps connections between different pieces of information (e.g., cause-and-effect, prerequisite relationships).
*   **Structured Output Generation:** Formats the organized memory into predefined structures like knowledge graphs, timelines, or hierarchical outlines.
*   **Gap Identification:** Analyzes the provided data set to suggest areas where more information is needed for a complete picture.

## Example Use Cases
*   **Project Post-Mortem:** Feed it all meeting notes and deliverables from a project. It will return an organized memory map showing decisions made, action items assigned, and unresolved dependencies.
*   **Research Synthesis:** Input several academic articles on a niche topic. The agent will structure the findings by key theories, contrasting viewpoints, and identifying consensus areas.
*   **Onboarding Documentation:** Provide raw departmental guidelines from various sources. It will synthesize these into a single, navigable 'New Hire Memory Guide' with clear ownership for each section.
