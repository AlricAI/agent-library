## Overview
OpenCouncil Assistant is a specialized AI companion built to serve as an indispensable resource for the OpenCouncil project team operating on Discord. Its primary function is to streamline communication, provide immediate access to project knowledge, and assist team members with day-to-day operational tasks.

This agent acts as a centralized point of information, ensuring that all team members remain aligned with the latest project details, discussions, and guidelines without needing to sift through extensive chat logs.

## Capabilities
*   **Project Knowledge Retrieval:** Quickly answers questions about OpenCouncil's goals, structure, and history.
*   **Task Summarization:** Can summarize key decisions or action items from long threads or meeting transcripts.
*   **Guideline Enforcement:** Provides instant reminders or summaries of established project protocols.
*   **Contextual Assistance:** Tailors responses specifically to the context of the OpenCouncil project discussions.

## Example Use Cases
*   **Onboarding New Members:** A new team member can ask, "What is the current sprint goal for OpenCouncil?" and receive an immediate, accurate summary.
*   **Meeting Follow-up:** After a long discussion, a user can prompt it with, "What are the three action items decided in this thread?" to get a clean list.
*   **Technical Query Resolution:** A developer can ask, "Where is the latest documentation for module X?" and receive direct pointers or summaries of the relevant sections.
