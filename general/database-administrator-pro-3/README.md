## Overview
This agent emulates a senior, expert Database Administrator with mastery across leading database systems, including PostgreSQL, MySQL, MongoDB, and Redis. Its primary function is to ensure data infrastructure achieves maximum reliability (targeting 99.99% uptime) while maintaining sub-second query performance.

It guides users through comprehensive lifecycle management, from initial production-grade setup and security hardening to advanced disaster recovery testing.

## Capabilities
*   **Performance Optimization:** Analyzes slow queries, designs optimal indexing strategies, tunes buffer pools, and manages statistics for peak efficiency.
*   **High Availability (HA) Implementation:** Configures robust replication patterns (Master-Slave, Multi-master) with automated failover mechanisms to prevent downtime.
*   **Disaster Recovery (DR):** Establishes comprehensive backup strategies, enabling point-in-time recovery and ensuring adherence to strict RTO/RPO objectives.
*   **System Hardening & Monitoring:** Performs security audits, implements connection pooling, and sets up detailed monitoring and alerting dashboards.

## Example Use Cases
*   **Scaling Out:** "Review our current MySQL setup for read-heavy traffic and recommend a load-balancing strategy using read replicas." 
*   **Disaster Readiness:** "We need to validate our PostgreSQL DR plan. Guide me through testing point-in-time recovery compliance."
*   **Tuning Bottlenecks:** "Analyze the query execution plan for this complex report and suggest index improvements to reduce latency by 50%."
