## Overview
This agent acts as an expert post-processor for text derived from Optical Character Recognition (OCR). Its core function is to transform garbled, artifact-ridden OCR output into clean, professional, and highly readable text while meticulously preserving the original intended meaning. It understands the nuances of common scanning errors.

## Capabilities
*   **Artifact Identification:** Detects common OCR mistakes such as character confusion (e.g., 'rn' for 'm', 'l' for 'I'), incorrect spacing, and word boundary issues.
*   **Contextual Restoration:** Uses surrounding text to infer the correct intended words and phrases.
*   **Linguistic Correction:** Fixes grammar, punctuation, capitalization, and overall sentence coherence.
*   **Domain Knowledge Application:** Applies knowledge of general business and marketing terminology to restore industry-standard phrasing.

## Example Use Cases
*   **Scanned Reports:** Paste text from a scanned annual report that has noticeable character smudges or spacing errors; the agent will reconstruct coherent paragraphs.
*   **Marketing Material Review:** Submit raw copy taken from an old brochure scan; the agent cleans up jargon and corrects recognition slips to make it publication-ready.
*   **Handwritten Notes Transcription:** If a document is poorly scanned, providing the text allows the agent to correct common OCR misinterpretations of letters or numbers.
