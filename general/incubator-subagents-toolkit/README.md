## Overview
This toolkit provides a suite of specialized, slash-command agents designed to augment the development and validation process for any AI incubator project. Instead of needing separate tools or context switching, these commands allow users to invoke expert analysis—from market research to code auditing—directly within the chat interface.

## Capabilities
*   **Market Research (`/inc-research`):** Conducts deep dives into competitor landscapes, domain availability, and community sentiment for new ideas.
*   **Design Review (`/inc-design`):** Provides structured feedback on UX, branding elements, and landing page effectiveness.
*   **Executive Review (`/inc-exec`):** Offers high-level sanity checks on business viability and strategic pivot decisions.
*   **Codebase Audit (`/auditor`):** Analyzes existing code against established patterns to maintain technical health as the project grows.
*   **News Briefing (`/news`):** Keeps users current with the latest developments in AI and startup technology at the start of a session.

## Example Use Cases
1. **Idea Validation:** To check if 'shipcheck.io' is viable, simply run: `/inc-research shipcheck.io - launch readiness audits for indie hackers`.
2. **Design Feedback:** After mocking up a landing page, use: `/inc-design Review this copy and structure.`
3. **Technical Health Check:** When adding a major new feature, preemptively audit the code with: `/auditor Check adherence to our component library standards in the latest commit.`
