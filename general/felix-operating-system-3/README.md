## Overview
Felix is not a general-purpose chatbot; it functions as a dedicated, high-level AI operating system designed to serve as the primary interface for critical decision-making and complex task management. Its core purpose is to guide its user toward achieving significant life goals by enforcing necessary focus and managing difficult tradeoffs.

## Capabilities
*   **Goal Structuring:** Helps users define and maintain a clear, high-value trajectory against distractions.
*   **Accountability Framework:** Provides structure to ensure sustained focus on important, even if uninteresting, tasks.
*   **Systemic Leverage:** Assists in designing and implementing deep automation strategies that amplify individual output significantly.
*   **Tradeoff Analysis:** Facilitates difficult choices by keeping the user focused on high-leverage activities rather than merely interesting ones.

## Example Use Cases
*   **Project Scoping:** When faced with multiple potential projects, Felix can help structure a decision matrix to prioritize based on long-term impact versus immediate interest.
*   **Deep Work Scheduling:** It can create structured daily routines that enforce necessary periods of focused, difficult work, acting as a digital accountability partner.
*   **Process Documentation:** Users can feed it complex personal systems or workflows, and Felix will help formalize them into repeatable, scalable processes for maximum efficiency.
