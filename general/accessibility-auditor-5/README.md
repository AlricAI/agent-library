## Overview
This agent functions as a senior accessibility tester, specializing in auditing digital products against the Web Content Accessibility Guidelines (WCAG 2.1/3.0). Its primary goal is to ensure that applications are usable by people with visual, auditory, motor, and cognitive disabilities, moving beyond simple compliance checks to achieve true universal design.

## Capabilities
*   **Comprehensive WCAG Auditing:** Validates adherence across Perceivable, Operable, Understandable, and Robust principles.
*   **Assistive Technology Testing:** Verifies compatibility with major screen readers (NVDA, JAWS, VoiceOver) and proper content announcement order.
*   **Keyboard Navigation Mastery:** Checks for complete tab order logic, visible focus indicators, skip links, and robust modal handling.
*   **Visual & Contrast Analysis:** Assesses color contrast ratios, text readability, and stable layout behavior under zoom.
*   **Cognitive Load Assessment:** Reviews content structure, language clarity, and consistency to minimize user confusion.

## Example Use Cases
*   **Pre-launch Audit:** Run this agent on a new feature build to generate a comprehensive report detailing all WCAG Level AA violations before deployment.
*   **Usability Testing Prep:** Use it to create an accessibility checklist for manual QA teams, ensuring no critical violation is missed during user testing sessions.
*   **Remediation Guidance:** Input existing code snippets or component descriptions and ask the agent to suggest specific ARIA attributes or structural changes needed for compliance.
