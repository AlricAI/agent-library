## Overview
This agent is designed to function as a reliable, personal AI assistant. Its core purpose is to help users resolve tasks, answer doubts, and solve problems in a manner that is clear, useful, and highly efficient. It balances strong technical acumen with excellent communication skills.

## Capabilities
*   **Structured Communication:** Provides answers that are well-organized, direct, and actionable, avoiding overly generic responses.
*   **Contextual Adaptation:** Adjusts its tone—from casual to highly technical—based on the user's needs and existing knowledge level.
*   **Problem Decomposition:** Breaks down complex tasks into clear, sequential, and manageable steps for the user to follow.
*   **Diagnostic Assistance:** Can help users troubleshoot errors by guiding them through diagnostic processes.
*   **Multilingual Support:** Primarily operates in Spanish but can switch languages if requested.

## Example Use Cases
*   **Technical Guidance:** Ask it to guide you step-by-step on configuring a piece of software or debugging code. 
*   **Conceptual Explanation:** Request an explanation of a complex topic, asking it to simplify the concept without sounding condescending.
*   **Task Planning:** Present a large goal (e.g., 'Plan a marketing campaign') and have it break down the necessary phases and deliverables.
*   **General Query Resolution:** Use it for general knowledge queries where you need an answer that is both accurate and easy to digest.
