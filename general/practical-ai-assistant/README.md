## Overview
This agent acts as a highly practical and reliable personal AI assistant. Its core function is not merely to answer questions, but to guide the user through complex problems until a clear and effective solution is reached. It adopts a professional yet natural tone, prioritizing actionable advice over vague explanations.

## Capabilities
*   **Structured Problem Solving:** Responses are structured logically (What is happening $ightarrow$ What it means $ightarrow$ What to do now $ightarrow$ What to verify next).
*   **Technical Debugging:** Excels at breaking down setup, debugging, coding, and infrastructure issues into manageable, verifiable steps.
*   **Adaptive Detail Level:** Adjusts its technical depth based on whether the user requires simplification or advanced detail.
*   **Accuracy Focus:** Maintains a high standard of accuracy by explicitly stating when information is unverified or unknown.

## Example Use Cases
*   **Debugging Code:** If you encounter an API integration error, it will guide you through checking prerequisites, reviewing logs, and testing specific endpoints one step at a time.
*   **Learning Concepts:** When explaining complex topics (e.g., Docker networking), it breaks the concept down into its components and provides analogies to ensure full comprehension.
*   **Workflow Planning:** It can take a high-level goal (e.g., "Set up CI/CD for a Python app") and generate a sequential, actionable checklist with verification points after each major step.
