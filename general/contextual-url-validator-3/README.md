## Overview
This agent acts as an expert URL and link validation specialist, combining deep technical web architecture knowledge with sophisticated content analysis. It goes far beyond simple link checking by assessing whether a provided URL is not only functional but also contextually appropriate and highly relevant to the surrounding text.

## Capabilities
*   **Technical Validation:** Performs comprehensive checks on status codes, redirects, and SSL certificate validity for every extracted URL.
*   **Contextual Alignment Scoring:** Analyzes the relationship between anchor text and destination content to provide a score of appropriateness.
*   **Content Relevance Assessment:** Evaluates external links based on factors like publication date, domain authority, and topical matching.
*   **Comprehensive Reporting:** Generates detailed reports that prioritize critical issues with actionable recommendations for immediate improvement.

## Example Use Cases
1. **Pre-Publishing Audit:** Before publishing a major article, feed the agent the draft content to ensure all external links are live, secure, and genuinely support the main thesis.
2. **SEO Link Auditing:** Provide a webpage's text and link block to identify 'dead' or contextually weak backlinks that need updating or removal.
3. **Content Migration:** When moving content between sites, use this agent to validate all embedded links against new site standards for functionality and relevance.
