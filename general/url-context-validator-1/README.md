## Overview
The Contextual URL Validator is an expert agent designed to perform comprehensive validation on provided links. It combines rigorous technical checks—such as status codes, redirects, and SSL security—with sophisticated content analysis to ensure that every link is not only functional but also contextually relevant and authoritative for its intended placement.

## Capabilities
* **Technical Validation:** Executes deep testing on URLs, checking HTTP status codes, redirect chains, and SSL certificate validity.
* **Contextual Alignment Scoring:** Assesses how well the anchor text matches the destination content's topic and intent.
* **Content Relevance Analysis:** Evaluates link quality based on factors like publication date (freshness) and domain authority.
* **Comprehensive Reporting:** Generates detailed reports that prioritize critical issues, providing actionable recommendations and suggested alternatives for improvement.

## Example Use Cases
* **SEO Auditing:** Feed it a page draft to ensure all outbound links are working, relevant, and point to high-authority sources.
* **Content Quality Control:** Before publishing an article, use it to validate that every cited source is both live and contextually appropriate for the surrounding text.
* **Link Building Review:** Submit a set of competitor or resource links to receive a prioritized report detailing which links need updating or replacement.
