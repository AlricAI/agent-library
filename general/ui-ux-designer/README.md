## Overview
This agent acts as an expert UI/UX designer, specializing in creating user-centered designs, robust design systems, and highly accessible interfaces. It guides you through modern design workflows, ensuring consistency across web, mobile, and desktop platforms.

## Capabilities
*   **Design Systems Mastery:** Implements atomic design using token-based architectures (e.g., Figma Variables) for scalable component libraries.
*   **User Research Integration:** Conducts user journey mapping, persona development, and usability testing planning to ground designs in real user needs.
*   **Accessibility Compliance:** Ensures adherence to WCAG 2.1/2.2 standards throughout the design process.
*   **Prototyping & Handoff:** Creates advanced prototypes with micro-interactions and optimizes asset generation for seamless developer handoff (Storybook integration).

## Example Use Cases
*   **Designing a New Feature Flow:** Provide a concept, and this agent will generate user flows, low-fidelity wireframes, and suggest necessary components for the design system.
*   **Auditing an Existing Interface:** Upload screenshots or descriptions of an existing UI; it will perform an accessibility audit against WCAG guidelines and recommend specific improvements.
*   **Building a Component Library:** Define your brand guidelines and core elements (buttons, cards); this agent will structure them into a documented, tokenized component library ready for development.
