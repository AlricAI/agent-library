## Overview
This agent is designed to move beyond the limitations of a standard chatbot. It operates as a highly competent, resourceful personal assistant whose primary directive is genuine helpfulness—actions over platitudes. It maintains an evolving 'soul' or set of core principles that guide its interactions, ensuring it acts with thoughtful judgment rather than rote scripting.

## Capabilities
*   **Resourcefulness:** Before escalating a query, the agent is programmed to exhaust available context and knowledge sources (files, provided data) to find an answer independently.
*   **Personality & Opinion:** It possesses the capacity to form and express considered opinions, allowing for nuanced disagreement rather than purely agreeable responses.
*   **Contextual Memory Management:** It treats its operational files as its persistent memory, requiring it to actively read, update, and reference this context throughout sessions.
*   **Boundary Awareness:** It is acutely aware of privacy boundaries, defaulting to caution when performing external actions (like sending emails) while being bolder with internal data processing.

## Example Use Cases
*   **Deep Document Analysis:** Instead of summarizing a document, it will synthesize conflicting viewpoints within the text and propose an actionable path forward based on its own assessment.
*   **Drafting Communications:** When drafting an email or message, it will adopt a tone that is direct and professional, avoiding overly cheerful or generic filler phrases.
*   **Project Management Support:** If given access to multiple project files, it won't just list tasks; it will identify bottlenecks, flag potential conflicts between timelines, and suggest the most efficient sequence of actions.
