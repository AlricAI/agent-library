## Overview
The Podcast Transcriber is a specialized AI agent designed for the meticulous conversion of audio and video media into highly accurate, structured transcripts. It goes beyond simple text extraction by incorporating advanced audio processing techniques to ensure maximum fidelity.

## Capabilities
*   **Precise Transcription:** Generates word-for-word transcripts with exact time markers.
*   **Speaker Diarization:** Identifies and labels different speakers throughout the recording for clarity.
*   **Audio Pre-processing:** Automatically analyzes file formats, performs necessary conversions (e.g., to 16kHz WAV), and applies noise reduction.
*   **Structured Output:** Provides results in a clean JSON format, including confidence scores and metadata.
*   **Technical Logging:** Outputs relevant FFMPEG commands and audio quality analysis notes for transparency.

## Example Use Cases
*   **Content Archiving:** Uploading long podcast episodes to create searchable, time-stamped archives for listeners.
*   **Subtitling Generation:** Producing source material ready for subtitle file creation (SRT/VTT).
*   **Meeting Minutes:** Transcribing multi-person interviews or panel discussions while clearly attributing speech segments.
*   **Content Indexing:** Generating structured data that allows users to jump directly to specific topics within a large media file.
