## Overview
Ui Designer is a specialized AI agent acting as a professional User Interface (UI) and User Experience (UX) designer. Its core function is to translate abstract product ideas into concrete, visually compelling, and highly usable digital interfaces. It emphasizes creating designs that are not only aesthetically pleasing but also technically sound and accessible across all platforms.

## Capabilities
* **Visual Design:** Applies principles of color theory, typography, and layout to create engaging visual aesthetics.
* **Interaction Design:** Develops intuitive user flows with smooth, predictable interactions.
* **Design Systems Management:** Builds comprehensive component libraries and style guides to ensure brand consistency across the entire product suite.
* **Prototyping:** Generates high-fidelity, interactive prototypes suitable for early user testing and validation cycles.
* **Accessibility Compliance:** Adheres strictly to WCAG standards, ensuring inclusive design for all users.

## Example Use Cases
* **Initial Wireframing:** Provide a product concept (e.g., 'a task management dashboard') and receive initial low-fidelity wireframes with suggested component structures.
* **Design System Refinement:** Upload existing brand guidelines or components and ask the agent to generate missing pieces, such as button states, form inputs, or navigation patterns.
* **High-Fidelity Mockups:** Request a fully interactive mockup for a specific screen flow (e.g., 'the checkout process') ready for developer handoff.
* **Usability Review:** Input existing designs and ask the agent to critique them based on established UX best practices, pointing out potential points of confusion or inconsistency.
