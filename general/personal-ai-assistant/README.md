## Overview
This agent is designed to function as a comprehensive and highly reliable personal AI assistant. Its core purpose is to assist users in resolving tasks, answering doubts, and solving problems with exceptional clarity, utility, and efficiency. It balances technical accuracy with natural, human-like communication.

## Capabilities
*   **Structured Problem Solving:** Breaks down complex issues into clear, actionable, step-by-step guides.
*   **Adaptive Tone:** Adjusts its tone to be professional yet conversational, avoiding robotic or overly generic responses.
*   **Contextual Guidance:** Can guide users through technical configurations or brainstorming sessions by proposing concrete and useful options.
*   **Diagnostic Assistance:** Helps diagnose errors by systematically questioning the user for necessary context.
*   **Multilingual Support:** Primarily operates in Spanish but can switch languages upon request.

## Example Use Cases
*   **Technical Troubleshooting:** "Mi servidor no conecta a la base de datos, ¿qué pasos debo seguir para diagnosticarlo?"
*   **Project Planning:** "Necesito crear un plan de marketing digital desde cero. ¿Podrías dividirlo en fases y tareas?".
*   **Conceptual Explanation:** "Explícame el concepto de blockchain como si tuviera conocimientos básicos de tecnología."

This agent excels when you need more than just an answer; you need a structured, intelligent guide to get the job done.
