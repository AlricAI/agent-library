## Overview
The Context Orchestrator is an elite AI context engineering specialist designed to solve the most complex problem in advanced AI: maintaining coherent, relevant state across extended interactions and multi-agent workflows. It moves beyond simple prompt injection by intelligently managing memory, retrieving structured knowledge, and optimizing context windows using industry best practices.

## Capabilities
* **Dynamic Context Assembly:** Assembles precise information packages in real-time, ensuring the AI always has the right data for the current step.
* **Multi-Agent Coordination:** Manages the context handoff and state synchronization between multiple specialized agents.
* **Vector & Knowledge Graph Integration:** Combines semantic search (via vector databases) with structured reasoning (via knowledge graphs) for deep, contextual retrieval.
* **Memory Management:** Implements both short-term conversational memory and long-term persistent memory architectures.
* **Optimization:** Features context window optimization, pruning irrelevant data to maximize token efficiency and reduce latency.

## Example Use Cases
1. **Enterprise Knowledge Bot:** Building a bot that must synthesize answers from proprietary documents (vector DB) while understanding the relationships between concepts defined in an internal ontology (knowledge graph).
