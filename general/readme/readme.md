## Overview
TripCraft AI is a sophisticated, multi-agent system designed to eliminate the complexity of trip planning. Instead of relying on a single prompt, it deploys specialized agents—such as Destination Explorer, Hotel Search Agent, and Budget Agent—to work together in a coordinated team effort.

This architecture ensures that every facet of travel, from finding local attractions to optimizing flight costs, is researched by an expert agent before being compiled into one cohesive plan.

## Capabilities
*   **Destination Research:** Uses deep web tools (like ExaTools) to find detailed information on landmarks, museums, and activities.
*   **Accommodation Booking Support:** Recommends hotels based on location, budget, amenities, and specific property features.
*   **Culinary Planning:** Suggests restaurants considering cuisine type, price point, ambiance, and dietary needs.
*   **Financial Modeling:** Creates detailed cost breakdowns, allocates budgets across activities, and suggests savings opportunities.
*   **Air Travel Logistics:** Plans flight routes, compares airlines, and optimizes connection schedules.

## Example Use Cases
1. **Family Vacation Planning:** A user inputs 'A week-long trip to Rome for a family with young children.' The system coordinates the Destination Explorer (for kid-friendly spots), Hotel Search Agent (for family amenities), and Budget Agent (to keep costs manageable).
2. **Business Trip Optimization:** For a corporate trip, it uses the Flight Search Agent to find optimal routes between multiple cities while the Dining Agent suggests appropriate business meal venues.
3. **Luxury Adventure Itinerary:** Combining high-end hotel searches with specialized activity recommendations ensures a seamless and premium travel experience.
