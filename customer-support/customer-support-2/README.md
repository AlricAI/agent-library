## Overview
This agent simulates an expert customer support specialist, designed to handle inquiries with a focus on rapid resolution and high user satisfaction. It follows a structured process that prioritizes empathy, clarity, and actionable steps.

## Capabilities
*   **Issue Diagnosis:** Reads complex issues to identify the root cause rather than just treating symptoms.
*   **Structured Response Generation:** Creates responses that include acknowledgment, numbered steps, alternative solutions, and clear next steps.
*   **Knowledge Base Management:** Generates content suitable for FAQs, detailed troubleshooting guides (including placeholders for visuals), and internal knowledge base documentation.
*   **Communication Tone Control:** Maintains a consistently friendly, professional, and empathetic tone throughout all interactions.
*   **Process Documentation:** Can draft canned responses and escalation criteria for team efficiency.

## Example Use Cases
1. **Handling a New Ticket:** Paste a detailed customer complaint; the agent will respond with an empathetic acknowledgment, followed by numbered troubleshooting steps and an expected resolution timeline.
2. **Creating Internal Docs:** Provide a common bug report; the agent will generate a comprehensive FAQ entry and a step-by-step guide for support agents to follow.
3. **Drafting Templates:** Ask it to create a 'Password Reset' canned response template, ensuring all necessary escalation contacts are included.
