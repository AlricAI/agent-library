## Overview
This agent acts as an expert customer support specialist designed for quick resolution and high satisfaction. It follows a structured process to ensure every customer interaction is handled with empathy, clarity, and actionable steps. From initial ticket triage to creating comprehensive knowledge base articles, this tool standardizes best-in-class support responses.

## Capabilities
*   **Issue Diagnosis:** Reads complex issues to identify the root cause rather than just treating symptoms.
*   **Structured Response Generation:** Crafts empathetic acknowledgments followed by clear, numbered step-by-step solutions.
*   **Documentation Creation:** Automatically generates FAQ entries and detailed troubleshooting guides (including placeholders for visuals).
*   **Process Management:** Provides canned response templates and defines explicit escalation criteria for complex issues.
*   **Tone Control:** Maintains a consistently friendly, professional, and solution-focused tone throughout all communications.

## Example Use Cases
*   **Handling a New Ticket:** Paste a customer complaint; the agent will respond with an empathetic acknowledgment, followed by numbered steps to fix it, and suggest creating an FAQ entry for future use.
*   **Building Knowledge Base Content:** Feed the agent several related support threads; it will synthesize them into a comprehensive troubleshooting guide.
*   **Drafting Outbound Communication:** Give it a technical solution that needs explaining; it will rewrite it into simple, non-technical language suitable for an end-user email.
