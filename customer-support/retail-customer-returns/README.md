## Overview
This specialized agent acts as an expert retail returns specialist, designed to manage the entire customer return lifecycle. Its core philosophy is that a positive return experience builds long-term customer loyalty, making it crucial for any modern retailer.

The agent handles everything from initial eligibility checks and policy enforcement to final refund issuance and value recovery from returned goods across in-store, online, and omnichannel touchpoints.

## Capabilities
*   **Policy Enforcement:** Accurately determines return eligibility based on established store policies (time windows, condition requirements).
*   **Return Processing:** Manages the physical flow of returns, including inspection, grading merchandise, and determining disposition.
*   **Refund & Exchange Management:** Processes refunds to original payment methods, issues store credit, or facilitates direct exchanges.
*   **Fraud Mitigation:** Identifies and flags potential return abuse patterns or suspicious transactions.
*   **Data Analysis:** Generates actionable insights regarding return trends to help reduce overall return rates for the business.

## Example Use Cases
*   **Customer Service Scenario:** A customer calls about an online purchase that arrived damaged. The agent verifies the policy, initiates a replacement order (exchange), and logs the damage report for vendor claims.
*   **Operational Audit:** Analyzing a batch of returns to identify if a specific product line frequently fails inspection due to packaging issues, allowing management to address root causes.
*   **Complex Dispute Resolution:** Handling an exception request where a return falls outside standard policy but warrants customer retention efforts, requiring careful documentation and managerial override suggestion.
