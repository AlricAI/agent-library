## Overview
This Buyer Success Agent is designed to act as a proactive, highly perceptive Customer Success Manager (CSM) for clients using Blueprint's hosted sessions and site-worlds. Its primary goal is not high-volume outreach, but rather maintaining 'lightweight' vigilance to catch subtle support signals or early signs of disengagement before they become critical issues.

## Capabilities
*   **Adoption Monitoring:** Tracks whether the buyer is actively using purchased features and accessed worlds.
*   **Health Diagnosis:** Assesses integration health by identifying if site-worlds load correctly or if data retrieval fails.
*   **Signal Detection:** Distinguishes between a quietly satisfied user and one who is becoming disengaged, requiring different levels of intervention.
*   **Opportunity Identification:** Naturally surfaces potential expansion needs (new sites, deeper coverage) based on observed usage patterns.
*   **Triage & Escalation:** Routes critical feedback regarding privacy, rights, or core product friction to the correct internal team immediately.

## Example Use Cases
*   **Disengagement Alert:** A buyer's weekly session count drops by 40% over two weeks. The agent flags this for review, suggesting a non-intrusive check-in rather than assuming they are merely 'busy'.
*   **Integration Failure Triage:** A report shows multiple failed attempts to load a specific site-world type. The agent correctly categorizes this as a potential Blueprint platform issue versus an environmental setup error.
*   **Expansion Nudging:** Usage data reveals the buyer repeatedly accesses documentation related to a modality not yet purchased. The agent flags this as a high-potential, low-friction upsell opportunity.
