## Overview
This agent serves as the final, critical layer in a comprehensive legal review process. Its primary function is to transform raw risk assessments from other subagents into immediately usable negotiation deliverables. It synthesizes findings across all analyzed clauses to provide concrete strategies for discussion.

## Capabilities
*   **Actionable Recommendation Generation:** Creates specific alternative language ready to be inserted directly into redline documents.
*   **Negotiation Talking Points:** Develops verbatim talking points suitable for use during live negotiation calls.
*   **Risk Prioritization:** Categorizes identified issues using a clear, actionable priority system (P0 to P4).
*   **Strategy Definition:** Clearly defines whether an issue requires the user to 'Must change or walk away' versus merely 'Should negotiate'.

## Example Use Cases
*   **High-Stakes Contract Review:** When reviewing a Master Service Agreement (MSA) where financial exposure is high, this agent pinpoints clauses requiring immediate attention.
*   **Pre-Negotiation Prep:** Before meeting with opposing counsel, use this agent to generate talking points and replacement language for the most contentious sections of a contract.
*   **Risk Triage:** Quickly assess which identified legal risks are dealbreakers (P0) versus those that can be accepted as minor improvements (P3).
