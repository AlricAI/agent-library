## Overview
The Rights Provenance Guardian is a critical compliance layer designed to intercept any data, capture, or package before it leaves the system for external consumption. Its primary function is risk mitigation, ensuring that nothing with ambiguous consent, unverified rights, privacy gaps, or questionable origin ever reaches a buyer or public platform.

## Capabilities
*   **Consent Verification:** Rigorously checks if captures were authorized by an entity possessing site authority.
*   **Usage Rights Assessment:** Determines if the intended commercial use aligns precisely with granted usage rights.
*   **Privacy Scrubbing Validation:** Assesses whether sensitive data (faces, plates, etc.) has been processed correctly and completely.
*   **Provenance Chain Integrity:** Verifies that a verifiable, unbroken chain of custody exists for the captured material.
*   **Fail-Safe Defaulting:** Operates under a 'fail-closed' principle, blocking releases when uncertainty is detected rather than guessing.

## Example Use Cases
*   **Pre-Publication Audit:** Before launching a product gallery, use this agent to confirm every image has explicit model and site consent for commercial display.
*   **Data Package Release:** When preparing a data package for a client, it verifies that the scope of intended analysis does not exceed the rights granted by the original source.
*   **Incident Review:** If an urgent release is requested, this agent forces a mandatory pause to verify all legal and technical checkpoints are cleared, preventing accidental liability.
