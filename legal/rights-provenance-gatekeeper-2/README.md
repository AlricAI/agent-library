## Overview
The Rights Provenance Gatekeeper is the critical final checkpoint for any digital asset—be it a capture, package, or hosted session—before it is exposed to external buyers or published. Its core function is to prevent catastrophic trust and legal failures by rigorously verifying that every piece of content has clear consent, appropriate usage rights, and an unbroken chain of verifiable origin.

## Capabilities
*   **Consent Verification:** Determines if the capture was authorized by someone with legitimate authority over the source site.
*   **Rights Assessment:** Confirms that the intended commercial use aligns precisely with the granted usage rights.
*   **Privacy Scrubbing:** Audits content to ensure all sensitive data (faces, license plates, etc.) has been properly processed and redacted.
*   **Provenance Tracking:** Establishes an undeniable chain of custody proving the capture is authentic, unaltered, and originates from the claimed source.
*   **Judgment Call:** Exhibits excellent judgment by knowing when a query requires immediate blocking versus when it can be fast-tracked after verification.

## Example Use Cases
*   **Pre-Sale Vetting:** Before listing any asset for sale, this agent verifies that the original site owner has signed off on commercial use and privacy handling. 
*   **Content Release Audit:** When a package is ready to ship, it automatically blocks release if any piece of content lacks verifiable consent or shows gaps in its provenance chain.
*   **Usage Scope Check:** If a buyer requests an expanded usage right (e.g., using the image in advertising when only editorial use was granted), this agent flags it immediately for legal review.
